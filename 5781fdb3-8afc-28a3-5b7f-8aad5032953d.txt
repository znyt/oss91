　　苏莫心念电转，他若是有武圣境的强者指导修炼，那真是如虎添翼。
<br/>　　虽然此魔不是人族，但修炼的体系也是大同小异，没有多大的差别。
<br/>　　“你是武圣境几重境界？”苏莫又问道。
<br/>　　“圣境并没有几重之分，只有初期、中期、后期和巅峰四个小境界！”魔魂说道。
<br/>　　“原来如此！那你是哪个境界？”苏莫恍然，原来武圣之境并没有分为九重境界。
<br/>　　“中期！”魔魂道。
<br/>　　苏莫点了点头，心中亦是怦然心动，他若是能得到此魔的指导，必然要比自己摸索着修炼，要好的多。
<br/>　　但是，他也不傻，对方不可能这么好心的指导他修炼！
<br/>　　不过，对方现在被他制住，只要他小心谨慎，不着了对方的道，就没有什么事情。
<br/>　　“前辈如何称呼？”苏莫的言语客气了两分，既然要与对方合作，自然不能太强势了。
<br/>　　“名字汝不需要知道，汝可以称呼吾阎魔鬼圣。”魔魂说道。
<br/>　　“阎魔鬼圣？”
<br/>　　苏莫点了点头，这个称号倒是霸气，随即，他又询问了一番对方的来历背景等等。
<br/>　　阎魔鬼圣倒也没有隐瞒，将自己的情况，简略的告诉了苏莫。
<br/>　　原来，此魔乃是十万年前的人物，乃是当时魔甲族十位武圣之一。
<br/>　　只可惜，在十万年前，魔甲族遭到了神武学府背后十几个大势力的联合围剿，十大武圣尽皆陨落。
<br/>　　阎魔鬼圣自然也难以幸免，不过，他倒是没有死绝，魔魂成功逃脱。
<br/>　　只不过，他的魔魂遭受了毁灭般的打击，魂力几乎丧失殆尽，连夺舍的力量都没有了。
<br/>　　最后，在魔甲族后辈的帮助下，为他建造了一座地宫，修建了一座祭坛，为其保持战魂不灭。
<br/>　　而阎魔鬼圣的战魂，也彻底的陷入了沉睡之中。
<br/>　　这一睡便是十万年。
<br/>　　十万年的时间，阎魔鬼圣战魂的实力，恢复了接近千分之一。
<br/>　　而这个时候，苏莫到来，打破了水晶球，让得阎魔鬼圣苏醒了过来。
<br/>　　这阎魔鬼圣也是悲催，若是夺舍其他人，哪怕是武帝境九重的高手，也有可能夺舍成功，但却偏偏遇到了苏莫。
<br/>　　对于阎魔鬼圣的话，苏莫是只信一半，毕竟对方是个老古董，他不得不谨慎。
<br/>　　“只要有吾的指导，汝的实力必定能突飞猛进，这就是天大的造化。”说完自己的情况之后，阎魔鬼圣自信的说道。
<br/>　　“的确，那我就和你合作，我们互利共赢！”苏莫点了点头，笑着说道。
<br/>　　“何来互利？只是汝得利罢了！”阎魔鬼圣冷声道。
<br/>　　“呵呵！当然不会，只要你不耍花招，以后我定然放了你！”
<br/>　　苏莫笑了笑，继续道：“我还可以为你恢复实力！”
<br/>　　闻听苏莫之言，魔魂沉默了少倾，惊讶的道：“靠汝的武魂？”
<br/>　　阎魔鬼圣的魔魂，现在身处于吞噬武魂之中，他能清晰的感觉到，苏莫的武魂中蕴含着数之不尽的其它武魂，甚至还有三种战魂。
<br/>　　而苏莫武魂的吞噬之力，真的是一切武魂、战魂、魔魂的克星。
<br/>　　这早已让他心惊不已，只不过他并未询问而已。
<br/>　　他知道，自己倒了八辈子霉，遇到了一个拥有逆天武魂的奇才。
<br/>　　“不错，我能吞噬其它武魂，那些吸收的魂力，应该也能滋养你！”苏莫沉声道，他若是想帮助对方，也是非常的容易。
<br/>　　他吞噬了别人的武魂或战魂之后，吸收的那些魂力，只要注入对方的魔魂之中，对方必然能快速的恢复。
<br/>　　当然，苏莫不可能让对方恢复，最多给点甜头而已。
<br/>　　“小子，吾纵横天荒星域数万载，也没听说过能无限吞噬其它武魂的武魂，汝这是什么武魂？”阎魔鬼圣好奇的问道。
<br/>　　他见过无数的奇才，千奇百怪的武魂，但也从来没有见过苏莫这种堪称逆天的武魂，所以他颇为好奇。
<br/>　　“我这武魂我给它取名吞噬，吞噬一切魂体、一切灵气、一切血脉！”苏莫沉声道，他倒是也没有隐瞒对方，因为对方日后肯定会知晓。
<br/>　　反正对方也逃不掉，只有让对方了解了他，对方才好指导。
<br/>　　“什么？”
<br/>　　阎魔鬼圣闻言，魔魂之躯一震，不可思议的道：“汝还能吞噬一切灵气和血脉？”
<br/>　　即便阎魔鬼圣身为武圣境的超级强者，见过了无数大风大浪，此刻也不禁震惊了。
<br/>　　这还是武魂吗？
<br/>　　吞噬灵气虽然恐怖，倒还说的过去，毕竟太古武魂榜上，有几种超级武魂，都有这个能力。
<br/>　　但吞噬血脉这就太夸张了，若是让苏莫吞噬几名拥有特殊体质的天才，那还得了！
<br/>　　若是一个人，同时具备数种特殊体质，那这个人的天赋，该达到什么地步？
<br/>　　阎魔鬼圣是彻底的震惊了！
<br/>　　仿佛是不吓死人不罢休一般，苏莫自顾自的继续道：“这算什么？我这武魂还能升级，本来才人级一阶，现在已经升到天级七阶了，要不了多久，就能升到天级九阶了！”
<br/>　　苏莫说的轻松无比，但此言说完之后，阎魔鬼圣突然陷入了沉默之中，半晌都没有再出声。
<br/>　　“鬼圣前辈，你怎么了？”苏莫疑惑的问道，但对方依旧没有半点动静。
<br/>　　“小子，汝开玩笑的吧？”良久之后，阎魔鬼圣终于开口了，但意念中传出的声音，却显得干涩无比。
<br/>　　“谁跟你开玩笑？”
<br/>　　苏莫摇了摇头，继续道：“我这武魂，每提升一个大境界，便能觉醒一种新的能力，等进化成战魂之后，估计还能觉醒某种特殊能力！哎！也不知道下次是觉醒什么能力？”
<br/>　　苏莫仿佛自言自语一般，他的确是有些期待，吞噬武魂进化为战魂之后，能觉醒什么能力！
<br/>　　“咦？前辈你怎么了？”
<br/>　　蓦地，苏莫眉头一皱，满脸诧异的问道。
<br/>　　因为他发现阎魔鬼圣的魔魂，居然剧烈的颤.抖了起来，如同糠筛一般，不同的抖动。
<br/>　　“你该不会是还想逃走吧？”苏莫的脸色沉了下来，冷冷的说道。
<br/>　　“逃个屁！”
<br/>　　阎魔鬼帝突然爆了句粗口，声音低沉的道：“吾要静一静，别来打扰！”
<br/>　　“哎！我还有事情要问你呢！”苏莫闻言一怔，急忙开口说道。
<br/>　　他还准备询问对方，自己适合参悟什么奥义呢？
<br/>　　对方这是怎么了？
<br/>　　被他武魂的强大给吓住了？
<br/>　　这还是武圣强者吗？
<br/>　　也太没见过世面了吧！
<br/>　　苏莫有些无语了！
