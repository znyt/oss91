
<br/>　　<br/>　　    丁武被吴洋连续划了十几刀之后，满身鲜血的往后踉跄着退了几步就“噗通”一声栽倒在了地上，吴洋连看都没看，翻身就跃上栅栏跳了过去。<br/>　　<br/>　　    同时，老马开车拐弯之后就过来了，车前两盏大灯一闪，吴洋和另外一人眯了眯眼睛，他跟对方说道：“分头走，你前面就是路口，你左我右”<br/>　　<br/>　　    两人迈步就往路口那边跑了过去，随即各自分开，老马开车过来后，午桥直接推开车门没等车停稳就下去了：“你们开车追一个，剩下那个我来……”<br/>　　<br/>　　    小区里，丁武倒在了地上，身前胸口处的肉全都被刀子给划烂了，完全就是一副皮开肉绽的状态，齐泉这时刚好赶了过来，一看丁武的样脑袋“嗡嗡”直响的蹲在地上，随即就看见了从他小腹里流出来的肠子，他脸瞬间就白了：“小武儿，你，你怎么样了？”<br/>　　<br/>　　    “泉，泉哥，我……”丁武说话的时候嘴里就往外喷着血沫子，明显有些气若游丝的感觉了。<br/>　　<br/>　　    “别，别说话了，叫车，我给你叫救护车”齐泉冒着冷汗拿出手机打了120，他们完全都没有料到吴洋这两人出手会这么悍，别说什么一言不合了，连对白都没有，直接在屋里就干死了一个，丁武追出来后没想到对方也是奔着要他命去的。<br/>　　<br/>　　    吴洋的这种姿态只能说明一个问题，对方人太狠辣了！<br/>　　<br/>　　    另外一头，外面路口处，老马开着车追上了吴洋，直接就跃上了马路牙子从人行道捋着追了过去，车擦着垃圾桶和停在人行道上的单车疯狂的全都给撞开了，吴洋跑着跑着伸手就把路上的共享单车给拽倒了，然后全都拉到了身后。<br/>　　<br/>　　    老马的车子过来后，前轮碾压着单车往前开了片刻，车轮子就被卡死了，几下油门都没有撞开，不得已他和王生就从车里跳了下来，再次追了过去。<br/>　　<br/>　　    “踏踏踏，踏踏踏”前方出现一座石桥，吴洋见状突然加速，片刻之后他人直接从桥上就翻了下去“噗通”一声一头扎进了河中然后就没影了。<br/>　　<br/>　　    等王生和老马追过来的时候，站在桥上张望，漆黑的河面上荡漾起了一圈圈的波纹，两人皱眉观察了片刻，在距离几十米外的水面，一道人影“哗啦”一下钻了出来，然后划动两手游向了对岸。<br/>　　<br/>　　    老马叹了口气，这时候人肯定是追不上了。<br/>　　<br/>　　    另外一头，午桥下车之后就朝着左边的路口追了过去，他和对方相隔一共不到十来米左右。<br/>　　<br/>　　    午桥边跑边伸手从路边拎起个旁边商家摆放在店门口的铁桶，然后猛地一抡就朝着对面飞了过去。<br/>　　<br/>　　    铁桶带着呼呼的风声飞了过去后，对方根本没意识到，就突然感觉后背一痛就被砸翻了，他倒底之后在地上滚了两圈，就要再次站起来，但午桥追来时右脚蓄力蹬着地面，人如离弦之箭一样射了过去，突然侧着身子，肩膀势大力沉的撞向了对方胸膛。<br/>　　<br/>　　    “咣当”午桥一撞直接就把刚爬起来的人给撞了个跟头，他大踏步的走过去说道：“给我老实眯着，我他么动手的时候你还在苞米地里唱你是一只小小鸟呢”<br/>　　<br/>　　    一米八十多身材壮硕的午桥别看鼻梁上卡着一副眼镜瞅着挺有文艺范的，但动起手来就跟子龙似的，很有猛感，他伸手就抓着地上那人的衣领子给人拉了起来，然后抬起膝盖一拉对方脑袋，就顶在了他的鼻梁骨上。<br/>　　<br/>　　    “咔嚓”一声脆响，鼻梁顿时就碎了，对方满不在乎的摸了下鼻子上的血迹，说道：“呵呵，你有能耐现在就弄死了我，整不死了，你以后得把我当爷给供着，怎么抓我的，你就得怎么原封不动的给我再送出去”<br/>　　<br/>　　    “你咋那么牛逼呢，你很有故事呗？”午桥斜了着眼睛说道。<br/>　　<br/>　　    “不一定是故事，但也肯定不是童话……”对方昂着脑袋说道。<br/>　　<br/>　　    片刻之后，老马和王生赶了过来，见午桥逮住了一个也松了口气，觉得这事办的还不算太丢脸，虽然是跑了吴洋可毕竟还有个落网的，也算是能说得过去了。<br/>　　<br/>　　    “我叫个车过来接我们，然后给他送到会所里先关起来，再看看陈重那边什么安排”老马叫人过来接他们后，又给陈重打了个电话过去。<br/>　　<br/>　　    “老陈，事办完了，不过挺可惜的是，吴洋那人太滑了让他给跑了，但我们也抓了一个跟吴洋在一起的”<br/>　　<br/>　　    陈重有点皱眉和不满的说道：“吴洋跑了？”<br/>　　<br/>　　    “嗯，他太精了！”<br/>　　<br/>　　    “行啊，跑了一个抓一个，你们没啥事吧？”<br/>　　<br/>　　    老马说道：“齐泉说被吴洋给捅死了一个，别的到没什么了”<br/>　　<br/>　　    死一个的时候，齐泉给老马打了电话告诉他吴洋跑了，等到他们追出去后，丁武半路被吴洋给捅了十几刀的时候，老马他自然就不知道这个状况了。<br/>　　<br/>　　    紧接着，老马就开始给附近的警局打电话，托关系把死人的事处理了下，争取别留下什么麻烦的尾巴，毕竟在国内的话死人的事是挺难处理的，是必须的得要立案了，但这种事对老马和陈重来说都比较好操作，主要就是拿钱办事，先给死者家属一笔善后的费用别让他们有意见就行了。<br/>　　<br/>　　    这么一来，老马就暂时没联系齐泉，因为他也没接到对方的电话，就估计没出什么大事，所以他也压根就不知道，齐泉哪有时间联系他啊，他正叫救护车把身中了十几刀的丁武往医院送呢。<br/>　　<br/>　　    丁武的伤很重，上了救护车的时候，人就已经昏迷不醒了，大量失血让他完全失去了意识，在救护车里氧气面罩挂上后医生就开始给他做心脏复苏急救措施。<br/>　　<br/>　　    幸好医院离的比较近，十几分钟就开到了，然后丁武直接被送进来手术室，齐泉焦急的等在外面，也没给老马和陈重汇报情况，打算等人抢救回来再说。<br/>　　<br/>　　    丁武进了手术室后半个小时左右，医生推开门，急促的说道：“去血库取血袋来，快一点，伤者流血过多……”<br/>　　<br/>　　    齐泉连忙问道：“大夫，我朋友他怎么样了？”<br/>　　<br/>　　    “状况不太好，伤势太重了，他先是失血过多，然后就是内脏受创，场子都被割出来了……”医生皱着眉头说道：“看手术吧，手术之后能挺过二十四小时，问题就不大了”
