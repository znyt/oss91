　　萧逸心中惊讶着。
<br/>　　但看了眼周围那些绚烂的冰雕。
<br/>　　又马上反应了过来。
<br/>　　毫无疑问，‘冰山’，是在刚才释放‘寒冰覆海斩’时。
<br/>　　悉数化为真气，被消耗掉了。
<br/>　　顺带一提，冰山火海，由真气凝聚而成。
<br/>　　实则，它们是比真液和真气，更高级的火属性力量。
<br/>　　以及冰属性力量。
<br/>　　但，萧逸能通过自身的转换，将其化为普通真气。
<br/>　　例如，用真气稳定他人的伤势。
<br/>　　用真气施展其它属性的武技，等等。
<br/>　　当然了，同等属性的真气，施展同等属性的武技，威力会更强。
<br/>　　例如使用火海力量，释放的火焰，就会更加强大。
<br/>　　使用冰山力量，释放的冰属性武技，就会更加恐怖。
<br/>　　长、宽、高，各一千六百多丈的冰山。
<br/>　　一丈为3米多。
<br/>　　如此堪称巍峨的冰山，竟在一瞬间消失。
<br/>　　如此大量的冰属性力量，竟在顷刻间消耗殆尽。
<br/>　　难怪刚才‘寒冰覆海斩’的威力，会恐怖如斯。
<br/>　　看着小世界内，原本‘冰山’的位置。
<br/>　　此刻空空如也。
<br/>　　萧逸不禁叹了口气。
<br/>　　“这次亏大发了。”
<br/>　　冰山火海，可不是武道修炼中应该存在的东西。
<br/>　　它出现的意义以及原因。
<br/>　　完全是因为要修复冰鸾剑的裂痕。
<br/>　　由此异变而生。
<br/>　　若全部消耗光，可就真的没了。
<br/>　　以往，萧逸使用火海力量时。
<br/>　　从不敢全部消耗殆尽。
<br/>　　他会留下一丁点儿‘火海’力量。
<br/>　　这一丁点儿，可能只有一丈大小。
<br/>　　对他的战斗，不会产生什么影响。
<br/>　　但，这一丁点儿，却是‘种子’。
<br/>　　在战斗过后。
<br/>　　萧逸能根据‘种子’的牵引。
<br/>　　吸收天地灵气，或自身的自我恢复。
<br/>　　快速恢复火海力量。
<br/>　　但现在。
<br/>　　冰山，是整个没有了。
<br/>　　全被冰鸾剑一次性挥霍光了。
<br/>　　冰山，简直就是生生被‘刮’去了。
<br/>　　而且还是刮地三尺那种。
<br/>　　一点儿都没有留下，更别说‘种子’了。
<br/>　　萧逸若想恢复‘冰山’。
<br/>　　只能再次通过吞服大量‘无杂质丹药’。
<br/>　　再次凝聚出来。
<br/>　　但，凝聚冰山火海，当时所消耗的无杂质丹药。
<br/>　　可是一个天文数字。
<br/>　　那是整整一个北山慕容家的全部底蕴。
<br/>　　“亏了亏了，真的亏大了。”
<br/>　　萧逸气得直跺脚。
<br/>　　拿出那颗‘武道冰晶’。
<br/>　　萧逸把它卖了的心都有了。
<br/>　　这颗武道冰晶，足有23分十成元界碑的传承。
<br/>　　一份传承，除了庞大的武道知识外。
<br/>　　还有玄阶低级至地阶巅峰等一些列齐全的武技。
<br/>　　23份，蕴含了多少武技，可想而知。
<br/>　　萧逸真想把它吞了。
<br/>　　然后得到这些武技。
<br/>　　悉数抄写出来，全部卖掉换钱。
<br/>　　武技，是这个世界最珍贵的东西之一。
<br/>　　当然，萧逸也只是随便想想。
<br/>　　这样一个‘宝贝’。
<br/>　　对自己本身是无用的。
<br/>　　毕竟自己已经有十成极界碑的武道传承。
<br/>　　还看不上这些元界碑传承。
<br/>　　但，如果将这23份传承，悉数加诸到一人身上。
<br/>　　绝对能培养出一个了不起的绝世天才。
<br/>　　自已用不着，日后拿回去萧家也是好的。
<br/>　　想罢，萧逸身影一闪。
<br/>　　来到那百多个黑魔殿武者冰雕的面前。
<br/>　　这些黑魔殿武者，不乏地元境武者。
<br/>　　但早已在冰封中气绝。
<br/>　　萧逸吸了他们的武魂力量。
<br/>　　取了他们的乾坤袋。
<br/>　　而后拿出武道冰晶，吸收了他们的武者精元。
<br/>　　最后，大手一挥。
<br/>　　漫天紫炎，倾泻而出。
<br/>　　将在场的所有冰雕燃烧殆尽。
<br/>　　方圆千米，一丁点儿的冰封痕迹都没有留下。
<br/>　　只残余一股炙热的灼烧气味。
<br/>　　做完一切。
<br/>　　萧逸点了点头，这才身影一闪。
<br/>　　彻底离去。
<br/>　　……
<br/>　　另一边。
<br/>　　各剑派长老和弟子，早已远远离去。
<br/>　　这段时间里，他们也渐渐恢复了一些实力。
<br/>　　却从未想过要回去帮萧逸。
<br/>　　只想着快速逃离。
<br/>　　恰在此时，一道人影快速朝他们而来。
<br/>　　正是叶铭。
<br/>　　萧逸之前赶去救人时，半路就将他扔下了。
<br/>　　他的速度又不快。
<br/>　　故现在才和各剑派长老与弟子碰上。
<br/>　　“叶铭。”
<br/>　　“师兄。”
<br/>　　北山郡裂天剑派长老和弟子，脸色一喜。
<br/>　　“我们还以为你死了呢。”
<br/>　　苗千火等人，激动地看着叶铭。
<br/>　　三长老也是欣慰地看着自己的爱徒。
<br/>　　“你们怎么在这里？”叶铭没有和他们一起激动。
<br/>　　而是追问道。
<br/>　　“易兄呢？”
<br/>　　“什么易兄？”苗千火问道。
<br/>　　“易霄易兄阿。”叶铭说道。
<br/>　　“我赶去流星郡猎妖殿，碰上了他，让他去救你们。”
<br/>　　“你们没有碰到他妈？”
<br/>　　“原来是紫炎。”苗千火等人面露恍然之色。
<br/>　　“就是他救我们出来的。”
<br/>　　“不过他让我们先走。”
<br/>　　“自己留下对付黑魔殿的人。”
<br/>　　“什么？”叶铭脸色一惊。
<br/>　　“易兄一人，如何能对付黑魔殿百多号武者。”
<br/>　　“加之还有一位天元境。”
<br/>　　“大家都恢复了不少实力了吧。”
<br/>　　叶铭看向其他剑派的长老，“快随我回去帮助易兄。”
<br/>　　剑派长老们，个个面露迟疑。
<br/>　　安云郡长老直接说道，“才刚刚逃出来，又回去？”
<br/>　　“你傻了吧？”
<br/>　　“紫炎易霄，声名在外，不会有危险的。”
<br/>　　“就是打不过，也肯定能逃掉。”
<br/>　　“你们…”叶铭跺了跺脚，急道。
<br/>　　“你们不回去，我回去。”
<br/>　　“说什么也不能让易兄一人，独自涉险。”
<br/>　　叶铭脚步刚动。
<br/>　　三长老一把拉住了他。
<br/>　　“叶铭，别冲动。”
<br/>　　“你这破玄一重的修为，去了也帮不上忙。”
<br/>　　“只会拖累他。”
<br/>　　“倒是我之前吩咐你的事，你办妥了吗？”
<br/>　　“额。”叶铭愣了愣。
<br/>　　三长老气道，“我不是让你去流星郡请帮手来救我们吗？”
<br/>　　“请完之后。”
<br/>　　“顺便在流星郡打探下有没有萧逸的消息。”
<br/>　　“他是否已经来了。”
<br/>　　叶铭面露难色，道，“弟子并没有打探萧逸师弟的事。”
<br/>　　“毕竟，要赶着回来…”
<br/>　　三长老打断道，“那你还管什么紫炎易霄。”
<br/>　　“赶紧随我们去流星郡，打探萧逸的消息。”
<br/>　　三长老严肃的表情。
<br/>　　让得尊师重道的叶铭，打消了去找紫炎易霄的心思。
<br/>　　三长老松了口气。
<br/>　　他就怕叶铭回去送死。
<br/>　　“易兄。”叶铭站在原地。
<br/>　　眺目远视，喃喃自语着。
<br/>　　面露担忧之色。
<br/>　　恰在此时，一道火红色人影，瞬间疾驰而来。
<br/>　　正是戴着幽魂面具的萧逸。
<br/>　　嗖，萧逸瞬间降落。
<br/>　　“易兄。”叶铭激动地叫了一声。
<br/>　　萧逸点了点头。
<br/>　　“紫炎易霄？”各剑派长老，立马一惊。
<br/>　　“怎么这么快回来了？”
<br/>　　“可是打不过那幽水老怪。”
<br/>　　“逃遁回来？”
<br/>　　萧逸白了他们一眼，“黑魔殿一行人，已悉数被易某斩杀。”
<br/>　　“走吧，我再送你们一程。”
<br/>　　“护送你们回流星郡。”
<br/>　　萧逸嘴上说着。
<br/>　　其实只是担心北山郡众人。
<br/>　　他现在‘冰山’已然消失。
<br/>　　战力大降，可没把握再对付天元境武者。
<br/>　　若黑魔殿还有人追来，那就麻烦了。
<br/>　　再者，他肩膀上那森森白骨的伤势。
<br/>　　也必需快点找个安全的地方治疗。
