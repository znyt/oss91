　　当晚，叶少阳三人回家就睡了，一觉睡醒，外面有人敲门，天已经大亮了。叶少阳揉着太阳穴过去开门，见是老郭，印象中好像刚分开似的，其实已经过了一个晚上了。
<br/>　　“电话也不接，敲门也不理，我都打算走人了！”
<br/>　　老郭抱怨着，手里提着两个很大的袋子走进来。
<br/>　　“这是啥？”
<br/>　　老郭打开袋子，里面全是成串的香肠、腊肉，还有几只咸鸡板鸭。老郭一个个全给挂在阳台的衣服撑上。
<br/>　　“这是你嫂子让我送的，知道你们几个大男人不可能上街去买，让我送了些年货过来，这不快过年了，总得有点过年的样子吧，喏，这一捆红纸是给你写对联用的，知道你字写的好，也就没买现成的。”
<br/>　　老郭自己去厨房倒了杯水，喝完也就走了。“我还得给家里送一些，对了年三十你们记得去我家啊，到时候我叫你们。”
<br/>　　老郭挥了挥手要走，突然另一间房间内响起四宝的声音，“哎少阳，你法器怎么乱放啊，赶紧拿回去！”
<br/>　　叶少阳听着好奇，走过去，推门进去，四宝缩在被窝里，一只手露在被窝外面，捧着个什么东西，迷迷糊糊地说道：“这东西硌了我半个晚上，你赶紧拿走，以后法器不要乱放啊，丢了看你哭吧！”
<br/>　　叶少阳皱眉上前，从他手中拿过那个东西，是一个四方形的大印，青绿色，半透明的玉质，上面刻着一些看不懂的符文，看上去像是古物，但是十分光洁，连一条裂缝都没有。
<br/>　　“这是什么，不是我的东西，你自己的吧。”
<br/>　　“我不用印，不是你的那就是白眉的。”四宝从被子里伸出脑袋，大声叫吴嘉伟的名字。
<br/>　　叶少阳把玉印丢回床上，转身要出门，脑海中还在想着这玉印好像在哪里见过……突然浑身触电般地僵住，几秒钟后，他扑到床前，把玉印又捡起来，双手托着，上下翻看起来，上面的符文自己不认识，但连起来看，像是一片汪洋水域，有山头若隐若现。
<br/>　　“操，这是……”叶少阳一把梧州自己的嘴，冲到客厅，老郭正穿鞋要走，叶少阳冲他招了招手。
<br/>　　“咋的了。”他一副神秘的样子吸引了老郭，赶紧过来。
<br/>　　叶少阳把窗户都关上，怕惊魂铃不保险，又画了几张血精符，贴在门窗上，再去吴嘉伟的房间把他踢醒，来到四宝的房间。
<br/>　　“你干啥呢一惊一乍的。”四宝揉着眼睛，靠在枕头上。
<br/>　　叶少阳一只手碰着玉印，高举起来，“大家猜猜，这是什么！”
<br/>　　“什么？”老郭皱眉打量着。
<br/>　　叶少阳望着他们，压低声音，一字一顿地说道：“山——海——印！”
<br/>　　十秒钟的窒息。
<br/>　　四宝腾地一下从床上坐起来，“没搞错吧！”
<br/>　　“是山海印，我见徐福用过，不会错的。”叶少阳难掩激动，“再说，如果不是山海印，为什么会出现在我家床上！”
<br/>　　瓜瓜也证明，之前见过的山海印就是这样的。
<br/>　　四宝皱眉道：“可是如果是山海印，又为什么会出现在你家床上？你不是说徐福没给你山海印吗？”
<br/>　　叶少阳把自己之前的猜测讲出来。
<br/>　　吴嘉伟和四宝都有点懵。
<br/>　　“也就是说，是徐福用山海印来到今天，把山海印放在了这里？”
<br/>　　得到叶少阳确定之后，四宝想了一会，道：“这么说，他去到的未来的时间，能与我们的时间产生联系，也就是说，他能够改变未来？这与我们当时推测的不一样啊。”
<br/>　　叶少阳想了一会，说道：“我觉得，两种情况都是并存的，既有平行世界，也有能发生连锁反应的时间线。徐福在被捕之前，去了未来……也就是今天，把山海印留在这里，然后我们一天天过来……”
<br/>　　吴嘉伟打断他的话，道：“也就说，昨晚，他来了，放下了山海印。”
<br/>　　叶少阳木讷地点点头，猛然想到什么，老郭却已经叫起来：“那如果昨晚你们半夜起床，不是能够正好看到他吗，可是真正的徐福，还在昭狱里，这世界就有两个徐福了？”
<br/>　　叶少阳倒吸一口冷气，道：“昨晚来的，是过去时间里的徐福！”
<br/>　　但是毕竟有两个徐福出现在了同一个世界，而且两个都是真的徐福，自己这条时间线上的徐福，现在在昭狱，而昨晚上送山海印来的那个，现在已经回到了过去的时间，不久就要被抓了……
<br/>　　这种感觉，还是让他们感觉说不出的怪异。
<br/>　　四宝傻傻地想了一会，说道：“那个，山羊，你想想，在徐福被抓之前，那时候我们没有山海印，徐福把山海印给了未来的我们……从我们当时来看，山海印岂不是被未来的我们给拿到了？”
<br/>　　“这……”叶少阳想了一下，当时……在未来的时间里，另一个“自己”拿到山海印，然后他的表现会是什么呢，肯定也跟现在的自己一样，吃惊之余，也会思考很多，他……还有当时的四宝、老郭白眉和瓜瓜，会不会也进行这样一场讨论？甚至……完全一模一样。
<br/>　　一股冰凉的恐惧，爬上大伙的心头。他们都想到了同一件事：如果，“未来”世界的他们，当时在看到山海印的时候，与他们现在的表现完全一样，那你就等于是人世间的一切事情都是安排好的？他们不过是沿着一个既定的轨道走下去？
<br/>　　他们现在所做的事，所说的话，甚至一个眼神，都是已经发生过的……叶少阳陡然想到，甚至这个世界的秋莹还活着的时候，在未来时间她已经死了，这还只是两个世界相差这不到一个月的时间（从徐福穿越时空到他们方才拿到山海印），再往后面追溯，还有无穷的时间，也就有无穷的世界。
<br/>　　例如，自己现在打算去灭了太阴山，但在未来的某个世界里，自己已经去做了，并且已经发生了结果，那么，不管自己去不去做这件事，结局早已注定，那自己去做这一切，还有什么意义呢？
