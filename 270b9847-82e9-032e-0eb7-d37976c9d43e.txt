　　<br/>　　可是大海的回答也让他有些郁闷，因为这个话就是二娃说的啊，没有毛病！顿时满满的郁闷！<br/>　　<br/>　　“长的快？那可能是因为吃的东西吧！”陈默说道。<br/>　　<br/>　　“吃的东西？二娃，这个在牲口吃的上面，可以简单些，但是千万不能喂一些乱七八糟的东西啊！”姚爷有些担心的说道。<br/>　　<br/>　　“哈哈！姚爷，你就放心吧！难道你还不了解我，绝对不会给这些牲口乱喂一些激素什么的！”陈默笑着说道。<br/>　　<br/>　　“那这些牲口为什么长的这么快？”姚爷决心打破沙锅问到底！<br/>　　<br/>　　“这就涉及到我的一些研究成果了。姚爷，你也知道我是生物专业毕业的，所以配置了一些生物营养液，不仅可以给植物使用，也能给牲口使用，而且还是无害的。不过就是因为配置不容易，所以不能大面积使用。”陈默解释的说道。<br/>　　<br/>　　“原来如此！”姚爷点点头：“只要不是乱七八糟的东西就好，你心里有数就成！”<br/>　　<br/>　　有些东西，涉及到陈默的商业秘密，姚爷也就不问了。他也知道，这个东西不能问，要是问了，就不好说了。<br/>　　<br/>　　“姚爷，你放心好了！我做事情可是用心去做，绝对不会干坑人的事情。你是多年的老把式，牲口有没有毛病，吃没有吃一些激素什么的，你还不清楚么？”<br/>　　<br/>　　陈默说的，姚爷也是心里清楚。就是搞不清状况，才会由此疑问的。他可是清楚，这里的牲口各个都是膘肥体壮，而且健康无比，绝对不是那种喂了什么乱七八糟的东西样子，要不他早就撂手不干了。<br/>　　<br/>　　这么多年的把式了，最最讨厌的就是现在人心不古，专门做一些坑人的事情。<br/>　　<br/>　　不过好在陈默这里他是看着长大的，而且二娃的德行还是不错的。而且他也是有眼睛和经验的，不是那么好糊弄的。<br/>　　<br/>　　“好，你这样说我也就放心了！”姚爷说道。<br/>　　<br/>　　“那就成！这里的牲口还要你多多费心呢，有什么事情你就提出来，要是大海解决不了的，你就来找我！”陈默笑着说道。<br/>　　<br/>　　“好！”姚爷点头答应！<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
