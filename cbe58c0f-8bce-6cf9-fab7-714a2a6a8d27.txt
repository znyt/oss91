　　苍穹神宫的发展，已经完全进入了正轨，九大宫之中，每一宫都有长老、弟子约两三万人。
<br/>　　除了长青宫和雷霆宫之外，另外七大宫尽皆有了传人。
<br/>　　整个苍穹神宫的发展，都呈现蒸蒸日上之势。
<br/>　　接下来的一段时日，苏莫暂时没有修炼，抽时间陪伴夕儿、陪伴父亲，日子过得也算清闲。
<br/>　　黄泉魔宗之人，仿佛彻底消失了一般，任凭天涯海阁之人如何查探，也查探不出丝毫的踪迹。
<br/>　　这让苏莫郁闷不已，黄泉魔宗已经成为了他的肉中之刺，不拔除这根刺，他实在难以安心。
<br/>　　因为黄泉魔宗，随时可能给予苍穹神宫巨大的伤害。
<br/>　　而且，黄泉魔宗和试炼者有关，不彻底铲除的话，到时候会横生变故。
<br/>　　但苍穹世界那么大，黄泉魔宗之人隐藏起来，没有特殊的手段的话，真的很难找到他们。
<br/>　　目前，除了找到黄泉魔宗之人以外，苏莫还有两个任务，分别是敲定雷霆宫和长青宫的传人，以及一统苍穹世界。
<br/>　　不过，后者不必着急，待苍穹神宫壮大到一定地步，自然会水到渠成。
<br/>　　那时候，其它势力大，也不会有多大的反抗之心。
<br/>　　关键是传人之事，他还要费些功夫。
<br/>　　雷霆宫苏莫已经有了人选，那便是他曾经的灵兽，小八。
<br/>　　至于长青宫，他本已经认定了宏青璇做为传人，但如今宏青璇遭此大劫，怕是难以再继任长青宫传人了，他只能另寻他人。
<br/>　　长青宫的弟子之中，已经有了几个不错的苗子，苏莫已经让青一重点关注了。
<br/>　　时间如白驹过隙，一日又一日，转瞬间便过去了五个月。
<br/>　　这五个月之内，苍穹世界无数的大势力，都有人来到苍穹神宫，送上贺礼，祝贺苏莫荣登苍穹世界第一强者，以此来向苏莫示好。
<br/>　　不过，苏莫除了亲自现身，面见了东海龙王之外，其他人一概不见，全部交由了帝释等长老负责接见，他安心修炼。
<br/>　　五个月的时间，苏莫的修为早已达到了武皇境四重的巅峰，但他身上从殷家缴获的资源，也已经消耗一空了，根本无力突破修为了。
<br/>　　天涯海阁以前虽然是巨富，但总部被灭，很多资源也不在了。
<br/>　　好在，有天涯海阁这个赚钱机器，足以很好的支撑整个苍穹神宫的运转了。
<br/>　　……
<br/>　　这一日，苏莫正在苍穹神宫一片僻静的花园中练剑。
<br/>　　他的剑凌厉万分，每一招每一式都玄妙莫测。
<br/>　　这几年以来，苏莫每时每刻身心都有巨大的压力，所以他拼命的修炼，一味的提升修为。
<br/>　　以至于，他对于各种武技，修炼的极少，就算修炼了也不精深。
<br/>　　不仅如此，就连五种帝级上品功法，他修炼的造诣也不高。
<br/>　　现在他有了足够的时间，便经常抽空练剑，精深剑法。
<br/>　　大苍穹剑法，蕴含一种天地万物、宇宙星空的玄妙意境，只有掌握了这种意境，才能发挥出这门剑法最强的威力。
<br/>　　而且，皇级以上的武学，只有融入奥义之力，才能更强大。
<br/>　　苏莫对于不朽奥义的领悟，虽然不错，但这种天地大道，实在浩瀚，他只能算是略知皮毛。
<br/>　　咻！咻！咻！
<br/>　　利剑闪烁，划破空气，犀利的破空声不绝于耳。
<br/>　　剑光时而如同浩瀚的天穹，时而如同广阔的大地，时而演化为漫天星辰，玄妙无比。
<br/>　　挞挞挞！！
<br/>　　就在此时，一阵沉闷的脚步声响起，一名面容很是苍老的老者，来到了苏莫的身后。
<br/>　　苏莫收剑伫立，转头看向老者，来人正是海明江。
<br/>　　此时的海明江，脸色颇为红润，身上的气息也更为深邃。
<br/>　　显然，修炼了帝级中品功法的海明江，实力有了很大的提升。
<br/>　　“海长老，有何事？”苏莫问道，目前海明江的身份，是光耀宫的大长老。
<br/>　　“宫主，查到了一丝黄泉魔宗的踪迹！”海明江抱拳说道。
<br/>　　海明江作为以前天涯海阁的阁主，一直在负责追查黄泉魔宗之人的踪迹，从来没有松懈过。
<br/>　　“哦？在哪里？”苏莫闻言，眸中顿时精光一闪，终于有消息了吗？
<br/>　　“禀宫主，数日前，我们的探子在北海青俞岛附近，发现了残留的尸气！”海明江沉声说道。
<br/>　　“残留的尸气？”苏莫闻言眼眸微眯了起来。
<br/>　　“不错！黄泉魔宗的强者，有很多都是尸罗殿之人，修炼尸功，而这残留的尸气，根据探子回报，颇为不弱，应该是武皇境强者所留！”
<br/>　　海明江点头道，天涯海阁的探子，很多人都是拥有特殊的武魂，对于气息感知极为敏锐。
<br/>　　苏莫点了点头，黄泉魔宗的确是与很多人修炼尸功，那大魔老便是一具天尸。
<br/>　　巫森虽然有空间宝器，但黄泉魔宗这么多人，显然不可能一直待在空间宝器之中，想必肯定会出来活动，或者是探查消息。
<br/>　　所以，有气息遗留，倒也不奇怪。
<br/>　　纵观整个苍穹世界，除了黄泉魔宗尸罗殿之外，好像也没有什么人会修炼尸功了。
<br/>　　“我出去一趟，在我归来之前，苍穹神宫任何人不得进出，护宫大阵时刻开启！”苏莫叮嘱道，这一段时日，浑天洛水大阵已经关闭了，毕竟一个偌大的宗门，不可能一直山门紧闭。
<br/>　　不过，他若是离开的话，威力安全起见，护宫大阵还是要开启的。
<br/>　　毕竟，说不定黄泉魔宗，就会趁着他离开的时机，前来偷袭苍穹神宫。
<br/>　　“宫主放心，不会有事的！”海明江点了点头，他倒是没有多少担心。
<br/>　　因为黄泉魔宗不可能来偷袭，巫森肉身毁灭，就算战魂重新夺舍一具肉身，没有几年的时间，也很难恢复实力。
<br/>　　而后，苏莫又交代了一番之后，便离开了苍穹神宫，径直向北海而去。
<br/>　　苍穹世界，四方海域辽阔无垠，无比广阔，加起来比整个苍穹大陆都要辽阔的多。
<br/>　　北海，便是四方海域之一。
<br/>　　偌大的北海，方圆亿万里，一望无际的海水，延伸到天地的尽头。
<br/>　　几日之后，苏莫便来到了北海，而后，他径直向青俞岛飞去。
<br/>　　他已经打听清楚了青俞岛的方位，此岛是北海范围内数百座大岛之一，位于北海偏西的位置。
