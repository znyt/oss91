　　苏莫心中充满了发苦，他已经够危险的了，没想到这拍卖会，居然还给他来这么一出。
<br/>　　什么狗屁真圣榜？他可不在乎这种荣誉，他要的是安全和平静。
<br/>　　但是，这样一搞，他以后怕是再也不能平静了！
<br/>　　这等于是在告诉巫族，他苏莫就藏在神武学府啊！
<br/>　　中央拍卖台上，玄净尊者并不知道此时苏莫的苦楚，正在详细的介绍。
<br/>　　“苏莫目前的修为，应该是武帝境四重，但此人的修炼速度，堪称无人能出其右，从准帝之境修炼到武帝境四重境界，也只是用了区区三年有余的时间而已！”
<br/>　　“目前，据老夫估算，其应该已经达到了武帝境五重，甚至是武帝境六重的境界！”
<br/>　　玄净尊者徐徐介绍，众人闻言心中震惊，从准帝之境修炼到武帝境四重，居然只用了三年多的时！
<br/>　　这个修炼速度，的确是惊世骇俗，一般的天才，从准帝之境修炼到武帝境四重，起码要二十年以上的时间，就算是妖孽天才，也需要七、八年的时间。
<br/>　　但是，仅仅是修炼速度快，还不至于排在真圣榜第一名吧，这还远远不够。
<br/>　　不过，这一次众人没有再开口打断玄净尊者，均是准备听对方说完。
<br/>　　若是苏莫真的有排在第一的资格，他们自然不会多说什么，但若是不够资格，那这所谓的第一，不够使狗屁而已，没有人会承认。
<br/>　　“此人战力无双，不仅在同境界无敌，就算是越一阶修为战斗，同样无敌！”玄净尊者继续说道。
<br/>　　众人听闻此言，心中均是有些嗤笑，同阶无敌？越一阶同样无敌？
<br/>　　这不过是个笑话而已！
<br/>　　因为在古灵星河，号称同阶无敌，甚至是越一阶无敌的天才，不在少数，这只是因为没有碰到同样的妖孽天才而已。
<br/>　　比如真圣榜上那些妖孽天才，很多都号称同阶无敌，这只是没有遇到能打败他们的同阶武者而已，可不是真正的无敌。
<br/>　　一般同样的妖孽天才，同等修为之下，大多都是很难分出胜负，这种情况下，只要不败，依旧认为自己同阶无敌。
<br/>　　玄净尊者身形伫立在拍卖台上，目光环视四方，发现众人对于他的话，并没有多少认同，他也没有在意。
<br/>　　毕竟，只是他一面之词，未免太空洞了。
<br/>　　“众位，老夫给大家看一段影像，此影像是我方商会一位武圣境强者，利用大神通手段，从神武学府中扩印而出！”
<br/>　　玄净尊者微微一笑，收起半空中的真圣榜，随即手掌一番，掌中出现了一块圆形的玉符。
<br/>　　微微注入一丝玄力，玉符中顿时爆射出夺目的光芒，直冲九霄。
<br/>　　而后，在拍卖台的上空，则是出现了一段清晰的影像。
<br/>　　影像中是一片大海，周围还有十几座小岛。
<br/>　　而此刻，一名个子不高，面色黝黑的青年，正从海面上暴冲而起，其身上雷光闪烁，手中拿着一柄硕大的雷刀。
<br/>　　而后，黝黑青年一刀劈下，威势狂暴无比，一刀浩大的闪电划破长空，直击下方海面上的青秀青年。
<br/>　　众人见此，心中稍稍惊讶，这名手握雷刀，状若雷神的黝黑青年，难道就是苏莫不成？
<br/>　　的确不弱啊！武帝境四重的修为，能发出如此强大的攻击，的确相当不凡。
<br/>　　但是下一刻，众人便都知道自己的猜测错误了，这名黝黑的青年，居然被下方那名青秀青年，一剑秒杀！
<br/>　　原来青秀青年，才是所谓的苏莫。
<br/>　　众人震惊，下方那名青秀青年，同阶修为之下，居然能一招秒杀强大的黝黑青年，这种战斗力却是恐怖啊！
<br/>　　影像不断的播放，青秀青年斩杀黝黑青年之后，又有一名身穿黑袍的青年上场与之对战。
<br/>　　所有人均是静气凝神，仔细的观看着影像。
<br/>　　而坐席上的苏莫，却是脸皮抽搐了起来，尼玛，连他在战界挑战太史世家派系的影像，都能播放出来，这拍卖会背后的势力，手段居然如此厉害！
<br/>　　刚才影像中播放的内容，正是他在战界之中，一剑斩杀雷灿的画面，现在则是开始对战林无影了。
<br/>　　谢彬面上亦是带着古怪之色，看了看影像，又看了看苏莫，不知其心中所想。
<br/>　　这画面他太熟悉了，还亲历了那次战斗，但现在回看，还是感到震撼异常。
<br/>　　另一边，一直面色平静的尤天韩，再次看到苏莫大发神威，击杀他们太史世家派系天才的画面，心中早已消散的怒火，再次死灰复燃。
<br/>　　他真恨不得，现在就出手，将苏莫击杀！
<br/>　　但是他不敢，苏莫现在不仅加入了皇祖圣朝派系，还得到了古天意的看重，不惜舍下血本，化解苏莫和他们太史世家派系的恩怨。
<br/>　　他若是杀了苏莫，怕是古天意都要亲自对他出手了！
<br/>　　空中的影像不断播放，苏莫如何斩杀林无影、如何斩杀高一重修为的滕金山，如何以一击之力，瞬杀七、八十名太史世家派系的弟子，所有的画面，均是清晰的展现在了众人的面前。
<br/>　　影像播放完毕，整个拍卖会场久久无语，强大！这名青秀青年的战力，极度强大，骇人听闻。
<br/>　　尤其是最后，其瞬间斩杀七、八十名武者的画面，最是让人震撼。
<br/>　　因为这七、八十名武者，不仅个个气息强大，其中有一小半人修为都比青秀青年还要高一重。
<br/>　　这是越级战斗之中，而大规模的秒杀，堪称恐怖！
<br/>　　最最主要的是，影像中的战斗，青秀青年一直淡然无比，显然还没有发挥出最巅峰的战斗力，其极限到底在哪里，根本没有展现出来。
<br/>　　少倾，影像消失之后，玄净尊者收回了玉符，朗声问答：“众位，你们看完了这段影像，都有何种感想？”
<br/>　　众人久久无语，整个拍卖会场，一片的安静，所有人均是感到震撼无比。
<br/>　　直到过去了十几个呼吸之后，才有人开口，只见前排的一名魁梧青年站了起来，沉声道：“玄净尊者前辈，我承认这人天赋妖孽，战力非常可怕，若是我和他同阶一战，只有被秒杀的份，但还是不认为他能排在第一名。”
<br/>　　“对啊！我也是这么认为，其一，此人修为才武帝境四重境界，其二具体的天赋和战力，到底达到了什么程度，仅凭一段影像，还是有些不足啊！”又有一人站了起来，朗声说道。
<br/>　　这两人的话语，基本上代表了大多数人的观点，其他人虽然没有开口，但均是点头赞同。
<br/>　　玄净尊者见此，面带微笑，略作沉吟，道：“这一段影像中，其实表露了很多的信息，或许武帝境和一般的武尊境武者无法察觉，但是瞒不过武圣境强者的眼力。”
<br/>　　“表露了什么信息？”众人闻听此言，不少人齐齐开口询问。
<br/>　　“此人——应该是四生战魂！”玄净尊者一字一句的说道。
