<br/>　　至于真圣之气，能得到更好，得不到也无所谓。

<br/>　　姜神夜面色平淡，仿佛丝毫不放在心上一般。
<br/>　　苏莫则是没有开口，巨摩挑战他的话，他只能应战，不挑战他自然更好。

<br/>　　“苏莫，就你吧！”巨摩大声说道，他之所以最终选择苏莫，是因为苏莫表现的更强势。
<br/>　　苏莫打败巫殇，何等的霸道，何等的震撼！
<br/>　　而姜神夜一直没挑战强大的对手，更没有人去挑战他，则是显得平凡了。
<br/>　　所以，巨摩选择了苏莫，击败苏莫比击败一直低调的姜神夜要来的更痛快。
<br/>　　苏莫闻言心中暗叹，他想要安稳结束考验，却是根本不可能了，既然对方要战，那就战吧！
<br/>　　唰！
<br/>　　苏莫身形一闪，瞬间落在了青色平台上，与巨摩隔空相望。
<br/>　　众人见此，顿时精神大震，巨摩和苏莫一战，这简直是此次考验最巅峰的一战了。
<br/>　　巨摩的战力毋庸置疑，强大的不可思议，一身力量恐怖绝伦。
<br/>　　苏莫的实力同样恐怖，强势击败巫殇，而且还能隐身，诡异无比。
<br/>　　这两人的战斗，没人能肯定谁能获胜，实力方面似乎巨摩更强一筹，但苏莫有隐身的能力，不是实力强就能战胜的。
<br/>　　总之，这一战，谁胜谁负，尚不可知。
<br/>　　所以人都暗暗期待了起来，谁能最终获胜，基本上就能奠定武帝境无敌了！
<br/>　　真圣界外，亦是一阵惊呼，终于有强强对决了，最巅峰一战要开始了。
<br/>　　“苏莫，你也会被淘汰！”
<br/>　　尤天韩盘膝坐在地上，眼眸森寒，仿佛要择人而噬。
<br/>　　他受的伤非常重，这具肉身险些崩溃，好在通过十几个时辰的疗伤，稍稍稳住了伤势。
<br/>　　但即便如此，没有两个月的时间，伤势都不可能恢复。
<br/>　　最主要的是，他不仅遭受重创，连真圣之气也无法获得了，所以他恨！
<br/>　　他希望苏莫也败了，也同样被淘汰。

<br/>　　苏莫在神武学府的每一次战斗，她都看了，几乎每一战苏莫都是绝地逆袭。
<br/>　　往往你认为苏莫必败了，但结果他还是胜了！
<br/>　　在苏莫的身上，战斗奇迹不断的出现，让得木璃导师自然而然的对苏莫产生了绝强的信心。
<br/>　　青色的平台上。
<br/>　　巨摩望着前方的苏莫，双拳一握，全身爆炸性的肌肉，如同一面面战鼓，发出咚咚的声响，极为惊人。

<br/>　　暴喝声如同天雷滚滚，巨摩比水缸还要巨大的拳头一抡而起，狠狠的轰响苏莫。
