　　“找虐！”
<br/>　　见这个妖族之人动用音波攻击，苏莫微微的摇了摇头，对方太弱太弱了。
<br/>　　他刚来到神域之时，实力都比此人要强很多。
<br/>　　眼见强大的音波之力袭来，苏莫也出手了，大袖猛然一挥。
<br/>　　随着大袖挥动，火浪席卷而出，直击巨大的雄狮。
<br/>　　火浪汹涌，焚化一切，音波之力都不能阻挡，立刻灰飞烟灭。
<br/>　　毫无意外，汹涌的火浪，结结实实的击在了雄狮的身上。
<br/>　　轰！
<br/>　　轰响震天，火浪炸开，伴随着雄狮的痛苦的吼叫。
<br/>　　只见强大的雄狮，直接被击飞了出去，并且差点被烧成了焦炭。
<br/>　　“回来！”
<br/>　　苏莫大手一探，直接将被重创的雄狮擒了回来。
<br/>　　随即，他将雄狮身上的空间宝器收取，随手就将其丢向了出口的门户。
<br/>　　眨眼之间，雄狮离开了迷宫城，一切都在极短的时间内完成，轻描淡写。
<br/>　　嗖嗖嗖！！
<br/>　　就在此刻，破空声响起，通道的尽头，出现了三道身影。
<br/>　　这三道身影，都是人族之人，显然正是追击刚才那名妖族青年之人。
<br/>　　“人呢？”
<br/>　　“难道已经出去了？”
<br/>　　三人没有见到妖族青年，立刻停了下来，微微有些疑惑。
<br/>　　不过，三人的目光，很快便锁定在了苏莫的身上。
<br/>　　“适才有妖族的惨叫，应该与此人有关！”其中一人凝视着苏莫，面色冷漠的说道。
<br/>　　在他看来，之前的那位妖族，很可能被眼前之人淘汰了。
<br/>　　那么，那件宝物，很可能被眼前之人得到了。
<br/>　　“小子，你得到了宝物？”
<br/>　　为首的一位高瘦青年，大步向苏莫走去，因为苏莫后面就是出口，所以他没有急着出手，免得苏莫逃跑。
<br/>　　另外两人，也大步前进，向苏莫不断的靠近。
<br/>　　“得到了！”苏莫淡淡的说道，不过，他还没有查看那位妖族的储物戒，所以也不知道有没有宝物。
<br/>　　但是，那位妖族既然被这三人追击，想必是有宝物。
<br/>　　“很好，将宝物交出来，我们不难为你。”高瘦青年渐渐接近了苏莫，沉声说道。
<br/>　　另外两位青年，也已经做好了出手的准备，他们要确认能擒住苏莫才会出手。
<br/>　　“你们也被淘汰了！“
<br/>　　苏莫淡淡的说道，言罢，他立刻出手了，出口的门户在移动，很快就要离开这处通道，他可没有时间浪费。
<br/>　　一掌出，同样的火焰大手，再次抓出。
<br/>　　火焰大手充满了整座通道，在木属性源力的配合之下，火焰大手的威力，强大到了极致。
<br/>　　“什么？”
<br/>　　“怎么可能？”
<br/>　　所谓强者一出手，便知有没有，苏莫的火焰大手一出，恐怖的威力立刻让得三人心中大骇。
<br/>　　逃！
<br/>　　一瞬间，三人立刻转身就逃，因为这火焰大手的威力，远超他们的实力水平，根本无法对抗。
<br/>　　所以，三人果断的选择了逃走。
<br/>　　不过，三人逃走的速度，很显然不急火焰大手的攻击速度。
<br/>　　只是眨眼之间，火焰大手便临近了三人。
<br/>　　“一起出手！”
<br/>　　为首的高瘦青年怒吼，随即三人纷纷仓促出手，想要联手挡住火焰大手。
<br/>　　轰轰轰！！
<br/>　　三人的攻击，纷纷轰击在了火焰大手之上，但是，却是如同鸡蛋碰石头一般，碰之既碎。
<br/>　　啊啊啊！！
<br/>　　惊恐的叫声响起，火焰大手如同火神降临，一把便擒住了三人，将三人捏的惨叫连连。
<br/>　　擒回三人，苏莫立刻开始搜索，将三人身上的储物宝器全部抢夺。
<br/>　　然后，三人的下场，和之前的狮族青年一样，被苏莫直接从出口丢了出去。
<br/>　　微微一笑，苏莫立刻开始检查战利品，不过，只有之前的狮族青年的储物宝器之中有宝物，其它三人的储物宝器之中，均是空无一物，甚至连兵器都没有一件。
<br/>　　或许这些人是没有信心，进入迷宫城，均是不带任何物品。
<br/>　　狮族青年的储物宝器之中，有一株神草，通体银色，气息晦涩，不知道是何物。
<br/>　　苏莫没有急着服用，收起神草和几件储物宝器，便立刻离开了，因为出口的门户，已经转移。
<br/>　　“这小子，真是不简单！”
<br/>　　迷宫城之外，龙城一直在留意着苏莫，见到此情此景，顿时眉头微皱。
<br/>　　因为，他发现自己看不透苏莫，苏莫击败几位对手，完全都是轻描淡写，实力的底线不知道在哪里。
<br/>　　“不简单啊！”龙烨叹声说道。
<br/>　　龙洌和龙双双，均是微眯着眼眸，他们倒要看看，龙腾的这个奴仆，有什么本事，胆敢守住出口抢夺宝物。
<br/>　　龙腾面色淡淡，嘴角带着一丝笑意，接下来会发生什么事情，他已经能预料到结果了。
<br/>　　迷宫城之中，苏莫依旧跟随着出口在移动，所有想要出去的人，必须要经历他的抢夺，只有司空炎能例外。
<br/>　　随着时间的流逝，浩大的迷宫城之中，五百一十将宝物，基本上全部被找到了。
<br/>　　血腥的争夺，依旧在迷宫城各处上演，实力不济之人，就算碰巧得到了宝物，也根本保不住宝物。
<br/>　　也有人为了宝物不被抢夺，得到神果、神草等物之后，便立刻吞服下去。
<br/>　　不过，有的吞服了神果的人能享受到神果的药力，但是有的人，吞服了神果之后，还是被人擒拿，连带着身躯都被别人炼化，凄惨无比。
<br/>　　但是，大多数的人，得到神果神草等宝物之后，都不会立刻吞服。
<br/>　　因为，在迷宫城里吞服，无法安静的闭关修炼，不能安心的享用药力，服用了之后也会白白浪费了宝物。
<br/>　　随着时间的流逝，迷宫城之中的人越来越少，很快便只剩下了几千人。
<br/>　　淘汰出去的人都是弱者，留下的人都是强者，优胜劣汰。
<br/>　　到了这个时候，大多数得到宝物之人，都开始寻找出口，想要离开迷宫城了。
<br/>　　毕竟，留下来太危险了，只有出去才算是真正的得到了宝物。
<br/>　　而一些没有得到宝物之人，或者一些对自己实力非常自信之人，则是更加拼命的开始寻找其他人，并出手抢夺。
<br/>　　“怎么都还不出来？”出口附近，苏莫闲庭漫步，跟随着出口在移动。
<br/>　　他现在得到了两件宝物，第一枚神果被他吞服了，效果还是非常不错的，到现在药力还未完全散去。
<br/>　　不过，等了这么长的时间，居然没有再遇到其他人。
<br/>　　当然，他也不急躁，那些人迟早要出去的。
<br/>　　踏！踏！踏~~
<br/>　　就在此时，苏莫的耳中，听到了一阵轻微的脚步声，好像是有人来了。
<br/>　　“哈哈，龙腾，你这个奴仆要遇到对手了！”迷宫城之外，龙城大笑了起来。
<br/>　　因为此时此刻，有一位厉害的奴仆，向出口所在的方位走去了。
<br/>　　这个奴仆的实力很强，不比他的奴仆巫铨差，绝对算是一个天赋妖孽之辈。
<br/>　　这个奴仆凭借强大的实力，以及很好的运气，得到了两件宝物，应该是想要出去了。
<br/>　　“这个奴仆不错！”龙腾微微一笑，强大的神念，关注着那位走向出口附近的奴仆。
<br/>　　这个奴仆他倒是知晓，名为阳云，是一位天阳族之人，主人也是龙族的一位妖孽。
<br/>　　天阳族是太古万族榜排名第十五的种族，无比的强大，只不过，这个阳云只能算是半个天阳族。
<br/>　　因为，阳云只有一半的天阳族血统，另一半是人族血统。
<br/>　　但是，即便如此，阳云的天赋也是非常强大，算是龙族之中，比较有名气的奴仆。
<br/>　　“阳云的实力，应该能和此人一较高下吧？”龙烨笑着说道，他倒是不认为阳云一定能胜，毕竟，守住出口的那位龙腾的奴仆，实力也非常强大。
<br/>　　“龙腾，你对这个奴仆这么有自信，我倒要看看，他能否击败阳云！”龙洌咧嘴一笑，朗声说道。
<br/>　　“呵呵，那你们就看着吧！”龙腾笑了笑，面上云淡风轻，这阳云的实力虽然挺强，但和苏莫相比，连提鞋都不配，根本就不在一个档次。
