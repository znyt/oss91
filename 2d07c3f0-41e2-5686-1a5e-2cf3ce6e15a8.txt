        童筱颖没想到傅夜沉会愿意陪自己一起向爷爷傅炎廷求情。
<br/>
<br/>        “大哥，你的好意，我心领了。你请回吧！”童筱颖黯然神伤地劝道。
<br/>
<br/>        傅夜沉淡淡地说道：“阿景是我的弟弟，我为了阿景而已。”
<br/>
<br/>        听完他说这话，童筱颖便不再劝了，劝多了反倒觉得她是在自作多情。
<br/>
<br/>        傅夜沉和童筱颖一起跪了整整一晚上。
<br/>
<br/>        然而，他们的“苦肉计”并未打动爷爷傅炎廷。
<br/>
<br/>        傅炎廷翌日起来床后，对于跪在自己房门外一宿没睡的童筱颖和傅夜沉，丝毫不予以怜惜，仍旧对他们俩视而不见地去做自己的其他事情。
<br/>
<br/>        这样的结果，傅夜沉早就心知肚明，但是，他还是愿意陪着童筱颖去了却她心里的心结。
<br/>
<br/>        “筱颖，起来吧！爷爷他……”傅夜沉的话说到一半。
<br/>
<br/>        童筱颖的身体摇摇欲坠。
<br/>
<br/>        傅夜沉心一横，直接从地上站起来，不顾自己膝盖上的麻痛，然后捞起童筱颖的身子，抱着她去了她的房间。
<br/>
<br/>        “我让厨房把早餐做好了，端过来。”傅夜沉坐在床边，看着面色惨白的童筱颖，温柔地说道。
<br/>
<br/>        童筱颖目光无神，默不吭声。
<br/>
<br/>        傅夜沉接着说道：“阿景的事情，我会再去想想办法。”
<br/>
<br/>        童筱颖这才有了点反应，抬眸看了傅夜沉一眼。
<br/>
<br/>        同样是跪了一宿一夜，傅夜沉似乎并无大碍，而她童筱颖浑身都难受得要命。
<br/>
<br/>        她饥肠辘辘不说，膝盖还痛得厉害。
<br/>
<br/>        “你先躺一会儿，我去给你拿药油过来。”傅夜沉接着说道。
<br/>
<br/>        童筱颖未吭声，直接躺下身去，并转身侧卧，背对着傅夜沉。
<br/>
<br/>        傅夜沉这才起身离去。
<br/>
<br/>        此时，傅夜沉接到了密友阿凯打来的电话。
<br/>
<br/>        “四少，您上次让我帮您查的事情，我查到了三个人。”阿凯的语气听起来非常严肃。
<br/>
<br/>        傅夜沉皱了皱眉，问：“哪三个？”
<br/>
<br/>        关于那个针对他傅夜沉的冒险者游戏幕后操纵者，他必须得尽快找出来。
<br/>
<br/>        “第一个，就是被你亲自送进监狱的毒枭秃鹰。第二个，是贩卖军火被你一举剿匪的林大。还有一个……”
<br/>
<br/>        “是谁？”傅夜沉的眉头锁得更紧了。
<br/>
<br/>        阿凯深沉地回答：“已经死了……”
<br/>
<br/>        “死了？”傅夜沉不解。
<br/>
<br/>        阿凯接着解释道：“你还记得三年前，那个一枪击毙的纨绔子弟吗？”
<br/>
<br/>        “我毙过很多坏人，我哪记得这么多？”傅夜沉不耐烦道。
<br/>
<br/>        如果对方是冲着他一个人来，他无所谓，大不了一死，但是，对方是冲着他心爱的女人来的，那么就不是他死这么简单了。
<br/>
<br/>        “死的那个男的叫凌飞翔，是……”阿凯娓娓道来。
<br/>
<br/>        傅夜沉听完他的讲述后，不禁皱起了眉头。
<br/>
<br/>        既然梁子已经结下来，就没有再挽回的余地了，那他以后，也只能小心行事。
<br/>
<br/>        只是令他不明白的是，他明明把自己的身份藏得那么深，为什么还会败露？
<br/>
<br/>        等到他拿着药油让家里的女佣端着早餐回来的时候，房间里的大床上早就没了童筱颖的身影。
<br/>
<br/>        在傅夜沉离开后，童筱颖就接到了一个陌生电话，但是电话那头的人却不令她陌生。
<br/>
<br/>        她约她见面，说是为了……
<br/>
<br/>        童筱颖想也没想，就答应她去见她。
<br/>
<br/>        离开傅家大庄园的时候，童筱颖跟伺候她的女佣说了自己的去向，但是她撒了谎。
<br/>
<br/>        所以，当傅夜沉询问童筱颖的下落时，女佣按童筱颖的原话如实回答道：“二少奶奶说，她回海滨城去找二少爷了。”
<br/>
<br/>        既然筱颖是去找昼景，那么他傅夜沉还有什么好说的？
<br/>
<br/>        傅夜沉能感觉得到，筱颖是真的不需要他。
<br/>
<br/>        他对她的一片真心，看来她是永远都不知道了。
<br/>
<br/>        童筱颖要去见的人，不是别人，而是苏漫雪。
<br/>
<br/>        自从事情的真相被傅老太爷傅炎廷知道后，苏漫雪就被傅炎廷囚禁在傅夜沉在临海城的依山别苑里，由刘管家和一个女保镖看管着。
<br/>
<br/>        苏漫雪知道自己无法再讨好傅老太爷，心里一直惶恐不安，又加之终日被囚禁，不得与外界联系，她心里更加方寸大乱。
<br/>
<br/>        她苦口婆心地向刘管家哀求，想打个电话给自己的闺蜜童筱颖，也就是傅家的二少奶奶。
<br/>
<br/>        刘管家确认电话无异常后，便将手机给了苏漫雪，并且要求苏漫雪当着他的面，将手机扬声器打开，他必须知道她们谈话的所有内容。
<br/>
<br/>        苏漫雪无奈，只好乖乖照做。
<br/>
<br/>        她拨通了电话，童筱颖这边是待电话响了许久后，才接听：“喂，您好，请问您找哪位？”
