　　或许是物必反的缘故，在那黑角域的中心位置，则是名震斗气大陆的迦南学院，只不过，在迦南学院之外的一定范围内，属于和平区域。
<br/>　　任何将黑角域里面的风气带到这里的人，都将会在第二天，成为迦南学院与黑角域交界线处一大树上的尸体，这么多年来，似乎从未有过例外。
<br/>　　据说，在那颗被称为死灵树树上，曾经被挂上了两名斗王一名斗皇的尸体。
<br/>　　慕清霄来黑角域的目的，自然是完成与药尘的约定，另外一个目的，自然是韩风手中的‘海心焰’。
<br/>　　现在，以他的实力已经能够轻松击杀韩风，自然不想在多等，异火代表着实力，自然是越多越好。
<br/>　　预控飞行了半小时左右，两人脚下的重重山峦就变得疏密起来，抬头看向远方，在视线的尽头，漆黑色的平原犹如一道黑线，将外面与里面的世界完全分割了开来。
<br/>　　“这就是传说中的黑域大平原吗？”
<br/>　　小医仙美眸中掠过一丝担忧，慕清霄捕捉到她的担忧，嘴角牵起一丝笑意，道：“不用担心，黑角域无人能奈何我。”
<br/>　　若是换做半个月前，慕清霄来黑角域，或许真的存在风险，但现在情况已经不同，他虽只是化神八重初期，可即便是高级斗宗，他也丝毫不惧。
<br/>　　依旧是如此平淡的语气，看似嚣张却极其自信，小医仙美眸痴痴的望着她，微微颔首。
<br/>　　放眼望去，一望无际的平原尽是单调的黑色，搭配略微昏暗的天空，一股压抑且令人心情烦躁的气氛，缭绕在平原上，在这种压抑的气氛下，难怪会滋生出混乱的规则。
<br/>　　两分钟后，站在百米上空，脚下已经能看到许多人影，以及无数的白色帐篷，慕清霄知道，这里只是黑角域的外围，因此并没有停留。
<br/>　　约莫十分钟，翻过一座山岭，而在山岭之脚，一座由漆黑巨石积累而成的庞大城市，若隐若现的出现了模糊的轮廓，在那城市的四方城门处，能够清晰的看见，密密麻麻的人群，犹如蚂蚁一样，汇拢着涌进城中。
<br/>　　来到城门约百米的地方，慕清霄轻轻落下，随后自储存空间中取出两挂黑袍，毕竟两人的容貌太过显眼，至于黑袍人，在混乱地带恐怕并不会少见。
<br/>　　慕清霄还好，但小医仙却不同，倾国倾城的容颜，在这混乱的黑角域中，简直就是大麻烦，随时都有可能遭遇袭击，至于目的不言明了。
<br/>　　小医仙乖巧的拿过黑袍套上，黑袍虽然有些大，但最起码能够将她整个包裹起来，甚至看不出她是男是女。
<br/>　　慕清霄要的就是这种效果，叮嘱一声后，两人就向着黑角域中有名的黑印城走去。
<br/>　　来到城门口，由于慕清霄隐匿气息的，小医仙乖巧的跟在他身边，倒也没有引起任何的注意。
<br/>　　缴纳了一些入城费用，两人就踏进黑印城中，目光在街道上扫了一圈，入眼的都是一些血拼用的武器，慕清霄都懒得看一眼。
<br/>　　两人身着黑袍走在路上，虽然不时有异样的目光投射过来，却也没有引起什么麻烦。
<br/>　　一路走下来，倒是能听到不少杂乱的信息，比如某某处售卖灵药很便宜，斗技，丹药，女人，血拼......
<br/>　　倒是‘拍卖行’三个字吸引了慕清霄的注意力，自从上次炼制过三品清灵丹后，他那破旧的药鼎就一直仍在储存空间里没换过。
<br/>　　倒不是不想换，而是没有找到好的药鼎，也不知道拍卖行有没有。
<br/>　　想到这里，慕清霄嘴角牵起一丝笑意，道：“我们先去拍卖行看看，到时候就去枫城。”
<br/>　　枫城，也是药皇韩枫府邸所在的城市。
<br/>　　小医仙微微点头，并没有开口说话，生怕给他带来麻烦，慕清霄揉了揉她隐藏在黑袍下的小脑袋，微笑道：“走吧。”
<br/>　　在街上找了一家地图商店，花费两百金币买了一份地图，虽然知道老板是黑商，但慕清霄也没有计较，他不是缺钱的主。
<br/>　　顺着地图的路线，通过曲折的街道，几分钟后就来到人流涌动的街道尽头，抬头望着映入眼中的巨大拍卖行，神情一愣。
<br/>　　乌坦城的米特尔拍卖场，比起这黑印拍卖行，简直是小巫见大巫，说不定真有让自己满意的药鼎也说不定。
<br/>　　黑印拍卖行的门口处，站着几十名脸色淡漠的黑衣劲装男子，腰间都佩着锋利的武器，锐利的目光，不断在进场的人群中扫视。
<br/>　　看了一眼人群，慕清霄就带着小医仙向拍卖行中走去。
<br/>　　由于黑角域秩序混乱，所以穿着黑袍进场的人有不少，都是担心自己拍得好物品后被惦记上，所以两人并没有引起注意。
<br/>　　顺着人流，慕清霄与小医仙就进入偌大的拍卖行中。
<br/>　　在拍卖行的内部，四周挂着巨大的屏幕，上面排列着各种参加拍卖的物品，只是略微的扫了一眼，并没有他感兴趣的东西。
<br/>　　目光扫过大厅，视线落在一个鉴宝室的门框上，随后就走了过去，小医仙寸步不离。
<br/>　　刚走进鉴宝室，便有一位穿着诱惑的待女走上前来，娇滴滴的声音透露着一丝妩媚，道：“请问两位是来鉴宝的，还是来做价格评估以便拍卖的？”
<br/>　　“后者。”
<br/>　　慕清霄言简意赅，语气平淡，毫无感情波动。
<br/>　　“请两位跟我来。”
<br/>　　待女倒是不在意，在拍卖行里，什么样的人她没见过，只是妩媚一笑，水蛇一样的腰肢便是扭动起来。
<br/>　　对于这种胭脂水粉，慕清霄一点不感兴趣，他虽然喜欢美女，但也是要看档次的，后者明显入不了他的眼。
<br/>　　跟在待女身后，走了十几米，前者就脚步就停在一个小密室前，恭敬道：“两位只需将所需要拍卖的东西交给里面的大师评估与审核，便能够在本拍卖行中获得相应的席位。”
<br/>　　微微点头，两人就向密室中走去，密室墙壁上镶嵌着月光石，光线极其明亮，刚进入密室，就有一道锐利的视线在两人身上扫过。
