　　“公孙师兄，这些剑气真是难以对付，一群剑气，居然懂得剑阵！”
<br/>　　沙漠之中，那五名武者，背对而立，每个人便是都催动这灵力，来抵挡这剑阵的攻击。
<br/>　　剩下这四人的实力，虽然不如那为首剑修，但是一个个的实力也都不比楚风眠曾经遇到过的那青衣男子要弱。
<br/>　　这些人身上所修的功法，显然都并非是平凡功法，至少也是来自于七大宗门之一的势力。
<br/>　　“居然是一群古月峡谷的弟子？”
<br/>　　楚风眠打量了半天，才终于认出了这一群人的身份。
<br/>　　七大宗门之中，最为低调的势力，便是古月峡谷，不过却并没有人，敢来小看这古月峡谷。
<br/>　　因为在七大宗门之中，最为古老的势力，便是这古月峡谷，古月峡谷据说可是有着远古时代的传承。
<br/>　　“复姓公孙，莫非就是中域百绝榜的第二人，公孙道？”
<br/>　　听到那些古月峡谷弟子，再叫为首那人公孙师兄，从楚风眠的脑海中，突然浮现出了这个人名。
<br/>　　如今的楚风眠虽然还没有去过中域，但是这中域百绝榜上人物的名字，如今楚风眠这边倒是听说过了。
<br/>　　中域百绝榜，一直便是代表着九域最为巅峰的人物，七大宗门之中，其中三个，便是位于中域。
<br/>　　所以想要从这中域百绝榜上，脱颖而出的人物，可都是经历了无数苦难的，虽然说这归墟天的名额，中域有着二十位，也并非是北域的十位。
<br/>　　但是实际上，想要进入到中域百绝榜的前二十位，实际上难度，比起进入什么北域百绝榜，西域百绝榜的前十，还要轻松的多。
<br/>　　能够从中域之中脱颖而出的武者，都是天才中的天才，而就在这无数天才之中，其中排名第二位的，便就是这眼前的公孙道。
<br/>　　眼前这公孙道的一身实力，简直是足以用深不可测来形容，就算是楚风眠贸然之间，都不一定愿意跟他交手。
<br/>　　如今看来他也是发现了关于无上剑气的秘密，才会带着这些古月峡谷的人，来抓捕降服无上剑气。
<br/>　　这无上剑气，对楚风眠有着大用，他是绝不可能放过的，但是眼前，这些古月峡谷弟子再跟这四十二道无上剑气交手，一时间，楚风眠也是不好参与进去。
<br/>　　“什么人？”
<br/>　　就在楚风眠思考之际，一声冰冷的声音，突然响起。
<br/>　　正是那公孙道的声音。
<br/>　　这公孙道，一双眼睛，正看向楚风眠所在的方向。
<br/>　　居然足以看穿我的隐藏？
<br/>　　楚风眠目光闪烁，似乎有些诧异，他的身影隐藏，让那些超越生死的老古董前来，都不一定可以说看得穿，没想到这公孙道，居然看穿了。
<br/>　　既然如此，那楚风眠也懒得在隐藏下去，只见楚风眠的身形，就在这天空之上，落到了这沙漠之中。
<br/>　　“没想到，居然有人也发现了这个地方。”
<br/>　　公孙道一边跟那些无上剑气战斗这，一边目光，却也是看向了楚风眠，打量了楚风眠一眼突然开口道。
<br/>　　“你叫什么名字？”
<br/>　　“楚风眠，北邙学院弟子。”
<br/>　　楚风眠平静的开口道，脸色极为平静。
<br/>　　“原来是北域百绝榜的榜首，怪不得也能来到这里。”
<br/>　　这公孙道一边开口，一边双眼中打量着楚风眠的目光，突然似乎是看到了什么，眼神中露出了几分冰冷到。
<br/>　　“楚风眠，我可以给你一个机会，现在你将你身上的那一道无上剑气，交出来，然后来投靠我们对付这些无上剑气，我可以放你离开，甚至在给你一笔好处。”
<br/>　　“当然，你若是不乖乖听话的话，今日我们便是先杀了你，在来对付这些无上剑气，如何选择，你自己应该清楚。”
<br/>　　这公孙道的话，虽然平静，但是实际上这话中的意思，却是霸道到极点。
<br/>　　这种平静，才是最为可怕的，仿佛是这种霸道，命令别人，都是一种理所当然的事情，平日里这公孙道，绝对就是一位狂傲到没边的人物，才会显得如此霸道。
<br/>　　“投靠你？”
<br/>　　楚风眠眼神冰冷，似乎是在思考这什么，冷冷的开口道。
<br/>　　“这样说来，除非是投靠你，今天我就别想活着离开？”
<br/>　　“不错，除非是投靠我，不然你，只有死路一条，而且你必须要将你身上的无上剑气交出来，你的身上，肯定是有着一道无上剑气，不然你根本无法寻找到这个地方。”
<br/>　　公孙道点了点头，斩钉截铁的说道。
<br/>　　“乖乖听话，我们可以给你些补偿，当然，这也是你唯一的机会，不然的话，你就得死！”
<br/>　　“公孙师兄，跟这样一个废物浪费口舌干嘛，一个就连生死大劫都没有度过的北域废物，北域百绝榜榜首？北域的人，不过都是一群土著，废物罢了，让他投靠您，太降低您的身份了。”
<br/>　　在公孙道身边，另外一尊古月峡谷弟子，阴冷的冲着楚风眠开口道。
<br/>　　“小子，现在识相一点，就乖乖跪下，臣服公孙师兄，作为公孙师兄的奴隶，也许表现的好一点，未必不会将你带到我们古月峡谷去，告诉你，你们北域，就是一处偏僻垃圾的地方，作为我们公孙师兄的奴隶，是你的富贵，这可是多少人梦寐以求的事情，你可知道？”
<br/>　　听到这话，楚风眠心中都极为冰冷。
<br/>　　这些七大宗门的弟子，看来是比起万年之前，一个个都要狂妄的多了，在这九域之中称王称霸久了，真以为自己都是什么大人物了。
<br/>　　这公孙道上来就是命令楚风眠交出无上剑气，要去投靠他。
<br/>　　眼前这古月峡谷弟子，更是狂妄的没变，要让楚风眠来做奴隶。
<br/>　　楚风眠目光一闪，看到那古月峡谷弟子，平静的开口道。
<br/>　　“话说的那么大，也不怕闪了舌头，古月峡谷？就算是你们谷主来求我，我都不会去，真以为你们古月峡谷能够在九域称王称霸，在我的面前，不过只是一群蝼蚁罢了。”
<br/>　　“找死！”
<br/>　　那刚刚开口的古月峡谷弟子，听到楚风眠的话，直接是被激怒了，脸色骤然一寒，一双眼睛，变得如同黑暗一般的深邃。
