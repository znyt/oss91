        傅昼景却毫不留情地甩开了季思妍的手。
<br/>
<br/>        他发现，自从他跟她想要断绝那方面的关系后，季思妍就完全像变了一个人一样。
<br/>
<br/>        她时而柔情似水，时而心机满满。
<br/>
<br/>        这让他完全分不清，到底哪个才是真正的她。
<br/>
<br/>        “还有，童筱颖和我哥哥季运恒也有一腿。我哥哥说，童筱颖之前在‘夜惑’里卖过唱。她跟我哥哥就是在‘夜惑’里认识的。而且，她跟我哥哥上过床！阿景，童筱颖这种女人，不值得你去爱。只有我，为了你，才是身心最干净的那个！我季思妍这一辈子，只有你一个男人！”季思妍信誓旦旦地说，并掏出手机，把那天季运恒发给她的照片和视频一并摆在了傅昼景的办公桌上。
<br/>
<br/>        照片里，是季运恒搂着童筱颖的“暧昧照”，更像是故意摆拍的。而后面的视频，是童筱颖被两只手铐锁在了车椅上，被季运恒撕破了上衣轻浮她的内容。
<br/>
<br/>        傅昼景看到这里，抬起迷茫的眼睛，一下子急怒攻心，突然紧紧地钳住了季思妍的手腕，疾言怒色地吼道：“我说过，不许你动筱颖，你为什么要动筱颖？联合你哥哥一起来‘欺辱’筱颖，是觉得我智商低，很好骗吗？”
<br/>
<br/>        筱颖委身于季运恒这件事情，他早就知道了。
<br/>
<br/>        但他知道，筱颖是为了帮他赚钱还债，才不得已那么做的。
<br/>
<br/>        当时，他气愤、恼火、情绪低落，完全是因为筱颖不相信他的实力。
<br/>
<br/>        如果，当时他没有被朋友出卖，身负巨额外债，他和筱颖也不会走到彼此都失了身的地步。
<br/>
<br/>        这件事情，一直是他心里的一根刺，一道疮疤。
<br/>
<br/>        而她季思妍现在在做什么？
<br/>
<br/>        将他的刺越扎越深，将他的疮疤越揭越痛。
<br/>
<br/>        季思妍也没想到傅昼景的反应会这么大，她以为他顶多只是伤心，然后接受童筱颖这份已经签好字的离婚协议书，但是，事情的结果，并不是她所料想的那样。
<br/>
<br/>        那两百万，她已经给童筱颖了。
<br/>
<br/>        如果，阿景不接受这份离婚协议书，那她岂不是人财两空。
<br/>
<br/>        季思妍心里顿时郁郁寡欢。
<br/>
<br/>        不过，话说回来，她的哥哥自从给她发了童筱颖的暧昧照和视频后，就在临海城失联了。
<br/>
<br/>        这当中，到底发生了什么事情？
<br/>
<br/>        “还有什么可说的吗？如果没有了，就给我滚！”傅昼景松开了季思妍的手，心烦意燥地呵斥。
<br/>
<br/>        最近他工作上不太顺利，就连身边的人，都给他整一些乱七八糟的事情。
<br/>
<br/>        季思妍怔愣地回过神来，双手不经意间地握成了拳。
<br/>
<br/>        既然他不仁，那么休得怪她不义。
<br/>
<br/>        她是不会就此放过童筱颖的！
<br/>
<br/>        季思妍愤愤不满地甩手离去。
<br/>
<br/>        傅昼景看着季思妍离开自己办公室的背影，黑眸意味不明地微微眯了起来。
<br/>
<br/>        几天前，大哥傅夜沉告知他被上头的人“盯”上了后，他跟爷爷打了一通电话，并跟爷爷做了一笔“交易”。
<br/>
<br/>        爷爷说，苏漫雪背叛了大哥傅夜沉，在外勾搭上了季思妍的哥哥季运恒。
<br/>
<br/>        这事是真是假，傅昼景并不关心。
<br/>
<br/>        所以，爷爷想要“做掉”苏漫雪，让他把苏漫雪认成童筱颖，他便听从爷爷的安排。
<br/>
<br/>        爷爷故意放了诱饵，命令刘管家对苏漫雪的所作所为，睁只眼闭只眼。
<br/>
<br/>        因此，苏漫雪才能和童筱颖轻而易举地互换了身份。
<br/>
<br/>        本来他是想自己去骗苏漫雪签法人转让协议，却没想到，季思妍会突然横插一脚进来。
<br/>
<br/>        刚刚他利用了季思妍，同时也利用了苏漫雪。
<br/>
<br/>        就在他为自己的“行径”暴露，而束手无策的时候，他也没想到，季思妍会突然来他这里闹这么一出。
<br/>
<br/>        他之所以故意激怒季思妍，其实心里已经断定季思妍会去找苏漫雪算账。
<br/>
<br/>        以季思妍的个性和她的背景，她一定会想办法让苏漫雪来承担这一切后果。
<br/>
<br/>        到时候，只要苏漫雪以童筱颖的身份被抓，那么他就安然无恙了。
<br/>
<br/>        这就是爷爷的计策，但同时，他得放弃“远东军机厂的项目”，并且离开傅氏集团，并带着童筱颖一起离开滨江城和临海城。
<br/>
<br/>        从此以后，他一无所有，身边只有筱颖了。
<br/>
<br/>        爷爷说，要他好好珍惜筱颖。
<br/>
<br/>        因为，筱颖为了能让爷爷救他，和大哥一起跪在了爷爷房门外一夜。
<br/>
<br/>        而这件事情，傅夜沉并不知情。
<br/>
<br/>        被关在依山别苑里的童筱颖也不知情。
<br/>
<br/>        傅炎廷下了一盘棋，而傅昼景也好，童筱颖也好，苏漫雪也好，都只不过是他手中的棋子。
