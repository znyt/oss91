　　嗖…
<br/>　　莫悠凭空而现。
<br/>　　“谢谢。”莫悠说了一声，语气有些莫名。
<br/>　　“谢什么？”萧逸淡漠道。
<br/>　　莫悠没有坐下，而是站着道，“若非萧逸师弟让青麟师弟离去，我如何有脸面现身？”
<br/>　　萧逸摇了摇头，“我没记错的话，这是莫悠师兄第二次谢我。”
<br/>　　“第一次，我承了，现在却后悔了。”
<br/>　　“这一次…”
<br/>　　萧逸停下了话语，不语。
<br/>　　莫悠看着萧逸那如若寒霜的脸色，皱了皱眉，亦不语。
<br/>　　半晌，莫悠率先张开了嘴巴，“我来，便是给萧逸师弟一个交代。”
<br/>　　萧逸淡漠出言，“莫悠师兄，可还记得第二轮比拼前，答应过我什么？”
<br/>　　“记得。”莫悠点了点头。
<br/>　　当初萧逸的原话是，‘第二轮比拼上，便拜托莫悠师兄照顾秦翊和童烨了’。
<br/>　　当时在萧逸看来，这二人是黑云学教中仅次于青麟的最不让人省心的弟子。
<br/>　　至于青麟，萧逸却未担心过。
<br/>　　因为他相信青麟的实力，即便五大首席，也休想奈何青麟。
<br/>　　而当初莫悠的话是，‘若这二人有半分闪失，他拿命来赔’。
<br/>　　现今，童烨和秦翊二人，确无半分闪失，顶多是轻伤。
<br/>　　唯独青麟，这个让萧逸放心，未拜托莫悠照顾的弟子，却重伤濒死。
<br/>　　萧逸没有明说。
<br/>　　但莫悠却明白他的意思。
<br/>　　莫悠脸上，闪过一丝愧色，“萧逸师弟该知晓，以我的性格，不会辩驳些什么。”
<br/>　　“若青麟之事，需要我用命赔，可以。”
<br/>　　萧逸摇了摇头，“莫悠师兄也该知晓，青麟现今无事，否则，你也无法站在这里与我一谈。”
<br/>　　莫悠脸色一凛，“想必，童烨他们，已经将天藏秘境内的事告知萧逸师弟。”
<br/>　　“若是换作萧逸师弟在那般情况，如何处置？”
<br/>　　“我仅一个妹妹，不帮？”
<br/>　　萧逸没有回答，而是语气霎时一沉，“15年前，上一届百院之争时，莫悠师兄忽然退赛，也是为了莫忧吧。”
<br/>　　“你知道？”莫悠脸色微变。
<br/>　　萧逸不语。
<br/>　　莫悠脸色肃然，点了点头，“不错。”
<br/>　　“15年前，上一届百院之争时，我还未满20，已力压天藏学宫十二峰首席。”
<br/>　　“哪怕是当时的天藏首席，也仅仅能在我剑下撑过一分钟。”
<br/>　　“其他各学院首席，连我一剑都接不住。”
<br/>　　“那时的我，一战成名，意气风发，甚至敢无视一切中域天骄。”
<br/>　　莫悠的脸上，尽是傲然之色。
<br/>　　这才是莫悠，虽平日里云淡风轻，但骨子里，却是傲气非凡，有着一切天骄该有的傲气，却比任何天骄更加自信，更加桀傲。
<br/>　　当然，他也有这个资本。
<br/>　　“也是那时。”莫悠脸上傲气，忽然褪去，取而代之的，是一抹复杂。
<br/>　　“天藏学宫通天峰长老寻上我。”
<br/>　　“便有了现今的天叠峰亲传弟子莫忧？”萧逸冷声打断道。
<br/>　　“不错。”莫悠点了点头，“我自幼与莫忧相依为命，一无势力，二无底蕴。”
<br/>　　“但只要我放弃第三轮比拼，天藏学宫可答应那时才年纪5岁的莫忧进入天藏学宫修习。”
<br/>　　“待遇，等同亲传弟子；甚至天叠峰长老亲自出手，前往万妖险地捉了一头魅心妖蝶，以供莫忧觉醒武魂之用。”
<br/>　　“也是那时起，有了现今的莫忧。”莫悠忽然咬了咬牙。
<br/>　　“萧逸师弟该知晓，武魂有影响武者性格之效。”
<br/>　　“若武者无法完全掌控武魂，便会受武魂侵袭；反之，武者能完全掌控武魂，便能将武魂之力发挥到最大。”
<br/>　　“魅心妖蝶，乃是最珍稀的妖兽之一。”
<br/>　　“作为武魂，位列紫色，莫忧才堪堪20岁出头，如何能完全掌控？”
<br/>　　“魅心妖蝶，是怎样的妖兽，萧逸师弟闯荡四方，该也知晓。”
<br/>　　莫悠一口气说罢，认真地看着萧逸，“此事，我希望萧逸师弟能明白，若要怪，便怪我。”
<br/>　　“我明白。”萧逸点了点头。
<br/>　　武者觉醒武魂，与自身资质、性格、际遇等等，都有关联。
<br/>　　反之，武魂也影响着武者。
<br/>　　这也是为什么修习火之一道的武者大多性格暴躁，修习剑之一道的武者，或者觉醒了剑类武魂的武者异常狂傲的原因之一。
<br/>　　当然，若是武者能够完整掌控武魂，便不会有这些影响；或者武者道心足够坚固，也能避免这样的影响。
<br/>　　不过，萧逸的脸色，仍旧冰冷至极。
<br/>　　“我明白。”萧逸淡漠地说着，语气霎时一凛，“但不代表我认可。”
<br/>　　“若我早知此事，十数天前，我绝不会答应退出第二轮。”
<br/>　　“当年莫悠师兄不过区区一句话，不，仅仅是‘放弃’二字，便能让得莫忧在天藏学宫受尽万千宠爱，嚣张跋扈。”
<br/>　　“她可以为之自傲，为之肆无忌惮、目空一切。”
<br/>　　“可这不代表我认可。”
<br/>　　不错，萧逸早已知晓15年前上一届百院之争的事。
<br/>　　不过，那是他在天藏地域风刹殿主殿查到的事。
<br/>　　故他堪堪赶在第二轮比拼结束前赶回来，就是担心有什么意外。
<br/>　　15年前，莫悠可以为了莫忧不顾一切。
<br/>　　15年后，他同样可以。
<br/>　　若他早知此事，第二轮前，绝不会答应退出比拼。
<br/>　　萧逸冷眼盯着莫悠，“你可知，青麟的浑身伤势中，哪怕是燃烧武魂，只要有个度，便还能挽救。”
<br/>　　“即便燃烧过度，也顶多是资质后退。”
<br/>　　“唯独破魂针，那是致命之伤。”
<br/>　　“天叠峰长老手中重宝，破魂针，当年曾创下秒杀圣皇境巅峰的战绩。”
<br/>　　“青麟如何能受？”
<br/>　　“心神溃散，行尸走肉的后果，莫悠师兄该很清楚。”
<br/>　　话音落下。
<br/>　　空气中的温度，陡然下降到零度以下。
<br/>　　莫悠感受着周遭冰冷的气息，却脸色不变。
<br/>　　“所以，我今日便是来给萧逸师弟一个交代。”莫悠沉声说道。
<br/>　　锵…
<br/>　　一道清脆剑鸣，陡然响起。
<br/>　　一道道锋锐剑意，顷刻笼罩莫悠。
<br/>　　嗤…嗤…嗤…
<br/>　　无数道锋锐剑气，肆虐在莫悠身上。
<br/>　　不过片刻，莫悠已遍体鳞伤，浑身是血。
<br/>　　“这算我还给青麟师弟。”莫悠脸色淡漠，“若还不够交代，我可以继续。”
<br/>　　锵…空气中的剑意，再次肆虐。
<br/>　　萧逸摇了摇头，“这不算交代，青麟也不需要这些。”
<br/>　　说罢，萧逸转身离去，并未看置身肆虐剑意中的莫悠一眼。
<br/>　　“萧逸师弟。”莫悠咬牙看着萧逸离去的背影。
<br/>　　萧逸摇着头，“你如何，那是学教老师和前辈该处理的事，我也管不着。”
<br/>　　“但青麟、童烨等其他学教弟子的事，我能管。”
<br/>　　“你想做什么？”莫悠脸色一变。
