        落以柔正襟危坐，乖乖地坐下。
<br/>
<br/>        “落小姐，请问你是来应征本公司的总裁助理的秘书的吗？”郑忠威淡定地从资料夹中挑出她的资料，好奇地睨着她。
<br/>
<br/>        “是。”
<br/>
<br/>        落以柔坐得端端正正、小手交叠在并拢的双腿上，腰杆挺直，坐姿呈九十度。
<br/>
<br/>        郑忠威从她的履历表中得知她父母不详，从小在孤儿院长大，心不禁微微地抽紧。才几岁的女娃儿就历经这么多人世坎坷、人情冷暖……
<br/>
<br/>        “你从大一起就开始做文员类的兼职，这又要学习又要打工，不累吗？”郑忠威的目光瞅着眼前瘦弱的女孩，口气不由得透露一丝像老父亲模样的温情。
<br/>
<br/>        “我需要工作。”落以柔不卑不亢地响应。
<br/>
<br/>        “缺钱？”郑忠威眉一挑、眼一沉，不悦地暗想，她怎么会瘦成这副德行，一百六十公分，才三十九公斤而已？
<br/>
<br/>        她是如何照顾自己？
<br/>
<br/>        “对！”落以柔点头。
<br/>
<br/>        “在校成绩好吗？”郑忠威直截了当地问。
<br/>
<br/>        落以柔掏出包包里的成绩单，她作为应届毕业生，偶尔有些顾主会要求看看她的成绩，所以应征时她通常会随身携带。
<br/>
<br/>        郑忠威挑着眉看完成绩单，不敢置信地盯着看似平凡的落以柔。
<br/>
<br/>        “只做助理秘书会不会太委屈你了？”郑忠威忍不住地出声，对于每科都得高分的她，不由得露出赞赏的眼神。
<br/>
<br/>        “贵公司其他职位要求的条件太高，我似乎只适合做助理秘书。”落以柔并不贪取不属于她的一切。
<br/>
<br/>        “很好，我决定聘请你了！”郑忠威挑着眉，很欣赏她不卑不亢、不强求的心态。
<br/>
<br/>        落以柔的应聘全过程，傅夜沉在自己的办公室里知道得一清二楚。
<br/>
<br/>        在这个女孩身上，真的可以找到许多筱颖的影子。
<br/>
<br/>        筱颖是孤儿，她也是孤儿，筱颖在校期间成绩优异也是勤工俭学，这个女孩亦是如此。
<br/>
<br/>        “周休两日，朝九晚五，中午十二点休息到一点半。薪水……一个月给你五千好了。”郑忠威接着说道，“你还有什么疑义吗？”
<br/>
<br/>        怎么会这样？一个月五千？
<br/>
<br/>        薪水高得让她不自觉半张开嘴，落以柔失神的瞅着郑忠威，怀疑是不是自己的耳朵听错了。
<br/>
<br/>        “有问题吗？”郑忠威又问道。
<br/>
<br/>        落以柔回过神来，连连摇头，并且非常郑重地道了声：“谢谢。”
<br/>
<br/>        “那么，现在就可以开始工作吗？”郑忠威微笑着询问道。
<br/>
<br/>        落以柔精神抖擞，义不容辞地回答：“嗯，可以！”
<br/>
<br/>        应征的当天即刻展开忙碌的工作，让落以柔忘却生涩和脚痛，专注于上司交代下来的诸多事务上。
<br/>
<br/>        一整天下来，除了用餐时间得以放下手中的工作外，其余时间都坐在计算机前奋战。
<br/>
<br/>        午夜十二点，落以柔终于将郑忠威交代的资料输入整理完毕，开始收拾了包包下班。
<br/>
<br/>        “当！”落以柔一跛一跛地踏出电梯，每走一步，麻辣地刺痛便更加明显。
<br/>
<br/>        她的小脸渐渐扭曲，脚踝的痛楚冉冉地往上，看样子，她是没时间去看医生了。
<br/>
<br/>        “你怎么才下班？”
<br/>
<br/>        冷冷的话语如浸人寒冰中，毫无情绪波动地从身后传来。
<br/>
<br/>        落以柔惊愕地回过头去，在半空中对上了刚从另一台电梯里走出来的傅夜沉的目光。
<br/>
<br/>        “大哥！”落以柔惊喜地唤了声，打趣地说，“你这不也才下班嘛！”
<br/>
<br/>        “跟我一起走，我送你！”
<br/>
<br/>        傅夜沉冷然地吭声，语调淡漠得探测不出任何的温度，只身往前走去。
<br/>
<br/>        “不用了，不用麻烦你了。啊……”落以柔不曾细想直接回绝，身子猛地站立，脚踝处袭上一股灼热的剧痛。她苍白着脸庞，咬紧牙关低喘呻吟。
<br/>
<br/>        “怎么？”
<br/>
<br/>        傅夜沉徐步回转，手臂里挂着灰色的西装外套，态度从容自在，似乎听到细微的吃痛声，目光瞥向她，毫无温度。
<br/>
<br/>        落以柔早在他返回的前一刻，敛起痛楚的模样，她知晓拒绝对她并无好处，时间已晚，无法忍着痛楚自行返家，只好接受他的好意。
<br/>
<br/>        “走吧！”傅夜沉没多问，往前开路。
<br/>
<br/>        离开傅氏集团大厦后，傅夜沉昂首大步往室外的停车位移动。
<br/>
<br/>        因为他要加班，所以他便让司机将他的车开出了地下室，方便他下班后直接去前坪取车。
<br/>
<br/>        而身后，落以柔缓慢的步伐落后了走在前头的他一大截的距离，他都未发觉。
<br/>
<br/>        傅夜沉一直不曾回头，自然无法注意到她那不稳的脚步、略显狼狈的姿态，只当是误认为她人长得娇小，自然走得比较慢。
