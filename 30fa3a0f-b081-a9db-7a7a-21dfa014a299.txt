　　于是道风坚定信念，等待机缘，直到那一次，镇元大仙前来告别，跟他讨论了一番生死，道风这才醒悟。
<br/>　　斩元神之道，在于舍身。
<br/>　　自己虽然一向不怕死，但要主动求死，心里肯定是有顾虑的，毕竟自己一身修为来之不易，但他越是在乎这些，就越是没办法斩去元神。
<br/>　　破而后立，死后还生。
<br/>　　首先就是要去死，而且要不带一点希望地去死，不能给自己留下一点活的希望，只有做到这样，才有斩元神的可能。
<br/>　　领悟过这一点之后，道风也踌躇过——他现在实力已经很不错了，三界之内也算是绝顶强者，是为了斩元神而去死一次，还是保持现有的一切。
<br/>　　换别人身在这种处境，十有八九都是不敢迈出这一步的。但道风是个追求极致的人，对于极端力量的渴望，时刻勾引着他的冲动。
<br/>　　后来，在得知了叶少阳身世的真相、明白了不管是他、还是叶少阳，其实都只是一个棋子之后，他心中的信念和信仰彻底崩塌，于是选择了这条路。
<br/>　　他筹划取得了龙晶，这件稀世珍宝，能帮助他聚拢精魄，聚魂重生……但这些都是传言，能不能做到，他也不知道。
<br/>　　况且，他本是斩过了魂魄的，虽然他斩尸之后，又自己创造出了三魂七魄，但跟一般的鬼魂已经不同，能否重生……他之前说五六分把握，还是说多了，最多也只有三分。
<br/>　　差不多没什么生还的希望。
<br/>　　想要斩却元神，也必须要有这种心态。
<br/>　　只是，如果死在这件事上，他是一点也不遗憾的。
<br/>　　试试吧。不试试怎么知道？
<br/>　　他跟叶少阳和杨宫梓等人道别，安排好风之谷的一切，为的就是如果自己挂了，至少能将他们的损失降到最低。
<br/>　　安排好这一切之后，他就需要一个契机了。
<br/>　　想要元神俱灭，倒是很容易，自己了断就可以了，但想要重塑元神，需要一份执念，这种执念不是自杀能给的，而是需要战斗。再说要是不能重生的话，自杀了就太浪费了，他就瞄准了法术公会，想要在死之前，帮叶少阳再做最后一件事，真死了也算有安慰了。
<br/>　　他想到了去找星月奴火拼，恰好得了秋莹的线索、假叶少阳又来找自己，于是将计就计，抱着赴死的心就去了。
<br/>　　两人听完他这讲述，惊得说不出话。
<br/>　　“所以，你现在已经斩三尸证道了？”芮冷玉用不可置信的口吻说道。
<br/>　　“是。”道风回答了一个字。
<br/>　　林三生和芮冷玉面面相觑，心中激动得不行。混元大道啊，那是一切生灵最终极的向往、但也是无数生灵不敢奢望的。
<br/>　　道风，居然真的做到了……
<br/>　　林三生刚要开口恭喜他，道风说道：“并没有这么简单，我现在只是堪破了大道，还没有真正走上去。我而今靠了这龙晶，勉强重塑了元神，我得先恢复肉身和魂魄，元神才能有所依附。”
<br/>　　林三生皱眉道：“还需重塑肉身和魂魄，那跟最初有什么差别？”
<br/>　　“差别是，这肉身和魂魄完全属于我自己，从此六道轮回困我不得，生死簿上也不再有我的名字。”
<br/>　　两人心中震惊，实在无法领会道风说的这种境界，大概就是所谓的超脱吧。像是他们，根本无法想象这种境界。
<br/>　　“就是说，你总算跳出了棋盘，不再是别人的棋子了？”
<br/>　　“还差最后一步。”道风说道，“我需要养息元神，重塑血肉，还需要一些东西，而今我却不能出去，只能待在这里，少阳那边的事，交给你们了。”
<br/>　　林三生皱眉道：“为何不能出去。”
<br/>　　“我离证道还差一步，如今出去，只是任人宰割。”
<br/>　　林三生笑道：“这你放心，保护你还是没问题的。”
<br/>　　“你保护不了我。”
<br/>　　“我有空界这千军万马，再说还有捉鬼联盟，怕什么来。”
<br/>　　“要杀我的那位，你们这些在他面前全都无用。”
<br/>　　林三生一下怔住，道：“你说的是……”
<br/>　　“酆都大帝。”
<br/>　　两人长大了嘴巴。
<br/>　　林三生道：“不会吧道风，你的身份……你如今证道，他应该欣慰才是，为何要加害你。”
<br/>　　道风轻笑，“我为棋子时，是他手中的‘车’，他为了整盘棋的胜负，自然不肯我死了，而且我跳出棋盘，大乱了他的部署，那他就容不得我了。”
<br/>　　林三生若有所思。
<br/>　　道风道：“我在这里，一时半会还算安全的。我有冷玉在这，你不用管我，只保护好山河社稷图，另外，你可去风之谷帮我寻找宫梓，之前我担心她阻止我，并未跟她透露一切，如今可以让她来了，只是别人不要提起。”
<br/>　　林三生点头道：“了然。”
<br/>　　随即提出，让道风和芮冷玉暂时住在自己那座道观里，里头还有不少自己的弟子，可以照顾他们。
<br/>　　道风答应下来。
<br/>　　林三生于是带着芮冷玉上山，来到这道观里——这道观已经被改名叫“隐贤观”，取招贤纳士的意思，与林三生最近招兵买马的战略符合。
<br/>　　因为道风被芮冷玉握在手中，正好林三生也不用跟弟子们解释了，只说芮冷玉在这边赞助，给她介绍认识了两个得力的弟子，挑选了一个房间住下。
<br/>　　临走之前，林三生问道风还需要什么吗。道风表示暂时不用，林三生就先告辞了。
<br/>　　芮冷玉把道风放在桌上，在房间里走来走去，四下打量着，感觉这里跟人间的道观差不多，厢房很清静。芮冷玉想起自己的出身，感叹道：“这下我可真是什么地方都住过了，人间，青冥界，鬼域，现在又到了这洪荒世界来，猛地想起来，可真是不可思议，人间有这种经历的，大概只有我一个了吧。”
<br/>　　“所以，你是转世鬼童。”
<br/>　　芮冷玉听了这话，更加感慨，说道：“照这么说，我还应该去太阴山了。”
<br/>　　随即想到叶少阳，对道风说道：“之前林三生说，他们不日就要进入古墓，去找少阳了，我想跟着一起去。”
<br/>(本章完)
