　　那老者说到这里，微微一顿，接着又道：
<br/>　　“首先，按照以往的考核惯例，咱们从一品炼丹师的考核开始！”
<br/>　　“凡是准备参加一品炼丹师考核的，请自己主动站到台上来吧！”
<br/>　　考核大台很大，足足有两个足球场大小，即便是几千人同时上台，也根本不在话下。
<br/>　　那丹师盟会的长老话音落下之后，那些参加一品炼丹师考核之人，当即纷纷跳到高台之上。
<br/>　　叶尘也没有想太多，也跟着跳了上去。
<br/>　　然而，
<br/>　　等他跳上台之后，这才发现情况有些不太对头。
<br/>　　原来，
<br/>　　周围这些参加一品炼丹师考核的，全都是十来岁的孩童，最大的也就十四五岁，最小的甚至才七八岁。
<br/>　　叶尘站在众人之间，顿时有一种鹤立鸡群的感觉。
<br/>　　饶是叶尘两世为人，看到这一幕，也不由得老脸微微发烫。
<br/>　　而很快，
<br/>　　台下的众人也发现了这个现象，顿时纷纷哄笑起来，
<br/>　　“我没有看错吧？你们快看那小子，至少也有二十岁了吧？居然也来参加一品炼丹师的考核？”
<br/>　　“天呐！这人的天赋，到底是有多差？”
<br/>　　“这么大岁数了，连一品炼丹师都还不是，还想走炼丹师这条路，这不是自讨没趣吗？”
<br/>　　“哈哈，此人的毅力，倒也值得我辈学习！”
<br/>　　……
<br/>　　台下众人越说越起劲，开始肆无忌惮的嘲讽起来。
<br/>　　甚至连蝉衣小姐和屠苏公子，两大丹道天才，也不由得侧目而视。
<br/>　　蝉衣小姐在叶尘身上扫了一眼，不由得轻轻摇了摇头，便将目光挪开，不再多看。
<br/>　　而那屠苏公子，脸上则露出一抹鄙夷之色，
<br/>　　“看来云丹城还真是没落了，真不知道这种垃圾，是怎么混进来的！这等资质，也妄想成为炼丹师，真是不知所谓！”
<br/>　　一旁的蝉衣小姐听到这话，顿时秀眉微微一蹙，忍不住反驳道：
<br/>　　“屠苏师兄，此言差矣！云丹祖师曾有云，有教无类！她老人家苦心建立云丹城，就是为真正热爱炼丹之人，提供一个可以修习炼丹之术的平台！”
<br/>　　“我观此人，虽然炼丹天赋不高，但修为并不比你我差！却依旧能顶着被众人嘲讽的压力，前来参加考核，显然是真的热爱炼丹之术！”
<br/>　　“更何况，这常言道，勤能补拙！以他的修为，活个几百岁应该不再话下，将来未必不能在炼丹上，有一番成就！”
<br/>　　屠苏公子听到这话，顿时气得不行，不由得冷冷一笑，道：
<br/>　　“蝉衣师妹的眼光，还真是独到的很呐！明明就是一个废柴，被你这么一说，好像连你我都不如他了呢！”
<br/>　　蝉衣小姐依旧淡淡道：
<br/>　　“凡事皆有可能！未来的事情，谁也说不准！”
<br/>　　屠苏公子顿时乐了，
<br/>　　“好呀！那我们就来打个赌，若是将来有一天，他的丹术水平真的能超过我，我改口叫你师姐！”
<br/>　　……
<br/>　　就在两人争论之时，台上的考核，已经在有条不紊的进行着。
<br/>　　一品炼丹师的考核内容十分简单，参加考核者，只要能任意炼制出一枚合格的一品丹药，便算是通过。
<br/>　　因为参加一品炼丹师考核的，都是一些十岁上下的孩童，本身的修为，基本上还在炼气境，自然也没有能力凝聚出内火，所以火焰和丹炉，都是由丹师盟会提供。
<br/>　　叶尘本以为，这些孩童年纪尚轻，成功率应该不会太高。
<br/>　　然而没想到，一连十几个孩童考核之后，居然只有五人，炼制出的丹药，因为差点火候，被评判为不合格，其他所有人全都通过考核！
<br/>　　不过仔细想想也对，
<br/>　　一品炼丹师的考核，没有明确的题目，只要能炼制一枚一品丹药，便算是考核通过，因而这些孩童，在前来考核之前，必定都是有着一定把握的，否则也不会前来。
<br/>　　而丹师盟会的工作人员，办事的效率也十分的高效，当即便给那些考核通过者，颁发了一品炼丹师的丹师宝卷。
<br/>　　望着那些拿着丹师宝卷，高高兴兴跳下台的孩童，就连叶尘也不由得暗暗感慨，
<br/>　　“这云丹城，不愧是整个修真界，都赫赫有名的丹师之都啊！”
<br/>　　很快，
<br/>　　终于轮到叶尘了，
<br/>　　由于他在人群之中，实在太过于显眼，台下很多人看到他走到炼丹桌前，顿时又纷纷起哄起来，
<br/>　　“你们都猜猜看，这小子能通过考核吗？”
<br/>　　“他都这么大岁数了，居然还在参加一品炼丹师的考核，多半是没有什么天赋，我看，悬！”
<br/>　　“嘿嘿，即便是通过考核，我猜他炼制的丹药，最多也就是勉强合格的粗制丹药！”
<br/>　　“话说今天参加考核的人虽然不少，但好像还没有一个人，能够炼出精品级别以上的丹药吧？”
<br/>　　……
<br/>　　同样的丹药之间，也是有品阶高下之分的，按照等级，从低到高，一共分为：粗制、普通、精品、完美！
<br/>　　而评判一个人的丹道天赋，除了丹药的本身品阶之外，还有一个重要的标准，便是丹药的精致程度。
<br/>　　同样的一枚丹药，精品级别甚至完美级别的丹药，自然更能体现出，一名炼丹师对丹道的理解水平高低。
<br/>　　在众人的议论声和嘲讽声中，叶尘已经开始开火、热炉，然后将各种药材，缓缓投入丹炉之中。
<br/>　　其实以他如今的炼丹水平，炼制一品丹药，哪里还需要借助外火，甚至连丹炉都不需要，直接以真元化炉即可。
<br/>　　不过既然考核规则如此，叶尘也不想节外生枝，他只想快点拿到丹师宝卷，然后尽快返回城外，将灵溪等人接回。
<br/>　　唰唰唰！
<br/>　　随着叶尘行云流水般操控着丹炉，台下众人的眼神，也开始慢慢发生了变化，嘲笑声也开始减弱了许多，
<br/>　　“他选的好像是，一品丹药中，难度最高的洗髓丹！”
<br/>　　“而且看这小子的手法，似乎也很娴熟啊！”
<br/>　　“哼！手法娴熟有什么用？炼丹可不光是看手法的！”
<br/>　　“就是！即便成功了又如何？区区一品丹药而已！”
<br/>　　……
<br/>　　众人看到叶尘的炼丹手法之后，虽然对他略微有些改观，不过依旧没有太放在心上。
<br/>　　毕竟，一品丹药，对于场中的绝大多数人而言，根本不值一提。
<br/>　　终于，
<br/>　　在众人的议论声中，叶尘这边也到了收尾阶段，
<br/>　　然而就在这时，
<br/>　　丹炉忽然发出“彭”的一声！
<br/>　　众人愣了片刻之后，忽然哄笑起来，
<br/>　　“不是吧？这小子炼制一品丹药，居然炸炉了？真是个人才啊！”
<br/>　　就连一旁的那名丹师盟会的长老，也不由得暗暗摇头。
<br/>　　就在他准备宣布，考核失败的时候，两眼忽然一瞪，死死的盯着叶尘跟前的丹炉，眼眸中透出一抹浓浓的惊喜之色，
<br/>　　“原来不是炸炉！他居然炼制出了……完美品级的丹药”
