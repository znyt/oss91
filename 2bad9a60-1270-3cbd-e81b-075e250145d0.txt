　　顺着李然手指的方向看去，可以看到有一辆吉普车和两辆罩着军绿色雨布的卡车停在进入军事监狱的检查哨前，等待着检查。
<br/>　　三个身上挂着自动步枪，穿着雨衣的士兵走出检查岗，拿着手电对三辆汽车进行检查并且通过证件，核对着汽车中司机的身份。
<br/>　　而在三个士兵在对汽车，还有司机身份进行检查的时候，罗昊、李毅峰、慕容飞、李然四个人已经是借着夜色和暴雨的掩护，潜行到了两辆卡车附近，等到两个士兵对卡车车厢中的货物完成检查后，四个人分成两组分别爬上两辆卡车。
<br/>　　这两辆卡车应该是给这个军事监狱运送给养的车辆，卡车中装得全都是蔬菜、鲜肉、还有一些罐头食物。
<br/>　　完成了对吉普车和卡车的检查，三辆汽车被放行，车厢摇晃中，汽车通过了监狱大门驶入这座军事监狱中。
<br/>　　如果不是因为正好遇到了运送给养的车队，想要进入这个守备森严的军事监狱，还真不是一件容易的事情，所以，就如罗昊一直坚信的那样，一场战争的胜利还是失败，运气非常重要，有些时候甚至是决定战争胜败的关键，对于特种作战而言，更是如此；而显然，这次罗昊他们的运气还不错。
<br/>　　汽车在军事监狱中停稳后，负责运送给养的士兵并没有立刻就把车上的给养给搬运下来，在这样糟糕的天气下，他们最想做的就是回到自己的宿舍，换上一身干净的衣服。
<br/>　　等到汽车周围的士兵全部离开后，罗昊、李然、李毅峰、慕容飞四个人分别从两辆卡车车厢中爬了出来，蹲在卡车四周，跟联合司令部取得联系。
<br/>　　“‘波塞冬’00，Alpha已经进入军事监狱，准备执行下一步行动！重复，Alpha已经进入军事监狱，准备执行下一步行动！”
<br/>　　罗昊试图通过无线电跟联合行动司令部取得联系，但是耳麦中却只是传来“嗞啦嗞啦”的杂音，并不能够听清楚耳麦中云熙的回答。在没有得到回复的情况下，罗昊又对着联合行动司令部进行了一次呼叫，但情况依旧得到改变，耳麦中除了杂音还是杂音。
<br/>　　罗昊扯掉自己挂在耳朵上的耳麦，对身边其他三个人，说道：“哥几个，我们跟‘老家’失去联系了，这里应该安装了信号干扰器，干扰了我们的无线电信号。”
<br/>　　“这也就是说，我们的无人机也不能使用了？”慕容飞对罗昊，问道。
<br/>　　“我想是这样的。”罗昊回答道：“除非我们能找到信号干扰器或者这个监狱的电源箱，并且关闭它们，否则我们身上的无线电设备和无人机都无法使用，总部也无法看到我们这边的情况。”
<br/>　　“现在我们只能靠我们自己了。”罗昊说道：“失去了无人机的帮助，我们只能靠自己去寻找这里的武装直升机和步兵战车。狐狸，你负责去寻找武装直升机，并且安装炸药；无常，你去寻找那三辆步兵战车。我跟妖瞳去找‘八爪鱼’，顺便寻找一下那个该死的信号干扰器，然后在找到‘八爪鱼’之后送它一颗‘小苹果’。”
<br/>　　“但愿小熙能跟我心有灵犀，让快速反应部队提前进行布置，做好了接应我们撤退的准备。”罗昊嘴里又轻声嘀咕了一句，对身边的三个人说道：“哥几个，行动吧。”
<br/>　　……
<br/>　　“尼米兹”级核动力航母，联合行动司令部。
<br/>　　云熙摘下挂在头上的耳机，对宋萱说道：“凤凰姐，我们跟Alpha小队失去联系了。”
<br/>　　“最后一次联系是什么时候？”宋萱对云熙问道。
<br/>　　“一小时十五分钟前。”云熙回答道：“在我把‘八爪鱼’的资料传给他的时候。”
<br/>　　“具体什么情况？”宋萱拿起放在桌子上的耳机戴在头上，然后让云熙播放了无线电通信的录音。
<br/>　　录音中只有一片杂音，根本听不清楚任何东西。
<br/>　　“他们的无线电受到了信号干扰。”宋萱摘下耳机，对云熙说道：“那座军事监狱中应该安装了信号干扰器，阻止外部无线电设备的使用，这是为了防止有外部人员对军事监狱内的情况进行探查。”
<br/>　　“我也这么认为。”云熙说道：“以西伦科特的本事，安装一个信号干扰器，对无线电信号进行筛选干扰，并不是一件难事。凤凰姐，我认为现在必须把这个情况告诉将军，并且让他的快速反应部队立刻部署到军事监狱附近，随时准备接应Alpha小队撤退。”
<br/>　　“我现在就去找詹姆斯将军沟通情况。”宋萱回答道：“我想他会答应我们的建议的。”
<br/>　　云熙拿起放在桌子上的手枪，塞进枪套中，从椅子上站起身来。
<br/>　　宋萱一把按住云熙的肩膀，问道：“你想要干什么？”
<br/>　　“我要跟快速反应部队一起去。”云熙回答道。
<br/>　　“但是你应该明白，你现在的身体并不允许你再次上战场。”宋萱盯着云熙，非常认真的说道：“而且在来这里之前，我相信你答应过承影不会直接参与到军事行动中，所以承影才会答应你来这里。你应该相信这支被詹姆斯挑选出来的快速反应部队，并且你也应该相信Alpha小队。”
<br/>　　云熙把手枪重新放在桌子上，然后向后捋了一下头发，对宋萱说道：“我明白了，我会在这里负责情报支援，直到他们安全归来。”
<br/>　　宋萱点了点头，转身去找詹姆斯进行沟通。
<br/>　　很显然，詹姆斯将军采纳了宋萱的意见，大约十分钟后，一架搭载了六管格林机枪的“黑鹰”直升机从航空母舰甲板上起飞，朝着军事监狱的方向飞去。同时，航母上的F/A-18E“超级大黄蜂”战机也是做好了起飞的准备，随时准备起飞，对军事监狱进行空袭，将那个属于影子部队的营地彻底摧毁。
<br/>　　云熙透过窗户看着那架从航空母舰甲板上起飞，朝着军事监狱方向而去的“黑鹰”直升机心中充满了无奈。
