　　戚老，幼年便加入了黄泉魔宗，一直到成为阴罗殿的护法。
<br/>　　他的职责便是负责阴罗殿的资源，将阴罗殿在外的各个据点中收集的资源，归纳统计然后送回黄泉魔宗。
<br/>　　所以说，戚老一大半的时间，都不是待在九幽魔渊。
<br/>　　而在黄泉魔宗的强者撤离之时，他正是身处外界，所以无法和黄泉魔宗的高层一起逃离。
<br/>　　所有在外的人都被抛弃了，无法联系上黄泉魔宗的高层，他自然也是无法联系上。
<br/>　　所以，戚老也是不清楚黄泉魔宗的高层，身处何地。
<br/>　　查探到的记忆，让苏莫非常的无奈，不过，他在戚老的记忆中，倒是查探到了其它的信息。
<br/>　　那就是关于巫山的信息，确切的说是关于李峰的信息。
<br/>　　李峰并没有死，其灵魂意识被封存在了身体之中，只要能灭了巫山的灵魂，便能让李峰重新掌控身体。
<br/>　　这个消息，是戚老听皇甫擎所说，绝对不会有错！
<br/>　　这对于苏莫来说，真的是非常重要的消息，若是能让李峰复活，那真是太好不过了。
<br/>　　不过，若是等到巫山达到武帝之境，凝聚出战魂，那李峰就真的彻底死了！
<br/>　　在戚老的记忆中，他是皇甫擎的心腹，知道不少皇甫擎的事情。
<br/>　　比如皇甫擎修炼了太古龙象诀，还比如皇甫曾得到苏莫的灵剑，欲用一种特殊的灵液，查出灵剑的材料配方等等。
<br/>　　用了小半个时辰的时间，苏莫将戚老的记忆查探了一遍。
<br/>　　当他松开手掌之时，戚老顿时倒在了地上，直翻白眼。
<br/>　　苏莫的眸中亦是闪过恍惚之色，直到过去了三个呼吸的时间，他的眸中才恢复了清明。
<br/>　　“这搜魂术，也不能多用啊！”苏莫低声自喃，他连续对屈覃和戚老搜魂，查探了太多的记忆，差一点迷失了自我。
<br/>　　这两人，一个活了三百年，一个活了两百年，记忆非常的浩瀚。
<br/>　　庞大的记忆画面，进入苏莫的脑海，让得他自己的记忆差点被淹没了。
<br/>　　刚才，有那么两个呼吸的时间，他甚至以为两人的记忆，就是他自己的记忆。
<br/>　　这是非常危险的事情，若是让两人的记忆占据主导，他就彻底了迷失了自我。
<br/>　　深吸口气，苏莫摇了摇头，将两人的记忆，彻底的抛却出了脑海。
<br/>　　转头看向屈覃，此刻的屈覃，脸色苍白，眸中亦是恍惚，即便过去了半个时辰，依旧没有回过神来。
<br/>　　苏莫的搜魂术刚刚修炼，造诣极低，将对方的记忆搅得天翻地覆，估计没有个把月的时间，对方很难恢复。
<br/>　　而戚老，更是不堪，双眼翻白，口中直吐白沫，整个人身体都在颤.抖，就算不会死，估计也很难恢复正常了。
<br/>　　嘭！
<br/>　　一声闷响突然响起，只见苏莫大手向屈覃狠狠的一抓，霎时之间，神情恍惚的屈覃，身躯直接被捏爆了，连渣都没剩下。
<br/>　　至于戚老，苏莫答应过不杀对方，自然不会动手。
<br/>　　随后，他命一名苍穹神宫的长老，将对方带离了苍穹神宫，丢到了一处妖兽密布的山脉之中。
<br/>　　至于神志不清的戚老，能否在山脉中活下来，那就看对方的造化了。
<br/>　　没有查到黄泉魔宗高层的线索，苏莫立刻传下命令，吩咐苍穹世界数万大势力，分别查探各自地盘，有没有黄泉魔宗高层的蛛丝马迹。
<br/>　　苏莫心中不免有些无奈，但也毫无办法，追查黄泉魔宗高层之事，只能慢慢进行了。
<br/>　　他现在的首要的任务，就是提升自己的实力，提升整个苍穹神宫的实力，面对试炼者大劫。
<br/>　　接下来很长的一段时间中，苏莫的身影淡出了苍穹世界各大势力的视线，再也没离开过苍穹神宫。
<br/>　　但他的名头，却是成为了苍穹世界的神话，无时无刻，都在被人谈论。
<br/>　　光阴荏苒，日月如梭，眨眼间便过去了两年的时间。
<br/>　　两年之中，苍穹神宫的实力，整整提升了两三个档次，武皇境的长老，达到了三千人之多。
<br/>　　而门下的弟子，也暴涨到了八十余万。
<br/>　　更为重要的是，钱副阁主修为大进，达到了准帝的地步。
<br/>　　期间，苍穹神宫三次大开山门，招收弟子，吸纳各方强者。
<br/>　　每一次都是亿万武者蜂拥而至，苍穹神宫每一次都是精挑细选，万里挑一。
<br/>　　值得一提的是，当初东洲很多的千骄榜天才，都加入了苍穹神宫，其中也有不少苏莫熟悉之人。
<br/>　　当初和苏莫一起加入天灵宗的无生，也并没有陨落在血罗殿之手，时隔数年之后，其靠着自己的努力，修炼到了武王之境，横跨了蛮荒山脉，来到了苍穹神宫。
<br/>　　经过两年的发展，苍穹神宫实力之强、威势之盛，简直如日中天。
<br/>　　而苍穹世界其它的一些大势力，因为很多天才和强者，都得以在苍穹神宫中修炼，或者是得到了苍穹神宫的重赏，他们和苍穹神宫的联系，亦是越来越紧密。
<br/>　　苍穹神宫，虽然远远没有达到远古之时的强盛，但也算是彻底复兴了，恢复了远古时代的地位。
<br/>　　……
<br/>　　一座云雾缭绕的山巅之上，一名青衣青年，仿若一颗不朽的青松，屹立在山巅之上。
<br/>　　青年便是苏莫。
<br/>　　苏莫的目光深邃悠远，仿佛穿透了层层浓雾，跨越了亿万里距离，遥望着天地的尽头。
<br/>　　他的身后，还伫立着九道身影，分别是司空炎、龙腾、翼晓晓、彩芸等九宫之主。
<br/>　　此时，九人之中，修为最高的是战乱，武皇境七重修为。
<br/>　　其次是龙腾，武皇境六重修为。
<br/>　　而后是司空炎、姜枫然、夕儿、和闻人天都，武皇境五重修为。
<br/>　　再然后，是翼晓晓，武皇境四重修为。
<br/>　　修为最低之人，是彩芸和青雅，一人武皇境一重，一人武王境八重。
<br/>　　九人望着前方的苏莫，个个面色郑重，时至今日，他们已经看不清苏莫的修为了。
<br/>　　甚至，若是不用眼睛去看，他们都感觉不到，身前还站着一人。
<br/>　　因为苏莫刚出关，就连作为妻子的夕儿，都不知道苏莫如今的实力了！
<br/>　　“你们的修为还远远不够！”苏莫轻叹一声，声音悠悠。
<br/>　　九人的修为提升速度，远远没有达到他的预期。
