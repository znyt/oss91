　　“想要收取天星河？一个小家伙怎么可能做到，我们圣龙秘境无数半步天人境强者，都曾经尝试收取天星河，都失败了，那楚风眠不过是一个新人，哪有这样的本事。”
<br/>　　又一尊圣龙秘境内半步天人境的强者，打量了一眼天星河开口道。
<br/>　　天星河，从圣龙秘境建立之初便是一直屹立在哪里。
<br/>　　曾经也有过不少的圣龙秘境强者，想要收取天星河，却都是失败了。
<br/>　　“这天星河，只怕是只有天人境的强者才有能力收取，要不就是少帝那样的存在，一个圣龙宗的新人小家伙，再强也到不了这个程度。”
<br/>　　这一尊圣龙秘境内部的半步天人境强者，缓缓的开口道。
<br/>　　“哼，不管那小畜生究竟收没收取天星河，那小畜生做的太过了，居然敢毁了执法堂，现在更是攻打六绝宗，圣院宗，简直不将我们圣龙秘境放在眼中，必须要将他镇压降伏才行，不然我们圣龙秘境的威严何在？”
<br/>　　一尊白发老者，冰冷的开口道，这一尊白发老者也是一尊半步天人境的强横人物。
<br/>　　他开口的每一句话，都如同审判，蕴含着规矩法理，这就是法理古帝，执法堂之中最为古老的老古董。
<br/>　　之前便是传闻法理古帝，在圣地之中苦修，试图突破九级古帝，现在却是成功了。
<br/>　　这法理古帝，已经是成功踏入了半步天人境，成为了这个时代的巅峰强者之一。
<br/>　　法理古帝在提到楚风眠的时候，声音之中的杀意已经是毫不掩饰。
<br/>　　楚风眠毁了执法堂，可以说毁了他的一切心血，这法理古帝出关，第一个要做的事，就是要诛杀楚风眠。
<br/>　　“一个小家伙，就算是有着几分资质，这做的也的确太过了，圣龙秘境四宗八堂传承了数万年，岂是他一个小家伙说灭就灭的。”
<br/>　　旁边的那一尊半步天人境老古董听到法理古帝的话，也是点了点头道。
<br/>　　“这小子必须要杀了才行。”
<br/>　　“可这小子毕竟是圣龙宗的弟子，要是杀了他，只怕会有麻烦，那云尊暂且不提，一个龙千绝，都无比棘手了，杀了这小子，龙千绝也要大闹起来。”
<br/>　　又一尊老古董开口道，他的语气之中有些担心。
<br/>　　“哼，一个龙千绝罢了，我们联手之下，杀他根本不成问题，如果不是念在此人也同为圣龙秘境同门的份上，我们早就可以杀了他。”
<br/>　　法理古帝冷哼一声道。
<br/>　　“至于那小子，我们可以不杀他，此人的身上有着大量的机缘，我们可以降服了他，让他为我们所用。”
<br/>　　“至于那龙千绝，如果不识时务的话，也将他一起杀了，我们已经祭炼了那个东西，就算是龙千绝，也翻不了天。”
<br/>　　一边交谈着，一边这一群圣龙秘境老古董，已经是飞到了天星河的边缘。
<br/>　　这一切的交谈，都被楚风眠听在耳中。
<br/>　　“这些老古董，联手祭炼了一个东西？可以用来对付半步天人境用，许多这么多的老古董联手祭炼，莫非是极道仙兵？”
<br/>　　楚风眠目光冰冷，心中却是在思考这。
<br/>　　龙千绝的实力，已经是踏入了半步天人境的层次，这些老古董的交谈之中，居然是说祭炼了那件东西，可以灭杀龙千绝。
<br/>　　那这件东西，如果是极道仙兵的话，那可就麻烦了。
<br/>　　稍有不慎，等待楚风眠的可能就是灭顶之灾。
<br/>　　不过也有可能不是极道仙兵，而是一切奇物，或者强大的天级灵器。
<br/>　　像是剑鼎那个层次的天级灵器，如果将全部威力施展出来，也同样可以对半步天人境的强者，造成极大的威胁。
<br/>　　不过这一次，那些老古董哪怕是有着在大的底牌，因为楚风眠已经是将天星河炼化到手了。
<br/>　　凭借天星河，楚风眠有着把握，将这些老古董，一网打尽。
<br/>　　楚风眠思考之际，那些老古董已经是来到了天星河的边缘。
<br/>　　这些圣龙秘境老古董做事，还是无比谨慎的，并没有轻易靠近天星河，而是就在不远处仔细的观察着天星河。
<br/>　　楚风眠虽然是将天星河炼化，可是现在楚风眠将属于他的印记隐藏起来，纵然是半步天人境，都无法看出。
<br/>　　那些老古董，观察了一下，也没有发现任何的异样。
<br/>　　“你去打开天星河。”
<br/>　　法理古帝开口，对着身边的一名九劫古帝开口道。
<br/>　　那九劫古帝听到这话，便是身形一动，靠近了天星河，拿出了一枚玉符，还是催动起来。
<br/>　　随着玉符的催动，天星河逐渐的打开，凝聚出了一道通往圣龙秘境的通道。
<br/>　　“没有异样，看来是那无不源为了让我们赶快过来，故意说的谎话，等见到他，也必须要好好教训教训他。”
<br/>　　看着天星河打开，一名半步天人境的老古董，冷哼一声说道，语气之中极为不满。
<br/>　　这一尊老古董，身上却是蕴含着一股股强横的龙威，却是一尊龙族妖神，万神龙。
<br/>　　万龙堂最为古老的老古董。
<br/>　　而在他身边的另外五名半步天人境强者，楚风眠也从青梦哪里，一一得知了他们的身份。
<br/>　　其中两人属于圣院宗，分别便是现在圣院宗的宗主凌山，还有大长老，王长老。
<br/>　　其他的三人则是。
<br/>　　五灵宗的宗主，炎乾龙。
<br/>　　六绝宗的宗主，韩圣。
<br/>　　武霸堂的堂主，霸斗元。
<br/>　　在加上法理古帝，万神龙。
<br/>　　一共七位半步天人境强者，跟在他们身后的那些九劫古帝，也几乎都属于这几个宗门堂口之中的强者。
<br/>　　这一次圣龙秘境的老古董，已经是尽数出动了。
<br/>　　“师弟，我们真的要动手？”
<br/>　　青梦在一旁，看到这庞大的阵仗，都有点打退堂鼓。
<br/>　　这可是足足七位半步天人境啊。
<br/>　　楚风眠虽然是在剑源界之中，曾经杀过一尊半步天人境强者荒鎏金，更是逼退了荒天王。
<br/>　　可那也只是单打独斗，现在楚风眠要面对的，却是足足七位，这可是截然不同的概念。
<br/>　　哪怕是同为大罗天星域之中四大势力的五行天，齐林山等，强者尽出，都未必可以奈何的了这七人。
<br/>　　论起高层实力，圣龙秘境可是还要强过五行天，齐林山。
