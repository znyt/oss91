那个女人走了进来，开始打扫卫生，替换垃圾桶里的袋子。
<br/>
<br/>        夏夕绾看着垃圾桶里那一堆被揉成了褶皱的白纸俏脸迅速一红，她当即上前道，“这个放下吧，待会儿我自己来。”
<br/>
<br/>        那个女人收回了手，“好的，那我进沐浴间打扫。”
<br/>
<br/>        沐浴间门被推开，那个女人走了进去，开始收拾昨晚换下来的衣服，夏夕绾抬头看的时候，只见那个女人低着腰，宽大的领口暴露了一片春光，那么多衣服她没拿，先伸手拿了陆寒霆换下来的…内裤！
<br/>
<br/>        夏夕绾迅速上前，伸手就将那个内裤给抢下了，“今天这里不用你打扫了，你先离开吧。”
<br/>
<br/>        那个女人看着夏夕绾，虽然有点留恋不想走的意思，但碍于夏夕绾陆太太的身份，还是离开了。
<br/>
<br/>        ……
<br/>
<br/>        很快陆寒霆就回来了，他身上是手工版的白色衬衫黑西裤，被熨烫的没有丝毫褶皱，优雅矜贵，今天他的心情是肉眼可见的不错，整个人英俊卓尔，神采奕奕，像块磁铁般让人移不开目光。
<br/>
<br/>        他进来的时候先看了一眼大床，床上已经没人了，她起床了，陆寒霆四周看了一圈，也没有看到她的人。
<br/>
<br/>        “陆太太。”他开口叫了一声。
<br/>
<br/>        没人回应他。
<br/>
<br/>        不过，沐浴间里发出了水流的声响，里面有人。
<br/>
<br/>        陆寒霆走进去，果然看到了那道纤柔的身影，夏夕绾站在盥洗台边，正在洗衣服。
<br/>
<br/>        她不知道洗了什么，两只小手洗洗搓搓的，弄了一手的泡泡。
<br/>
<br/>        陆寒霆来到她的身后，单手抄裤兜里从后面轻轻的推了她一下，“陆太太，你怎么不理人？”
<br/>
<br/>        “哼！”夏夕绾动了一下，那意思是别碰我！
<br/>
<br/>        陆寒霆觉得奇怪了，他勾起薄唇嗓音低醇的笑道，“陆太太，你过分了啊，昨晚你还往我怀里钻，现在下了床就不认人了，我应该没有得罪你吧，你怎么好像生我气了？”
<br/>
<br/>        夏夕绾贝齿咬了一下红唇，“我怎么敢生陆总的气，陆总是谁啊，陆氏集团的总裁，逼格就是跟人家不一样，就连清洁阿姨都能甩出了人家几条街！”
<br/>
<br/>        她阴阳怪气的话里可是溢满了醋味，这个小醋坛又打翻了。
<br/>
<br/>        陆寒霆不明所以，“陆太太，你说的哪个清洁阿姨？”
<br/>
<br/>        “就是那个肤白貌美，胸器能跟霍璇有得一拼的清洁阿姨了，陆总，难道你不清楚？人家可是惦记着你呢，伸手就来拿你的内裤！”
<br/>
<br/>        陆寒霆还真不知道她说的是哪个清洁阿姨，他这个休息室确实是有人来打扫的，不过这都是生活助理在全权安排。
<br/>
<br/>        陆寒霆伸手抱住了她纤软的腰肢，“那你真的让她拿走了我的内裤？”
<br/>
<br/>        “没有，幸亏我眼疾手快抢下来的！”说着夏夕绾拎起了他的内裤，“没看到我正在洗吗？”
<br/>
<br/>        陆寒霆刚才没看出来她在洗他的内裤，也没有想到她会在这里给他洗内裤。
<br/>
<br/>        “陆太太，昨晚去超市给我买牛鞭汤，今天跟一个清洁阿姨抢我的内裤，你怎么好意思的？”
