        &lt;h3&gt;第三卷第六十九章  陈子乔请客&lt;/h3&gt;
<br/>
<br/>        陈子乔订好了晚上吃饭的地方，给大家都发了地址。
<br/>
<br/>        “子霖，你早到一会啊，招呼一下他们，我可能要迟到一会。”
<br/>
<br/>        陈子乔在等乡镇企业家--常萧林。
<br/>
<br/>        常萧林的父亲是中医医生，在洛城是非常有名气的，母亲就是马总的姑妈。
<br/>
<br/>        常萧林原本也是学的中医，但是面对着那些个瓶瓶罐罐，满屉子的各种药材，心里莫名的发怵，他怕和爸爸一样一辈子和中药打交道。
<br/>
<br/>        在改革开放时期，在表哥有了一些能力的时候，他咬咬牙，辞了单位的工作，下海了。
<br/>
<br/>        开始承包一些小工程，找一些农民工砌墙修路，没资金了找银行贷款，慢慢的越做越大，在表哥的介绍下娶了陈子乔生了对双胞胎儿子，后来，变成一个企业家，日子也是很幸福的。
<br/>
<br/>        下午公司开会，陈子乔说妻弟过来要一起吃饭，他对陈家不好也不坏，陈家的人似乎个个心高气傲，他们除了过节会聚到一起之外，其实也是很少来往的，主要也是陈子乔太忙，家里的事情都管不了，更不用说娘家的事情了。
<br/>
<br/>        陈子乔上次就说妻弟来了，但是他太忙，也就没有见上，开完会，直接回家，接了陈子乔一起去饭店吃饭。
<br/>
<br/>        “子乔，子霖一个人来的？”
<br/>
<br/>        “公司的人也来了几个。说是前期调研呢。”
<br/>
<br/>        “看来，子霖他们是想拿下这个项目？”
<br/>
<br/>        “表哥给傅海生打过招呼了，而且傅海生还是子昂的同学。”
<br/>
<br/>        “那应该问题不大吧？”
<br/>
<br/>        “子霖说也有几家竞争呢。”
<br/>
<br/>        “慢慢来吧。”
<br/>
<br/>        “你在家里，子霖他们你也多照顾一点，我娘家也就这么几个亲人。”
<br/>
<br/>        “嘿嘿，子乔，现在开始为娘家人着想了？”
<br/>
<br/>        “你说什么话呢，我这不是从嫁给你之后，都是为了你们家里拼死拼活的，自己娘家里都不曾照顾。”
<br/>
<br/>        “老婆，我的好老婆，我当然是知道你的好啦。”常萧林伸出手，捏了一下陈子乔的手，陈子乔漂亮，但是也霸道，当初能娶到她还是表哥的功劳，要不然他常萧林是娶不到这么漂亮的女人。
<br/>
<br/>        俩个人到达饭店的时候，大家基本都到齐了。
<br/>
<br/>        陈子乔看见傅海生旁边的夏颜微微的愣了一下，她是没有想到夏颜也会来洛城。
<br/>
<br/>        陈子霖在见到傅海生和夏颜的时候早已经将不满深深的埋藏起来了，他似乎是什么事情都没有发生过。
<br/>
<br/>        今天是四姐做局将大家召集到一起，目的是为了他，他也是很明白的，这个世界上的圈子本来就这么小，相互之间能认识也不足为奇，况且他们都是泉城人。
<br/>
<br/>        “表哥，傅总，夏总！夏总是什么时候到的，都不提前给我说一声。”
<br/>
<br/>        “我到了好几天，想和海生多呆几天，过一过二人世界。”夏颜微笑着说。
<br/>
<br/>        陈子乔给马总打电话说下午一起吃饭的时候她都听见了，她知道陈子乔是为了陈子霖，不过陈子霖上午对她动了手，她岂能让他如愿拿到项目？
<br/>
<br/>        夏颜笑得如花似玉，像秋风里的野玫瑰，摇曳生姿，一桌的男人眼巴巴的看着她，她拿眼斜了一下马总，又甜甜一笑。
<br/>
<br/>        “你们夫妻真是恩爱得令人羡慕啊，萧林，我去西城了，你以后也要经常去看我和表哥俩。”陈子乔也不含糊。
<br/>
<br/>        “那是当然，你去西城，咱就将家安在西城，这几天我已经在西城买好了房子，就等着装修了，装修好了，我们就搬过去。”常萧林配合着他的妻子撒了一地的狗粮不说还毫无声息的炫了一下富。
<br/>
<br/>        陈子乔微微一笑，他们家萧林总是会在外人面前给足她面子，虽然她打心眼里不怎么喜欢这个憨憨的男人，但是人家现在是企业家，有做生意的潜质，而且和马总是表兄弟。
<br/>
<br/>        “都坐下啊，今儿个高兴，第一是为了陈子霖陈总接风，第二呢也是我和子乔要去西城了，这也算是我们小范围的聚餐。”马总说话了。
<br/>
<br/>        马总首座，傅海生和陈子霖坐在马总的两边，陈子乔坐在陈子霖的旁边，夏颜坐在傅海生的旁边，江俞轩坐在夏颜的旁边，常萧林挨着陈子乔坐着，其余的人是另外坐一桌。
<br/>
<br/>        原本陈子霖不坐在现在的位置上的，他坐的那个位置是陈子乔来坐的，但是陈子乔和陈子霖换了位置。
<br/>
<br/>        陈子乔表明了她对弟弟的看重。
<br/>
<br/>        她要调走的，洛城还是她陈子乔的天下，她弟弟在这里就是代表着她，
<br/>
<br/>        夏颜皱了皱眉头。
<br/>
<br/>        傅海生一直很少说话，今天上午发生了那样的事情，他内心对陈家存了愧疚。
<br/>
<br/>        无酒不成宴，桌上打开的茅台酒散发着淡淡的酒香，但是对夏颜来说，却似毒药，她酒量不好，昨天出丑了，今天还是不要喝的好。
<br/>
<br/>        马总拿起酒杯：“来，第一本，为我们的相识干杯。”
<br/>
<br/>        夏颜不举杯，她拿起了水杯。
<br/>
<br/>        “夏总，今天怎么不喝？”江素轩看着夏颜不拿酒。
<br/>
<br/>        “我.....”
<br/>
<br/>        “夏总昨天喝得不错，今天莫非是看不起桌上的人？”陈子霖也看着夏颜说。
<br/>
<br/>        “马总，我喝不了酒。”夏颜娇滴滴的向马总求助，傅海生微不可察的皱了一下眉头。
<br/>
<br/>        “昨天你都喝了，今天就喝不了？”平时看着很聪明伶俐的一个人，今天怎么就笨了呢？马总可不想将他们的关系让别人知道，尤其是傅海生在旁边。
<br/>
<br/>        “人家昨天喝得有点多嘛！”甜腻的娇滴滴的撒娇声。
<br/>
<br/>        “端起吧，喝前三杯，不喝就是不给我面子。来，大家先干了这一杯。”马总的声音有点僵硬。
<br/>
<br/>        陈子乔冷冷的看着马总和夏颜。
<br/>
<br/>        “第二杯，欢迎陈总来洛城投资，虽然我明天就走了，但是洛城是我的家乡，希望陈总帮着傅总将他们单位信息化项目做好，做成功。”这是在表明态度。
<br/>
<br/>        “第三杯，庆祝我和子乔调到省城，无论怎么说，我们也算是迈出了洛城。”三杯一下肚，大家的脸都红润润的。
<br/>
<br/>        旁边桌上的人喝了三杯酒之后，都过来他们桌子上敬酒了。
<br/>
<br/>        一圈下来，每个人都喝了不少，酒桌上，只要你拿起酒杯，就不分男女了，虽然夏颜是女的，也算领导，但是大家敬酒了，她也不能不喝，这一下，她又有点飘飘然了。
<br/>
<br/>        “傅总啊你和陈总合作的怎么样？”马总兴致勃勃。
<br/>
<br/>        “今天技术人员才进场，过一段时间才能出方案。”傅海生一直不太说话，气氛尴尬的他不知道怎么说。
<br/>
<br/>        夏颜酒量不行，还要跟着他一起来，坐在他的旁边眼睛却盯着马总，尤其是现在的那个眼神，简直就是赤/裸/裸.
<br/>
<br/>        “对，前期的方案一定要做好，这样才能在竞争中立于不败之地。陈总啊，你们可是要加油啊。”
<br/>
<br/>        “是，马总，谢谢您的关照。”
<br/>
<br/>        “马总，颜颜敬您一杯酒，颜颜以后也得您关照呢。”
<br/>
<br/>        “夏总是女中豪杰啊，酒量不错嘛，啊哈哈哈。”
<br/>
<br/>        “马总，您就不要夏总夏总的叫了，我是谁你不知道吗？我是您的颜颜啊，我们两家可是世交。”
<br/>
<br/>        “哈哈，好，好！颜颜她妈和我是朋友。”马总尴尬的解释着。
<br/>
<br/>        “子乔啊，陈总啊，另外一个部门的项目你们就不要参与了吧，有一个朋友找到我了，推不掉。”马总对陈子乔说。
<br/>
<br/>        夏颜满意的喝了手中酒，看了一眼陈子乔。
<br/>
<br/>        “无所谓的，马总，现在还是前期调研中，谁能做不一定呢，后期肯定是要参加招投标的。您放心，我们不会让您为难的。”陈子霖站起来敬了马总一杯酒。
<br/>
<br/>        陈子乔并不知道弟弟和夏颜之间产生了纠葛，但是，夏颜的话似乎是说给她听的。
<br/>
<br/>        夏颜的一句话，挑明了她和马总的关系，酒桌上的人已经心知肚明了。
<br/>
<br/>        傅海生的脸就像是冬天的冰。
