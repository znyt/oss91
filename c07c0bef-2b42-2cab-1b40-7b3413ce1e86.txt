　　洪狱出手。
<br/>　　一掌轰然打出，在这一掌凝聚出的一刻，突然周围出现了无数黑雾。
<br/>　　这黑雾，并非是魔意，而是更加阴邪之物，极恶之物。
<br/>　　在这无数黑雾的围绕之下，一掌打出，周围的无数人看到这一幕，都是心中一寒，仿佛陷入到了无比恐惧之中去。
<br/>　　“炼狱魔掌！”
<br/>　　这一掌打出，楚风眠身边的白萱脸色一变道。
<br/>　　“这是洪狱的绝学，相传他曾经在虚空海中，得到过大奇遇，学会了一身炼狱魔功，这炼狱魔功，甚至还要超越恒古魔域，恐怖至极，一旦被打出，就会被无穷无尽的痛苦，折磨致死！”
<br/>　　这炼狱魔掌，一旦是打在武者的身上，就会产生无穷无尽的痛苦。
<br/>　　这痛苦，就足以让许多心智不坚定的武者，活活疼死。
<br/>　　炼狱魔功的痛苦，可不止是身体上的，更是来自灵魂上的，就算是圣者，都无法抵挡。
<br/>　　“小子，准备接受无穷无尽的痛苦吧，我会让你求生不得，求死不能的！”
<br/>　　洪狱狰狞的笑着，他要以这炼狱魔功，彻底的折磨楚风眠。
<br/>　　“懂得点废物魔功，就敢在我面前耀武扬威？”
<br/>　　楚风眠看着洪狱的一掌打来，脸色平静到了极致。
<br/>　　“小小魔功，给我破！”
<br/>　　“吼！”
<br/>　　楚风眠口中，淡然吐出一个字，一声龙吟，猛然响彻而起。
<br/>　　在这一声龙吟中，蕴含有着精粹之际的战龙龙威，远古战龙之力，克制一切阴邪。
<br/>　　炼狱魔功，一眼逃脱不了被克制的可能。
<br/>　　这炼狱魔功虽然厉害，但是在楚风眠的战龙之威面前，却是根本不可能有着多少作用。
<br/>　　在这一声龙吟之下。
<br/>　　洪狱的炼狱魔功，凝聚出的一掌，顿时直接消散开来，其中的所有力量，都被直接化解。
<br/>　　周围所有围观武者，皆是大吃一惊。
<br/>　　这洪狱出手，可是没有任何留情，反而可以说是动用了全力，就连炼狱魔功都动用了。
<br/>　　这一招之下，至少也有着打算轰杀楚风眠的意思。
<br/>　　这一招炼狱魔掌，在虚空海的神力境武者之中，能够挡下的，不足十人，甚至是许多人挡下，都要付出极大的代价。
<br/>　　像是这样，单单口中一个字，便是将炼狱魔掌的力量直接化解的。
<br/>　　简直是闻所未闻，前所未见。
<br/>　　“此人到底是什么来历？他先是击败了欧阳烈，抽取了欧阳烈的圣者精血，现在居然有足以抗衡洪狱！”
<br/>　　“我们北域中，还从未听说过这样一号人物。”
<br/>　　“能够跟洪狱抗衡，此人的实力的确恐怖。”
<br/>　　“不过他能够化解洪狱的炼狱魔功，也并非是实力多么强大，而是他的功法，隐隐约约正是克制炼狱魔功，才可以如此轻松的化解。”
<br/>　　“看看吧，洪狱的实力可没有这么简单，天盟的大人物，就算是没有炼狱魔功，也是站在了定点的人物。”
<br/>　　这短暂的交手，便是引起了无数议论。
<br/>　　洪狱这样天盟的大人物，已经是很少出手了，没想到他出手，居然还被楚风眠抵挡住了，这让无数人都是惊讶万分。
<br/>　　不过却没有任何一人看好楚风眠，洪狱的实力，远远没有表现出的这般简单，炼狱魔功被克制，但是洪狱不动用炼狱魔功，也依然是可怕至极的人物。
<br/>　　在众人看来，楚风眠能够从洪狱的手中，到底能撑过几招。
<br/>　　“龙族武技？”
<br/>　　看着他的炼狱魔功被化解，洪狱一瞬之间，也是瞬间明白过来。
<br/>　　这龙族武技，的确是对一切魔功，都极为克制。
<br/>　　他的炼狱魔功，碰到这龙族武技，的确是天生被克制，没有发挥的余地。
<br/>　　楚风眠看着洪狱，摇了摇头道，似乎眼神中都尽是不屑。
<br/>　　“天盟的人，只有这等实力吗？那未免也太过有些让人失望了。”
<br/>　　“狂妄！”
<br/>　　洪狱听到楚风眠的话，猛然长啸一声到，语气中杀机连连。
<br/>　　“小子，看来你真是得到了一些奇遇，就不知天高地厚了，懂得几分龙族武技又如何？不用炼狱魔功，在我面前，你也只是一只蝼蚁罢了！”
<br/>　　“兵！”
<br/>　　从洪狱的手中，猛然只见，一根漆黑无比的棍子，便是凝聚出来。
<br/>　　这一道漆黑棍子出现，凭空便是一下子冲着楚风眠砸了过去。
<br/>　　这一棍打出，看似平常，其中却是分化万千，足足是化为了三千道虚影。
<br/>　　每一道虚影，都是虚虚实实，令人摸不清到底是真是假。
<br/>　　“三千虚幻！洪狱的兵道！居然达到了这一步，这三千虚影，每一道都是半真半假的，每一道都有可能是真！想要抵挡，必须要将这三千虚影，全部抵挡住才行！”
<br/>　　众人看到洪狱的手段，都是吃了一惊。
<br/>　　这一招，的确是强大无比，除非是将三千虚影，同时挡住，不然的话，便是就会陷入到围攻之中。
<br/>　　在这种级别的对抗之中，陷入劣势，便是如同万劫不复一般，很难还有翻盘的可能，只能是节节溃败。
<br/>　　“小子，你激怒了我，如今你也要为我的怒火，而付出代价！死吧！”
<br/>　　洪狱狰狞的大吼一声，三千黑棍虚影，同时落下。
<br/>　　“三千虚幻？就敢来大放厥词，可笑！”
<br/>　　楚风眠大步踏出，面对着三千黑棍虚影袭来，不闪不避，直接迎了上去。
<br/>　　“他要做什么？”
<br/>　　“莫非要硬抗三千虚幻？”
<br/>　　“此人这是在找死！三千虚影，逃都难逃，正面硬抗，以一己之力，怎么可能对抗的了三千！”
<br/>　　面对这三千虚幻，一般人所想，都只能是逃。
<br/>　　尽可能少面对其中虚影，来对抗。
<br/>　　这种正面硬抗的，简直是疯子一般的行为。
<br/>　　就连李长老，都是皱了皱眉。
<br/>　　洪狱的三千虚幻，他也知晓，这一招最为强大的，便是正面一击。
<br/>　　正面抗衡三千虚幻。
<br/>　　神力境武者。
<br/>　　必死！
<br/>　　“剑！”
<br/>　　就在众人看向楚风眠，感觉下一刻楚风眠就要被三千虚幻，直接打成肉泥的一刻。
<br/>　　楚风眠轻声呼唤，在他的手中，便是剑意凝剑。
<br/>　　“一剑万重！”
<br/>　　随着楚风眠剑锋落下一刻，其中剑气，猛然爆发出来。
