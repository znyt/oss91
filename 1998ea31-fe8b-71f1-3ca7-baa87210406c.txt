　　归墟天，可是远古遗迹，这种远古遗迹中，很有可能留有许多机缘。
<br/>　　这奇遇，比起虚空海中的古洞府，还要庞大，这种机会，楚风眠必须要争取。
<br/>　　夜天君都在寻找机会，度过生死大劫。
<br/>　　若是让夜天君先一步度过生死大劫，很有可能楚风眠便是会遭到灭顶之灾。
<br/>　　所以楚风眠也必须要赶在夜天君前，度过生死大劫。
<br/>　　倒时候不说横扫整个北邙学院，但是在弟子一代中，他将会没有任何敌手。
<br/>　　“楚风眠，你出来了？”
<br/>　　楚风眠心中思考之际，寒月漓的双眼，便是缓缓睁开，对着楚风眠开口道。
<br/>　　“玄北老祖，对你说了些什么？”
<br/>　　对于楚风眠突然冒出来的师尊，寒月漓倒是并没有太多意外。
<br/>　　在当初武胜国度的时候，寒月漓便是知晓，楚风眠有一个玄北圣者的师尊。
<br/>　　只是没想到当初的玄北圣者，居然是北邙学院的一尊老祖。
<br/>　　在武胜国度这样的小地方，居然会有一尊踏上生死台阶的老祖，这也的确是有些骇人听闻了，这种奇遇，楚风眠也正好撞上了。
<br/>　　“师尊告诉我了一些，关于归墟天的事情，还有前往武王城，参加武斗大会的事。”
<br/>　　楚风眠开口道。
<br/>　　对寒月漓，他还是没有什么好隐藏的，直接便是将交谈的内容，全部告诉了寒月漓。
<br/>　　还有楚风眠得到关于颜古龙帝洞府中的奇遇，跟众圣堂中北邙圣君传承的事。
<br/>　　不过关于北邙圣君就是曾经楚风眠大师兄剑凌的事，楚风眠还是隐瞒了下来。
<br/>　　这件事，关系重大，要是现在传出去楚风眠是剑道之主的弟子。
<br/>　　七大宗门的人，也会不惜一切的代价抓捕楚风眠的。
<br/>　　告诉寒月漓，只能是给她徒增灾难，这个秘密，如今只有楚风眠一个人知晓。
<br/>　　“颜古龙帝，北邙圣君，没想到你的机缘，也丝毫不比我小。”
<br/>　　寒月漓不由的感叹一声开口道。
<br/>　　“我之前来到北邙圣域中，误入到了一处圣地之中，意外的到了一枚玄寒石斛，将我身体中的力量，大部分的都激发了出来，顺势度过了生死大劫，才有着如今的实力。”
<br/>　　“本以为我的奇遇已经不小了，没想到还是无法跟你相比，就连北邙圣君的传承，你都足以得到。”
<br/>　　玄寒石斛。
<br/>　　这个名字，楚风眠自然也不会陌生，这玄寒石斛，那是天地圣药之一。
<br/>　　比起楚风眠曾经得到过的万年火莲，品级还要高得多。
<br/>　　正好属性为寒，被寒月漓得到，激发出了她玄寒之体无数的本院之力，令她的实力暴涨，可以算得上是最为适合寒月漓的奇遇了。
<br/>　　也怪不得寒月漓刚刚进入学院不久，便是度过了生死大劫，原来是还有着这种奇遇。
<br/>　　只是比起楚风眠的奇遇，这一枚玄寒石斛，却是算不得什么了，单单是一把祖龙至尊剑，都足以是让无数老古董都为之发狂了。
<br/>　　不过楚风眠得到的奇遇，可都是他拼死争抢而来的，在颜古龙帝洞府中，楚风眠都差一点是死在了不死魔尊的手中。
<br/>　　也算是楚风眠命大，才活了下来，有着如今的这一切。
<br/>　　两人正在交谈之际，突然寒月漓的脸色，却是微微有些变化，似乎是有些恼怒。
<br/>　　“怎么回事？”
<br/>　　看到这寒月漓脸色突然的变化，楚风眠出声询问道。
<br/>　　“是一个烦人的家伙，武道世家司空家的司空弘，在北域中偶然跟他碰到，此人便是一直想要追求我，向我提亲，已经拒绝了，如今又来了。”
<br/>　　寒月漓的脸色有些难看的说道。
<br/>　　“司空家的老祖，跟我的师尊冰漓长老关系不错，所以也不好撕破脸，没想到他居然这次又来了，直接找到我师尊，要向我提亲。”
<br/>　　武道世家？司空弘？
<br/>　　听到这个名字，楚风眠冷笑一声。
<br/>　　跟着司空家，楚风眠还真是有缘，之前楚风眠可以得到祖龙至尊剑，进入到颜古龙帝洞府，也是全靠了那司空轩，最后司空轩对付楚风眠，便是被楚风眠直接轰杀。
<br/>　　没想到如今这司空家，居然有人不要命的人，来找死了。
<br/>　　寒月漓看向楚风眠开口道。
<br/>　　“师尊现在让我回去，怎么办？”
<br/>　　“我跟你一起回去。”
<br/>　　楚风眠冷冷的开口道，脸色也是变得阴沉了许多。
<br/>　　“我倒是要看看，到底是谁，敢来打我女人的注意！”
<br/>　　楚风眠的话，虽然阴沉，但是让寒月漓听见，心中也是不由一暖。
<br/>　　“走。”
<br/>　　两道遁光，瞬间消失，在这北邙圣域之中穿梭。
<br/>　　北邙圣域之大，不亚于一方大陆，两人飞行了半个时辰，终于在前面，出现了一座巨大无比的山峰。
<br/>　　这山峰，布置了无数禁制，显然便是寒月漓的师尊，冰漓长老的潜修之地，虽然不如楚风眠师尊玄北圣者的所在之处，却也是修缮的极为完美。
<br/>　　楚风眠跟寒月漓的遁光，缓缓的来到山峰周围，落下来到了山峰之上。
<br/>　　在这山峰上，只有这一座宫殿。
<br/>　　二人在宫殿外，等待了一阵，一道身影，却是从其中，缓缓的走了出来。
<br/>　　这是一名看起来也就二十岁左右的女子，看起来极为年轻，比起寒月漓稍大一些，不过这女子身上的气息，却是似乎有着一些超脱生死的意味。
<br/>　　跟那陆千刃相比，气息便是在伯仲之间，同为生死台阶的第一重碎命之境，不过比起陆千刃还是要青涩一些，看起来像是不久才突破到碎命之境的。
<br/>　　楚风眠一眼之间，便是分析出了这冰漓长老的实力。
<br/>　　这冰漓长老，修行的也是一种冰系功法，跟寒月漓极为相似。
<br/>　　“徒儿，你回来了。”
<br/>　　冰漓长老出来，便是看到了寒月漓，欣喜的说道。
<br/>　　“你突然匆匆离开，是去哪里了？”
<br/>　　“师尊，我……”
<br/>　　寒月漓听到冰漓长老的话，脸色一愣，似乎有些不好解释。
<br/>　　她匆忙离开，便是知晓楚风眠前往了生死台的消息，前去救人的，匆忙离开，都没有给一个交代。
<br/>　　“见过冰漓长老。”
<br/>　　还是楚风眠突然出声，才打破这寂静。
