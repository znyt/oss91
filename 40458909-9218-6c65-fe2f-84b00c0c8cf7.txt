　　“哦，一两啊。没有。”
<br/>　　小二眼睛一下子就直了。
<br/>　　“你意思是……”
<br/>　　“就是没钱，不过我也不是吃霸王餐的，你们老板娘在不在，你把她叫来，我亲自跟她说。”
<br/>　　“咱家老板娘忙着，我说这位爷，吃饭给钱，天经地义，您还是把钱付了……”
<br/>　　叶少阳过去拉起小九，往楼下走。
<br/>　　小二拦不住，也不敢动手，就对着下面大叫：“来人啊来人啊，有人吃白食啦！”
<br/>　　没等两人下楼，楼下通往后堂的帘子掀开，走出来几个矮壮的厨师，冲两人瞪眼。
<br/>　　小九拉着叶少阳的胳膊，道：“我用点手段？”
<br/>　　“不用。”叶少阳冲那几个厨师喊道，“你们老板娘呢，让她出来。”
<br/>　　“你吃白食见老板娘做什么，知道我们老板娘心善啊，大伙先把他拿住！”一个厨师卷着袖子就要冲上来。
<br/>　　这时候后堂帘子掀开，一个系着围裙的女子走出来，一边去拽那个厨师，口中喊道：“老邓你们不要闹事，一顿饭吃不穷我，没准人家是忘了带钱呢，问清楚再说！”
<br/>　　说着抬头扫了叶少阳一眼，“年轻人，你要是忘了带钱去回家拿去，或者下次一块算也行，你快走吧。”
<br/>　　叶少阳笑道：“哪有你这么做生意的，都像这样吃白食，你早晚得关门。”
<br/>　　“你这客人，得了便宜还这么……”猛然间意识到到什么，后半句话也咽了回去，往前冲了两步，抬头打量叶少阳。
<br/>　　叶少阳歪头冲她笑着。
<br/>　　“姐，两年不见，你不认识我了。”
<br/>　　老板娘一下子就懵了，手捂着胸口，一张脸因为激动涨得通红，几秒钟后，眼泪一下子就出来了，“少阳，真的是你！”
<br/>　　叶少阳换了发型，衣服也换了，之前没仔细看，她竟然没能认得出来。
<br/>　　“姐！”
<br/>　　叶少阳看她哭了，也不闹了，三两步走过去，伸手为她擦眼泪。
<br/>　　总算是见到了。
<br/>　　小寡妇翠云。当初星宿海一战，自己身受重伤，被徐福用山海印带到民国十一年，昏迷在翠云家门外，多亏了她悉心照料，自己捡回一条命，之后翠玉为了自己，离开了村子，一路上像个亲姐姐似的对自己百般照料，自己当初离开民国时，连面都没跟她见上，本来还以为可能是终生的遗憾了。
<br/>　　有了山海印，叶少阳除了窥探小时候的自己，第一个就想到了翠云，在自己这条时间轴上往前查找，找到了一九二五年，也就是民国十三年，找到了这位实际比自己大“一百多岁”的大姐。
<br/>　　之前带着小九穿越到民国闲逛，看到那些相似的时代特征，他就忍不住了，这才导演了一出吃霸王餐，想要给翠云一个惊喜。
<br/>　　“少阳啊，真的是你啊！”
<br/>　　眼泪刚被擦干，翠云又哭起来，倒把边上几个厨师和小二闹的一脸懵，不知所措地站在边上。
<br/>　　翠云的情绪总算缓和下来一些，在叶少阳肩上擂了几拳，“少阳你可真狠心，当初就那么走了，连一句话也不留下！”
<br/>　　叶少阳愧疚道：“姐对不起，当时我是实在来不及跟你道别，只好让毛小方给你带话，你看我这不是来看你了吗。”
<br/>　　“带话有什么用！”翠云瞪了他一眼，“行啦，姐知道你走得匆忙，不是真怪你，行了不要站在这了，老沈你们快去忙，哦对了，这就是我那个弟弟，你们的老板。我经常跟你们说的。”
<br/>　　客人变成了老板，大家目瞪口呆，但还是恭敬地打了招呼。
<br/>　　“行了我先带他回家，你们接着忙，有人点西湖醋鱼，就说我今天有事不在。”说完拉着叶少阳就往外走，声称要带他回家。
<br/>　　“哎哎，姐，我还有个朋友呢。”叶少阳回头招呼小九过来。
<br/>　　翠云一下子愣住，突然笑起来，拉着小九的手说道：“妹子你可别生气，我这刚见到少阳，太激动了，就忘了还有个人，还是个如花似玉的大姑娘，你是……”
<br/>　　小九有些羞涩地看了叶少阳一眼，叶少阳也有点卡壳，不知道怎么介绍，说门人什么的，翠云一个普通人是听不懂的。
<br/>　　翠云却以为他们是害羞，脸上露出知心大姐的标准笑容，“这怕是你媳妇啊，好好，少阳你可真行，找个这么漂亮的媳妇，弟媳怎么称呼？”
<br/>　　“我……”小九有点局促。
<br/>　　叶少阳干咳两声，“你就叫她九儿吧。”
<br/>　　“好，九儿，跟姐走，去姐家里。”
<br/>　　翠云拉着两人就出了门，朝对面的胡同走去。叶少阳回头看到招牌上的“仙云楼”三个字，道：“姐，你这名字取得可以啊，跟你名字正好对应。”
<br/>　　翠云笑道：“我哪里懂这个，这是毛道长取的名字，这酒店也是他张罗开起来的，我一个妇道人家，我哪里懂这些。”
<br/>　　叶少阳笑道：“我看出来了，刚才我吃白食，你还想免单来者。”
<br/>　　说话间，翠云带他们走进了胡同，来到一栋宅子前面，这死一栋二层楼的老宅，带一个院子，很普通，但是里面收拾的很干净，连一些杂物都靠墙摆的整整齐齐。
<br/>　　翠云带他们进堂屋，去烧了水，然后拿了一大把瓜子花生之类的零食，就放在小九前面，热情地招呼她吃，自己就搬了凳子坐在她对面，看着她笑，弄得小九反倒有点不好意思。
<br/>　　在叶少阳询问下，翠云讲述了他离开之后的经历：自己当时在杭州城跟人学做菜，一边等着他们回来，结果毛小方、云儿倒是回来了，叶少阳和道渊真人却没回来。
<br/>　　道渊真人回了龙虎山，而叶少阳，她也听毛小方说了，是回到了他来时的年代。
<br/>　　那段时间，她很失落，以为这辈子再也见不到了，但她内心深处却不愿放弃，总觉得只要等下去，少阳一定会来的。
<br/>　　说到这里，她眼眶又红了，急忙擦去，冲叶少阳笑道：“你看，这才过了两年，我就把你给等到了。也是老天保佑，少阳，你还是好好的，就是瘦了一点。”
