　　一路飞行。
<br/>　　萧逸带着小星，来到了破玄城之外的雷鸣荒泽。
<br/>　　寻了一处偏僻的高山后。
<br/>　　萧逸拿出了四根引雷木。
<br/>　　“引雷木？”小星惊呼一声。
<br/>　　他作为雷属性妖兽中的佼佼者，加之拥有远古雷兽的一丝血脉。
<br/>　　对这种雷属性的天材地宝，极其敏感，一眼就认了出来。
<br/>　　“主人，你到哪寻了这四根宝贝回来？”小星满脸崇拜之色地问道。
<br/>　　“借的。”萧逸随口回答一句。
<br/>　　不错，他之前特地去药王谷一趟，借来引雷木，就是给小星准备的。
<br/>　　他知道，绝对有人要对付萧家。
<br/>　　虽然萧家现在风平浪静，但，这必然只是暴风雨前夕的平静。
<br/>　　以他现在的速度，从破玄城刚拿回萧家，顶多十分钟的事。
<br/>　　故，他并没有太过担心。
<br/>　　这时，萧逸大手一挥，一道庞大的灵气屏障，包裹了整座高山。
<br/>　　方圆数千米之内，无人能靠近。
<br/>　　随后，将四根引雷木插在地面。
<br/>　　霎那间，整个天空电闪雷鸣，风云大作。
<br/>　　平时的引雷木，并不会显露功效，引动雷电。
<br/>　　只有将它们放于地面，让它们接触大地的气息，方能生效。
<br/>　　引雷木，作为最珍稀的天材地宝之一。
<br/>　　哪怕被采摘下来多年，里头的生机也不会消散，反倒是被锁于木身之内。
<br/>　　一旦将它们重新插于地面。
<br/>　　木身内的生机，便会顷刻爆发。
<br/>　　这等同于变相地将它们重新‘栽种’。
<br/>　　自然的，便也发挥了它们引动天雷的功效。
<br/>　　“嘶。”小星倒吸一口凉气。
<br/>　　“主人，这引雷木，不会是最珍稀的天材地宝之一。”
<br/>　　“就这般引动的天地雷电，竟然比我全力释放的雷电还要强得多。”
<br/>　　萧逸笑笑。
<br/>　　引雷木，作为无限接近八品阶品的天材地宝。
<br/>　　能被药王谷用来结阵封印无数星辰之火，它们的功效，自然极强。
<br/>　　小星的控雷手段，其实并不比引雷木差。
<br/>　　只不过，他现在才破玄九重，自然远远比不过引雷木。
<br/>　　“好了，别废话了。”萧逸沉声道。
<br/>　　“这四根引雷木，是要还回去的。”
<br/>　　“抓紧时间修炼吧。”
<br/>　　说罢，萧逸原地坐下打坐。
<br/>　　下一秒，漫天雷电，带着狂暴气息，汹涌袭来。
<br/>　　那种雷霆万钧的威势，如若毁天灭地。
<br/>　　哪怕小星是天星雷鳄，也满脸凝重。
<br/>　　轰…轰…轰…
<br/>　　无数道爆响声响起。
<br/>　　一条条雷蛇，从天而降。
<br/>　　狠狠地劈在四根引雷木之上。
<br/>　　而在引雷木旁边的萧逸以及小星，自然受到了波及。
<br/>　　小星，瞬间被天雷轰得脸色潮红，显然是受了轻伤。
<br/>　　萧逸，则岿然不动，毫发无损。
<br/>　　时间，渐渐过去。
<br/>　　引雷木，一旦重新接触大地气息，焕发生机。
<br/>　　木身之内的生机，会逐渐被触发。
<br/>　　待得生机完全触发，那时降下的雷电，将极其恐怖。
<br/>　　果然，不过十数分钟后。
<br/>　　天雷，变得更加密集，也变得更为狂暴。
<br/>　　小星，已经口吐鲜血。
<br/>　　半个时辰后。
<br/>　　天雷，已化作了一条条粗壮的雷蛇，宛若吞噬天地。
<br/>　　小星，再也无法保持人类形态。
<br/>　　只能立刻变回妖兽之躯。
<br/>　　妖兽的身躯，本身就极强强大。
<br/>　　只有变回妖兽躯体，小星才能发挥出最强的实力。
<br/>　　轰…轰…轰…
<br/>　　小星那狰狞的皮肤盔甲，顿时被天雷轰得破碎。
<br/>　　不过十数分钟。
<br/>　　偌大一头天星雷鳄，被轰得里焦外嫩，甚至一阵如同‘烤熟’了一般的气息传出。
<br/>　　“张嘴。”萧逸低喝一声。
<br/>　　连忙抛出数粒高品丹药，扔入小星嘴中。
<br/>　　丹药效力，顷刻发挥。
<br/>　　小星原本被轰得里焦外嫩的躯体，以肉眼可见的速度，快速修补着。
<br/>　　不多时，竟恢复如初。
<br/>　　不得不说，对比人类，妖兽的身躯，无比强大。
<br/>　　这是妖兽最大的优势。
<br/>　　也难怪无数年前，上古时期的某位武道至尊，费尽心血，都要创出‘修罗战体’这样的逆天功法。
<br/>　　让人类也可以拥有妖兽的强大之躯。
<br/>　　半天后。
<br/>　　整个高山上空，弥漫着一股剧烈的雷电气息。
<br/>　　那些轰向小星的雷电，虽让小星受着折磨。
<br/>　　却也让得它的修为在不断提升。
<br/>　　对于天星雷鳄来说，这些雷电，可是大补之物。
<br/>　　而萧逸，他本身并无法吸收这些雷电气息。
<br/>　　他也在天雷之下修炼，硬抗天雷，只是借助天雷，淬炼肉体。
<br/>　　他没有用元力护身，甚至将寒冰铠甲也脱下了。
<br/>　　漫天雷电，早就将他的上衣轰得破碎。
<br/>　　但，雷电打在寒冰铠甲上，却奈何不了寒冰铠甲半分。
<br/>　　萧逸自然只得脱下。
<br/>　　上身那结实的身躯，让得他本就帅气的脸庞，显得更加魅力十足。
<br/>　　漫天雷电之下，却损不了他分毫。
<br/>　　仿佛让他沐浴于雷电之下，似极了战神下凡。
<br/>　　时间，一分一秒过去。
<br/>　　雷电，并未能让他提升修为。
<br/>　　却让他的肉体，不断得到淬炼，不断得到提升。
<br/>　　三天后。
<br/>　　萧逸忽然停下了修炼，将地面的四根引雷木拔起。
<br/>　　天空上那滚滚天雷，顿时消散。
<br/>　　“怎么了主人？”小星一阵不满。
<br/>　　这三天时间，他可是得尽了好处。
<br/>　　虽然吃了不少苦头，但修为，大大飙升。
<br/>　　破玄九重的修为，已突破至地元一重。
<br/>　　短短三天，便能突破地元境，着实恐怖。
<br/>　　当然，这只局限于小星乃是天星雷鳄，拥有远古雷兽的一丝血脉，才这般天赋惊人。
<br/>　　雷电，是他最好的大补之物。
<br/>　　当然了，仅仅如此，还不够。
<br/>　　那般恐怖天雷之下，哪怕小星，也支撑不了半个时辰。
<br/>　　还是仰仗于萧逸那大量的高品疗伤丹药。
<br/>　　这才让得小星的妖兽之躯，能够在不断损坏受伤的情况下，又不断复原，不断撑下去。
<br/>　　总之，两者结合，才有了小星如今这般恐怖的修炼速度。
<br/>　　另外一点。
<br/>　　萧逸的肉体，也明显有了变化。
<br/>　　地元五重的肉体力量，达到了地元六重水准。
<br/>　　不错，他的肉体力量，仅有地元五重。
<br/>　　还是上次与血谷主交手，得了他乾坤袋里头的大量高级妖兽精血后，才晋升的地元五重。
<br/>　　之后，并没有什么机会得到太过高级的妖兽精血。
<br/>　　修罗战体的修炼，暂时停了下来，肉体力量，自然也没怎么提升。
<br/>　　而这时，萧逸之所以收回引雷木。
<br/>　　是因为，他明显感觉到，猎妖令有异。
<br/>　　母令上，明显感知到萧家萧离火身上的子令有异动。
<br/>　　嗖，萧逸身影一闪，瞬间御空离去。
<br/>　　不过十分钟，便已回到了萧家。
<br/>　　果然如他所想，有人要对付萧家。
<br/>　　此刻的萧家，正被数千陌生武者包围着。
<br/>(本章完)
