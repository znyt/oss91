　　眼见苏莫急速远去，褚彦放弃了出手，转头对沁儿道：“沁儿，我暂时饶他一次，但若是此人接下来，再敢故意接近我们，我定然不饶！”
<br/>　　言罢，褚彦看向骆妤，问道：“骆妤，你觉得呢？”
<br/>　　“就这样吧！”骆妤点了点头，她也不是凶狠之人，既然这个段惊天走了，她也懒得再去处理。
<br/>　　“褚彦哥，你也无需这么戒备，此人虚神境修为，根本不会伤害到我们！”沁儿微笑着说道。
<br/>　　她之所以不担心什么危险，自然是因为她们三人修为高、实力强。
<br/>　　“防人之心不可无！”
<br/>　　褚彦摇了摇头，他也没有过多唠叨，沉声道：“我们走吧！”
<br/>　　随即，三人没有过多逗留，再次启程。
<br/>　　……
<br/>　　高空之上，苏莫加快了速度，免得被褚彦三人追上，到时候又有不必要的误会。
<br/>　　不过，飞行了几日，他也没发现褚彦三人追来，便继续一边飞行，一边参悟了起来。
<br/>　　他的吞噬奥义，已达八阶中期，进展虽然不算奇快，但也不慢。
<br/>　　又过去了半年，苏莫终于进入了轮回神朝境内。
<br/>　　一进入轮回神朝，他立刻降落而下，在低空飞掠前进。
<br/>　　是非之地，贸然来此，他必须先弄清楚情况。
<br/>　　飞掠了一个多时辰，他在一处山脚下，发现了一处小镇。
<br/>　　于是，苏莫向小镇飞了过去。
<br/>　　在小镇中待了半个时辰，苏莫便离开了小镇，继续飞行。
<br/>　　“天王宫、地王宫、人王宫、兽王宫、魔王宫、鬼王宫！”
<br/>　　苏莫低声自喃，他在小镇之上，打听到了很多重要的信息。
<br/>　　小镇上的人，虽然修为都不高，基本上都是虚神境以下的人，但是，对于轮回神朝的事情，却是都一清二楚。
<br/>　　轮回神朝六大王宫，除了天王宫、地王宫和人王宫之外，剩下的三大王宫，名为是兽王宫、魔王宫和鬼王宫。
<br/>　　天、地、人三宫，均是人族掌控，兽王宫乃是妖族掌控、鬼王宫居然是太古万族榜排名十三的鬼族掌控。
<br/>　　而魔王宫，并不是魔族，而是修炼修罗魔道的人族。
<br/>　　六大王宫，分布在轮回神朝六个方向，坐镇整个轮回神朝。
<br/>　　最主要的是，帝一魂便在天王宫，拓跋青云在地王宫。
<br/>　　根据苏莫打听到的消息，拓跋青云此人，就出生在地王宫，而且是地王宫宫主的嫡系血脉。
<br/>　　所以，整个地王宫，才会非常拥护拓跋青云，全力对抗帝一魂。
<br/>　　据说，帝一魂当初弱小之时，有多次险些被地王宫的强者镇压。
<br/>　　但是，帝一魂非常强大，无比逆天，数次化解危难。
<br/>　　而现如今，帝一魂早已彻底的成长了起来，名震神域，基本上已经很难被人镇压了。
<br/>　　而现在，帝一魂在人王宫，担任太宫主，地位与天王宫宫主并驾齐驱，甚至凌驾于宫主之上，已经得到了整个天王宫的支持。
<br/>　　而这些年，天王宫和地王宫，更是冲突不断，究其原因，自然便是因为帝一魂和拓跋青云。
<br/>　　不过，现如今，对于地王宫而言，局势并不妙。
<br/>　　因为以前，其它四大王宫，都抱着观望的态度。
<br/>　　但现如今，在很多事情之上，四大王宫，渐渐和天王宫开始走近。
<br/>　　“奇怪！”
<br/>　　苏莫心中有些惊疑，帝一魂是轮回主魂，此事在轮回神朝，几乎人尽皆知。
<br/>　　但是，其它四大王宫，居然不全力帮助帝一魂，这就有些耐人寻味了。
<br/>　　“难道说，其它王宫根本不想要轮回天尊归来？”
<br/>　　苏莫低声自喃，但若是这样的话，他们为何不帮助拓跋青云，将帝一魂彻底镇压，那不就完事了。
<br/>　　留着帝一魂，若是帝一魂将来成功了，其它几宫不是祸患无穷？
<br/>　　而若是想让轮回天尊归来，那几宫的强者，直接助帝一魂融合拓跋青云，不就行了吗？
<br/>　　苏莫有些想不太明白，感觉不合情理。
<br/>　　再者，帝一魂的存在不是秘密，其轮回主魂的身份，想必各大神朝都知晓。
<br/>　　而各大神朝的强者，对此好像莫不关系，这更加的匪夷所思了。
<br/>　　难道说，轮回天尊身前，没有敌人也没有朋友，不然为何没有超级强者干预此事？
<br/>　　这一切，都是谜团，而这些事情，普通的轮回神朝之人，根本无法知晓。
<br/>　　“有些扑朔迷离啊！”
<br/>　　苏莫深深的吸了口气，不过，不过怎样，他都要保护好自身，强大起来才能应对。
<br/>　　不管有什么秘密，不管有什么原因，他都不能被别人融合。
<br/>　　嗖！
<br/>　　苏莫加快了速度，低空飞掠，急速而行，向人王宫的方向而去。
<br/>　　不错，他就是要去人王宫，正是褚彦、沁儿、骆妤三人所在的一宫。
<br/>　　他有这个决定，并不是因为沁儿三人，而是经过了综合的考量。
<br/>　　首先，天王宫和人王宫，是绝对不能去的。
<br/>　　其次，兽王宫基本上都是妖族，魔王宫都是魔修，鬼王宫是鬼族，他若是去了这三大王宫，实在太显眼了，简直与找死无异。
<br/>　　所以，他只能去人王宫。
<br/>　　去人王宫的目的，自然是想办法获得人王宫的支持。
<br/>　　此举乃是兵行险招，不能操之过急，必须徐徐图之。
<br/>　　一旦有变，他要立刻离开。
<br/>　　毕竟，人王宫强者的太多，目前他并不清楚。
<br/>　　苏莫已经打听清楚了，人王宫坐落在轮回神朝东部的人王城，实力强大。
<br/>　　人王宫，共有一位宫主、两位副宫主、十位长老，以及三十万弟子。
<br/>　　宫主和副宫主，以及长老，全部都是神王境的强者。
<br/>　　而三十万弟子，虽然不多，但是，其中却也是强者众多。
<br/>　　不过，人王宫不是一般的宗门势力，很少在外界招收弟子。
<br/>　　其内的弟子，均是如同家族传承一般，大多都是宫内强者的后代。
<br/>　　据闻，人王宫内，有大大小小的家族上千个之多。
<br/>　　当然，为了保持人王宫的发展，人王宫还是会吸纳外界的天才。
<br/>　　只不过，招收的太少，二十年招收一次，一次只有十个名额。
<br/>　　“距离招收弟子，还有十五年！”
<br/>　　苏莫低声自喃，有些郁闷，他来的并不巧，距离下届人王宫招收弟子，还有十五年的时间。
<br/>　　而他，根本不可能等待十五年。
<br/>　　“或许，那位沁儿姑娘，可以帮我！”想到那个娇小单纯的可爱女子，苏莫脑中闪过思索之色。
<br/>　　那个褚彦和骆妤，肯定不会帮他，但是那个沁儿却是有这个可能。
<br/>　　而目前，那三人应该还再后面，没有回到人王宫。
<br/>　　想到此处，苏莫立刻加速，向人王宫而去。
