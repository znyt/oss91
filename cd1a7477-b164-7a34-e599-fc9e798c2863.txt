　　随着苏莫一指点出，霎时之间，一道长达万丈的犀利指芒，破空而出。
<br/>　　这道指芒的速度，快到了无法形容的地步，洞穿了虚空，飞跃了第五山，跨越了第六山、第七山、第八山、第九山、掠过了无数万里，直接击中了界障。
<br/>　　轰隆！
<br/>　　一声闷响，好似从天际的尽头传来，神域的界障应声而破，直接被洞穿，裂开了一个硕大的口子。
<br/>　　“现在的界障之力，只有以前的九成，想必过段时间还会减弱！”苏莫见此，声音低沉的说道。
<br/>　　然而，此时此刻，全场鸦雀无声，寂静的可怕！
<br/>　　所有的人都瞪大了眼眸，不可思议的望向界障所在的方向，虽然界障所在之地，已经超出了他们的视线范围，但是，他们都能感知到界障破了。
<br/>　　因为，此时此刻，界障的波动之力，远超任何人攻击的时候，绝对是界障被破开，才能发出如此大的波动。
<br/>　　身在第五山，一指破界障！
<br/>　　这种实力，让得所有人为之骇然，为之恐惧。
<br/>　　要知道，他们之中，没有任何人有实力破开界障，而苏莫距离界障如此之远，随意一指破开界障，这种实力已经强大到让人无法想象了。
<br/>　　众人心中骇然，他们也暗自猜测过，苏莫如今的实力达到了什么地步。
<br/>　　但是，现在看来，只能用深不可测来形容了！
<br/>　　姜神夜暗暗的叹了口气，当初，他和苏莫之间，差距不是很大，但现如今，可谓是天壤之别了。
<br/>　　“那我们时候可以去神域？”
<br/>　　片刻之后，有人面带期盼的问道，既然界障开始减弱了，那他们的机会也都来了。
<br/>　　“只要你们能破开界障，随时可以去！”苏莫朗声说道，他可不会出手，帮助这些人破开界障。
<br/>　　因为，若是只身实力不够，就算进入了界障之内，也到达不了神域。
<br/>　　所以，这些人想要去神域，还要靠他们自己。
<br/>　　对于神路的这些霸天族、不死族、幽冥族等人，苏莫并不记恨，因为这些人和神域的那几大神朝，完全没有任何往来。
<br/>　　“不过，我要提醒你们的是，神域更换阵基的时间有限，这个时间可能是十年左右，也可能会稍久一些，你们自己掂量吧！”
<br/>　　苏莫再次朗声道，言罢，他背负着双手，转身向苍穹神宫之内走去。
<br/>　　不过，刚刚走出两步，苏莫又停下了脚步，转身望向虚无神和姜神夜，道：“两位要不要进来坐坐？”
<br/>　　虚无神和姜神夜闻言，顿时微微一怔，不过，两人均是立刻点头。
<br/>　　随后，两人便快步走出，跟着苏莫走进了苍穹神宫之中。
<br/>　　其他的所有人，见苏莫离去，也没有再追问，苏莫虽然说得不多，但是，最主要的已经讲的很清楚了。
<br/>　　少倾，众人纷纷散去，他们都要为前去神域做准备了。
<br/>　　……
<br/>　　宫殿中。
<br/>　　苏莫和虚无神、姜神夜，分宾主落座。
<br/>　　“两位，多年不见，别来无恙！”苏莫环视两人，微笑着说道，这两人当初都对他有恩，他自然不会忘记。
<br/>　　姜神夜还好说，天赋强大，去往神域丝毫没有问题，而虚无神就不行了，修为才虚神境二变。
<br/>　　这个修为，就算界障威力大减，怕是虚无神也没有机会。
<br/>　　所以，这一次，苏莫也是想帮一下对方。
<br/>　　“苏莫，现如今，你与我们已经不在一个层次，没想到你还能单独邀请我们！”虚无神叹了口气说道。
<br/>　　“师兄，你这说的是哪里话？若是没有你，当初我已经死在了巫族的手中！”苏莫沉声说道，他知道，现如今他和虚无神之间，已经有了无法逾越的鸿沟。
<br/>　　但是，恩情他是不会忘记的。
<br/>　　想到恩情，苏莫不由的有想到了真贤道圣，这位道宗的圣长老，当初对他的帮助也很大。
<br/>　　“当初我们是同门，这都是我应该做的！”虚无神道。
<br/>　　“或许吧！”苏莫微微点头。
<br/>　　“苏莫，神域是什么情况，真神之境，又是什么情况？”姜神夜开口问道，这是他迫切想要知道的情况。
<br/>　　“神域并不大，共有九大神朝……！”苏莫徐徐开口，向姜神夜讲解神域的大概情况，以及真神之境的具体境界。
<br/>　　“神王！”
<br/>　　听完苏莫的讲述，姜神夜低声自喃，眼眸之中露出了向往之色，道：“不知道有生之年，我有没有机会达到这个地步？”
<br/>　　“以你的天赋，有很大的希望！”
<br/>　　苏莫笑着说道，随即，他直言道：“今日让你们来，是要给你们送个造化。”
<br/>　　“什么造化？”姜神夜和虚无神闻言，惊疑不定的问道。
<br/>　　“你们先去找两种元神，契合你们自身武道的元神，速去速回！”苏莫沉声道，为两人融合元神，他也算是报当年之恩了。
<br/>　　虚无神和姜神夜，对视一眼，一脸懵逼，完全不知道苏莫要干什么。
<br/>　　“找到元神之后，便回来找我！”苏莫微微一笑，不再多言，闪身离去。
<br/>　　虚无神和姜神夜疑惑不已，但是，既然苏莫有这个要求，他们也只能先照办了。
<br/>　　随后，两人离开了苍穹神宫，前去选择元神。
<br/>　　苏莫并未等待太久，他和木璃温存了片刻，便感知到姜神夜和虚无神回来了。
<br/>　　苏莫再次接见了两人，为两人融合元神，增强天赋。
<br/>　　一个时辰之后，两人再次离开了苍穹神宫，离开之时，两人均是面色潮红，神情振奋，显然，元神的融合非常成功。
<br/>　　接下来的日子，苏莫过的非常清闲，抽空去了一趟玄荒星域，找到了真贤道圣这位曾经的师兄。
<br/>　　真贤道圣的修为，还不到圣王之境，非常的地下。
<br/>　　苏莫同样没有吝啬，为对方融合了战魂，而且是一次性融合了两种战魂。
<br/>　　随后，他才返回了神路。
<br/>　　在这段时间之中，界障的威力再次大减，已经有少量种族的强者，开始攻击界障，并且成功的进入了神域。
<br/>　　在神路待了两个月的时间，苏莫不准备再逗留了，和一众苍穹神宫之人，一一惜别。
<br/>　　并且，他告知众人，霸天族的强者，不能留了。
<br/>　　神路的霸天族，都是苍穹神宫的奴仆，而如同力霸、力铜等人，实力非常强劲，都是有能力进入神域的。
<br/>　　这些人进入神域，有害无利。
<br/>　　随后，苏莫便带着龙腾、司空炎和池儿，向界障所在之地飞去。
<br/>　　本来，他想带一部分苍穹神宫之人同行的，但是，考虑到神域的局势，便让众人过几年再去神域。
<br/>　　界障之处，苏莫四人伫立在一起。
<br/>　　“池儿，你的实力最弱，要小心一些！”苏莫目光看着池儿，郑重的告诫道。
<br/>　　二长老开启的神域通道，要在阵基更换完毕之后，才会再次开启，所以，他们要反回神域，只能从界障之处。
<br/>　　“放心吧，师尊！”池儿点了点头，她的修为已经达虚神境第三变，足够应付了！
<br/>　　而且，她很有信心，神路的这些虚神境四变的强者，都没几个人是他的对手。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即环视池儿、龙腾、司空炎，郑重的说道：“我们到神域之后，会被分散开来，出现在不同的飞升池之中，至于出现在哪个神朝的飞升池，无法确定。”
<br/>　　“若是出现在五大神朝怎么办？”龙腾皱眉说道，他倒是不知道还有这一茬子。
<br/>　　“出现在五大神朝还是好的，若是苏莫你出现在天王宫的地盘，那就完蛋了！”司空炎警告道。
<br/>　　“放心，不会有事的！”
<br/>　　苏莫微微一笑，怎么可能这么巧合出现在天王宫，出现在亘古魔国等五大神朝，倒是很有可能。
<br/>　　不过，他丝毫不担心，各大神朝的飞升池，都是独立建造的，由于常年无人飞升，都快荒废了，一般只有个别修为最低的人看守，没有人会认出他。
<br/>　　“我想说的是，你们到神域之后，就返回龙都城吧，我暂时不回去！”苏莫沉声道。
<br/>　　“你要去哪？”司空炎急忙问道，不回龙都城，那危险将百倍增加。
<br/>　　“时空界域！”
<br/>　　苏莫微微一笑，随即手指一化而出，瞬间击在了界障之上。
<br/>　　哧啦！
<br/>　　庞大的界障，如同幕布一般，瞬间便苏莫撕裂出一道长达数十丈的口子。
<br/>　　“我们走！”苏莫招呼一声，身形一闪，率先飞入了界障之中。
<br/>　　而后，司空炎三人，仅随其后。
