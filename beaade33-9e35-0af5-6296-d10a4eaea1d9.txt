都是武者。你要知道，武者和普通人的治疗是不一样的，有些时候就是药材都要换成上年份的才成。如果这样子，其实每个人10万元，恐怕有的连个药钱都不够，更不要说五万元了！”<br/>　　<br/>　　“那你说，要怎么做？”袁若珊一项，感觉陈默说的也有些道理，所以就直接问道。<br/>　　<br/>　　“珊姐，这么着吧，治疗费用我可以不要。但是需要什么药材，需要你们来出，而且每治疗好一个人，你得给我寻找些特殊的药材，你看这样如何？”陈默其实今天打得注意就是这个，让特管局的人为自己寻找一些药材，这要比他被动的找黄老寻找药材要快捷的多。<br/>　　<br/>　　“特殊的药材？”袁若珊有些疑问。<br/>　　<br/>　　“这样，你跟我来一下！”<br/>　　<br/>　　陈默带着袁若珊来到了别墅的一个房间，这里都被陈默腾空后放置药材，就和中药房的药材墙柜一样。<br/>　　<br/>　　拿出几张纸，上面是陈默手头有的药材名称。将这个递给了袁若珊。<br/>　　<br/>　　“这是我手头有的药材，你要提供的药材要出去这个上面标注的。另外就是必须是活株或者种子。当然，如果是些稀有的药材，那么无论什么样子，都可以。”<br/>　　<br/>　　如果已经被炮制成药用的，那么怎么也不能再种植了。要是拿过来的话，陈默就想办法入药好了，反正他也需要配置一些药材的。<br/>　　<br/>　　“好，就按照你说的办！我回去后会让局里满足你的要求！”<br/>　　<br/>　　对于陈默的这个要求，她并没有什么意外。中医，对于中药的需求，当然是比较大的。而且现在好的中药材也不好找，既然陈默也不是要一些特殊的药材，只是这些登记过的药材除外，那么还是好找的。<br/>　　<br/>　　要是他要些武林中武者都有用的药材的话，还不如给他出诊费呢。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
