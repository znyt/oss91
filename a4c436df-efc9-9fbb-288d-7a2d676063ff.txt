　　“钱副阁主，你在等我？发生什么事了吗？”苏莫疑惑的问道。
<br/>　　“是阁主让我在此等你！”
<br/>　　钱副阁主点了点头，满脸凝重的道：“的确是有大事发生了，黄泉魔宗宗主，晋升武帝之境了！”
<br/>　　“什么？”苏莫闻言心中一震，面现愕然之色，黄泉魔宗的宗主，居然这个时间点突破到了武帝之境？
<br/>　　这也太会挑时间了吧！
<br/>　　“什么时候的事情？”苏莫立刻沉声问道。
<br/>　　“就在昨日！”钱副阁主叹道。
<br/>　　“昨日？”
<br/>　　苏莫闻言皱眉，诧异的问道：“昨日才突破的修为，这么快消息就传回来了？
<br/>　　按理来说，黄泉魔宗的宗主，若是真的突破了修为，消息想传到苍穹神宫，最快也要几日时间吧！
<br/>　　“苏莫，你有所不知，在苍穹世界，只要有人踏入武帝之境，或者是有妖兽晋升八级，便会出现天地异象，整个苍穹世界的人都能看到。”
<br/>　　“哦？什么异象？”苏莫闻言惊奇，他还从未听说过，晋升武帝会出现天地异象。
<br/>　　“天地之间出现白虹贯日的异象，这就是宣誓着有武帝出现，任何人都能看到。”钱副阁主道。
<br/>　　“我知道了！”
<br/>　　苏莫点了点头，又问道：“海阁主和帝宫主还有东海龙王，是否在宫中？”
<br/>　　“阁主和帝宫主都在，东海龙王并不在！”钱副阁主道。
<br/>　　随即，苏莫不再询问，快速的向苍穹神殿掠去。
<br/>　　“苏莫，这是你上次让我收集的炼器材料，全部在这枚储物戒中。”钱副阁主追了上来，递给苏莫一枚储物戒。
<br/>　　“哦？”苏莫闻言讶然，天涯海阁的总部都被灭了，对方居然还记着这事，并且还将炼器材料带了过来。
<br/>　　“多谢了！”苏莫没有拒绝，伸手接过了储物戒，他也没有给对方购买材料的灵石，因为在他心里，已经将天涯海阁纳入了苍穹神宫之内。
<br/>　　而后，苏莫便急速飞向了苍穹神殿，然后命人找来了帝释和海明江。
<br/>　　……
<br/>　　苍穹神殿中，只有苏莫和帝释、海明江三人。
<br/>　　海明江和帝释的脸色沉重无比，显然都是因为黄泉魔宗宗主突破了武帝之境，而心中担忧。
<br/>　　“两位，不必担忧！”苏莫见此，微微一笑说道，他心中倒是没有太多的担忧。
<br/>　　“苏宫主，我们如何能不担忧？那巫森突破到了武帝之境，天下无敌了，再没有人能阻挡！”海明江叹道，苍老的脸庞上布满了愁容。
<br/>　　等到巫森杀过来，他们所有人怕是都难活命。
<br/>　　虽然观苏莫的气息，已经如愿的突破到了武皇境四重境界，但还是不可能是武帝强者的对手。
<br/>　　“苏莫，你可有什么好的应对之策？”帝释沉声问道。
<br/>　　苏莫闻言沉吟了片刻之后，道：“你们也不必太过担心，就算黄泉魔宗的宗主突破到了武帝之境，也不太可能攻入苍穹神宫！”
<br/>　　苏莫并没有太过担心，苍穹神宫的浑天洛水阵，可不是摆设。
<br/>　　“我们总不能坐以待毙吧！”
<br/>　　帝释摇了摇头，继续道：“巫森突破到武帝之境，整个苍穹世界为之哗然，估计有很多大势力，都要向黄泉魔宗靠拢了！”
<br/>　　“这些无需在意！”苏莫摇了摇头，其他势力不足为惧，唯一要考虑的就是如何对付黄泉魔宗宗主巫森。
<br/>　　“老朽估计，要不了多久，巫森必然会杀来苍穹神宫，我们要早做打算！”海明江道。
<br/>　　“不错！这护宫大阵虽然强大，但也架不住攻击，若是巫森联合十数名准帝一起攻击，这水幕大阵不太可能挡的住！”帝释道。
<br/>　　苏莫闻言沉默了，两人说的有道理，他不能坐以待毙。
<br/>　　万一，黄泉魔宗真能攻破浑天洛水阵，那可就完蛋了，他的爱人、亲人、朋友，怕是一个都活不了。
<br/>　　可是，如何能拖住黄泉魔宗、拖住巫森呢？
<br/>　　黄泉魔宗，定然不可能给他们时间了。
<br/>　　还有就是，黄泉魔宗肯定也不会放过东海蛟龙一族，若是率先杀去东海，怕是整个蛟龙一族都要灰飞烟灭了。
<br/>　　苏莫沉吟了片刻，脑中想出了一个不是办法的办法。
<br/>　　“两位，立刻向外传出消息，就说我已经拥有了武帝的实力！”苏莫目光如电，郑重的说道。
<br/>　　“武帝的实力？”帝释和海明江闻言一怔，均是面带愕然之色，苏莫有了武帝的实力？
<br/>　　这不太可能吧？
<br/>　　虽说苏莫武皇境三重修为之时，就有准帝的实力，但准帝和真正的武帝，相差太远了。
<br/>　　就算苏莫现在提升到了武皇境四重境界，实力大增，也不太可能拥有武帝的实力吧！
<br/>　　苏莫没有理会两人置疑的目光，继续说道：“另外，再传出消息，就说我要在十日后，在帝玄山脉和巫森一战！”
<br/>　　帝玄山脉，便是昔日帝玄宫所在的山脉。
<br/>　　“和……和巫森一战？”帝释和海明江两人，均是被苏莫的话语惊呆了，完全没有想到苏莫居然要与巫森一战。
<br/>　　“苏宫主，你疯了吗？”
<br/>　　海明江长身而起，大声告诫道：“你根本不知道武帝强者是多么的恐怖，你这样无异于自寻死路！”
<br/>　　“苏莫，此法不可行！”帝释也是摇了摇头。
<br/>　　两人根本不支持苏莫的这个办法，与武帝强者一战，这简直是蠢到家了。
<br/>　　苏莫若是有武皇境五重的境界，他们不会阻止，但武皇境四重境界，想抗衡武帝强者，简直是痴心妄想。
<br/>　　“我没有疯！”
<br/>　　苏莫摆了摆手，道：“黄泉魔宗估计几日后便会杀过来，到那时整个苍穹神宫都会被围困，如你们所说，浑天洛水阵都不一定能挡住。”
<br/>　　“若是如此的话，我们更加的危险！”
<br/>　　“所以，为了整个苍穹神宫的安全，我主动邀战，并离开苍穹神宫，巫森应该不会来苍穹神宫，毕竟，杀我才是他们的首要目标。”
<br/>　　苏莫徐徐说道，向两人讲述他心中的想法。
<br/>　　帝释和海明江两人闻言沉默了，两人并不傻，自然知道这个道理。
<br/>　　不过，知道归知道，但是苏莫真的和巫森一战的话，那绝对是有死无生！
<br/>　　苏莫若是死了，那一切都完了，他们也必死无疑！
<br/>　　可以说，现在的苏莫，已经成为了他们的主心骨。
