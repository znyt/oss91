        夏凌焰也忍不住来了个哆嗦，然后咽了咽口水，只身去开那间卧室的门。
<br/>
<br/>        童筱颖尾随其后。
<br/>
<br/>        卧室门打开后，夏凌焰走进去看，发现只是墙壁上的相框掉下来，才松了口气，于是走过去将相框和照片从地上捡了起来。
<br/>
<br/>        估计是这么多年过去了，相框后面的胶也老化了。
<br/>
<br/>        童筱颖见地上一地的玻璃碎渣，于是转身去了厨房，拿来了撮箕和扫帚。
<br/>
<br/>        当她再次回到卧室里时，只见夏凌焰正拿着一张A4纸大小的照片发呆。
<br/>
<br/>        童筱颖也好奇地凑过去看了看，忽然间觉得这照片上的女孩令她觉得很眼熟，好像在哪里见过。
<br/>
<br/>        夏凌焰回过神来看了看童筱颖，又看了看手里的照片，接着又将手里照片放到了童筱颖的脸庞比对着。
<br/>
<br/>        “我去！你跟我姐长得可真像啊！”夏凌焰不禁感慨道。
<br/>
<br/>        “像吗？”童筱颖夺走夏凌焰手中的照片看了看。
<br/>
<br/>        上面的女孩，看起来也就二十岁左右，笑起来的样子特别甜。
<br/>
<br/>        要说她和这相片里的女孩像，其实也就眉眼间有些像罢了。
<br/>
<br/>        看到这里，童筱颖忽然间想起了什么来，下意识地问道：“你姐姐以前是不是在‘夜惑’里工作过？”
<br/>
<br/>        “你怎么知道？”夏凌焰顿时有些难以置信地瞪着童筱颖。
<br/>
<br/>        童筱颖也有些诧异地对视着夏凌焰的眼睛。
<br/>
<br/>        居然会这么巧！
<br/>
<br/>        “你姐姐叫什么名字？还有，你姐姐是不是已经嫁人呢？你姐夫叫什么名字？你是不是还有一个亲妹妹啊？”童筱颖一连串地发问，将夏凌焰问得哑口无言。
<br/>
<br/>        不过，为了满足她童筱颖的好奇心，夏凌焰一一回答道：“我姐姐叫夏玄烟，她确实是嫁人了，我姐夫叫冷逸峰，我只有一个亲姐姐，没有亲妹妹。不过，还有两个堂妹。一个叫夏玄炘，一个叫夏玄灵。噢，对了，夏玄灵就是夏凌焕的妹妹，不过是收养的妹妹，没有血缘关系。”
<br/>
<br/>        “原来，婴宁的原名叫夏玄烟，十爷的原名叫冷逸峰。昼景跟我说过，他的父亲叫冷卫峰，还有个堂叔叫冷岩峰……八爷又说十爷也是姓冷的。也就说，十爷其实也是大哥和昼景的堂叔之一。他们幕后对大哥虎视眈眈，看中的其实是傅氏集团！傅家大庄园里有他们的眼线。他们在暗，大哥在明。大哥岂不是很危险？！”童筱颖自言自语起来。
<br/>
<br/>        夏凌焰见童筱颖压根就没认真听自己说话，于是一脸不悦地问道：“你在碎碎念叨着什么啊？”
<br/>
<br/>        “不过话说回来，你除了夏玄烟这一个亲姐姐以外，就真的没有其他亲姐妹了吗？那为什么你姐姐要去‘夜惑’里工作？目的是为了找妹妹？”童筱颖随即纳闷道。
<br/>
<br/>        夏凌焰深沉地叹了口气，一脸忧伤地回答道：“我姐姐去‘夜惑’就是为了找那个最小的堂妹。因为，‘夜惑’里还有外界人不知道的地下夜场，专门拍卖一些年轻又漂亮还未开苞的女孩，成年的未成年的都有。所以，我姐姐想去那里探情况。当年，我九叔和九婶，生了个超级漂亮的小女娃。大家都说，这小女娃长得跟我姐姐出生时一个样。我姐姐也特喜欢这个小妹妹，天天去九叔家看她陪她玩。有一天，我九婶带着我姐姐还有那个小丫头一起在楼下小区里玩。然后我九婶的大儿子在外婆家玩受伤了，被送去了医院，我九婶心念大儿子急着去医院，想着我姐姐也有八岁了，于是让我姐姐把小丫头带回楼上去。哪知道小丫头被人贩子抱走了……他们大人回来后，我姐姐也复述不清那坏人的样子，就连是怎么被抱走的，我姐姐也说不清，只知道哭，报了警也未把那小丫头找回来。哎——我九婶差点还疯了。而我姐姐，也遭到了家族里的人的排挤。就连我爸妈也对我姐姐……”
<br/>
<br/>        “所以，这是婴宁的执念……也成了十爷的执念……”童筱颖弱弱地应道，顿了顿后，又问道，“那你姐姐是怎么死的？”
<br/>
<br/>        “不知道。”夏凌焰微微耸肩，“或许那个冷逸峰知道。不过，冷逸峰跟我们家早就已经交恶了。很多年都没见过面了。”
<br/>
<br/>        “如果是这样的话，那十爷为什么一口咬定我就是婴宁的妹妹？”童筱颖表示不理解。
<br/>
<br/>        夏凌焰忽然打了个响指，指着童筱颖，高兴地咧嘴道：“你跟我姐长得那么像！说不定，你就是当年那个被我姐姐弄丢了的小堂妹！”
