　　清晨，朝阳初生。
<br/>　　新房之中，苏莫换上了一身月白色的长衫，夕儿玉手轻拂，为他抚平衣衫上的褶皱。
<br/>　　经过一个多月的修养，苏莫的精气神虽然没有完全恢复，但也没有什么影响了，所以便结束了静修。
<br/>　　和夕儿一.夜温存，他不仅没有丝毫的疲惫，反而觉得精神百倍。
<br/>　　“夕儿，你们水月宫招收了多少弟子？”苏莫笑问道。
<br/>　　“三千余人，不过，再加上以前的老弟子重新划分，现在水月宫共有弟子近三万人！”夕儿笑着说道。
<br/>　　“嗯！”
<br/>　　苏莫微微颌首，苍穹神宫原来的那二十万弟子，也是按照各自的天赋属性，重新进行了划分，这样将更加适合发展，也更加的适合管理。
<br/>　　虽然目前长青宫、雷霆宫和厚土宫还没有传人，但有守护傀儡帮助管理，也不会出什么岔子。
<br/>　　“对了，苏莫哥哥，战乱目前在宫中，已经等了你一个月！”夕儿想到了什么，突然开口说道。
<br/>　　“战乱？”苏莫闻言恍然，他这才想起来，上次在帝玄山脉，他可是要求战乱来苍穹神宫。
<br/>　　而后，苏莫便和夕儿告别，离开了房间，前去面见战乱。
<br/>　　……
<br/>　　一间古朴的偏殿之中，苏莫和战乱分宾主落坐。
<br/>　　“战乱，许久不见，别来无恙！”苏莫面带笑容打量着战乱，越看越发现对方非常适合厚土宫。
<br/>　　战乱乃是双生武魂的天才，拥有巨人武魂和大地武魂，天赋绝对是巅峰存在，非常适合厚土宫。
<br/>　　“苏莫，你找我有何事？”战乱端坐不动，身形挺拔，即便是面对苏莫这个苍穹世界无敌的存在，他依旧面不改色。
<br/>　　“找你，自然是有事！”
<br/>　　苏莫点了点头，沉吟了一番，郑重的道：“你加入苍穹神宫如何？
<br/>　　战乱闻言一怔，随即立刻摇了摇头，道：“这不可能，我生于战神殿，长于战神殿，不可能背叛战神殿！”
<br/>　　战乱是有原则的人，他不会受人压迫。
<br/>　　“没有人要你背叛！”
<br/>　　苏莫轻笑一声，继续道：“要不了多久，苍穹世界将会大一统，所以，根本没有背叛一说！”
<br/>　　“什么？你要吞并所有的大势力？”战乱闻言心中一惊，眼眸眯了下来，他没想到苏莫的胃口如此之大。
<br/>　　其实，苏莫有了如今的实力，苍穹神宫已经无敌，其它大势力肯定会表面上臣服。
<br/>　　但这还不够，他要的是一统，不过，他也不急于一时。
<br/>　　“大劫将至，不得不为！”苏莫叹了口气，事实上若不是因为大劫，他根本懒得争霸，和自己心爱之人逍遥自在多好。
<br/>　　“大劫还有八、九年的时间，再说了，你就算统一了整个苍穹世界，也无济于事！”
<br/>　　战乱摇了摇头，面对试炼者降临，苍穹世界的强者只能竭力躲藏，才有机会逃过杀身之祸，对抗无异于自寻死路。
<br/>　　他不认为苏莫是为了对抗大劫，绝对是其自身的野心和欲.望。
<br/>　　“难道你就甘心成为试炼者屠宰的羔羊？难道你就甘心苍穹世界世世代代成为巫族圈养的猎物？难道你就不想反抗？”苏莫目光灼灼，直视战乱的眼眸，连问三个问题。
<br/>　　战乱闻言皱眉，他当然不甘心，但不甘心又能怎样？
<br/>　　按照战神殿历代先辈的记载，巫族乃是不可抗衡的存在，他们只有躲藏、逃避才有生还的希望。
<br/>　　“回答我，你甘心吗？”苏莫骤然一声冷喝，声浪滚滚，一股霸道、威严的气势在他身上冲天而起。
<br/>　　“不甘心！”面对苏莫突然的变化，战乱神情一怔，下意识的说道。
<br/>　　“既然不甘心，那就跟着我，打败试炼者，冲出苍穹世界，追寻我们应有的自由！”苏莫面色郑重，声音低沉无比。
<br/>　　打败试炼者，冲出苍穹世界，追寻我们的自由——
<br/>　　苏莫的话语，在战乱的心神中回荡，浩荡不休。
<br/>　　战乱心中震撼，他没想到苏莫居然有如此野望，这，可能吗？
<br/>　　这种想法，上古之时的无数先辈，无不想做到，也是为之努力。
<br/>　　但无数万年来，世世代代，都没有人能做到。
<br/>　　反抗，只会遭到血腥的屠杀，根本没有任何列外。
<br/>　　所以，如今的苍穹世界，面对试炼者大劫，想的不再是反抗，而是怎样躲藏，怎样逃避！
<br/>　　“你能做到？”战乱怔怔的问道，虽然他知道苏莫的天赋恐怖异常，盖压万古，但他还是不太相信，苏莫能完成这个伟大的壮举。
<br/>　　因为自古以来，有太多太多惊才绝艳的天才，都最终饮恨，陨落于试炼者之手。
<br/>　　比如独孤世家的先祖，十亿魔剑的锻造者，独孤殇。
<br/>　　在上古时代，独孤殇便是盖世妖孽，最终无敌于天下，为了对抗试炼者，更是锻造出十亿魔剑这种恐怖魔器。
<br/>　　但最终的结果，独孤殇依旧是以失败告终。
<br/>　　不过，若是真的有完成这个壮举的可能，战乱不会拒绝，他也算是如今的这个时代，最杰出的天才了，肯定要扛起肩上的责任。
<br/>　　“不成功，便成仁！”苏莫淡淡的说道，他一定会去做，这不仅是为了拯救苍穹世界，也是为了他自己的命运。
<br/>　　而且，苏莫知道，以他的修炼情况，在苍穹世界根本不可能突破到武帝之境。
<br/>　　因为苍穹世界，资源已经严重匮乏，气运也已经消耗殆尽。
<br/>　　这也是目前的苍穹世界，很难出现武帝的原因，出现一个巫森便已经是奇迹了。
<br/>　　他想在苍穹世界突破到武帝之境，千难万难，无法晋升武帝，便无法救活芊寻月。
<br/>　　所以，综合各种情况，苏莫一定要离开苍穹世界。
<br/>　　望着苏莫眼中的决然之色，战乱心绪涌动，久久不能平复，他感觉到自己体内的热血，仿佛已经被调动了起来。
<br/>　　“你有什么具体的计划吗？”战乱沉声问道。
<br/>　　“任何的计划只是徒劳，只有强大自身，才有胜利的希望！”
<br/>　　苏莫眸中精光闪烁，继续道：“苍穹神宫有大量远古顶尖武学，以我们的天赋，八、九年的时间足够了！”
<br/>　　听闻苏莫的话，战乱沉默了，心中不断的思索。
<br/>　　不错，苍穹神宫作为远古霸主级势力，拥有大量远古至强武学，这没有人怀疑。
<br/>　　若是有了这么多的武学，再集合整个苍穹世界的资源，以他们的天赋，几年之内，实力便能达到一个极致。
<br/>　　到时候，说不定真的有胜利的希望。
<br/>　　最主要的是，有苏莫这个天赋超越常理的天才，完成这个壮举，并不是没有可能。
<br/>　　而且，现如今的时代，随着苍穹世界的实力越来越弱，降临的试炼者，实力也是越来越弱，远不能和上古时代相比。
<br/>　　“好，我愿意加入苍穹神宫！”
<br/>　　沉默了良久之后，战乱重重的点了点头，他，心动了！
