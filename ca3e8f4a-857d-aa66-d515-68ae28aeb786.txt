　　没等叶尘开口，易山河当即高呼道：
<br/>　　“叶帅！邪魔外道的话不可信，大不了我们今日和他同归于尽，您尽管出手，不要管我们！”
<br/>　　唐威等人也反应了过来，也跟着道：
<br/>　　“是呀！尘哥，只要你能为我们报仇，我们也算死得其所了，你就算答应他条件，他也未必会放过我们的！”
<br/>　　“闭嘴！”
<br/>　　魔祖听到这话，直接一声暴喝，
<br/>　　噗！噗！噗！
<br/>　　众人顿时全都如遭雷劈，一个个鲜血狂喷，便再也说不出话来了。
<br/>　　以魔祖的实力，想要杀掉几个凡人，简直不要随意，
<br/>　　“桀桀！叶狂仙，你可要想要好了，机会我只给你一次，倘若你不同意，那本祖这就杀了他们！”
<br/>　　说着，魔祖虚手一抓，五人顿时再次纷纷发出一道惨叫。
<br/>　　一时间，叶尘顿时大为踌躇，
<br/>　　倘若答应魔祖的条件，那就等于，人为刀俎，我为鱼肉，沦为对方的阶下囚，恐怕这辈子都难以翻身。
<br/>　　但是倘若他不答应，眼前五人必死无疑，以魔祖的实力，想要杀掉他们，简直易如反掌，仅仅只需要一道念头即可。
<br/>　　就在叶尘无比纠结之时，却看到‘萧若曦’，忽然冲他露出一抹淡淡的微笑，低声喃喃说了一句，
<br/>　　“看来这一切都是天意……”
<br/>　　说完这话，‘萧若曦’的双手竟然又开始暗暗结印！
<br/>　　片刻后，
<br/>　　萧若曦的娇躯之上，忽然金芒大盛！
<br/>　　继而，
<br/>　　一道金色虚影，自萧若曦的身体中猛然飞出！
<br/>　　叶尘看到此景，似乎想也到了什么，顿时两眼一瞪，连忙高呼了一声，
<br/>　　“曦月！不要！”
<br/>　　原来，那金色虚影，正是曦月的残魂，在施展了某种逆天秘术之后，所呈现出的形态！
<br/>　　曦月的残魂，虽然经历月光宝鉴的时空轮回，实力大大削弱，可毕竟也还是堂堂渡劫期的大修士，一旦催动秘术，自然威力不俗。
<br/>　　只是，一旦施展这门秘术之后，曦月好不容易恢复的残魂，也会再次受到反噬，之前叶尘所做的一切努力，恐怕就要化为乌有了！
<br/>　　“这是——什么东西？”
<br/>　　连魔祖也吓了一跳，不由得微微一呆。
<br/>　　而就在他一愣神的功夫，那道金色虚影突然朝他一指，
<br/>　　“灵虚破！”
<br/>　　伴随着一道冰冷、淡漠的女子声音，在整个天空之上回荡。
<br/>　　轰隆！
<br/>　　一道犹如神话般的璀璨光芒，瞬间便击中了魔祖的眉心，
<br/>　　魔祖顿时浑身一颤，感觉自己的识海，犹如被万钧雷霆反复蹂躏了千万遍，大脑竟然瞬间空白！
<br/>　　经过片刻的宁静之后，
<br/>　　“彭！”
<br/>　　魔祖的整个脑袋，轰然爆炸开来！
<br/>　　一时间，周围所有魔界强者们，个个目瞪口呆，惊愕的说出话来。
<br/>　　任谁也没有想到，明明已经稳操胜券的魔祖大人，居然在转眼之间，被人打爆了脑袋！
<br/>　　而这个一招将魔祖灭杀之人，竟然是那群人质之中，最不起眼的一个凡人女子！
<br/>　　“魔祖大人……死了？”
<br/>　　这一切，发生的太过于诡异，太过匪夷所思！令人完全摸不着头脑，就像是在做梦一般。
<br/>　　不光魔界众强者懵了，就连易山河和李虎等人，也个个一脸懵逼，
<br/>　　“我去！这是……嫂子出手了吗？没想到嫂子的实力，竟然也这么厉害！”
<br/>　　惊愕过后，四人个个精神大震。
<br/>　　然而，他们又哪里知道，出手的哪里是萧若曦，而是萧若曦体内的曦月！
<br/>　　“曦月！”
<br/>　　叶尘也是足足愣了好一会，方才大吼一声，闪电般冲到那道，已经变得十分虚淡的金色虚影跟前，
<br/>　　“曦月！你怎么这么傻！你怎么这么傻！……”
<br/>　　别人不清楚，叶尘又岂会不知道，以曦月如今的残魂状态，强行施展‘灵虚破’，虽然足以灭杀了魔祖，可是他神魂必然大大受损，甚至将永远无望复原，恐怕用不了多久，就要魂飞魄散！
<br/>　　那道美丽的金色虚影，却冲着叶尘微微一笑，
<br/>　　“叶尘，你不必难过！当初我动用月光宝鉴，送你离开之时，就已经怀着必死的决心，如今多活了这么长时间，又有什么不满足的呢？”
<br/>　　“更何况，我本就不属于这个世界，一直被规则所排斥，等你重返修真界之后，一定记得去找八百年前的我，那个才是真正的我……”
<br/>　　说完这番话之后，曦月的残魂变得越发虚淡，声音越来越弱，最终缓缓闭上了双眸，似乎一阵风就能把她吹散！
<br/>　　“不！曦月！我不会让你死的！绝不！！”
<br/>　　叶尘大吼了一声，手掌一翻，吞天鼎便被召唤了出来，
<br/>　　“收！”
<br/>　　曦月的残魂，顿时便被叶尘吸入了吞天鼎之中。
<br/>　　“主人，这道残魂太过虚弱，即便是在我的鼎内空间之中，恐怕也支撑不了太久！”
<br/>　　吞天鼎的器灵奥维斯，开口提醒道。
<br/>　　这一点，叶尘自然比谁都清楚，
<br/>　　“为今之计，只有尽快进入修真界，说不定可以找到解救曦月的办法！”
<br/>　　一念及此，叶尘恨不得立即赶往珠峰脚下，由通天路进入修真界。
<br/>　　然而，
<br/>　　急火攻心之下，再加上之前强行透支身体，叶尘忽然感觉两眼一黑，
<br/>　　“噗！”
<br/>　　狂喷了一大口鲜血之后，叶尘身上传来一股极度的虚弱感，连同气息也开始急速下滑！
<br/>　　却是逆生百转的效果，终于达到了极限……
<br/>　　“这小子好像受了重伤，快！大家趁机杀了他！”
<br/>　　周围的魔界众强者，看到这一幕之后，顿时个个精神一震，纷纷大呼小叫着，开始对叶尘出手了！
<br/>　　“找死！”
<br/>　　本就心情差到极点的叶尘，顿时暴喝一声，又强行提起一口真元，直接杀入那些魔界强者群中。
<br/>　　嘭嘭嘭！
<br/>　　暴怒之下的叶尘，出手丝毫不留情，每出一拳一脚，便会有一名魔界强者被打爆！
<br/>　　“逃！快逃！”
<br/>　　众人看到叶尘依旧如此威猛，顿时又吓得纷纷四散逃遁。
<br/>　　“还想跑？晚了！”
<br/>　　叶尘将速度催动到极致，在空中留下道道残影，直到将所有魔界强者，全部灭杀，方才罢休。
<br/>　　“噗！”
<br/>　　杀光所有人之后，叶尘顿时再次喷出一口鲜血，继而一头栽落在地上，直接晕厥了过去。
