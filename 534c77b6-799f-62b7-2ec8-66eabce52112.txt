        &lt;h3&gt;第三卷第一百三十四章  夏颜的心思&lt;/h3&gt;
<br/>
<br/>        夏颜看着江俞轩急匆匆的离去，嘴角噙着一丝笑意。
<br/>
<br/>        上次同学聚会，并没有将陈子昂怎么样，相反还被陈子昂打脸了，虽然她知道陈子昂开了公司，可是她怎么都没有想到陈子昂早在几年前就已经是业界大佬了。
<br/>
<br/>        陈子昂这几年是有什么奇遇吗？冯睿在十几年前帮他们的时候并不知道是陈子昂，冯睿和唐红莲还有傅海生的爸爸是校友，冯睿比他们高了两级，关系也不是很亲密，但是唐红莲想干的事情绝对是要干成的。
<br/>
<br/>        这些年他们也偶有来往。
<br/>
<br/>        当夏颜拿着耀眼的钻石项链放在冯睿的面前，说着感谢冯睿的话时，冯睿的嘴巴张得像鸡蛋一样大。
<br/>
<br/>        “冯姨，不，应该叫冯姐姐。谢谢你当初帮了我，要不是您帮助我，我现在还不知道过着一种什么样的生活呢。”
<br/>
<br/>        “你说当初你顶替了陈子昂？”冯睿从震惊之中回过神来。
<br/>
<br/>        “是啊，要不是您，我现在的这个位置应该是陈子昂的吧！”夏颜微笑着将项链挂在冯睿的脖子上。
<br/>
<br/>        这个项链可是马总给她买的，花了不少钱呢。
<br/>
<br/>        冯睿什么话都说不出来，当初唐红莲找她想认识某部门的人的时候，但是她却不知道替换的人是谁，阴差阳错之中，她和陈家算是结下了仇。
<br/>
<br/>        她和陈子建已经离婚了，没有想到陈子建真是狠心，不愿意给她道歉还不理她，再加上母亲他们的拆迁款下来了，老母亲怕陈子建会分他们的拆迁款，鼓动着她和陈子建连面都没见，就离婚了。
<br/>
<br/>        当夏颜来找她的时候，看着价值不菲的项链，想着和陈家的纠葛，冯睿心里就像是被抹了辣椒水一样火辣辣的疼。
<br/>
<br/>        于是，冯睿能知道陈子昂多少情况就全部告诉了夏颜。
<br/>
<br/>        原来陈子昂是被人抛弃的，原来陈子昂的依仗就是江家大少----江俞轩。
<br/>
<br/>        她回来之后又调查了江俞轩的情况。
<br/>
<br/>        江家大少前几年很低调，也就是近几年才展露头角，可是一露面就拿到了西城的改造项目，震惊业内。
<br/>
<br/>        而且江家大少要结婚的对象是张家的张倩楠，而不是陈子昂，那么陈子昂和江家大少的关系就是隐秘的，如果这个料爆出去会带来多么大的震惊？
<br/>
<br/>        江家愿意将这个料爆出去吗？肯定是不愿意的。
<br/>
<br/>        豪门之间没有爱情，江家和张家及有可能是联姻。那么江家大少为了事业和家族也会舍弃陈子昂的。
<br/>
<br/>        陈子昂啊陈子昂，你将会再一次被人抛弃！
<br/>
<br/>        夏颜联系了江俞轩，果不其然，江俞轩同意和她合作，虽然她的语言里多了那么一丝丝的威胁。
<br/>
<br/>        此时，夏颜和冯睿坐在陈子悦那一家豪华的西餐厅里，桌上放着精致的甜点和牛排。陈子悦和闫欢在西城南门城墙脚下开了一家豪华的西餐厅。
<br/>
<br/>        随着经济水平的提高，人们对食物的追求也会越来越多样化，陈子悦和闫欢开了一家颇具档次的西餐厅，这个西餐厅的定位普通人家是消费不起的，一楼是一些高档品牌的化妆品，二楼是名牌皮具及手包。
<br/>
<br/>        三楼是西餐厅，四楼是名牌服装。
<br/>
<br/>        五楼是棋牌及休闲的地方。
<br/>
<br/>        夏颜陪着冯睿，在一楼买了一套雅诗兰黛的化妆品，在二楼买了一个lv的包送给了冯睿，然后俩个人准备在三楼吃完午饭再去四楼转转。
<br/>
<br/>        “冯姨，听说你和陆家张家关系都很好呢。”夏颜问冯睿，马总虽然也是大树，但是相对比起陆家和张家来说，那就是小树苗。
<br/>
<br/>        “还好吧，就是闺蜜的关系。”冯睿切着牛排，淡淡的说，和陈子建离婚了，以后去见弥青云和蔡少芬还能拿些什么东西？
<br/>
<br/>        之前的东西都被陈子建上缴了。冯睿怎么都想不通当初她怎么就看上了陈子建哪个憨憨？想来想去，她归结于陈子建长了一张霍霍人的脸。
<br/>
<br/>        “冯姨，你太厉害了，和她们都成了闺蜜。”夏颜不知道冯睿和陆、张两家的关系，满心满脸的羡慕，要是她能上搭她们任何一家都可以了。
<br/>
<br/>        “也没有什么厉不厉害的，她们之前在西城的时候都是听我的，我说什么就是什么，我们三个经常一起逛街、买东西，她们的爱好我都知道。”冯睿拿起餐巾纸擦了一下嘴。
<br/>
<br/>        “真心羡慕你有那么好的闺蜜，陈峰以后的工作都不用你担心了。”夏颜可是知道陈峰前一段时间调到了一个不错的职能部门。
<br/>
<br/>        “颜颜，你是有什么想法吗？”冯睿能在社会上混这么多年，她不是傻子，她是干什么吃的？对于夏颜的想法她也能理解的。
<br/>
<br/>        “我也很想认识你的那俩个好闺蜜呢。”夏颜笑着说。
<br/>
<br/>        “你们家背靠着马总这颗大树，想要什么没有啊？”冯睿有点不了解夏颜，现在的马总可是西城第一人物，而马总和唐红莲可是好朋友。
<br/>
<br/>        不过话说回来，夏颜就算是给她再高的工作，恐怕她也干不了，这一张脸和肚子里的货不怎么匹配啊。
<br/>
<br/>        “冯姨。”夏颜的脸一热：“马叔叔那人你是知道的，到了一定的位置怕也是靠不住的。”
<br/>
<br/>        冯睿低着头，蔡少芬虽然不怎么待见她，但是还是给她办了事情的，弥青云的作用就是牵线搭桥，后期陈峰的工作肯定还是会麻烦蔡少芬的，如今她没有了陈子建，手头上就没有了稀罕玩意。
<br/>
<br/>        夏颜虽然工资不高，可是有唐红莲，能给她送那么昂贵的项链，估计也不缺稀罕玩意，更不会却钱。
<br/>
<br/>        “也好，我改天去北京了和她们说一下你，等时机成熟了就带你去见见她们。”冯睿思考了一下开口。
<br/>
<br/>        “谢谢冯姨，如果她们有什么需要的或者是喜欢的你给我说一声。”夏颜喜笑颜开。
<br/>
<br/>        她夏颜其实不缺钱也不想要特别的权利，就是要看着陈子昂以后的下场，只要和陈子昂有一点点不好的关系她都想利用上。
<br/>
<br/>        最好是将陈子昂的孩子让张函的老婆知道，并且误会那孩子是张函的，那样生活是不是更加精彩？
<br/>
<br/>        她要看江俞轩会不会和她说的一样，不参与洛城的那个项目或者是高价弃标，如果江俞轩真的弃标了，那就不用动江俞轩了，对于江俞轩她还是抱着一层好感的。
<br/>
<br/>        如果要竞争那个标的话，那她也是可以在西城给他们制造一些困难，最起码马总还是会照顾她的情绪的。
