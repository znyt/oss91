　　继而，
<br/>　　等杨国强反应过来之后，直接一脚便把高建人踢翻在地，
<br/>　　“一派胡言！来人！把这个不知道死活的东西，给我丢进天江喂鱼！”
<br/>　　随着杨国强一声令下，门外顿时冲进来两名黑衣壮汉，眼看就要将高建人拿下。
<br/>　　然而就在这时，
<br/>　　叶尘的声音却突然响起，
<br/>　　“等等！让他把话给我说清楚！”
<br/>　　杨国强听到叶尘这话，脸色顿时煞白，连忙陪着笑脸道：
<br/>　　“叶仙人，这家伙自知犯下大错，就胡乱攀咬，他的话可当不得真！”
<br/>　　说完这话，杨国强当即又冲着那两名黑衣壮汉，使了一个眼色。
<br/>　　那两人都是杨国强的贴身保镖，武功修为自不必说，自然明白了自家老板的意识，当即伸手按在高建人的重要穴位，想要将他悄悄打晕，避免他继续说下去。
<br/>　　然而，
<br/>　　两人才刚要动手，忽然胸口纷纷一痛，似乎被一股无形的力量击中，登时便倒飞了出去。
<br/>　　“杨国强，我说了，让此人把话说清楚，怎么？你连我的话都敢不听？”
<br/>　　叶尘随手打飞了两人之后，瞥了一眼杨国强，冷冷开口道。
<br/>　　杨国强感受到叶尘身上的怒气，顿时吓得浑身一颤，险些没有瘫软在地上，连忙躬身道：
<br/>　　“不敢！叶仙人误会了！我只是……只是觉得此人胡说八道，他的话根本不足为凭！”
<br/>　　叶尘轻哼一声，淡淡道：
<br/>　　“是不是胡说八道，我自有决断，你给我退到一旁！”
<br/>　　“是是！”
<br/>　　杨国强不敢违抗，只好乖乖退到一边。
<br/>　　那高建人此时也反应过来了，杨国强方才分明就是想要杀他灭口！
<br/>　　一念及此，高建人当即连滚带爬，来到叶尘的跟前，磕头如捣蒜，道：
<br/>　　“叶仙人！我说！我全说！其实针对菲菲小姐的这一切，全都是杨总让我干的，此事真的与我无关啊！”
<br/>　　一旁的庄菲菲突然插口道：
<br/>　　“你说的杨总，是华艺天皇娱乐的执行总裁杨国威？”
<br/>　　高建人连忙不迭的点头，
<br/>　　“正是！菲菲小姐可还记得，之前杨总想让去陪公司的一位重要客户，但是被您给拒绝了，杨总之后虽然表面上没有说什么，但其实大为恼火，觉得你驳了他的面子，便让我想办法，好好调教调教你，否则以我区区一个总监，怎么敢明目张胆的和您为难？……”
<br/>　　随着高建人一五一十的将事情讲了出来，一旁的杨国强顿时脸色大变，有好几次想要打断高建人的话，可慑于叶尘的威严，愣是没敢开口。
<br/>　　叶尘听完之后，望向庄菲菲，
<br/>　　“他说的可是真的？”
<br/>　　庄菲菲眼眸中闪过一抹恨意，点了点头道：
<br/>　　“确有此事！当时我还纳闷，杨总向来好面子，为何会如此轻易放过此事？没想到！真是没想到啊！”
<br/>　　这时，
<br/>　　一旁的杨国强坐不住了，
<br/>　　毕竟杨国威，可是他的亲弟弟！
<br/>　　“叶仙人，此人一看就是奸诈之辈，他的话可不能轻易相信！”
<br/>　　叶尘瞥了一眼杨国强，嘴角露出一抹冷笑，
<br/>　　“这个简单！把那位杨总请过来，一问便知！”
<br/>　　“这个……”
<br/>　　杨国强顿时大为踌躇，
<br/>　　以他对自己这个弟弟的了解，此事十有八九应该是真的！
<br/>　　而且以他弟弟一贯骄横的脾气，若是不知道叶尘的身份，怕是要倒大霉。
<br/>　　就在杨国强纠结之时，叶尘指了指跪在地上的高建人，
<br/>　　“你来打电话，如果此事真的如今所说，我便饶你一条小命！”
<br/>　　高建人闻言，顿时连忙磕头道谢，
<br/>　　他算是看出来了，虽然对于叶尘的身份，他始终猜不透，但绝对是一个超级牛逼的存在，连杨国强在他的面前都大气不敢喘。
<br/>　　为了保住自己的性命，他也只好把杨国威推出来了。
<br/>　　很快，
<br/>　　电话拨通，高建人打开了手机免提，电话那头传来一道中年男子不耐烦的声音，竟然还夹杂着女子的喘息声，
<br/>　　“老子现在正忙着呢，有什么事？快说！”
<br/>　　高建人两眼微微一转，连忙谄笑道：
<br/>　　“杨总，好消息啊！庄菲菲已经答应所有的要求，您让我办的这件事，成了！”
<br/>　　电话那头，顿时传来杨国威充满欣喜的声音，
<br/>　　“此话当真？那小娘们真的答应了？哈哈哈！”
<br/>　　“庄菲菲这个小娘们！竟然敢跟老子摆谱，还以为她多有骨气，如今还不是乖乖服软了？”
<br/>　　“看着吧，要不了多久，这个不听话的小娘们，就会乖乖爬到老子的床上来！嘿嘿嘿！”
<br/>　　说到这里，电话那头传来一道肉体撞击的声音，以及一名女子的娇喘声~~
<br/>　　就算是傻子也能猜到，此时的杨国威，正在做着某种床上运动。
<br/>　　一旁的庄菲菲听到这话，顿时气得浑身娇躯发颤。
<br/>　　而杨国强则一脸惨然之色，心中暗道一声，
<br/>　　“完了！”
<br/>　　很快，
<br/>　　在叶尘的示意之下，高建人继续开口忽悠道：
<br/>　　“杨总，此事虽然已经谈成了，可是这庄小姐有个条件，必须要您亲自来，她才肯签约，您看……”
<br/>　　电话那头的杨国威顿时大为不耐，
<br/>　　“怎么这么麻烦？”
<br/>　　高建人又一番好说歹说，杨国威总算是答应前来了，
<br/>　　“好吧好吧，你把地址发给我，我这边忙完了，就立即过去！”
<br/>　　声音落下的同时，杨国威又猛然发出一道低吼，而那道女子的娇喘声，也变得急促起来。
<br/>　　又过了少许，
<br/>　　“啊~~”
<br/>　　随着一阵尖叫声过后，电话那头总算恢复了平静，显然已经完事。
<br/>　　庄菲菲三人早已经听的面红耳赤，其余众人也全都满脸尴尬之色。
<br/>　　……
<br/>　　大约又等了半个小时，
<br/>　　这位华艺天皇娱乐的执行总裁，总算是赶了过来。
<br/>　　“菲菲小姐，别来无恙啊！”
<br/>　　杨国威走进包厢，一眼便看到了国色天香的庄菲菲，两眼顿时直冒婬光。
<br/>　　杨国威正要走过去，忽然一旁冲出一人，直接快步上前，然后抬手就是一巴掌，
<br/>　　“握草尼玛……”
<br/>　　杨国威顿时大怒，不过等他看清楚眼前打他之人，瞬间就蔫了，
<br/>　　“大、大哥，您怎么在这儿？”
<br/>　　杨国强两眼一瞪，几乎要喷出火来，
<br/>　　“孽障！你竟然敢背着我，做出如此伤天害理之事，真是气死我了！还不赶紧给我跪下，向庄小姐磕头道歉！”
<br/>　　杨国威顿时懵了，捂着脸，不甘道：
<br/>　　“大哥，你在说什么呀？不错，是我要整这女人，可是她不过就是一个戏子而已，你至于这么生气吗？”
