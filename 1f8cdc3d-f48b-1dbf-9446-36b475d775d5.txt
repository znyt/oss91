        &lt;h3&gt;第三卷一百三十七章  奇怪的甸村&lt;/h3&gt;
<br/>
<br/>        陈子澜带着郑培，让司机开车送他们去甸村。
<br/>
<br/>        去甸城的路可以说是山路十八弯，曲折蜿蜒，他们到达甸村的时候已经是中午12点多了。
<br/>
<br/>        甸村其实是甸城下属一个县城的一个小镇子，不过这个镇子地处偏远，又处在深山老林之中，村子里的人口也不多，地形复杂，村子里的人都是住在上山，村子下面有一处山凹，山凹处修了门楼。
<br/>
<br/>        自从几年前这里发生了清洗事件之后，这里就已经将村口重新加固了，而且村口还安装了岗亭，来这个村子里的人本身不多，这里也不是旅游的地方，所以陈子澜他们来到这个地方自然就会引人注目。
<br/>
<br/>        这里的人们不像泉城农村的人，有了外地人到来会热情的招呼你喝茶或者还热情的邀请你去吃顿饭，这里的人们对外来的人充满警惕和冷漠。
<br/>
<br/>        当陈子澜他们到村口的时候，被挡了下来，不让他们的车进去，开车的师傅就在村口等他们。
<br/>
<br/>        陈子澜和郑培到村子里时，一帮面黄肌瘦的半大小孩围了过来。
<br/>
<br/>        陈子澜问：“你们都是这个村子里的人吗？”
<br/>
<br/>        几个小孩并不说话，只是冷漠的看着陈子澜他们。
<br/>
<br/>        他们俩个人走到哪里，孩子就跟到哪里，碰见一个大人，陈子澜想咨询一下当地的情况，问一下这里有没有吃晚饭的地方，可是对方好似听不懂她的话，依然是冷冰冰的看着她。
<br/>
<br/>        陈子澜心里疑虑更重，和郑培对视了一眼，又朝村子深处走去。
<br/>
<br/>        几个孩子一声不吭的跟着他们，那冷森的寒气让陈子澜和郑培心里发毛，但是他们还是想找一点有价值的地方，顺着蜿蜒的山路一直朝着半山走去，半山上才是一排排居民房。
<br/>
<br/>        可是很多家都是关门闭户，大中午的家里都没有人似的，偶有一两家是开着门，可是里面的老人或者的妇人看见他们背后的小孩，都挥挥手像赶鸭子一样，赶着他们“去、去、去。”
<br/>
<br/>        陈子澜和郑培转了一圈，什么情况都没有了解到，他们又沿着山路走了一圈，郑培似乎隐隐约约看到了群山后面还有房子，便指着远去说：
<br/>
<br/>        “陈总，你看，那边好像有房子，似乎是很不错的房子？”
<br/>
<br/>        陈子澜顺着郑培的指点朝远去看去，隐隐约约好像是有房子：“走，我们再去高的地方看看。”
<br/>
<br/>        俩个人合同那几个默默不说话的小孩一起走了大概一公里，到达一个小山顶，模糊的房子有些清晰起来。
<br/>
<br/>        “那边应该是另外一村子，这个村子和另外的一个村子相邻呢。”郑培看着远去，冬天的下午有些模糊。
<br/>
<br/>        陈子澜看着山路，越上去路越窄：“嗯，不知道能不能过去？”
<br/>
<br/>        “感觉那座房子还挺豪华的。”
<br/>
<br/>        “也许是有钱人在这里盖的别墅？”
<br/>
<br/>        “这两个村子相邻，虽然气候不错，但是来这里住是不是离城里太远了？而且你看看这个村子，也不像是有钱的人家呢。”郑培小声的分析。
<br/>
<br/>        “小朋友，你们知道那个房子吗？”陈子澜再一次问跟着他们的小朋友。
<br/>
<br/>        “没有，我们是孤儿，就在这个村子长大的。”这一次终于有个孩子回答他们的问题了。
<br/>
<br/>        “你们.......”陈子澜看着眼前的几个孩子有些心疼，难怪这些孩子都是面黄肌瘦，很冷漠。
<br/>
<br/>        “你们住在哪里呢？”郑培问其中一个大一点的孩子。
<br/>
<br/>        “诺。”孩子指了指村子里远一点的一个院子。
<br/>
<br/>        “我们可以去看看吗？”陈子澜问
<br/>
<br/>        “嗯。”几个孩子带着陈子澜俩个人走向那座院子。
<br/>
<br/>        推开门，院子里还有几个孩子，只不过这几个孩子比跟着陈子澜他们的孩子要好很多，脸上都很红润，而且要健康很多，但是每个人的眼里都很冷漠，看人的眼神就像是冬天的寒冰。
<br/>
<br/>        他们本来是坐着正在说着什么，但是看见陈子澜他们进来了，都站了起来，看着陈子澜他们充满了敌意。
<br/>
<br/>        “不好意思，小朋友，打扰你们了，我们只是路过，想过来看看有没有晚饭可以吃。”陈子澜抱歉的说。
<br/>
<br/>        “你们过来。”有一个大概1.60米身高的孩子指着跟着陈子澜的孩子说。
<br/>
<br/>        几个孩子走过去一排排的站在他的身后。
<br/>
<br/>        “我们这里没有吃的，你们去别的地方吧。”
<br/>
<br/>        陈子澜和郑培退出了院子。
<br/>
<br/>        “这个村子有些奇怪。”
<br/>
<br/>        “是的，那些孩子特别奇怪。”
<br/>
<br/>        “他们不像是普通的孤儿，而且最后的那几个孩子也不像是很孤苦的孩子，他们好像是这几个孩子的头目。”陈子澜心里的疑虑越来越重。
<br/>
<br/>        “我在他们的院子里闻到了味道。”郑培的声音有些激动。
<br/>
<br/>        “你确定？”陈子澜急切的问。
<br/>
<br/>        “我确定。”郑培点点头。
<br/>
<br/>        “如果说你闻到了味道，甸城的人为什么不来这里？”陈子澜问郑培也像是问她自己。
<br/>
<br/>        俩个人看了一眼刚刚离开的院子。
<br/>
<br/>        “走吧，回去再想办法。”他们俩个人什么事情都做不了。
<br/>
<br/>        下到村口，司机坐在车上已经睡着了。
<br/>
<br/>        陈子澜敲了敲车门。
<br/>
<br/>        “师傅，开门。”
<br/>
<br/>        打开车门俩个人坐进去，司机一句话也没有说，等着俩个人坐定，便发动汽车离开。
<br/>
<br/>        中午俩个人没有吃饭，又爬山，现在已经是下午六点多，俩个人多少有点疲惫，郑培便靠着眯着眼。
<br/>
<br/>        陈子澜看着远去的群山和越来越暗的天空，心里模模糊糊升起一丝不好的感觉。
<br/>
<br/>        这个村子充满了古怪。
<br/>
<br/>        她拿起手机，给在甸城的同事打了个电话，告诉他们她在返程的路上，大概晚上8点多能回去。
<br/>
<br/>        晚上十点多，陈子澜和郑培还没有回去，晚上十二点多还没有回去。
<br/>
<br/>        甸城的同事只好求助廖国华。
<br/>
<br/>        廖国华披着衣服过来：“怎么回事？”
<br/>
<br/>        “我们的俩个同事下午6点多打电话说在甸村朝回走着，到现在还没有回来。”
<br/>
<br/>        “我就告诉过你们，不要远离，这里冬天山路不好走。看看，出事情了吧？”廖国华虽然抱怨着，但是还是安排了几辆车去甸村方向去找他们。
<br/>
<br/>        沿路找着，给陈子澜他们打电话，电话响着却无人接听。
<br/>
<br/>        俩个同事心急如焚，他们心里莫名的感觉不安。
<br/>
<br/>        “你说他们会不会出事情？”一个同事担心的问。
<br/>
<br/>        “应该不会，应该是手机没有电了。”另外一个同事安慰着，但是说出的理由让他自己都不相信。
<br/>
<br/>        几辆车拉着警报沿路寻找，在一个拐弯处，发现有急刹车的痕迹，路旁明显的有翻车的痕迹。
<br/>
<br/>        大家停车，沿着山体寻找。
