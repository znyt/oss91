　　　　杜天易这忽如其来的手段不但将牛妖王弄的整个人不好，就连阳台上的涂小安跟张天师都甚是震惊。
<br/>　　牛妖王的铁棍，虽然不知道是什么材料建筑而成，但是绝对刚硬神威无比。
<br/>　　之前的剑修罗子杰，他手中的青色宝剑也算不凡，但还不是被铁棍一敲，敲成两半。
<br/>　　杜天易能将铁棍变成麻花绳，足够惊艳了。
<br/>　　“给我意识洞察”涂小安眼睛明亮起来，他终于等到杜天易使出本体能力了。
<br/>　　要说这样的三阶生灵，没点特殊能力，他是打死不相信。
<br/>　　“叮...杜天易，身份妖精管理局组长，等级30级，三阶生灵级别”
<br/>　　“觉醒能力，元素金属性：可掌控世间任何的金属物质”
<br/>　　涂小安一眼洞察之后，数据显化，顿时振奋起来。
<br/>　　金木水火土，五行属性，没想到这杜天易还真是一鸣惊人，既然觉醒的是五行之一的元素金属性。
<br/>　　这让涂小安还真的有点没想到，怪不得能成为妖精管理局的第一人。
<br/>　　其实杜天易的强，是毫无争议的，在一城之内绝对是横着走，无所顾忌，要不是涂小安让他看不出深浅，要不是牛妖王变成了他的战宠，重重因素冲击下，让杜天易不得不有所顾虑。
<br/>　　论战斗力，他可不逊色任何人，只是他不想把事情闹大而已。
<br/>　　谁知涂小安步步紧逼，他也深知双拳难敌四手，唯有退，才有最好的选择，不过如今既然他退不了，只能迎战。
<br/>　　“大牛小心了，对方是金属性元素掌控者”涂小安冲着下方愕然的牛妖王喊了一句。
<br/>　　“金属性？”牛妖王闻言大吃一惊，瞬间醍醐灌顶，怪不得对方能将自己的铁棍变成麻花绳似的。
<br/>　　恐怕天底下任何兵器在他的手中，都能跳个舞给你看。
<br/>　　杜天易脸色骤变，没想到自己不过略施小计，就暴露了自己的能力，目光朝着阳台看去，只见那位清秀少年浅笑吟吟的望着他。
<br/>　　看他的眼神很古怪，好像在看什么稀世珍宝，充满了贪婪。
<br/>　　“阁下，我杜某无意与你为敌，不管你是人，还是妖，都与我无关，我跟牛妖王更是多年合作关系，何必弄的生死相向”杜天易看着阳台上观望的涂小安，一脸的俨然。
<br/>　　涂小安只是看着他，并未回话。
<br/>　　杜天易继续道：“可是杜某曾经有得罪阁下的地方”
<br/>　　这年轻人莫名其妙的敌意，让杜天易很是不理解，要知道他不但是三阶生灵级别，还是妖精管理局的组长，没有天大的仇恨，得罪自己真的不明智。
<br/>　　只要自己不死，那么温城就无他容身之处，况且想他死，太难了。
<br/>　　“你没有得罪我的地方，相反，我现在有点喜欢上你了”涂小安靠在阳台上，笑眯眯的说。
<br/>　　杜天易脸色阴沉：“你什么意思”
<br/>　　喜欢？这两个字，让他心里一阵鸡皮疙瘩，阴寒一片。
<br/>　　这年轻人莫非有什么特殊的嗜好？
<br/>　　正当杜天易胡思乱想的时候，涂小安悠悠然的话传下。
<br/>　　“我想收你做战仆，你可愿意？”
<br/>　　“战仆？”杜天易闻言便一愣，旋即一张脸顿时扭曲狰狞起来，仿佛受了莫大的侮辱。
<br/>　　他是何许人也，妖精管理局的第一人，超凡脱俗的三阶生灵，无论是身份，还是的地位，都何等尊贵。
<br/>　　收他做战仆，国家第一人恐怕都说不出这样大言不惭的话。
<br/>　　泥人还有三分火，霎时间，杜天易身上的白色气焰沸腾到了顶点，从而转化成了金色，这是他的金属性异象显化。
<br/>　　寻根之间，一股窒息感笼罩在整个别墅，别墅在晃动，地面也在晃动，一道铁片破土而出，悬浮空中，别墅一面墙上，直接塌陷，有很多金属物质朝着杜天易飘忽而去。
<br/>　　他这是全力毫无顾虑的运转体内金元素属性能力。
<br/>　　“狂妄小儿，你在逼我，不知天高地厚，给劳资去死”
<br/>　　杜天易一声怒吼，眼中熊熊烈火，只见被他控制的铁棍直接朝着阳台砸去。
<br/>　　涂小安等人诧异在别墅的异样中，回神而来，直接一个个飞身离开。
<br/>　　轰...铁棍砸来，二楼阳台顷刻间不复存在。
<br/>　　涂小安落到别墅大院，顿时蹙眉，牛妖王的铁棍携带着一股惊人的力量射来。
<br/>　　如今，这根铁棍完全在杜天易的掌控之中，他是金属性掌控者，天下金属都是他的武器。
<br/>　　不得不说，这种能力非常的变态，变态到完全不逊色涂小安掌握的其他元素能力。
<br/>　　铁棍射来的速度快如闪电，几乎在涂小安刚一落地，就已经到了眼前。
<br/>　　“给我定”
<br/>　　涂小安顾不得闪躲，眼睛浑然一眯，空间凝固，这根铁棍直接骤停在了他的三尺之内，悬空而立。
<br/>　　擒贼先擒王，杜天易知道解决了这年轻人，自己才有逃离的胜算。
<br/>　　另外，他也想试一试这年轻人到底有什么本事，既然连他都看不出深浅。
<br/>　　铁棍凝固而不动，杜天易微微诧异，一张脸更加的扭曲，金属性力量全面控制铁棍的攻击。
<br/>　　铁棍光华流转，棍身发出金属铿锵声，朝着涂小安突进，三尺变成了二尺，不到半米的距离。
<br/>　　涂小安的意识力彻底投放在铁棍身上，后背冷汗已经溢出，这铁棍还能突进，这点很糟糕。
<br/>　　“给我去死”
<br/>　　杜天易一声爆喝，铁棍如脱缰的野马，涂小安的意识力控制不住，一道黑影射来。
<br/>　　说时迟那时快，涂小安眼瞳深缩，身子微微一侧，铁棍从他的脖子擦肩而过，直接插到了后方的院墙上。
<br/>　　轰...整面院墙塌陷。
<br/>　　这时，别墅院内，都是嗤嗤声响，鬼哭神嚎似的，无数金属物质从各个地方脱落，如同成千上百道光束同时切割人体血肉一般的可怖声响，朝着涂小安射去。
<br/>　　涂小安那里还敢在装逼，这杜天易既然比他想象的要强的多。
<br/>　　牛妖王不如他也，那么强，之前为什么那么怂？
<br/>　　涂小安整个人腾空而起，躲避无数射来的光束金属物。
<br/>　　可旋即，这些金属物失去目标，全部笼罩在了杜天易的身上，他整个人就像是一块大型磁铁，将别墅中任何金属物质全部吸来，吸到自己的身上。
