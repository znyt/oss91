　　苏羽连忙叫人将那名晕倒的服务员抬出去安顿，而涂小安不断的观察手中的这条金蚕蛊蛇，此蛇的毒牙牙齿尖锐而细长，一看，就知道是一种致命的毒蛇。
<br/>　　让它进入你的身体，它会如一个吸血鬼吸光你的一切，要让它咬一口，你会立刻中毒而亡。
<br/>　　不过涂小安发现这并不是一条成年的蛊蛇，还在发育期。
<br/>　　到底是谁将这条蛊蛇下在了余子童的身体内呢？
<br/>　　是白家的人吗。
<br/>　　涂小安目露思忖，很关键的一点是，余子童的猫并不在。
<br/>　　那等超等生物，谁看了能不觊觎呢。
<br/>　　这中间发生了什么事情，恐怕只有等余子童醒来了告诉他了。
<br/>　　涂小安一只手抓了金蚕蛊蛇，另一只手中出现了一颗蛇莓，然后塞入了余子童的口中，这东西入口即化。
<br/>　　要是什么都不做的等余子童醒，那不知道要的到什么时候了。
<br/>　　可一颗蛇莓被余子童吞进，他一点苏醒的迹象都没有，这家伙的身子现在太虚了，金蚕蛊蛇几乎已经将他的身体榨干。
<br/>　　涂小安也不敢再给他吃蛇莓，怕虚不受补，一下子身子承受不了，还是循循渐进吧。
<br/>　　“小安兄弟，这蛊蛇你准备怎么处理”苏羽心有余悸的看着他手的蛇问。
<br/>　　涂小安打趣道：“苏哥你这店不是野味馆吗，拿去给客人煲蛇羹吃”
<br/>　　苏羽一听，脸色都白了，连连摇头，如此歹毒的蛇，他可不敢拿去给客人吃，会吃出人命的。
<br/>　　这个钱，他可不敢赚。
<br/>　　“主人，这蛇不错，不如给老牛煲蛇羹吃吧”牛妖王此时一双铜铃大眼锁定到了金蚕蛊蛇上，非常的贪婪。
<br/>　　这头色牛到是有眼力。
<br/>　　这条蛊蛇不管如何阴邪，但有一点不得不承认，现在这条蛇经过了在余子童体内的蛰伏，吸食人的血肉，精元，已经非同小可，它将是一条价值非常高的蛇种。
<br/>　　无论它的蛇毒，蛇血，蛇肉，还是其他器官，都能作为珍贵的药用。
<br/>　　大补之物。
<br/>　　金蚕蛊蛇的来历，涂小安一时半解，他需要找个人问问才行。
<br/>　　无视了牛妖王的要求，涂小安拿出手机，对着金蚕蛊蛇拍了一张照片，然后给一个人发了过去。
<br/>　　很快对方的信息来了，很是惊叹道：“金蚕蛊？？？”
<br/>　　涂小安笑了，果然不愧是蛇博士，对于蛇的知识，无人能比。
<br/>　　“这种蛇，你何处得来”
<br/>　　“人的身体中！”涂小安简单明了。
<br/>　　对方的信息：“我们见一面，你在哪”
<br/>　　涂小安直接将自己的地点告诉了对方，所谓的蛇博士陈默，只要关于蛇，他都无比的热衷，更别说是这种金蚕蛊。
<br/>　　已经不是什么不世出的毒蛇，而是不应该存在的毒蛇。
<br/>　　他本人也对这种蛇很有兴趣，先了解了解这种蛇，在等余子童醒了，就知道这蛇出自何人之手了。
<br/>　　一个小时后，蛇博士被人推着轮椅来到了味鲜阁包厢中。
<br/>　　涂小安跟苏羽的身份介绍了一下，然后手中的金蚕蛊交到了陈默的手中，陈默抓着蛇一阵抚摸，确定是金蚕蛊之后，一阵唏嘘道：“没想到我有生之人还能看到这种蛇，这是本人最不想看到的一种蛇种了”
<br/>　　说话间，陈默的目光投放到了晕迷的余子童身上，他骨瘦如柴，生命气息很弱，不用说，就知道是被蛊蛇给残害的。
<br/>　　“看来博士对这种蛇还挺了解的”涂小安看着他问。
<br/>　　陈默道：“说不上了解，还是第一次见，只不过曾经在古籍上见过这种蛇而已”
<br/>　　“这种蛇是用非常阴邪之术创造出来，据说方法是从苗之一族的巫师手中传出来”
<br/>　　“一条金蚕蛊的制作过程是非常的麻烦跟繁琐，基本分为三个步骤，第一先将一百种品种高的毒蛇一起放在一个瓮缸中密封起来，让它们自相残杀，吃来吃去，吃到最后只剩下一条，此蛇的形态颜色会发生改变，形状像蚕”
<br/>　　“第二，在用多种毒虫，如蜈蚣、蜥蜴、蚯蚓、蛤蟆等十二种毒虫放在缸中饲养这条毒蛇，饲养整整一年的时间，让这条蛇变得剧毒无比，凶残无比”
<br/>　　“最关键是第三点，需要用大量的活人血让此蛇浸泡其中，而且必须是孩童的血，每日一换，经过七七十九天，等此蛇的皮肤似金黄，才算成功”
<br/>　　一条金蚕蛊诞生之后，要嘛被密封沉睡，要嘛就是用活人的身体做寄居饲养，直至它吸食光人的精华，人死后，它才会离开，寻找新的一具身体。
<br/>　　陈默慢慢的说出来，但是奇怪他看向金蚕蛊的眼神却是温柔的。
<br/>　　涂小安听了蹙眉道：“为什么要如此煞费苦心的创造一条金蚕蛊，它应该有别的大作用吧”
<br/>　　“那是自然，说了你可能不信，据说这种金蚕蛊是不死的，它的蛇血能让人容颜不老，每一日一口金蚕蛇血，你的容貌直至死去，也如当初”
<br/>　　此言一出，一旁的苏羽眼睛瞪大起来，能让人容颜不老，岂不成了灵丹妙药，这条蛇简直就是一座金矿呀，有非常大的商业价值。
<br/>　　他暗暗的后悔，刚才涂小安说给他，他却没敢要。
<br/>　　“还有呢”涂小安却很冷静的问。
<br/>　　陈默看了他一眼：“当初我抓你，跟吸血蛇，你应该知道，我就是想培养一条绝世蛇种出来，为了就是利用绝世蛇种的蛇毒来研究制作抗癌药物”
<br/>　　“其实真的不需要那么麻烦，因为金蚕蛊的毒就是世上最好的抗癌药物了”
<br/>　　苏羽又眼睛凸起，前半句话他没听懂，但后半句话听懂了，能治疗癌症，那可了不得。
<br/>　　“既然博士知道这点，为何...”涂小安疑惑，虽然制作金蚕蛊的方法有点阴邪，还要用活人的血，但也不是完全不能尝试。
<br/>　　毕竟这种蛇的价值太高了，有点牺牲也说的过去。
<br/>　　按照博士的话，蛇血可让人容颜不老，蛇毒可以抗癌，无论哪一条，都让人不可抗拒，绝对都是史诗级的突破，能名流千史，成为人类的伟人。
<br/>　　这个世界会彻头彻尾的改变，光容颜不老，这一点就值得一个人付出全部身价，在所不惜。
