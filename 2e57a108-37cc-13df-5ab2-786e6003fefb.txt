　　“两大好处？不知……是什么？”公孙轩辕心念一动，若有所思。
<br/>　　“在你入门后，吾便一直观察着你，你的悟性、资质、气运都很好，不过百年时间，便踏过别人的一生，但是终究是因为时间尚短，心性方面还有些欠缺，比不上那些修炼数万年的修士。”
<br/>　　“吾观你今后命途多舛，会遭遇强大的对手，度得过，则乘龙飞天，度不过，则万事皆休！为此，我便送你……一场梦！”
<br/>　　“一场梦？”公孙轩辕不解，不知道一场梦有什么好的。
<br/>　　不过，他还没来得及说些什么，便见元始一挥玉袖，梦幻般的色彩在他眼前闪过，自己便失去了只觉，躺倒在地……睡着了！
<br/>　　恍惚之间，当他再次醒来之时，他成了一个普通的书生，背上一篓书籍进京赶考。
<br/>　　在路上，经过一个荒庙，遇上了一个美貌如仙的女子，两人经过一场灾难，便爱上了对上，之后便展开一场泣人泪下的爱情。最终，书生被女子的父亲给害死了，女子闻言之后，在出嫁的路上，用剪刀结束了自己的生命。
<br/>　　第二世，他化身一个女子，倾国倾城，风姿绝艳，却一生孤苦无依，父母早亡，沦落风尘，好不容易嫁给一人爱她的人，却因为自身的绝世容貌，而导致惨死。在将仇人杀害的那一天，她也结束了自己的生命。
<br/>　　第三世，他成了一代将军，杀伐果断，一生征战无数，为自己的王朝立下赫赫战功，但因为自己功高盖主，最后被自己的大王给杀了。
<br/>　　第四世，他成了海边的一个渔民，一生靠打渔为生，最后，他活了六十三岁，头发花白，在出海打渔的过程中，葬身海中，被鱼吃了。
<br/>　　第五世，他成了一个士兵，一生都在战场上厮杀，最终战死沙场，被人洞穿胸膛。
<br/>　　第六世，他成了一国之主，可整天却只知道贪图享乐，贪恋美色，乱用佞臣，杀还忠良，最后他的一个个臣子们都跑到了敌国去了，反而带兵攻回了自己的王都，最后一刻，他后悔莫及，但一切都太晚了，一杯毒酒，结束了自己的生命。
<br/>　　地七世，他成了一个家奴……
<br/>　　第八世，他是一只妖怪……
<br/>　　第九世，他是一块石头……
<br/>　　第十世，他成了一个**……
<br/>　　第十一世……
<br/>　　……
<br/>　　一次次的轮回，或男或女，或草或木，或蝼蚁，或绝世高手，或奴隶，或国王，或……
<br/>　　世间万象，在一场堪比真实的大梦之中不断轮回着，灵魂的沉沦，精神的交锋，心灵的淬炼……流转而过。
<br/>　　一切看起来都是那么的真实，宛若亲生经历一般，让人感觉不出丝毫的虚假。
<br/>　　直到，第一百世……
<br/>　　第一百世，他出生一个叫有熊氏的弱小部落，他的名字叫公孙轩辕。
<br/>　　他出生几十天便能言，少时思绪敏捷，青年时敦厚能干，成年后聪明坚毅。
<br/>　　时蚩尤暴虐无道，兼并诸侯，当时天下共主神农已经衰落，各部落之间相互攻击，战乱不断，生灵涂炭。他毅然肩负起安定天下的责任，从姬水起兵，向着周围的部落进发，以弱小之势，不断征战，不断壮大，最后联合许多部落，联合炎帝，与蚩尤等部落大战。
<br/>　　但终究是因为实力悬殊的原因，败多胜少。一次次的落败，并没有使他失落、放弃，反而让他更进一步，激发出他内心的豪气，他坚忍不拔，大发生产，强盛国力。他占据大义，于是，更多的人前来投靠他，他的队伍渐渐扩大。
<br/>　　最后，双方战于逐鹿，双方战士英勇无畏，战斗十分惨烈。轩辕在大将风后，雨师、力牧、旱魃等人的辅助下，终擒蚩尤而诛之，诸侯尊为天子，取代炎帝，成为天下共主，因有德天下，故称之为黄帝。
<br/>　　后来，炎帝不满黄帝为天下共主，企图夺回失去的地位，终于起兵反抗。炎、黄二帝发生火并，决战于坂泉之野。经过三场恶战，黄帝得胜。自此，黄帝为天下共主的地位确立，号令天下，凡是不从的部落，皆以天子的身份前去讨伐。
<br/>　　黄帝在位期间，威震四方，扩疆域，兴杀伐，国势强盛，八方莫不拜服，四海皆低头。
<br/>　　晚年时期，黄帝寻仙问道，最后在昆仑山的一个小山头上，见到了一位广成仙人。他向他请教长生之道。
<br/>　　然而，广成天尊只回答了四句话，便消失不见了，那四句话为：
<br/>　　“如梦如露亦如电，似虚似实更似尘
<br/>　　大梦千秋百世情，堪破虚实长生仙！”
<br/>　　广成天尊是不见了，但黄帝却愣住了，那四句话犹如洪钟一般响彻在他心神深处，将他从无尽深渊之中拉起来。
<br/>　　顿时，一道道信息犹如潮水一般，向他涌来，冲击在他心神深处，让他的内心一阵触动。
<br/>　　“啊——”
<br/>　　黄帝一声大叫，他的头实在是痛得厉害，一下子便有无数信息冲了过来，任谁也无法承受住它们的侵袭。
<br/>　　百世信息一下子便回到了他身上，百世轮回，让他的内心受到巨大的冲击，犹如醍醐灌顶一般，一下子，他便想起了许多。
<br/>　　“公孙轩辕……黄帝……广成子……昆仑山……元始天尊……”
<br/>　　那一幕幕场景，在他心头浮现，就仿佛是经过了千万年的时间，由点点滴滴的模糊记忆，到逐渐清晰起来。
<br/>　　“原来，我便是公孙轩辕啊！”黄帝喃喃自语道。
<br/>　　“可是，我已经醒了，为何还没有恢复？难道……真的已经过去了千万年？”轩辕看着自己已经显得垂老的身影，有些疑惑道。
<br/>　　“不！我不相信！我要醒来！”
<br/>　　他大声喊道，可惜，却无济于事，他还是原样，一尘不变。
<br/>　　渐渐地，他沉默了，他开始思考自己的人生，他将前百世的经历一遍又一遍地温习，渐渐地，他开始恢复淡然，那双眼睛之中，透露着看透一切的淡然。
<br/>　　不知过了多久，当他再次醒来，他的心灵变得通透许多，变得更加透彻了。
<br/>　　百世轮回之中，感悟人生，哪怕是一棵草，一粒尘，都有自己的人生。百世人生百世情，百世感悟，在这一刻全部化为一道神秘的力量，流淌在他心间。
<br/>　　“原来……如此。”轩辕喃喃低语道。
<br/>　　“大梦谁先觉？平生我自知！”
<br/>　　突然，一道洪钟大吕般的声音响起，在山谷之间回荡不休，久久不绝。
<br/>　　一瞬间，轩辕便再次恢复到青年模样。
<br/>　　一场梦，便已经……醒了。
<br/>　　……
