　　　　山野集团内部。
<br/>　　蛇博士陈默的办公室走进了一位黑袍男子，宽大的袍子将男子的全身皮肤都遮挡的严严实实，哪怕是他的脸庞也笼罩在黑袍之下，显得非常神秘。
<br/>　　到有点像武侠小说中的杀手。
<br/>　　黑袍男子走路都是低着头，让人根本看不到他的样子。
<br/>　　“师父”
<br/>　　黑袍男子走进办公室，张口发出了极度沙哑的声音，让人听的头皮发麻，鸡皮疙瘩掉一地。
<br/>　　陈默坐在沙发上，看着走进来黑袍男子，微微吸了一口气，开口：“泽儿，近些天，你体内蜂毒没有在发作吧”
<br/>　　“只要“吃药”就不会发作”
<br/>　　他说话间，稍稍的抬了抬下巴，黑袍下，露出了一张面目全非的脸，简直就是十级毁容，比厉鬼还在吓人。
<br/>　　此人便是被涂小安用数千只蜜蜂同时蛰咬后，神奇般活下来的江行泽。
<br/>　　不得不说此人的命很硬，被数千只蜜蜂同时蛰咬，还能活下来，但虽然他捡回了一条小命，可他的全身，没有一块部位是完整的，坑坑洼洼，千疮百孔，体无完肤，非常的狰狞。
<br/>　　他只能用长长宽宽的黑袍加身，否则出去，就是装鬼吓人。
<br/>　　陈默的眉头紧蹙，自己的徒弟小命虽然被他救回来了，但却变成人不人鬼不鬼，性情更是大变，沉默寡言。
<br/>　　最让人揪心的是，他命是救回来了，但却身中蜂毒，毒入骨髓，随时随地都有暴毙的可能。
<br/>　　唯有一样东西才能压制住他体内的蜂毒，那么就是变异吸血蛇的血。
<br/>　　当初陈默就是利用吸血蛇的血将江行泽从鬼门关强行给拉回来，但他没想到，自己的徒弟蜂毒中的太深，每日都必须要喝吸血蛇的血才能压制毒性，渐渐形成了一种依赖。
<br/>　　一日不喝蛇血，就会毒发生不如死，随时没命。
<br/>　　可以说，他是彻底离不开吸血蛇了，一人一蛇，绑定在一起。
<br/>　　“泽儿，你这样下去也不是办法，师父联系了国外的一家先进医院，你到时候去治治，看看有没有办法彻底解了你体内的蜂毒”
<br/>　　黑袍男子闻言，却摇了摇头：“师父，我没事，不用去”
<br/>　　陈默站了起来，望着他：“你是人，人怎么可以每日已喝蛇血存活”
<br/>　　江行泽现在只要一日没有喝到吸血蛇的血，就会立刻死亡，绝对没有意外。
<br/>　　陈默不愿意自己的徒弟如此人不人鬼不鬼的活着，最为关键的是，他发现江行泽每日喝着吸血蛇的血，他无形之中跟吸血蛇产生了某种联系。
<br/>　　一人一蛇变的很亲近，隐约间，陈默都发现自己快控制不住吸血蛇了。
<br/>　　这点是最让他忌惮的。
<br/>　　他费尽心思将吸血蛇培养成绝世蛇种，这条蛇一定要完全在他的掌控之中。
<br/>　　“师父，你觉得我现在还是人吗，人怎么可能比厉鬼还要吓人”黑袍下的江行泽露出无比怨毒的眼神，心中隐藏着滔天的仇恨。
<br/>　　陈默开口：“泽儿，你变成现在这般样子，师父也要责任，没有保护好你，你先去国外那家医院治疗，如果能治好你体内的蜂毒，你还可以在去做全身性的整容手术，已现在的技术，将你改头换面完全不是问题”
<br/>　　“不去，我体内的蜂毒无所不在，不可能解的了，离开了吸血蛇，我就会死，难道师父你想我死吗”
<br/>　　江行泽直接顶撞过去，语气中毫无尊敬的意思。
<br/>　　“这件事情，师父不是在跟你商量，三天后，你就出国治病，到时候我会给你准备一部分的蛇血，让你不至于毒发”
<br/>　　江行泽面目全非的脸挂着一丝冷笑，却道：“师父，我要去吃药了”
<br/>　　吃药吃药，说的好听，他其实是要去喝吸血蛇的血了。
<br/>　　他像是喝血成瘾，一日不喝，自己都不快活。
<br/>　　成为绝世蛇种的吸血蛇，它的血是非常的珍贵，蕴含能量，每日喝着，江行泽发现自己的蜂毒不但不会发作，他还气力大增，轻松就能举起五百斤重的石头。
<br/>　　他的身体被吸血蛇的血不断的强化改造。
<br/>　　最关键的一点，吸血蛇开始听命于他，一人一蛇好像有血脉相连的感觉。
<br/>　　所以，江行泽是绝对不会出国的，因为他还要报仇，找一条蛇报仇。
<br/>　　旋即，江行泽转身就走，招呼都不打。
<br/>　　“泽儿，师父的话还没说完呢”陈默见此，隐隐不快。
<br/>　　黑袍身子一顿。
<br/>　　陈默压制心中的火气，道：“近日，我们山野集团有数位员工离奇消失，你知道怎么回事吗”
<br/>　　“不知道”
<br/>　　丢下三个字，他直接快步走出了办公室。
<br/>　　陈默眉宇深锁，深深的看着江行泽离开的背影，直到他消失，才不得已收回。
<br/>　　山野集体近日非常的不太平，有好几名员工离奇的消失了，好像人间蒸发，活不见人，死不见尸。
<br/>　　员工的家属不断来山野集团来闹事，希望给一个交代，为什么人来你们公司上班，却把人给上没了。
<br/>　　山野集团也找不到原因，为了不让名誉受损，只能用大量的金钱来安抚员工的家属。
<br/>　　员工的家属用金钱来打发，可山野集团内部却慢慢谣言四起，有人说，这栋大厦中隐藏着一头吃人的怪兽。
<br/>　　消失的员工就是被怪兽给吃了。
<br/>　　又有人说，其实这怪兽就是山野集团专门研发出来的，作为很多产品的应用。
<br/>　　总之，山野集团现在是人心惶惶，每日都有不少员工辞职。
<br/>　　“我的绝世蛇种，是用来救命的，不是用来害命的，就算你是我的徒弟，也不能坏了我的规矩”
<br/>　　蛇博士陈默安静的坐着，像是下了某种决定。
<br/>　　........................
<br/>　　山野集团秘密基地，一间开阔的独立空间内，光线十分昏暗，这里像是常年不见天日，连空气都是浑浊的，隐晦间还有一股潮湿和血的味道，散发这一种死亡的气息。
<br/>　　外人很难相信，温城第一上市集体内部，还隐藏着如此一间宛如牢笼的地方。
<br/>　　让人心生恐惧。
<br/>　　里面时不时还有一股低沉的嘶吼声发出，好像关押着什么洪水猛兽。
<br/>　　到是应了谣言中的吃人怪兽之说，
<br/>　　这个地方之隐秘，就算是山野集团的高层，也没几个人知道，仿佛成了最大的秘密。
