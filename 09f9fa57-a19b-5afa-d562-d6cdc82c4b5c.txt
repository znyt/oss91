　　第747章
<br/>　　裂痕这种东西，一旦产生，就不会轻易消失。陈梦与徐问天从茶摊离开时，二人不自觉拉开了一段距离。
<br/>　　陈梦虽然有些狂，但他并不傻。越想越觉得自己这位师父路数有些不对，谁知他会不会抽冷子给自己一斧头，用自己的人头，做他进身之阶。
<br/>　　徐问天停住了脚步，陈梦回头看了一眼，道：“师父，怎么不走了？”
<br/>　　徐问天窥见陈梦的手不自觉搁在身后，随时可以亮出斧头的姿势。他摇摇头，道：“徒儿，你我师徒缘尽于此，就此别过吧。”
<br/>　　陈梦一怔：“师父何出此言，我一身本事都是您老人家传授，还未来得及报答。”
<br/>　　“不要提报答的事情了，当初不是老爷搭救，我已经冻死在路边，这十年授艺，也算是还清救命之恩，从今日起，你我两不相欠。”
<br/>　　徐问天知道他和陈梦的隔阂已经产生，二人可以装作不知，但隔阂却不会消失。终有一日，问题会爆发出来，那便是不可挽救的地步。
<br/>　　及时分道扬镳的好，两不相欠，各奔江湖。
<br/>　　陈梦咬了咬牙，他心里何尝不知这一点。恼恨的是程大雷，三言两语，就断送了师徒情分。
<br/>　　“师父，弟子蒙您传授，有这一身武艺。只想着日后能报答师父，给您老人家养老送终。谁料想今日情分已尽，请受徒儿一拜，全弟子孝敬的心。”
<br/>　　陈梦在长街上屈膝跪地，这一幕引得路人侧目，可陈梦不管不顾，双手摁地，毕恭毕敬向徐问天行了一礼。
<br/>　　再抬起头来，街道上人来人往，已经不见徐问天的身影。
<br/>　　“连这一拜之礼也不愿受么？”
<br/>　　陈梦站起身，拍了拍膝上的灰尘，再抬起头，少年脸上已有几分惆怅。
<br/>　　徐问天离开之后，想办法寻到程大雷的踪迹，继续缀在程大雷身后。
<br/>　　他仍旧不死心，想要做到这一点也并不困难。因为徐问天发现，程大雷三人背后至少有七双眼睛，当然，算上徐问天之后，就有了八双眼睛。
<br/>　　这些眼睛盯着他的一举一动，当然，这些眼睛究竟属于谁，徐问天就看不穿了。
<br/>　　程大雷路过脂粉摊，没过多久，兜售脂粉的姑娘就被领走了。他打赏路边乞丐几文钱，不多时那乞丐就不见了。
<br/>　　李乐天，百里胜，或者是各路藩王？究竟谁才是这些眼睛背后的主人。
<br/>　　程大雷显然也意识到这一点，带着徐神机与刘发财一路闲逛。看上去真如外地人第一次来到长安，对一切都感到新鲜。
<br/>　　偶尔与路边的商贩闲聊几句，偶尔走进首饰铺，瞧一瞧从宫里传出的新花样。
<br/>　　徐问天不远不近的缀在他们身后，心里仍旧在琢磨着，如何借程大雷的势让自己出头。
<br/>　　忽然前面传来乱糟糟的声音，人群向前面涌去。程大雷是个爱凑热闹的人，刘发财和徐神机也不例外。见到三人涌向前去，徐问天快走几步，也凑了过去。
<br/>　　前面是城隍庙，在城隍庙前有一片空地。此刻人群中间围着两个人，都是魁梧身材，赤手空拳，却有一种剑拔弩张的气氛。
<br/>　　程大雷听周边人议论，方才知道是怎么会事。这两人，一个三角眼的汉子是撂地卖艺的，一个锦衣华服的中年男人，却不知是从哪里来的。
<br/>　　却原来先是那中年男人出言挑衅，说什么你这两下子也敢在长安城吃饭，那三角眼汉子气不过，二人先是言语相争，眼看就要动起手来。
<br/>　　长安城百姓都愿意瞧个热闹，不停在一旁欢呼起哄。程大雷被好几双眼睛跟着，也是有些无聊，也想借此打发一下时间。
<br/>　　华服男子和三角眼此刻已经动起手来，二人一板一眼，竟然颇有章法。这不得不令程大雷刮目相看，市井之中也有高人呐。
<br/>　　最后结果谁也没想到，竟是那卖艺的穷酸汉子击败了华服男子，重重一脚将他踹翻在地。
<br/>　　周围响起一片喝彩声，汉子双手一抱拳：“诸位乡亲父老，兄弟王涛，初到贵宝地，只为混口饭吃。并非有意伤人，只不过他一再相欺，还请各位有钱的捧个钱场，兄弟但凡能吃一口饱饭，也记得各位的恩情。”
<br/>　　周围诸人看了一场热闹，纷纷慷慨解囊，程大雷也施舍出二两银子。
<br/>　　人群渐渐散去，那名叫王涛的汉子也收拾包袱离开，场上也不见了华服男子。
<br/>　　徐问天揉了揉眼睛，忽然发现随着人群散去，自己眼前也看不见了程大雷。
<br/>　　不仅是他，人群中的几个探子也都跟丢了人，站在原地，四处寻找。
<br/>　　徐问天一连找出好几条街，也没发现程大雷。却见前方大柳树下，出现一伙人。
<br/>　　那华服男人和王涛赫然出现在其中，两人关系似乎还很亲密，丝毫没有刚才剑拔弩张的气氛。
<br/>　　街道的另一头，程大雷三人悄悄出现。刘发财转着眼珠，道：“大当家，我说什么来者，这两人肯定是一伙的，演一处戏给大家看。另外还有些喽啰混在人群中，瞧瞧，今天肯定有不少人丢了银子。”
<br/>　　刘发财本身就是以骗术行走江湖，对这些江湖上下九流的把戏比程大雷懂得更多。
<br/>　　“啧，还真是读万卷书，走万里路。”徐神机叹了一声，眼神中有种跃跃欲试：“大当家，咱们要来一场黑吃黑？”
<br/>　　“什么话。”程大雷白了他一眼：“本当家什么身份，马上就要封王的角色，岂能和几个小贼计较。”
<br/>　　声音刚落地，只见柳树背后走出几个人。
<br/>　　“呔，此山是我开，此树是我栽，你们在长安城吃饭，拜过我的码头没有？”
<br/>　　王涛一伙人面面相觑，他见对方只有三人，嘴角露出一抹冷笑。
<br/>　　“小子，摆谱摆错地方了吧，再不滚，小心爷爷花了你。”
<br/>　　程大雷没有路见不平的打算，谁料有人想这么做。他往前行了几步，在看清楚对方模样后，无可奈何的叹了口气。
<br/>　　怎么是他。
<br/>　　冒出来抱不平的非别，正是落凤城的逍遥王李行哉。
<br/>(本章完)
