　　各方对于苏莫的追查，紧锣密鼓，因为秦陨和紫霄的传闻，让得此事更加的扑朔迷离。
<br/>　　云龙城，府邸的密室之中，苏莫在静心的修炼。
<br/>　　他的修为，已达三品低级，吞噬法则，也已经达到了三阶初期。
<br/>　　不过，修为达到这个地步，即便他目前资源充足，也不能轻易再做突破。
<br/>　　不过，他参悟吞噬法则，在悟道神丹的辅助之下，进展极快。
<br/>　　时间匆匆，眨眼之间，三个月便过去了。
<br/>　　这一日，龙腾来到了府邸，敲响了密室的石门。
<br/>　　苏莫闻声，便结束了修炼，打开了密室的石门。
<br/>　　“苏莫，你看谁来了。”龙腾面带微笑，指了指身侧。
<br/>　　他的身边，一袭蓝衣的绫裳，亭亭玉立，绝世的容颜，优雅的身段，风华无双。
<br/>　　“你怎么知道我在这里？”苏莫见此，顿时惊讶的问道。
<br/>　　这么短的时间，绫裳居然找到了这里?
<br/>　　“我去了轮回神朝，也去了混沌神朝，最终才找到这里。”绫裳面色郑重的说道。她为了找到苏莫，可是耗费了不少的时间。
<br/>　　好在，她知道苏莫和龙族关系匪浅，也知道苏莫有龙腾这个至交，才找到了这里。
<br/>　　“苏莫，我来是有要事告诉你！”绫裳叹了口气，面色阴沉了下来。
<br/>　　“关于紫霄是不是？”苏莫眸中精光一闪，他能猜的道，绫裳来此，十有八九是因为紫霄。
<br/>　　“不错，紫霄说，若是五年之内，你不去找他，他就会伤害师尊，若是十年之内你不去找他，师尊会死！”绫裳沉声说道，面色凝重不已。
<br/>　　“紫霄！”苏莫闻言，忍不住握起了双拳，即便他早有心理准备，此刻也不禁心中愤怒。
<br/>　　“他对于终极神道和终极神剑，志在必得。”绫裳道。
<br/>　　苏莫闻言，深深的吸了口气，沉默不语。
<br/>　　他的计划已经实施了，若是成功，还是有希望救回芊寻月的。
<br/>　　至于紫霄，将来，定然不会让其好过。
<br/>　　“绫裳，我已经有了计划，你也不必担心！”苏莫他了口气说道。
<br/>　　“什么计划？”绫裳沉声问道，她实在想不明白，苏莫能有什么计划。
<br/>　　想要从紫霄的手中，将芊寻月救出来，这难度简直无法衡量。
<br/>　　除非，有高级神王出手帮助苏莫，但是，就算是有高级神王出手，也不一定能救回芊寻月。
<br/>　　“你还是别问了，静观事态的发展吧！“苏莫摇了摇头，倒不是他不信任绫裳，而是此时比较复杂，他自己都没有捋清楚，只能走一步看一步了。
<br/>　　绫裳闻言，沉默不语，她知晓苏莫心思缜密，既然有了计划，她也不再多问。
<br/>　　“需要我做什么？”沉吟了片刻，绫裳沉声问道。
<br/>　　“你回到终极神王殿之后，若是终极神王殿的高层询问，你一口咬定，我是紫霄的密门弟子便可。”苏莫道。
<br/>　　“这个消息，果然是你放出去的！”绫裳闻言恍然，这段时间，她也听闻了这个传闻。
<br/>　　“不错，我要让紫霄身败名裂，成为众矢之的。”苏莫眸中精光一闪，现在只是开始，他后续还有一系列的计划。
<br/>　　“我明白了！”绫裳点了点头，她也不傻，大概明白了苏莫的计划。
<br/>　　但是，这样好像还不足以对付紫霄，看来苏莫还有后续的计划。
<br/>　　“既然你已经有了计划，那我就先回去了！”
<br/>　　少倾，绫裳叹了口气，她离开终极神王殿，时间已经不短了，必须尽快返回。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即又道：“司空炎去了终极神王殿，你暗中照应一番！”
<br/>　　“好！”绫裳点头，她拥有宏青璇的记忆，自然知晓司空炎。
<br/>　　随后，绫裳没有过多逗留，离开了云龙城，返回终极神王殿。
<br/>　　龙腾和苏莫闲聊了两句，便也返回了龙都城之中。
<br/>　　苏莫继续修炼，日子过的无比平静，时间悄然飞逝，眨眼之间，便又过去了数月。
<br/>　　不过，苏莫虽然在修炼，但是，这数月之间，他通过分身，对于外界的局势，却是知晓的一清二楚。
<br/>　　这数月之中，轮回神朝发生了几件不大不小的事情。
<br/>　　先是天王宫的一名弟子，将地王宫的一名弟子打成重伤，后又有两名地王宫的弟子，杀了一名天王宫的弟子，最后，又有一群天王宫弟子和一群地王宫弟子火拼，造成不小的死伤。
<br/>　　这几件事情的发生，让得天王宫和地王宫之间，闹的很不愉快，两宫的长老人物纷纷震怒，都差点互相出手。
<br/>　　不过，好在两宫之间，都比较克制，虽然心中恼怒，但均是互相忍耐了下来。
<br/>　　而在终极剑域，也有大事发生，绫裳和紫霄回到终极神王殿之后，立刻被终极神王盘问。
<br/>　　一番盘问之下，紫霄根本没有办法证明，自己和秦陨无关。
<br/>　　各方调查，以及绫裳的证词，都证明了紫霄和秦陨有关系，秦陨极为可能是紫霄的密门弟子。
<br/>　　对此，终极神王大怒，检查了紫霄的储物空间和随身的空间宝器，看看是否有终极神剑的存在，以及是否有传闻中那位秦陨的爱人，不过，并没有看到。
<br/>　　此事，没有确凿的证据，紫霄也无法开脱。
<br/>　　最后，终极神王命紫霄，没有吩咐暂时不得离开终极神王殿一步。
<br/>　　这一切，让得天池各个界域的各大势力为之惊讶，因为，紫霄在此事之中，完全脱不开干系。
<br/>　　不过，对于此事，其它大势力，倒是没有什么行动。
<br/>　　毕竟，若是终极剑道和终极神剑在秦陨的身上，他们还能抢夺，若是在紫霄的身上，则没有办法抢夺了。
<br/>　　首先，紫霄是中级神王，势力强大；其次，终极剑道和终极神剑本就是终极神王殿之物，他们没有理由去抢夺。
<br/>　　“很好！”
<br/>　　密室之中，苏莫睁开了眼眸，眸中闪过犀利的精光。
<br/>　　一切都在预料之中，那么，他就能进行下一步的计划了。
<br/>　　随后，他立刻闭上了眼眸，心神沟通远在天池神朝的分身，进行下一步的计划。
<br/>　　五年的时间，足够他施展计划，将紫霄逼进深渊之中。
