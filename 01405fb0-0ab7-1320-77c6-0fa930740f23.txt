　　　　“我们被老鼠群给包围了”
<br/>　　苏羽突然感觉后脊梁发凉，一道吱吱的声音愈来愈近，一个回头，看到一只大老鼠在空中扑向了他，露出一排尖锐的牙齿。
<br/>　　“老板小心”
<br/>　　他身边的黑衣保镖一拳砸出，快如闪电，直接将这只大老鼠打飞了出去，印在了一面石墙上，直接成了一团肉泥。
<br/>　　他这一拳，不下一千公斤的力量。
<br/>　　“还真被你这个乌鸦嘴说中了”余子童看到那么多老鼠，脸色严肃了起来，大喊：“宝贝，快来护驾”
<br/>　　还在大快朵颐的黄金瞳听到余子童的叫唤，直接出现在余子童的面前，看着周围包围过来的老鼠，不但不惊，反而兴奋起来，浑身的金毛更为的炸立，有王者之威势。
<br/>　　“宝贝，去全部给我咬死，一只不留”
<br/>　　余子童发号施令，这些老鼠简直作死，有自己的黄金瞳在，还敢出来放肆。
<br/>　　“嗷...”
<br/>　　这只猫气势滔滔，冲着老鼠群一声怒吼，暴戾乖张，倏然地，老鼠群停滞，直接被震慑住了。
<br/>　　有一句成语是虎狼之势，但这只猫更为的牛逼，拥有老虎之威，狮子之势，往那里一杵，谁敢乱动。
<br/>　　而且猫对老鼠天生就具备一定的威慑力。
<br/>　　“苏老板吾怕，有我这只猫在，再多的老鼠也不过尔尔”
<br/>　　余子童看到黄金瞳震慑住了蠢蠢欲动的老鼠群，含笑的看着脸色有点苍白的苏羽。
<br/>　　苏羽忙道：“全靠余大师了”
<br/>　　“苏老板放心，今天之后，余某保证你的工地在无一只兴风作浪的老鼠”余子童双目一凝，这句话说的霸气侧漏。
<br/>　　现在虽然老鼠多大百来只，但问题还并不算大，主动权还在余子童的手中。
<br/>　　此时，只闻随着黄金瞳一声虎啸，像是猛虎出山似的冲进了老鼠群，那叫一个大杀四方，区区一只猫有万军丛中取上将首级之气势。
<br/>　　没有一只老鼠是这只猫的一招之敌，黄金瞳的猫爪锋利之极，见血封喉，一只只肥大的老鼠不是被它一口咬死，就是被它的猫爪被抓死。
<br/>　　短短几个呼吸，既然有十几只肥大的老鼠死在了这只凶煞的猫手中。
<br/>　　有时候，甚至让人会错愕的以为，这是一只迷你形的老虎。
<br/>　　涂小安盯着这只大杀四方的猫，眼中有神采，微微的看头，此猫真的不错，勇猛无敌。
<br/>　　如果这只猫步入二阶，成为猫精，就更加的厉害了。
<br/>　　譬如海棠是一只算是家养的猫，没有凶性，那么这只黄金瞳却充满凶性，比老虎还凶猛。
<br/>　　海棠活了一百六十岁还无法证道，但眼前这只猫，将来绝对会有证道的可能。
<br/>　　只要别半路夭折。
<br/>　　随着黄金瞳的一夫当关万夫莫开，反之，苏羽跟余子童的脸色却越发的难看了起来。
<br/>　　为什么呢。
<br/>　　因为工地上的老鼠数量既然越来越多，成百上千的增加起来，虽然后面出现的老鼠并没有上一批那么的肥大，但是架不住数量多。
<br/>　　一只猫在厉害，但也只有一只，双拳难敌四手。
<br/>　　另外大量新出现的老鼠如同蝗虫过境，朝着他们四人潮水般的涌来了，须根之间，工地的地面被染了色，整个空间变得杂乱无章，非常的混乱，都是老鼠的尖叫声。
<br/>　　整个工地似乎都要垮塌下来似的，一股窒息感侵袭过来，苏羽跟余子童撑着惊骇的眼神望向不知道数量的老鼠群朝着他们汹涌而来。
<br/>　　“老板，这工地中肯定藏着一只鼠精在背后指挥”
<br/>　　黑衣保镖还算冷静的说。
<br/>　　而余子童却喊道：“宝贝快来这里”
<br/>　　先不说背后真有鼠精指挥，就现在光对付这些老鼠群，都让人产生无力感。
<br/>　　都说人多力量大，任何动物，一旦数量达到一定的点，就起了灾难性的作用。
<br/>　　黄金瞳此时依然在大杀四方，但是它被上百只老鼠围着，一时脱不开身。
<br/>　　嗷...
<br/>　　只见黄金瞳身上迸发一股骇人的气浪，直接将上百只老鼠席卷震飞了出去，其画面简直如同烟花绽放。
<br/>　　这只猫凶猛异常，同时也非常的忠心，想回到余子童的面前，保护他。
<br/>　　可惜，刚震飞上百只老鼠，旋即，直接又来了一批，将它包围。
<br/>　　工地上的老鼠数量还在陆续增加。
<br/>　　“老板站在我后面去”
<br/>　　黑衣保镖怒刷存在感，终于到他发挥作用的时刻，一双铁拳在手，将飞扑而来的老鼠拳拳打飞出去，一双腿更是不断的横扫，非常的忙碌。
<br/>　　余子童此刻见黄金瞳不能过来，变戏法一般，手中不知何时已经拿上了一柄长剑。
<br/>　　他左右张望，不断的挥舞，一只只老鼠被劈成了两半，嘴巴还不闲着：“苏老板，有我在，没有老鼠能近你的身”
<br/>　　说着他还看了一眼涂小安：“你就自求多福吧”
<br/>　　涂小安淡淡一笑，耸了耸肩膀。
<br/>　　只见黑衣保镖跟余子童左右的护着苏羽，不让老鼠大军杀过来。
<br/>　　“小安兄弟，快来我这里”
<br/>　　苏羽见涂小安并没有随着他们退后，眼看很快要将被潮水般的老鼠群淹没。
<br/>　　“余大师，你去帮帮我的小安兄弟，别让他受伤，我不需要你保护”
<br/>　　苏羽虽然知道涂小安是个修武者，但并不知道他到底有什么厉害之处。
<br/>　　只所以找他，就是看在他手中有条觉醒蛇。
<br/>　　余子童一口拒绝：“他与余某没任何的关系，苏老板不是说他也是修武者吗，要是死在这些老鼠手中，也只能怪自己没用”
<br/>　　开玩笑，那小子都要被老鼠大军淹没了，自己去救，找死吗。
<br/>　　“那也不能让我的小安兄弟被老鼠给咬死啊”
<br/>　　苏羽急了，莫非自己叫他来帮忙，还害了他性命不成。
<br/>　　如今，现场的老鼠数量在短短的一段时间内，已经多的让人心生绝望，这才是真正的鼠灾呀，他们的灾难来临。
<br/>　　这般看来，之前工地死几个工人，是再过正常的一件事情了。
<br/>　　因为这里的老鼠太多了，而且还太凶了。
<br/>　　不禁让人怀疑一句成语，叫胆小如鼠，工地里的老鼠简直胆大如虎，一反常态。
