　　武英，是一名青年，一席黑衣，他的手上，并没有任何刀剑兵器，而是有着一双拳套。
<br/>　　在北邙圣域之中，武英还有这一个称呼，便是戮血拳皇。
<br/>　　戮血二字，便是代表着武英的性格，狠辣无比，面对任何有威胁的人，甚至是碰到一些天才，都会残忍扼杀，手中鲜血无数。
<br/>　　楚风眠的手中，虽然也是沾满了鲜血，但是他从来不杀无关之人，跟着武英，截然不同。
<br/>　　而后面的拳皇二字，便是代表着武英的拳法强大。
<br/>　　擂台上，武英跟楚风眠相对而立，看着楚风眠，武英的脸上露出了几分冷笑。
<br/>　　“小子，没想到我居然有着如此好运气，居然是让我碰到了你。”
<br/>　　楚风眠，可是他们圣党之中最大的仇敌，是夜天君的死敌，谁可以杀了楚风眠，都可以得到极大的好处。
<br/>　　如今武英看向楚风眠的目光，完全便是看向一只肥羊一样，只要是杀了楚风眠，足以是让他从夜天君哪里得到无数的好处。
<br/>　　至于楚风眠的实力？
<br/>　　楚风眠的实力表现出来的虽然惊人，但是在武英看来，却是不值一提。
<br/>　　楚风眠的实力，从来没有被武英放在眼中过。
<br/>　　毕竟武英可是北域百绝榜的第九位，能够在北域百绝榜上进入前十位的，都是实力足以达到生死境巅峰的强者。
<br/>　　“这楚风眠，居然是碰到了武英。”
<br/>　　下方的许多武者，看到这一场战斗，都是不由的关注了起来。
<br/>　　楚风眠本来是默默无闻的，不过接连的战斗，也将楚风眠的实力给展现了出来，可以说是这一届的黑马之一。
<br/>　　他的擂台战，足以是吸引无数武者围观的。
<br/>　　更何况关于楚风眠得到了北邙圣君的传闻，已经是开始在武斗大会之中传播出去，虽然大部分武者都没有相信，不过无风不起浪，起码楚风眠的确是得到了众圣堂中的传承才行。
<br/>　　之前的几战之中，楚风眠都是强势的将对手碾压击败，一招胜利。
<br/>　　所以在场的无数武者，也不清楚楚风眠到底是什么样的真实实力。
<br/>　　如今碰到武英，众人终于是激动起来，武英可是北域百绝榜前十的人物，实力强大无比，不管是谁面对他，都要动用全力。
<br/>　　这一战，定然是极为精彩。
<br/>　　只是几乎全部的武者，心中都看好武英。
<br/>　　毕竟武英，可是北域百绝榜上前十的人物，而且并非是突然崛起的新人，而是已经在北域百绝榜上带了十年的人物了。
<br/>　　从下方一直是爬到如今的前十顶端，武英的实力，毋庸置疑。
<br/>　　楚风眠之前表现出来的实力，的确是极为可怕，但是却也只是轰杀了古傲寒，罗狞。
<br/>　　这二人在北域百绝榜上，只是中游水平，如今让武英出手，也足以是一招将他们轰杀。
<br/>　　除此之外，楚风眠根本没有太过耀眼的成绩。
<br/>　　众人无比激动，大多也是想要看看楚风眠这批“黑马”的真正实力，而至于胜负，却是没有多少人关系的。
<br/>　　因为在大部分的人看来，这一战，是楚风眠必败无疑的，没有任何的悬念。
<br/>　　下方真正支持楚风眠的，也只有寒月漓一人，只有寒月漓清楚，以楚风眠的实力，击败这武英，没有任何的问题。
<br/>　　楚风眠真正的对手，是夜天君，风无心等人。
<br/>　　这武英，还不算。
<br/>　　听着众人的议论声，武英的脸色也是无比得意。
<br/>　　单单是从气势上，他便是已经压倒了楚风眠。
<br/>　　“小子，本来念在你我都是同门弟子，我还是想要放你一条活路的，只是可惜，你不知天高低吼，居然敢来得罪圣主。”
<br/>　　武英看着楚风眠，脸上露出几分阴冷，轻蔑的开口道。
<br/>　　“你是自己来找死的，可怨不得我，你现在自杀，还可以留你一个全尸，不然的话我怕我一会动用力量太大，直接将你的身躯给轰成炸！”
<br/>　　“是吗？这句话应该是我说才对，你还有什么话，快点说完吧，不然的话你也要去陪那古傲寒，罗狞他们了。”
<br/>　　楚风眠轻描淡写的开口道。
<br/>　　面对着武英，他的脸色轻松无比。
<br/>　　“找死！”
<br/>　　武英的脸色，瞬间狰狞起来，冷喝道。
<br/>　　“死到临头，还敢嘴硬，一个得到些奇遇的新人小家伙，还真以为足以跟我们圣党抗衡？招惹到了我们圣党，今日便是你的死期！什么所谓的传承？在真正的实力面前，什么都不是！死吧！”
<br/>　　武英长啸一声，一身武道，便是完全的展现了出来。
<br/>　　这武英所修的，便是纯正的武道，纯正的力量，这种纯正的武道，修炼起来，没有任何的捷径可言。
<br/>　　但是一旦修炼成，则也是几乎没有弱点，除非是以更加强大的力量击败他，不然任何手段，都无法击败武英。
<br/>　　“武逆天地！”
<br/>　　武英浑身武道，凝聚在拳锋之上，这一拳猛然冲着楚风眠轰击过来。
<br/>　　这一拳，看似平常。
<br/>　　但是其中蕴含的武道，却是玄妙无比，一拳打出，足有千万种变化。
<br/>　　不管是楚风眠如何应对，都可以用最为玄妙的一招，轰击楚风眠。
<br/>　　这武英的武道，已经是达到了千变万化的地步，他打出的一拳，都不是寻常的拳风了，足以变化成种种力量，这只有在武道造诣上极高的强者，才能做到这一步。
<br/>　　不过这一拳的威力，却是全部都在纯粹的力量上，并没有任何的变化。
<br/>　　显然这以前，是武英在试探，试探这楚风眠的实力，到底是到达了哪一步。
<br/>　　“以力相撞？好？”
<br/>　　面对这武英的一拳，楚风眠的脸色轻松无比。
<br/>　　比起力量，他可是不怕任何人，拥有战龙血脉，凝聚远古战龙身的楚风眠，简直就是一条远古战龙。
<br/>　　虽然以他现在的实力，只能算是一条幼龙，但是一尊武者，敢来跟远古战龙来比拼力量，那不是在自寻死路？
<br/>　　“战！”
<br/>　　楚风眠长啸一声，远古战龙血脉被全部催动起来，楚风眠右手之上，也是凝聚无数灵力。
<br/>　　猛然之间，同样也是一掌，直接便是轰击了过去。
