        &lt;h3&gt;第四卷第十三章  霸道总裁上线&lt;/h3&gt;
<br/>
<br/>        送走陆宇霆夫妇之后，寒伯安和李长卿迎来了第二个客人——沧岚雪。
<br/>
<br/>        寒伯安坐在自己豪华的办公室里，李长卿站在他的身后。
<br/>
<br/>        跟在沧岚雪的身后的还有苓希。
<br/>
<br/>        苓希已经对寒伯安没有任何想法了，他们苓家没有必要再绑着寒家，她也没有必要赔上自己的幸福，看到了寒伯安的真面目之后，她突然就释然了，那几年贴着寒伯安，寒伯安对她爱理不理冷冰冰的，原来不是自己不够好，而是寒伯安另有所好。
<br/>
<br/>        这一次能够陪着沧岚雪一起来，有好几个原因，第一：寒家和苓家是亲戚；第二：陆宇霆夫妇已经和寒伯安的关系很好了，第三：寒家已经和谷家签订了合同，他们几家已经是一条线上的人了。
<br/>
<br/>        话说沧岚雪回到甸城之后，见到了谷老大，汇报了在美洲展开的工作情况，并且拿出了寒家和他们签订的合同。
<br/>
<br/>        “你这合同只有李长卿的签名，也没有签章，可以吗？”
<br/>
<br/>        “这个李长卿是寒伯安的助手，代表的也是他们的公司，他们在美洲的时候是没有带着公章，我们的业务大多是从国内往出走，在国内签章更合适。”
<br/>
<br/>        “你确定？”
<br/>
<br/>        “确定，寒伯安让我回来让你签字盖章之后直接拿到上海去盖章就可以了，另外，那个李长卿需要一点好处。”
<br/>
<br/>        “要多少？”
<br/>
<br/>        “他说我们这个事情要担风险而且是长久的合作，最起码得这个数。”沧岚雪伸出了四个手指头。
<br/>
<br/>        “去找财务预支吧！”
<br/>
<br/>        沧岚雪预支了款项，但是他没有打给李长卿，而是装进了他的一个小妹妹的腰包，当然，他还是会给李长卿一些的，不过不是现在，而是签章之后再给。
<br/>
<br/>        “苓希，你和沧总认识吗？”李长卿看着苓希明知故问。
<br/>
<br/>        “我曾经在沧总公司工作过。”
<br/>
<br/>        “曾经？不是现在？”
<br/>
<br/>        “现在也在，不过现在我代表的是苓家和他们有合作，以后他们在上海的业务将会由我来和你们对接。”苓希微微抬着下巴，看着寒伯安和李长卿。
<br/>
<br/>        “哈哈哈，苓家这是麻雀变凤凰，攀上高枝啦？”李长卿毒舌的说。
<br/>
<br/>        “彼此彼此，寒家不是也已经签了合同吗？”苓希想也不想的怼李长卿。
<br/>
<br/>        “我们寒家本来就是高枝，你们上巴着我们签合同，这个合同还没有签呢，就趾高气扬起来了？”寒伯安的声音像冬天的寒冰，让苓希和沧岚雪感觉脊背一凉。
<br/>
<br/>        当宫阳将吴润竹的话带给寒伯安的时候，寒伯安就明白了沧岚雪后面的人是谁了。
<br/>
<br/>        寒伯安让他的弟弟寒冰玉秘密的调查了一番，吴家有过卑劣的手段，培养过很多自己的人却没有和甸城的人有过来往，那么前一次的事件也就不是吴家所为，相反，他寒伯安被人利用了。
<br/>
<br/>        在几大商界大佬里面，虽然王家和陈家已经没落了，但是其身后的残余势力还在，他们有可能联合起来，利用他寒家，看起来陈家和王家是吴家的左膀右臂，但在错综复杂的环境中，吴家进行自保，没有伸出援手，也有可能对吴家心生怨恨，投靠别的家族也未尝不可。
<br/>
<br/>        既然知道了背后的人是谁，寒伯安也就不愿意在虚与委蛇了。
<br/>
<br/>        他寒伯安从来都不是愿意违背自己良心的人，陆家和谷家怎么牵扯和他没有关系，他们之间有什么猫腻也和他没有关系，但，想将他寒伯安拖下水是万万不能的。
<br/>
<br/>        他们的这个行业除非是有正轨的官方文件能动了他们，一般的私人是不能动他们的，寒家屹立在行业的顶端不是说全靠个人私欲，他们靠的是一个责任心，对国家、对企业、对个人的一颗责任心。
<br/>
<br/>        谷家和陆家想靠他们的暗箱操作来对付他实在是太不了解他们了。
<br/>
<br/>        苓家自以为攀上陆家和谷家就麻雀变凤凰，真是太可笑，就怕最后是怎么死的都不知道！
<br/>
<br/>        “寒总，我们是合作关系，就不分彼此了吧。”沧岚雪尴尬的笑着。
<br/>
<br/>        “我们有合作吗？长卿，吩咐人下去，在今天之内收回苓家所有的业务！”寒伯安冷冰冰的说。
<br/>
<br/>        “是，沧总，你先坐，苓总，你也先坐一会，等我去安排一下工作，然后回来和你们签合同。”李长卿笑眯眯的对沧岚雪和苓希说。
<br/>
<br/>        “寒伯安？你什么意思？”苓希睁大了眼睛。
<br/>
<br/>        “字面上的意思！我让你看看到底谁才是高枝！”
<br/>
<br/>        李长卿转身去安排工作去了。
<br/>
<br/>        “你当真？”苓希颤抖着声音，如果寒伯安收回寒家在苓家的业务，那么苓家就会被打回原形，他们攀上谷家和陆家也会失去作用。
<br/>
<br/>        “沧总，我将苓家所有的业务收回来，抽调一个部门配合你工作你看可行？”寒伯安并没有理会苓希。
<br/>
<br/>        “寒总公司内部事务，我们不会干涉的。”沧岚雪听说寒伯安将苓家的业务全部收会和他们合作，这对他们有利而无害，毕竟会少了苓家的这一层关系，也少了一下费用，更安全更可靠。
<br/>
<br/>        “那沧总就等一会，等长卿过来了咱们在谈合作的事情。”寒伯安客气的说。
<br/>
<br/>        可是苓希坐不住了，她站起来就走。
<br/>
<br/>        走到门口，被寒伯安的保镖拦下来了：“很抱歉，没有寒总的命令谁也不能离开这里！”
<br/>
<br/>        “寒伯安！”苓希大喊。
<br/>
<br/>        “别大呼小叫的，你要知道丑小鸭永远是丑小鸭，不可能是天鹅，就算偶尔充当一次天鹅，因为心术不正，飞不起来的。”
<br/>
<br/>        “我要给姑父说。”
<br/>
<br/>        “哈，你忘记了你姑父是我爸爸？你觉得儿子重要还是你们苓家重要？”寒伯安拿起一支雪茄，放在鼻子底下嗅了嗅，点燃，抽了一口，吐出了圆圆的烟圈。
<br/>
<br/>        办公室死一片的寂静，沧岚雪重新审视着寒伯安，现在的寒伯安好像不是原来的那个寒伯安啊。
<br/>
<br/>        当李长卿从外面走回来的时候，寒伯安不耐烦的说：“两个小时，你的速度是不是太慢了！”
<br/>
<br/>        “是有点慢，不过已经全部收回！刚刚召开了新闻发布会，苓家之前所做的业务和我们没有任何关系，从今天起，苓家和寒家也无瓜葛！你看看新闻？”
<br/>
<br/>        “不看了！沧总和苓总等了你好久，你来完成你的工作吧！”寒伯安站起来看也不看沧岚雪和苓希就出门了。
<br/>
<br/>        “寒总！”沧岚雪有些后悔带着苓希过来了，没有想到寒伯安竟然真的就收回来苓家手中的事务。
<br/>
<br/>        寒伯安头也不回的走了。
