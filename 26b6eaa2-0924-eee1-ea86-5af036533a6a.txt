　　　　涂小安怎么总是感觉自己在被这个丫头占便宜，这一次二次三次的。
<br/>　　长此以往，自己来温城光帮她档桃花，都挺忙的了。
<br/>　　心里暗暗决心，这是最后一次，实在不行，让这丫头在众多桃花中，随便选一个。
<br/>　　“你就是照儿的男朋友？”
<br/>　　眼前一个胖乎乎的人冲着涂小安发狠，努力的装出一副凶狠的样子。
<br/>　　不过说实话，这胖子白白净净，体重二百以上，可看上去还挺可爱的。
<br/>　　属于别人家的大胖儿子。
<br/>　　“这个来头很大？”涂小安见胖子身后四五个保镖，轻声的问。
<br/>　　李清照偷偷的说：“山野集团董事长的儿子，你说的”
<br/>　　山野集团可谓无人不知，涂小安之前还阴差阳错的成为了山野集团实验室的“小白鼠”。
<br/>　　但现在他跟蛇博士陈默已经和解，他又有自己父亲故友的身份，算是他的长辈了。
<br/>　　来到温城，涂小安还寻思的要不要去山野集团走一趟，看看蛇博士。
<br/>　　他对温城熟悉，自己也可以在向他打听打听一些隐晦的事情。
<br/>　　还没去山野集团，就碰到了山野集团的公子哥，涂小安笑了起来，有点意思。
<br/>　　“喂，问你话呢，不管你是不是照儿的男朋友，给你一百万，你离开”
<br/>　　什么叫富二代，什么叫财大气粗，这便是。
<br/>　　“我不是她男朋友”
<br/>　　涂小安淡淡的开口，李清照立刻瞪了他一眼，这不是拆台吗。
<br/>　　东方袁一听，乐呵呵起来，看来被自己吓唬到了，说出了实话。
<br/>　　李清照有没有男朋友他怎么会不知道。
<br/>　　“我是她哥”
<br/>　　涂小安眼眸划过一丝玩味，对着胖子说：“我妹妹不喜欢胖子，你减完肥在来追吧”
<br/>　　“呦，原来是大舅哥呀”东方袁的态度立马好了起来，但一听减肥，蹙眉道：“那我该减多少呀”
<br/>　　“减一半吧”
<br/>　　“什么”东方袁一个二百多斤的胖子蹦跳了起来，喊道：“你这不是要我的命吗”
<br/>　　涂小安憋着笑的摇了摇头：“看来你也没有很喜欢我妹妹，这点事情都办不到，妹妹，我们走”
<br/>　　说着，涂小安拉着李清照转身就走。
<br/>　　李清照走着，嘟着嘴：“我什么时候成为了你的妹妹”
<br/>　　“不然你还真想当我的女朋友呀”涂小安道。
<br/>　　“臭美”
<br/>　　两人走出了十几步，胖子东方袁抖动着浑身肥肉跟了上来。
<br/>　　“大舅哥，我请你跟照儿吃饭吧，让我尽一下地主之谊”
<br/>　　李清照翻了一个白眼：“大你个头，乱叫什么，我们不吃”
<br/>　　“现在都是中午饭时间了，随便吃点呗”东方袁眼睛眯成了一条缝，看着涂小安。
<br/>　　涂小安也摇了摇头：“吃饭就免了，等你减完肥在找我吃饭吧”
<br/>　　东方袁愕然的愣在原地，他这是在歧视自己吗，胖点怎么了。
<br/>　　他胖他有钱呀。
<br/>　　等涂小安跟李清照远远的消失在校门口的时候，李清照诧异道：“奇怪，这东方袁今天怎么那么好打发呀”
<br/>　　涂小安看着她：“这你就没经验了吧，说什么我是你男朋友，这很容易引发一场不必要的纠纷”
<br/>　　“我现在说是你哥，对方不就安分下来了吗”
<br/>　　“哦...”
<br/>　　李清照恍然大悟，给涂小安点了一个赞，这家属的身份比男朋友好使。
<br/>　　“其实我说另外一个身份或许会更好使”
<br/>　　“什么身份”
<br/>　　“你爸”
<br/>　　“涂小安，你又占我便宜”
<br/>　　身后传来李清照气急败坏的声音。
<br/>　　一个小时后，涂小安跟李清照分开了，他想了想，还是约蛇博士出来见一面。
<br/>　　但涂小安并没有去山野集团，而是跟博士约在山野集团对面的一家咖啡馆会面。
<br/>　　值得一说的是，温城大学跟山野集团不过隔了两条大街而已，都是在最中心的繁华地段。
<br/>　　已经坐在咖啡馆中的涂小安，选一个开窗的位置，很快，他就看着一个助手推着蛇博士而来。
<br/>　　但助手没有跟进来，涂小安起身过去推着蛇博士来到了自己的位置上。
<br/>　　只见蛇博士陈默复杂的看着眼前这位清秀少年，谁能想到这位看着年纪轻轻的小伙子，是一条无比巨大的巨蟒所化。
<br/>　　“没想到你真的来了，而且比我想的还要快”
<br/>　　那天两人在路灯下交谈后，也就过去了半个来月。
<br/>　　“我昨天就到了，正如博士说的，我继续留在白镇，恐怕会引来不少修武者，我到不是怕，就是觉得白镇已经不适合我在待了”
<br/>　　涂小安想要证道，在白镇升级已然太慢，在这个大都市，有修武者，有不少觉醒类的精怪，甚至是三阶生灵（妖怪），这里才有意思。
<br/>　　陈默一听他昨天就来了，突然皱眉起来：“昨日温城大学忽然群蛇所至，莫非跟你有关系”
<br/>　　在他的眼中，涂小安是一条蛇王所化，也只有他有这种手段，号令群蛇，莫敢不从。
<br/>　　“我昨天确实在温城大学，但群蛇跟我没关系”
<br/>　　涂小安实话实说：“群蛇来到温城校园，是因为一只猫精”
<br/>　　“猫精？”蛇博士疑惑了起来，猫精跟群蛇有什么关系。
<br/>　　涂小安笑了笑，他知道自己要是不说出前因后果，任由蛇博士脑洞大开，恐怕也想不到。
<br/>　　“猫精从白家的手中夺走了一颗蛇王证道后留下的蛇王石，它将蛇王石吞入了肚子，于是，猫精所到之处，便是群蛇所致”
<br/>　　蛇博士惊奇：“还有这等事情，那后来呢”
<br/>　　“猫精想利用蛇王石上的力量证道，可惜失败了，而这事情刚好被闲来无事的我碰到，我就将猫精说服，将蛇王石从它体内弄了出来”
<br/>　　“蛇王石在哪里，给我看看”
<br/>　　证道蛇王留下的蛇王石，对于陈默这种蛇类研究者来说，简直是大宝贝。
<br/>　　涂小安指了指自己的左眼。
<br/>　　蛇博士开始不明白，但观察了一会涂小安的左眼跟右眼，发现两只眼睛有着明显的区别。
<br/>　　那只左眼有一股强大的威势，连他这种蛇类掌控者，都有胆战心惊的感觉。
<br/>　　“莫非所谓的蛇王石，就是这只左眼”
<br/>　　蛇博士何等聪明，立刻看明白了。
<br/>　　“没错，这块馅饼被我接住了”
<br/>　　蛇王石对于涂小安来说，就是天上掉馅饼，对猫精来说或许也是，只是它接不住，差点还被压死。
<br/>　　涂小安稳稳的接住了，并且一口吃了消耗掉。
