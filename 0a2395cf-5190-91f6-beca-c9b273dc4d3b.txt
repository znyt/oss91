　　“冯长老，你真想知道？”苏莫面带微笑，望着满脸好奇的冯光。
<br/>　　“当然！”冯光等着一双小眼睛，等待苏莫回答。
<br/>　　“那好，我就告诉你！”
<br/>　　苏莫点了点头，继续道：“我和真贤长老是朋友，很好的朋友，这够了吗？”
<br/>　　“朋友？”冯光闻言，肥胖的脸颊抽了抽，他可不相信。
<br/>　　试问，一名武帝境的武者，怎么可能和一名武圣境的强者成为朋友？
<br/>　　完全就不是一个层次的人，根本就没有可比性。
<br/>　　随即，冯光也没有再多问，既然苏莫不告诉他，他再问下去的话反而不美。
<br/>　　不多时，冯光带着苏莫，来到了山腰部位，这里建筑连绵，布满了一座座精致的院落。
<br/>　　“苏莫，你以后就住在此地吧！我先去为你解决入宗身份问题！”冯光指着前方的一座院落说道。
<br/>　　“嗯，麻烦冯长老了！”苏莫向对方道谢。
<br/>　　“客气了！客气了！举手之劳而已！“冯光笑着摆了摆手，他可不嫌麻烦，只要是能讨好真贤长老，这点小事不值一提。
<br/>　　随即，冯光便转身离去，前去被苏莫办理入宗的身份事宜。
<br/>　　苏莫则是推开院落的大门，走进了院落之中。
<br/>　　这院落不大，方圆约莫三、四十丈左右，其内有不少房间，干净整洁。
<br/>　　“这是什么？”苏莫将院落的角落之中，玉石地面上刻画着繁杂的符文，不由的有些疑惑。
<br/>　　“居然是阵法！”走进一看，苏莫顿时认了出来，这是一个小型的阵法，但具体是何用，还不清楚。
<br/>　　随即，他投了几快下品灵晶，放进了阵法的卡槽之内，阵法中顿时爆射出一道光芒，而后光芒四散而开，在院落之上形成了一个极其淡薄的气罩。
<br/>　　“不错！”苏莫见此，面上露出了一丝笑容，这气罩有防御和隔音的效果。
<br/>　　虽然这阵法没有多强，但却也大有好处。
<br/>　　这种阵法，看来每个核心弟子的院落中都有，毕竟修为到了这个层次，修炼之时很容易造成大的动静，从而影响其他人。
<br/>　　有了这种阵法，院落内的情况，就不会传递到院落之外，这对于苏莫来说也是好事。
<br/>　　院落中有石桌和石凳，苏莫坐了下来，思考自己目前的情况。
<br/>　　现在，他成功的成为了道宗核心弟子，距离太阴族已经更近了一步，只要成为真传弟子，就有机会去太阴族修炼，就能见到青璇了。
<br/>　　不过，他想要成为真传弟子，实力怕是不够，而且精神力大损，又让实力下降了不少。
<br/>　　看来接下来一段时间，他要努力修炼，争取尽快突破到武帝境九重境界了。
<br/>　　只要突破到武帝境九重，打败一名真传弟子，取而代之，就问题不大了。
<br/>　　在苏莫思索之间，没过多长时间，冯光长老便再次回来了。
<br/>　　苏莫立刻撤掉院落的阵法，将对方请进了院落中。
<br/>　　“苏莫，搞定了，这是你身份牌！”冯光长老面带笑容，递给苏莫一块玉牌。
<br/>　　“有劳冯长老了！”苏莫收起了身份玉牌，向对方道谢。
<br/>　　“对了，本宗内新晋的核心弟子，都有奖励，你虽然是刚加入宗门，也有奖励！“冯长劳说道。
<br/>　　“什么奖励？”苏莫随口问道，他对于奖励倒是不太在意。
<br/>　　“奖励可以自由选择，虽然对你可能用处不大，但聊胜于无嘛！”冯光笑道。
<br/>　　在道宗，只要晋升武帝之境，就能成为核心弟子，所以奖励都是低阶武帝的标准，对于苏莫而言效用不大。
<br/>　　“好，去哪里领取？”苏莫点了点头，随意的问道。
<br/>　　“去本宗的宗武殿！”冯光说道，言罢，他又和苏莫闲聊了几句，告知了宗务堂的位置，便告辞离开了。
<br/>　　冯光走后，苏莫也没有急着去领取奖励，开启了院落的阵法，便进入房间中修炼了起来。
<br/>　　他取出大量的下品灵晶，开始了不断的吞噬，修为开始了快速的增长。
<br/>　　他倒不是要冲击武帝境九重，而是准备先将修为提升到武帝境八重的巅峰，然后等到沉淀的足够了，便一举冲破境界瓶颈。
<br/>　　时间流逝，苏莫不断的吞噬，一小堆一小堆的下品灵晶被他吞噬，直到吞噬了约十五万下品灵晶之后，他的修为达到了武帝境八重的巅峰。
<br/>　　“消耗真是越来越大了！”苏莫叹了口气，他身上的下品灵晶只有十万出头了，这还是加上搜魂的那名太阴族武者身上的灵晶。
<br/>　　也不知道这十万出头的灵晶，能不能支撑他突破到武帝境九重境界。
<br/>　　少倾之后，苏莫关闭院落的阵法，离开了院落，向宗务殿走去。
<br/>　　他也知道宗门的晋升核心弟子的奖励不会太高，但目前他灵晶已经不多了，有奖励也不会放弃。
<br/>　　山道上，不少道宗弟子来来往往，交谈声、议论声，甚至是大小声，均是不绝于耳。
<br/>　　“这道宗弟子虽然不多，但总体实力确实强大！”苏莫有些感慨，他发现这些道宗的弟子，普遍修为很高，比神武学府强大太多了。
<br/>　　宗务殿，位于问道山的山腰偏下的地方，是一座恢弘的大殿，人来人往非常热闹。
<br/>　　走进宗务殿，苏莫发现大殿中人头攒动，足足有百余名弟子，热闹非凡。
<br/>　　打量了片刻，苏莫便明白了，原来这些弟子都是来从宗武堂长老手中接任务的。
<br/>　　望着这些弟子的身影，不由得，苏莫回想到了多年之前，他在风凌岛之时，接取宗门任务的经历，是何等的相像。
<br/>　　苏莫没有急着去要奖励，而是在一旁等候，准备待众人接完任务，他再领取奖励。
<br/>　　“郑长老，有没有适合我的任务？”
<br/>　　就在此时，一声粗犷的声音突然响起，只见大殿外大步走进来一名青年。
<br/>　　这名青年身穿月白色长袍，身材高大，体型非常的魁梧壮硕。
<br/>　　“是吕罡！”
<br/>　　“吕罡居然也来接任务了？”
<br/>　　“吕罡可是只接超难度任务！”
<br/>　　大殿内的众人见到高大青年，顿时惊讶了起来，而后纷纷让开了道路。
<br/>　　苏莫打量了一番高大青年，发现此人的修为高达武帝境九重的巅峰，距离武尊之境，也就一步之遥了。
