　　“无妨，你们无需脱离原来的宗门！”
<br/>　　真贤道圣摇了摇头，继续说道：“万界山的弟子虽然不多，但遍布于四大星域，万千星河之中，每个人都有自己的身份！”
<br/>　　“原来如此！”姜神夜和苏莫心中恍然，同时对于万界山的实力感到心惊。
<br/>　　这样一个隐藏在暗中的势力，门人弟子遍布四大星域，全部都是最顶尖的天才，那这个万界山的能量，估计大的可怕。
<br/>　　“所以，你们无需有什么顾虑，加入我万界山对于你们来说，是千载难逢的机遇！”真贤道圣面带微笑道。
<br/>　　“前辈，我们若是加入了万界山，日后需要做什么？”
<br/>　　苏莫沉声问道，这个万界山与一般的势力不同，此地几乎看不到人影，门人弟子在外界都有自己的身份。
<br/>　　若是这样的话，万界山如何凝聚门下弟子的信念？难道只是单方面的付出？
<br/>　　这根本不太可能，所以苏莫有此一问，
<br/>　　“这个问题问的很好！
<br/>　　真贤道人闻言，赞赏的看了苏莫一眼，笑道：“你们加入万界山之后，就是万界山的记名弟子，会有入门奖励；其后，万界山不会再给予你们任何的赏赐，只有你们完成万界山分配的任务，才会得到相应的赏赐。”
<br/>　　“什么任务？”苏莫和姜神夜异口同声的问道。
<br/>　　“任何任务都有可能！”
<br/>　　真贤道圣说道，言罢，他继续道：“万界山除了界主之外，所有人都是弟子，分为记名弟子、正式弟子和亲传弟子，你们完成任务会有积分累计，积分达到一定的地步，你们就能晋升成为正式弟子，将来有希望继续晋升界主真传弟子！”
<br/>　　苏莫闻言心中恍然，对于万界山的运作，心中也大概清楚了。
<br/>　　成为万界山弟子，就要去完成万界山的任务，从而获得万界山的奖励。
<br/>　　“要提醒你们的是，加入了万界山，终生不得脱离，否则只要身在这个宇宙，就没有逃脱的可能，无论修为有多高！”真贤道圣说道，其面上带着自信。
<br/>　　终生不能脱离，无论修为有多高！
<br/>　　苏莫和姜神夜闻言惊疑，这万界山居然如此霸道，还是盲目的自大！
<br/>　　苏莫心中沉吟，他已经有了加入的想法，这万界山绝对是强大的势力，加入之后肯定有好处。
<br/>　　最主要的是，加入万界山，他就能对抗巫族大祭司。
<br/>　　姜神夜眸光微闪，并不为所动，这万界山或许很强大，但并没有让他加入的足够理由。
<br/>　　他贵为三神宗重点培养的核心弟子，实在没有加入万界山的必要，虽然两者不冲突，但他也不想被万界山所束缚。
<br/>　　“怎么样？你们两人考虑的如何？”真贤道圣微笑着问道。
<br/>　　“前辈，我是古灵星河三神宗的弟子，不知你有没有听说过三神宗？”姜神夜问道。
<br/>　　“三神宗？本道自然知晓！”
<br/>　　真贤道圣点了点头，他负责管理古灵星河和附近几个星河的真圣碑，换言之负责在那几个星河中物色弟子，自然知晓古灵星河巅峰大势力三神宗。
<br/>　　“前辈，既然您知晓三神宗，应该知晓三神宗的实力，所以，我没有理由加入万界山吧！”
<br/>　　姜神夜没有直接的拒绝，而是较为委婉的说道，毕竟，他现在身在此地，而这个万界山又如此的隐秘，万一他直接拒绝的话，对方杀人灭口就完了。
<br/>　　“当然有理由！”
<br/>　　真贤道圣微微一笑，仿佛看穿了姜神夜的想法一般，继续道：“你们三神宗的虚无神，天赋不错，也是我万界山的弟子，而且是正式弟子，将来有希望成为界主亲传弟子。”
<br/>　　“什么？”姜神夜闻言一怔，顿时面现呆滞之色。
<br/>　　“这不可能！”
<br/>　　少倾之后，姜神夜回过神来，顿时满脸不信之色。
<br/>　　虚无神，乃是他们三神宗的三大虚神境大能之一，也是三神宗最年轻的一位虚神境大能，五千年前晋升虚神境，威震古灵星河，震慑亿万星辰。
<br/>　　如此大能人物，怎么可能是万界山的弟子？而且还只是一名正式弟子，不是亲传弟子！
<br/>　　姜神夜根本不信，在他的心中，成为虚神境大能，已经基本上站在了宇宙的巅峰，遨游太虚，纵横环宇。
<br/>　　试问，如此强大的大能人物，怎么可能甘心做万界山的一名正式弟子？
<br/>　　若真是如此的话，那这个万界山、以及这个万界山的界主，该强大到了何等恐怖的地步？
<br/>　　“没有什么不可能！”
<br/>　　真贤道圣摇了摇头，叹道：“本道武圣境后期修为，同样是万界山的正式弟子，想要晋升亲传弟子，还有很远的路要走！”
<br/>　　他加入万界山一千多年了，修为从武帝境达到了武圣境后期，为万界山贡献极大，但或许是天赋的原因吧，想晋升为亲传弟子可能性极小。
<br/>　　姜神夜闻言沉默了，他不太相信对方的话，但又不得不信，对方似乎没有必要骗他。
<br/>　　姜神夜心中沉吟，对方之前也说了，万界山是一个隐藏的势力，不为外人所知。
<br/>　　现在他来到了这里，对方又详细告知了万界山的情况，若是他不加入，貌似离不开了。
<br/>　　这真贤道圣看似平易近人，颇为健谈，但姜神夜也是聪明人，可不相信对方又这么好。
<br/>　　他若是拒绝加入万界山，估计对方就不会这么客气了！
<br/>　　“苏兄，你意下如何？”姜神夜看向苏莫。
<br/>　　苏莫沉默了少倾，目光直接望向真贤道圣，沉声道：“前辈，我愿意加入万界山。”
<br/>　　苏莫直接作出了决定，他与姜神夜不同，对方有三神宗这个靠山，得到三神宗的大力培养，而他什么也没有。
<br/>　　同时，他也和姜神夜有同样的想法，他若是不加入万界山，今日绝对离不开此地。
<br/>　　所以，他没有过多考虑，便同意了加入万界山。
<br/>　　“苏莫，你很明智，相信你不会后悔！”真贤道圣微微颌首，苏莫的天赋，在之前的真圣界测试中，他已经看的一清二楚，绝对是不可多得的妖孽。
<br/>　　“姜神夜，你呢？”
<br/>　　随即，真贤道圣又看向姜神夜，等待对方的回答。
<br/>　　虽然姜神夜的表现不如苏莫惊艳，但也是很妖孽的天才，所以他同样是相当看重。
