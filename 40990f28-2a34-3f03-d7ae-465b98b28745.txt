        &lt;h3&gt;第三卷第一百九十九章  我的甜只给我的安男&lt;/h3&gt;
<br/>
<br/>        寒家生意遍布世界各地，虽然国内放了年假，国外分公司却没有放假，寒伯安也不会闲着，在他的印象中好像自己接手寒家的事业以来他都没有休过周末。
<br/>
<br/>        放年假了他是不用去公司，工作还得照干，在安家陪着安男和老丈人守岁，那种融洽和温馨的日子真的很适合他们，寒伯安都不想离开，在安家呆大年初二，才和安男离开安家回到自己的家里。
<br/>
<br/>        安男：“不回去看你爸爸了吗？”
<br/>
<br/>        寒伯安：“要回去的，不过不是今天，今天是我们私人世界。”
<br/>
<br/>        安男：“不是还有工作要做吗？”
<br/>
<br/>        寒伯安：“现在你就想让我工作？安男小姐姐，能不能给我吃上一顿甜甜的饭，这样我才有动力去干活嘛。”
<br/>
<br/>        安男脸红红的：“难道这几天你都没有吃饱饭？”
<br/>
<br/>        寒伯安：“你说呢？”
<br/>
<br/>        夜色苍茫，红纱幔帐，一点朱唇暗绛，胜雪秀域几千里，嘤哦漫川，苍溪清吟，与清风万里，携玉手并肩，揽巫山风云。
<br/>
<br/>        安男水汪汪的丹凤眼微微的眯着，似睡非睡，那一汪碧潭恰似春雨后的甘甜，寒伯安亲点安男的鼻尖悄悄的离去。
<br/>
<br/>        “工作去吗？”安男慵懒而沁人心扉的语音。
<br/>
<br/>        寒伯安微微一笑：“冰玉在欧洲好像有些事情，下午的邮件我还没有回复，我看看去，你先休息。”
<br/>
<br/>        “我等会起来做宵夜。”
<br/>
<br/>        “宵夜就不必了，加餐是可以的，我还想吃甜甜的加餐。”
<br/>
<br/>        .........
<br/>
<br/>        寒伯安打开邮件，邮件里是寒冰玉最近一段时间在欧洲的情况，弟弟对他这个大哥是很尊敬的，俩个人有问题都会是以邮件的形式来沟通，他们的邮件有一套自己专属密码，这套密码是落雪专门让人设置的。
<br/>
<br/>        落家有自己的信息化部门，里面人才济济，尤其是落雪接手落家的工作之后，更是注重信息化建设，他们的信息网络是整个落氏王国坚强的后盾。
<br/>
<br/>        寒冰玉详细的诉说了他和落雪在欧洲的一切，他能感受到弟弟的快乐也能感受到弟弟那种患得患失的心情，在邮件的末尾，寒冰玉这样写到：“吴家和落家看似融洽，其实让人不可捉摸，尤其是落雪的态度，如果说她爱着吴彤，那么她和我就不会这么亲密，她似乎是在谋划着一件大事情，但是她又不肯让我知晓，我能做的就是备受煎熬的陪着她闹。哥，寒家存在这么多年，请你务必擦亮眼睛，远离是非，保存自己的实力！”
<br/>
<br/>        寒伯安删掉了弟弟的邮件，心下黯然，他们寒家确实是大家族，大企业，可是面对着一切不可预估的时候，谁能独善其身？
<br/>
<br/>        每一个光环下都是努力的结果，每一个光环下都是满腔热血，每一个光环下都是斗智斗勇，谁能躺在家族的功劳簿上一辈子？
<br/>
<br/>        别人看到的风光满面后面，实际上有多少个彻夜不眠？
<br/>
<br/>        寒伯安打开一封又一封的邮件，回复着思考着，前路漫漫，每一步都要规划着，每一步都得小心翼翼，弟弟给他提了个醒，擦亮眼睛远离是非，可是他怕是不能远离是非，独善其身了。
<br/>
<br/>        现在局势错综复杂，各大家族相互制衡，稍不注意便一发而动全身，只要有一点风吹草动都会引发不可预估的结果。
<br/>
<br/>        吴家之前对他的试探，还有陆宇霆夫妇来上海的拜访，都是在给他释放一种信号，想要拉拢寒家，如果寒家和落家都被吴家、陆家所用，还有谁能够抗衡？
<br/>
<br/>        他和陈子昂已经是一条线上的人，在商业合作上已经很多年了，如今又发现陈子昂和安男之间的关联，无论是从道义上还是从私人关系上，他都做不到对陈子昂的不闻不问，安男虽然说过，在陈子昂和他之间，她会选择他的，如果他真的不去过问陈子昂的事情，安男心里怕是会内疚的。
<br/>
<br/>        陆家不会容忍陈子昂母子的存在，无论是从颜面上还是利益上，陈子昂母子对他们来说都是一根刺。
<br/>
<br/>        寒伯安将自己窝在沙发里，点上一支雪茄，深深的吸了一口，闭上眼睛沉思着。
<br/>
<br/>        良久，他打开电脑，给弟弟寒冰玉发了一封邮件：
<br/>
<br/>        “落雪的朋友陈子昂亦是你嫂子安男的舅家姐妹，如今孩子的事情有可能会暴露，期间关系错综复杂，如果暴露会影响几个家族的动荡，问一下落雪该如何解决？陈子昂想先将孩子送到落家庄园，但也是暂时之计。”
<br/>
<br/>        “明天我们将要启程美洲，落家美洲分部出了问题，急需我们过去解决，无暇顾及陈子昂，如果想送孩子到欧洲庄园可自己联系落叔叔，另外，雪儿和陈子昂不来往也是变相的保护。”寒冰玉的信息回得很及时。
<br/>
<br/>        寒伯安关了邮件，点开了另外的一封邮件：“寒总知悉，美洲分部接到一笔订单，疑似有违禁物品，但是对方的来头比较大，陆！”
<br/>
<br/>        寒伯安：“无论是多大的来头，只要有违规之事，一律拒接！”
<br/>
<br/>        肩上传来了温软的热，寒伯安抓着安男的手：“怎么就起来了？”
<br/>
<br/>        “你不知道现在几点了吗？”安男轻轻的给寒伯安揉着肩膀，寒伯安感觉僵硬的肩膀立马轻松了很多。
<br/>
<br/>        “我没有注意呢。”寒伯安胳膊一伸将安男拉到身边，让安男坐在她的腿上，两眼灼灼的看着安男。
<br/>
<br/>        安男用手捂着寒伯安的脸：“早上六点，你已经熬了一整夜。”
<br/>
<br/>        寒伯安轻轻的捉住安男的手：“有老婆在身边，这个日子真是行云流水，过得很快，做事情也是事半功倍，你看我将这几天积压的工作一晚上就做完了。嗯，老婆，我今天任你差遣。”
<br/>
<br/>        安男：“你这话说的，太阳都羞红了脸，今天怕是个阴天。”
<br/>
<br/>        寒伯安：“风起云涌，沧海桑田，无论晴天阴天你都是我的艳阳天！”
<br/>
<br/>        安男：“没发现，你现在的话是越来越甜，我好像掉进了蜜罐。”
<br/>
<br/>        寒伯安：“我的甜只给我的安男，给我的小--心---肝。”
<br/>
<br/>        天阳真的是羞红了脸，躲在云层里偷偷的看，东方的鱼肚白一片黑一片暗，一片明亮一片灿烂。
<br/>
<br/>        寒伯安将心思深藏，所有的忧虑都不会让安男看见，安男只做他快乐幸福的爱人就好。
