        “我没有惹星儿不开心。”明司寒十分笃定又委屈地回答道。
<br/>
<br/>        顾千凝听到他这样只可意会不可言传的声音，差点没忍住笑出声：“那好吧！待会星儿心情好了，我让她给你回电话。”
<br/>
<br/>        “好。”明司寒应了声后，便将电话还给了赵子晋，末了，他还纳闷地问了一句，“你说，我要怎么哄星儿开心？”
<br/>
<br/>        赵子晋用一种难以置信的眼神看着明司寒。
<br/>
<br/>        都说谈恋爱的时候，女人会变傻。
<br/>
<br/>        他们倒是觉得，这谈恋爱的时候，连星儿那女人没傻，明司寒这男人倒是傻了。
<br/>
<br/>        “寒哥，以我这身经百战的恋爱经验告诉你，这女人，如果不是你惹她不开心了，你就不要去哄。”仲正浩凑上前来，搭上明司寒的肩膀，一本正经地说教。
<br/>
<br/>        寝室里，确实也就他仲正浩谈过的女朋友数量最多。
<br/>
<br/>        “为什么？”明司寒不解。
<br/>
<br/>        同样赵子晋和邢乐游也不解。
<br/>
<br/>        仲正浩咧嘴一笑：“这女人吧！宠着是没问题，但是不能骄纵她。懂吗？”
<br/>
<br/>        懂！
<br/>
<br/>        可结果……
<br/>
<br/>        明司寒还是去买了鲜花，守在女生宿舍楼下，打算用来哄连星儿开心。
<br/>
<br/>        仲正浩和赵子晋还有邢乐游不禁摇头感叹：“这男人真傻了！”
<br/>
<br/>        楼上宿舍里，生气归生气，但也不能不吃饭。
<br/>
<br/>        连星儿从床上爬起来后，和顾千凝她们三一起下了楼。
<br/>
<br/>        出了宿舍楼，她们三看到明司寒后，连忙默契地挽着手，将连星儿一个人给“抛弃”了。
<br/>
<br/>        “星儿，我们先去吃饭啦！就不等你了。”顾千凝笑着说道，然后和赵安彤还有路娅玟一起先走了。
<br/>
<br/>        连星儿看着手捧着粉玫瑰的明司寒，一想到爹地的那件事情上，心里就莫名其妙地感到很膈应。
<br/>
<br/>        她今天一直在想，是不是她喜欢上明司寒，也是爹地算计好了的。
<br/>
<br/>        又或者说，是明司寒和她爹地串通了什么事情。
<br/>
<br/>        连星儿的内心此刻是五味杂陈。
<br/>
<br/>        “这花，是用来，哄我的小公主开心的。”明司寒向连星儿迎了上去，走到她跟前时，将手中的粉玫瑰递向她。
<br/>
<br/>        连星儿看着这花，迟迟未接。
<br/>
<br/>        明司寒心里有些紧张地打起了小鼓。
<br/>
<br/>        良久后，连星儿接过粉玫瑰，看着明司寒强颜欢笑道：“谢谢啊！我很喜欢。”
<br/>
<br/>        明司寒这才松了口气，微微一笑：“那，我们一起出去吃晚饭？”
<br/>
<br/>        “不了！我想跟千凝她们一起在食堂里吃，嗯，就这样吧！谢谢你的花啊！”连星儿一手抱着花，另一只手抬起来摆了摆。
<br/>
<br/>        她转身之际，脸上的笑容渐敛，只身往食堂那边走去。
<br/>
<br/>        明司寒愣在原地，看着连星儿渐行渐远的背影，有些忐忑不安。
<br/>
<br/>        翌日。
<br/>
<br/>        上课期间，老师点名，连星儿未到。
<br/>
<br/>        课间休息的时候，明司寒看着手机，犹豫着要不要给连星儿打电话的时候，同班的一个女生突然凑过来，站在课桌间的走道上，看着明司寒，娇滴滴地问道：“明司寒，这次的全国大学生IT技术大赛，是以五人为组。我听说，你这一组只有四个人，可否让我加入？”
<br/>
<br/>        明司寒看都不看女生一眼，冷冷地拒绝道：“不好意思，我们组已经满员了。”
<br/>
<br/>        “我已经打听过了，你们这组就只有你们寝室四个人。你放心，我不会给你们拖后腿的。我……”
<br/>
<br/>        “我女朋友已经加入了。”明司寒再次冷冷地回绝。
<br/>
<br/>        女生一听他这话，悻悻地瘪了瘪嘴唇，默默地离开了。
<br/>
<br/>        现在谁不知道他明司寒的女朋友是连星儿。
<br/>
<br/>        这个女生被拒，其他女生在私底下忍不住对她冷嘲热讽。

<br/>        直到上午的课程上到一半，连星儿才抱着课本偷偷地从后门进来，然后弓着身子快速走到明司寒身旁的座位上坐下。
<br/>
<br/>        明司寒惊愕地看着连星儿，连星儿偏头冲他咧嘴一笑。
<br/>
<br/>        他那颗一直悬着的心，在看到她脸上的笑容后，终于可以放下了。

<br/>        上午的课程结束后，明司寒跟连星儿解释道：“我昨天提出想你跟我同居，其实是为了让你跟我们一起参加大学生IT技术大赛。”

<br/>        坐在一旁的张子晋，第一个反对：“阿寒！你别开玩笑了！她什么都不懂，你让她加入我们，岂不是给我们拖后腿吗？”

<br/>        “就是！”仲正浩十分赞同地点了点头。

<br/>        邢乐游却不以为然：“她可以帮我们翻IT术语词典啊！”
<br/>
<br/>        明司寒打了个响指，很赏识地看着邢乐游，挑了挑眉：“还是你懂我的想法。”
