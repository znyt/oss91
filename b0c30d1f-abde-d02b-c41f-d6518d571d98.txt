　　苏莫话音一落，翼晓晓立刻小脸一呆。
<br/>　　她刚说苏莫打不过象叔叔，苏莫便让象叔叔和她师父一起上。
<br/>　　苏莫大哥疯了吗？
<br/>　　这不是自己找虐吗？
<br/>　　象叔叔和师父联手，天下间只有一人有实力可战，那便是她师父的大哥，这蛮荒山脉三位蛮荒大妖之首。
<br/>　　对于她师父的大哥，翼晓晓也是从未见过，但听她师父说过其实力。
<br/>　　光头大汉和黑衣中年人，亦是面色一滞，被苏莫突然的话语，弄的反应不过来。
<br/>　　少倾，两人回过神来，光头大汉顿时一声怒喝，道：“岂有此理，简直太狂妄了，小子，你真以为自己无敌了吗？”
<br/>　　光头大汉真的怒了，苏莫的言语和神态，居然完全没将他们放在眼里，真的狂妄至极。
<br/>　　黑衣中年人虽然没有说话，但其淡漠的眼眸中，却是闪烁着危险的寒光，让人不寒而栗。
<br/>　　“是不是狂妄，一战便知！”苏莫轻笑道。
<br/>　　“小子，本帝倒要看看，你有何狂妄的资本！”
<br/>　　光头大汉怒喝一声，随即身形一闪，急速掠上远处高空。
<br/>　　至于黑衣中年人，则是没有行动，他们两人还不屑于联手。
<br/>　　苏莫见此，便立刻飞天而起，同样向远处高空飞去。
<br/>　　片刻之间，光头大汉和苏莫便飞临了天穹之上，虚空凝望。
<br/>　　两人身上的气势无比狂暴，澎湃如海，如妖似魔，形成浩大的狂风，呼啸长空，纵横万里。
<br/>　　方圆万里之内，高空上浓郁的云层，如同春雪般消融。
<br/>　　吼！吼！吼！！
<br/>　　一望无际的蛮荒山脉之中，无数惊恐的兽吼声响起。
<br/>　　方圆数万里之内，无数的妖兽都感觉到了这两股恐怖的气势，吓的四散奔逃。
<br/>　　“师父，你说苏莫大哥能打的过象叔叔吗？”翼晓晓小脸上布满了忐忑，向黑衣中年人问道。
<br/>　　“不清楚！”黑衣中年人淡淡的摇了摇头，他的眼眸中充满了凝重。
<br/>　　他本以为苏莫不可能是三弟的对手，毕竟才武皇境五重的修为，但现在一观苏莫的气势，居然丝毫不比他三弟弱。
<br/>　　但具体实力如何，还要一战才能知晓。
<br/>　　天穹之上。
<br/>　　苏莫眸光如电，直视身上金光闪耀的光头大汉，面上带着无敌的自信。
<br/>　　这位蛮荒大妖虽然强大，但也不可能是他的对手，不过，他也想领教一番，远古战象的实力。
<br/>　　“小子，接我一拳！”
<br/>　　光头大汉陡然一声怒喝，而后整个人化为了一颗金色的流星，向苏莫疾驰而来，速度快到极致。
<br/>　　光头大汉瞬息而至，来到了苏莫面前，而后，一只金光闪耀的拳头，金刚捣杵，直击而出。
<br/>　　这一拳威势并没有多恐怖，因为这一拳力量极为凝练，完全是肉身的力量。
<br/>　　但这一拳力量之大，如同十万大山，又如万里江河，无可阻挡。
<br/>　　轰！
<br/>　　一拳出，空间被洞穿，金色的拳头裹挟毁天灭地的力量，直击苏莫的胸膛。
<br/>　　苏莫面色不变，瞬息之间，他的身上同样是金光闪耀，刺眼夺目。
<br/>　　而后，他的拳头抬起，金色的拳头三色玄力缭绕，同样洞穿空间，向光头大汉的拳头迎击而去。
<br/>　　瞬息之间，两只金色的拳头，如同两颗缩小的星辰，狠狠的撞击在了一切。
<br/>　　轰！
<br/>　　一声爆响，震动九天十地，两人周身千丈之内的空间，瞬间被震成了虚无。
<br/>　　汹涌的三色气浪，如同飓风扫荡，逆卷九天。
<br/>　　嗖！嗖！！
<br/>　　苏莫和光头大汉的身形，均是受到强大的反震之力的冲击，纷纷向后飞退。
<br/>　　不过，苏莫只是后退百丈，而光头大汉却是暴退数百里，孰强孰弱，一目了然。
<br/>　　“什么？”止住身形的光头大汉，满脸横肉的脸庞上充满了震惊，他居然被击退了！
<br/>　　他早已看出苏莫实力不弱，所以刚才一拳并未留手，力量之大，足以镇压一般的人族准帝。
<br/>　　而如此强大的一拳，不仅没有击败苏莫，反正是他自己被击退了！
<br/>　　这怎么可能？
<br/>　　下方。
<br/>　　黑衣中年人面无表情，但其眼眸中也是流露出深深的震撼。
<br/>　　苏莫的实力、苏莫的成长速度，实在惊人。
<br/>　　两三年前，苏莫第一次来到蛮荒圣山之时，不过是真玄境的蝼蚁，短短两三年的时间，居然就成长到了这等地步！
<br/>　　“看来他就是上天的宠儿！也是苍穹世界的希望！”黑衣中年人心中急思，大哥的愿望，若是能得到此人之助，也不是不可能完成。
<br/>　　“好厉害！”
<br/>　　就在此刻，一声娇呼打断了黑衣中年人的思绪，只见翼晓晓两只水灵灵的大眼睛中，充满了惊喜之色。
<br/>　　“嘻嘻，象叔叔这下要吃瘪了！”翼晓晓满脸喜色的说道，能看到光头大汉吃瘪，她非常的兴奋。
<br/>　　光头大汉作为准帝级的蛮荒大妖，在蛮荒山脉万兽膜拜，这种吃瘪的情况，可是难得一见。
<br/>　　翼晓晓感觉非常的振奋，自己这个曾经的主人，真是厉害啊！居然能将象叔叔打落下风。
<br/>　　天穹之上。
<br/>　　“前辈，一击已过，还要战吗？”苏莫遥望远处的光头大汉，沉声问道。
<br/>　　对方的实力，也是让他颇为惊讶，此妖之强，怕是都快接近真正的武帝了。
<br/>　　刚才那一拳，他动用了肉身的力量，配合上强大的玄力，本以为一拳足以击败对方，没想到居然只是将对方击退百里！
<br/>　　而对方那一拳，力量之大，直接将他拳头中的玄力都震散了。
<br/>　　“小子，刚才一击不算，我们再来！”
<br/>　　光头大汉闻言，立刻大喝一声，一拳被击退，让他感觉非常的愤怒。
<br/>　　言罢，光头大汉的身躯，如同刺破苍穹的利箭，直冲九天。
<br/>　　而后。
<br/>　　哞！
<br/>　　一声沉闷的吼叫声，陡然响起，声震十数万里，威凌天地。
<br/>　　下一刻，一头无比庞大的巨象，出现在了天穹之上。
<br/>　　此象之大，世所罕见，身长足有四、五百里，仿佛一座大山横亘在天地之间。
<br/>　　此象的四肢，如同四根擎天之柱，支撑起整个天穹。
<br/>　　其粗大的长鼻，长约两百里，惊人无比，轻轻一甩，周身的大片空间为之崩碎。
<br/>　　这是，远古异种，神兽后裔，远古战象！
