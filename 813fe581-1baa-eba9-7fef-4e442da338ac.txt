        “哪天，把你男朋友一起约出来，让我和你哥哥也瞧瞧合不合适？”唐倩兮就像个当家长的女人一样，对小妹诺筱颖说得条条是道。
<br/>
<br/>        诺筱颖只能“嗯嗯嗯”地敷衍了事。
<br/>
<br/>        一看傅昼景就是出身高贵，她诺筱颖可高攀不起啊！
<br/>
<br/>        倘若情非得已，诺筱颖决定去“租”个男友，来搪塞哥哥嫂嫂。
<br/>
<br/>        诺筱颖平安回家，唐倩兮给正在公司里上班的诺司峻回了个电话，好让他放心工作。
<br/>
<br/>        面对白天都能在家无所事事的嫂嫂唐倩兮，诺筱颖好奇地问道：“嫂嫂，你是做什么的？”
<br/>
<br/>        “夜魅娱乐城人事部里跑龙套的！”唐倩兮漫不经心地回答。
<br/>
<br/>        诺筱颖不太理解这个职业，只是微微一笑。
<br/>
<br/>        说白了，唐倩兮的工作，就是物色各路美女进入夜魅娱乐城里从事不同的工作。
<br/>
<br/>        比如说她诺筱颖，长得漂亮嗓子不错，那么她介绍进去就仅仅只是唱歌。至于要不要从事其他服务，完全由对方的自主意愿。
<br/>
<br/>        夜魅娱乐城从来不强迫手下的人做他们不愿意做的事情。
<br/>
<br/>        但如果遇上了什么大人物，那又会是另一种做法了。
<br/>
<br/>        总而言之，越是繁华的城市，某个光鲜艳丽的地方便越是复杂。
<br/>
<br/>        唐倩兮心里的难处，她诺筱颖是无法理解的。
<br/>
<br/>        “嫂嫂，我哥哥能有你相陪，是他这一生的福气！”诺筱颖微笑着说。
<br/>
<br/>        在她眼里，不管唐倩兮从事什么工作，唐倩兮这个女人，都是一个好女人。
<br/>
<br/>        唐倩兮会心一笑，摸了摸诺筱颖的头，问道：“会做饭吗？”
<br/>
<br/>        “嗯！”
<br/>
<br/>        “你嫂嫂我饿了！”
<br/>
<br/>        “那我马上去给你做饭！”
<br/>
<br/>        诺筱颖说做就做，连忙进了厨房。
<br/>
<br/>        这丫头真的是太可爱了！
<br/>
<br/>        唐倩兮不禁感慨，无意间看到诺筱颖脚腕上还贴着膏药，有些心疼地问道：“脚还痛吗？”
<br/>
<br/>        “已经好很多了！”诺筱颖拿起电饭煲的内胆，回头看着唐倩兮微微一笑。
<br/>
<br/>        说真的，诺筱颖这丫头挺招她唐倩兮喜欢，单纯、勤快、听使唤。
<br/>
<br/>        能有这么一个乖巧懂事的妹妹，又何不是一种幸福？
<br/>
<br/>        唐倩兮正非常满意地吃着诺筱颖做的饭菜时，一听说诺筱颖要另外租房子住，顿时激动了：“不如，我和你哥，跟你一起另外合租一套房子吧！”
<br/>
<br/>        “合租？”诺筱颖讷讷地眨巴着眼睛。
<br/>
<br/>        唐倩兮如小鸡啄米般点了点头，然后伸出手指来指了指楼上：“我们这幢大厦叫地利公寓，原本是傅氏集团的员工房。但是，傅氏集团旗下的员工一个个都太有钱了，自己买得起房，压根就不用住这公寓。所以，这幢公寓便对外出租。”
<br/>
<br/>        “再所以呢？”
<br/>
<br/>        “这幢公寓一共有十七层，一到五层是单身公寓，六到十层是两室一厅，十一层到十五是三室两厅，十六层和十七层则是空中别墅。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “我听说，楼上两室一厅的房子还有一套是空着的！”
<br/>
<br/>        “嫂嫂说了这么多，意思就是想我们一起租那个两室一厅的房子？”
<br/>
<br/>        唐倩兮抿嘴一笑，猛地点点头。
<br/>
<br/>        诺筱颖二话不说，便答应了：“嗯，好！”
<br/>
<br/>        唐倩兮顿时喜上眉梢，实则心底早就乐开了花。
<br/>
<br/>        倘若跟诺筱颖住一块的话，家务活，她便不用干了，而且每天回家，还有饭吃，多好！
<br/>
<br/>        而在诺筱颖看来，暂时和哥哥嫂嫂住一块，她会比较有安全感。
<br/>
<br/>        “等你毕业后，你就留在滨江城工作吧！”唐倩兮又接着说。
<br/>
<br/>        诺筱颖微笑着点了点头，她也觉得滨江城这个地方不错，就是不知道她能在这个城市待多久。
<br/>
<br/>        看房子和搬家的事情，便交给了哥哥诺司峻和嫂嫂唐倩兮。
<br/>
<br/>        诺筱颖下午去了傅氏集团旗下地利置业的人事部报到。
<br/>
<br/>        在谈及实习期工资的问题，诺筱颖本不抱太多的希望。
<br/>
<br/>        谁知对方人事部部长竟然主动开口跟她说：“你想要多好工资，我便给你开多少工资！”
<br/>
<br/>        听到这样大跌眼镜的话，诺筱颖目瞪口呆。
<br/>
<br/>        “那个……部长，您……是不是说错话了？”
<br/>
<br/>        没有哪个公司的主管，会对一个刚来应聘的实习生，说出这样不切实际的话吧？
<br/>
<br/>        其实坐在诺筱颖对面的人事部部长，也非常想抽自己两巴掌，但是，总经理那传达下来的命令，他不得不服从。
<br/>
<br/>        看着这个长相娇好，声音甜美，对人彬彬有礼的女孩。
<br/>
<br/>        人事部部长猜想着，这个诺筱颖，十有八九是总经理正在追求的女孩子。
<br/>
<br/>        如果是女朋友的话，压根就不用介绍工作这么麻烦。
<br/>
<br/>        所以，人事部部长猜测的是“正在追求”。
<br/>
<br/>        看着人事部部长对自己一脸纠结的样子，诺筱颖内心也很纠结。
<br/>
<br/>        因为，遇上这种问题，诺筱颖心里揣摩着，这位部长是不是故意在拿这件事情来考核她。
<br/>
<br/>        她在学校里就有道听途说，面试官都喜欢出一些稀奇古怪的问题来考核一个人是否能胜任他们公司的职位。
<br/>
<br/>        诺筱颖心里盘算着，想着自己在临海城的那家装饰装修公司实习期的工资才每个月两千块钱，到这里是，是不是也一样呢？
<br/>
<br/>        可是，滨江城的物价比临海城还贵，每个月两千，她还得交房租，还要了自己日常所需的开销，并且她还得赶紧把钱还给傅昼景，这负担是得多大啊！
<br/>
<br/>        纠结来纠结去，诺筱颖也没能纠结个所以然来。
<br/>
<br/>        人事部部长见诺筱颖半天不回话，不由地笑着说：“梁小姐，您别见外，工资的事情，你就放着胆子说！你说多少，就是多少！”
<br/>
<br/>        “那……”诺筱颖弱弱地伸出了四根手指头。
<br/>
<br/>        “四万？！”人事部部长惊愕。
<br/>
<br/>        这小丫头大学还没毕业了，口味就这么大，更何况她还是个专科大学，在傅氏集团工作的员工，文凭最低的也就三本大学毕业的。
<br/>
<br/>        诺筱颖连忙摇了摇头，讷讷地解释道：“不是四万，是四、四千……可以吗？”
<br/>
<br/>        “没问题！”人事部部长不禁会心一笑，庆幸这小姑娘能有自知之明。
