        千味料理屋。
<br/>
<br/>        傅昼景将车开入这家餐馆的前院停车场里时，一阵微风吹过，满院的樱花犹如蝴蝶纷飞。
<br/>
<br/>        童筱颖从副驾驶座上下来后，还闻到了空气里浮动的淡淡的花香，沁人心脾。
<br/>
<br/>        滨江城四季如春，这里的樱花可以常年盛开。
<br/>
<br/>        傅昼景下车后，无意间看向童筱颖的时候，顷刻间，看出了神。
<br/>
<br/>        一朵樱花悄无声息地落在了她的鬓角上，秀发如泼墨的瀑布，而她却微微仰着头在欣赏着樱花树。
<br/>
<br/>        殊不知，她素颜清雅，比这樱花还美。
<br/>
<br/>        童筱颖回眸，见傅昼景站在车旁望着自己发呆，于是大步流星地蹦到他跟前，抓起落在他肩头的樱花花瓣，摊开手来吹到了他的俊脸上。
<br/>
<br/>        “噗嗤”一声，她笑靥如花。
<br/>
<br/>        傅昼景倏地回神，会心一笑，伸手便将正在跟他调皮的童筱颖搂入了怀中，俯身、偏头，轻轻地啄了一下她粉嫩的脸颊。
<br/>
<br/>        童筱颖身心一怔，双颊瞬间浮泛起两抹红润。
<br/>
<br/>        他给了她这世上，最纯洁、最美好的爱情……
<br/>
<br/>        傅昼景薄唇微扬，捉住童筱颖的手，牵着她在这樱花纷飞之中漫步，向料理屋的大门口走去。
<br/>
<br/>        他两刚进大门，门口的服务员便给他俩一人一个发了一副假面眼镜，说今天里面待会会有活动。
<br/>
<br/>        童筱颖对戴这假面眼镜已经习以为常，从服务员手中接到假面眼镜后就立即戴给了傅昼景看。
<br/>
<br/>        “昼景，好不好看？还能认出我来吗？”童筱颖笑不拢嘴，又俏皮地问道。
<br/>
<br/>        傅昼景忍俊不禁地回答：“傻丫头，我当然认得出你来！”
<br/>
<br/>        她当着他的面戴的，他又岂会认不出？
<br/>
<br/>        不过，倘若她不是当着他的面戴的，他只怕不一定能认出来吧！
<br/>
<br/>        毕竟这种假面眼镜，就只露出两个眼洞和一张嘴，着实很难分辨谁是谁。
<br/>
<br/>        服务员领着他俩脱了鞋，去了大堂一个靠近后院小池的小方桌前，堂内客人还比较多，且大多数是年轻的男女。
<br/>
<br/>        童筱颖跟着傅昼景刚在小方桌前席地而坐，正对面的大屏风前，就有一个穿着和服的女人，手里拿着话筒，面带微笑地“点名”。
<br/>
<br/>        当那个女人遥手指向童筱颖时，一旁立即就有两个女服务员过来邀请童筱颖去屏风后。
<br/>
<br/>        童筱颖下意识地看向傅昼景，傅昼景冲她微笑着点了下头，她才起身跟着那两个女服务员过去。
<br/>
<br/>        半晌后，一列身着清一色的粉色和服，身段高矮几乎一致，全都盘着发髻，面带银色假面眼镜的女人陆陆续续地从屏风后站到了屏风前。
<br/>
<br/>        台下所有男士瞬间看傻了眼。
<br/>
<br/>        “今天是我们XXX先生，向XXX小姐求婚的日子。那么，我们就给XXX先生出个难题。你要如何在我们这众多美女中，找出那个你心仪的XXX小姐呢？”女主持人笑盈盈地问道。
<br/>
<br/>        坐在大堂榻榻米上其他年轻的男女，一个个都拍手起哄。
<br/>
<br/>        别说那位要求婚的XXX先生，就连他傅昼景坐在自己座位前，都难以分辨出屏风前那排身着清一色和服，面戴假面眼镜，站姿一致的美女们中到底哪个是他的筱颖。
<br/>
<br/>        “如果，XXX先生一次就猜中了。那我们给XXX先生免单，如果XXX先生一次就猜错了。那么XXX先生将会送在座的每桌一份鲜三文鱼刺身。”女主持语调激昂。
<br/>
<br/>        大堂里随即响起一阵欢声笑语。
<br/>
<br/>        那位XXX先生还真第一次就猜错了，在座大片对他那因为牵错女友的呆萌行为哄堂大笑。
<br/>
<br/>        XXX先生没找到自己的女友，女主持人又让在座的男士把自家女友牵回座位上。
<br/>
<br/>        傅昼景不得不从小方桌前起身，去把站在屏风前的筱颖找回来。
<br/>
<br/>        跟他一起上台的其他男士，看到面前这一排难以分辨出谁是谁的女人，一个个都纳闷地搔头挠耳，谁都不敢先手牵人，就怕牵错。
<br/>
<br/>        傅昼景一一将这些女人打量完后，挪步走到站在最左边的那个女人面前，二话不说，便抓起这个女人的手，拉着她往自己的餐位上走去。
<br/>
<br/>        既然有人开了头，其他男人似乎也豁出去了，一人一个牵着屏风前的女人回到各自的餐位旁。
<br/>
<br/>        女主持人一声令下，被牵走的女人们这才摘下自己脸上的假面眼镜。
<br/>
<br/>        大堂里，瞬间又是一片“哈哈”的笑声。
<br/>
<br/>        童筱颖摘下假面眼镜后，冲傅昼景做了个鬼脸。
<br/>
<br/>        傅昼景忍俊不禁地抬起手来，爱怜地在她的头顶揉了揉：“我就知道是你！”
<br/>
<br/>        “你是怎么认出我来的？刚刚我去那后台的时候，主持人是叮嘱过我们，不许我们出声，不许我们做任何记号的！”童筱颖看着傅昼景的眼睛，满心期许地问。
<br/>
<br/>        傅昼景会心而笑地答：“当然是对你的心电感应啊！”
<br/>
<br/>        当然，在座还真就有牵错女友的。
<br/>
<br/>        大堂里，哭笑不得，热闹非凡。
<br/>
<br/>        其实，他也不知道自己是怎么找到她的，总而言之，他把所有的女人都仔细打量了一遍后，就对站在最左边的那个女人有一种说不出道不明的感觉。
<br/>
<br/>        既然对她有特殊的感觉，那么他索性就把她给牵回来了。
<br/>
<br/>        还好，他牵中了！
<br/>
<br/>        傅昼景暗自在心底松了口气。
<br/>
<br/>        其实，童筱颖心里也感到很欣慰，不，不仅仅只是欣慰，她还很开心。
<br/>
<br/>        免费赠送的三文鱼刺身已经上了桌，傅昼景夹了一片放在了童筱颖的碗中。
<br/>
<br/>        哪怕童筱颖不喜欢吃这生鱼片，她仍旧面带微笑着夹起生鱼片，沾了点芥末，淡定自若地塞入嘴中。
<br/>
<br/>        嘴里的滋味虽然不好受，但她心里却是甜的。
<br/>
<br/>        傅昼景也并未发现，她童筱颖其实不爱吃日本料理。
<br/>
<br/>        真正爱一个人，是会改变自己的喜好的吧！
<br/>
<br/>        童筱颖发现，自己为了昼景，可以改变自己，可以隐忍自己，可以为他做一些，连自己都预想不到的事情。
<br/>
<br/>        这一天，童筱颖觉得自己待在昼景的身边，觉得自己过得既充实又快乐。
<br/>
<br/>        饭后，他还陪她逛了街，甚至还带她去看了电影。
<br/>
<br/>        童筱颖原本以为，晚饭过后他会继续留在家里陪她，结果……
