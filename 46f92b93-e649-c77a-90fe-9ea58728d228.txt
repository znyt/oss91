孔真儿整个人一怔，在夏夕绾如此犀利而凌厉的气场里完全败下了阵。
<br/>
<br/>        “夏…夏夕绾，我没有想到你竟然这样的卑鄙，你这样跟李玉兰母女她们有什么区别？”孔真儿脸色苍白的指责道。
<br/>
<br/>        夏夕绾看着她，“怎么着，你搞了我爸，我还要跟你说一句你辛苦了？”
<br/>
<br/>        “…”孔真儿无语，夏夕绾的伶牙俐齿真的一再刷新她对她的认知。
<br/>
<br/>        “你知道李玉兰怀上这一胎有多难吗，她和夏妍妍将所有的期望都放在了这一胎上，现在孩子因为你没了，你以为她们会放过你，我刚才只不过将她们会对你做的事情给你提前演练了一遍，如果你非要问我和她们有什么区别，这区别当然是有的，你对于她们而言毫无价值，她们必然将你赶尽杀绝来泄愤，但你对于我而言还很有价值，说出我想要的，我可以对你袖手旁观。”
<br/>
<br/>        孔真儿气的浑身颤抖，这一切明明都是夏夕绾的设计，但是她在她面前毫无还手之力，只能被动的挨打。
<br/>
<br/>        孔真儿冷笑了一声，“夏夕绾，难道你没有想过我还有第三种选择，现在你爸被我迷得团团转，我完全可以转正当你后妈！”
<br/>
<br/>        夏夕绾像是听到了什么好听的笑话，“你可做不了我的后妈，毕竟我跟夏振国都断绝父女关系了，不过你想转正当夏妍妍和夏小蝶的后妈我还是举双手赞成了，没想到你打算这样帮我，我很感动。”
<br/>
<br/>        “…”
<br/>
<br/>        这一下孔真儿气的连话都说不出来了，当夏妍妍和夏小蝶的后妈，她想一想都发抖，而且夏家现在就是一个空壳子，夏振国完全就是靠女儿来养的，她才二十多岁，怎么可能嫁给一个五十多岁的男人？
<br/>
<br/>        她本来想的是在夏振国身上赚足钱，然后离开这个城市，找一个又高又帅自己又爱的男人来接盘。
<br/>
<br/>        夏夕绾一双盈亮的澄眸将孔真儿所有的算计和心机都看在了眼里，唇角勾出了一抹冷笑，“孔真儿，我爸给你不少钱了吧，这些钱不香吗，趁李玉兰母女还没有找你算账的时候，赶紧说吧，我的耐心也是有限的！”
<br/>
<br/>        孔真儿的心理防线已经被击破了，她现在还有时间带着夏振国给她的钱逃走，“好，夏夕绾，我告诉你。”
<br/>
<br/>        ……
<br/>
<br/>        医院回廊里，护士将李玉兰从手术室里推出来，夏妍妍已经紧急赶过来了，她紧张的看着医生，“医生，我妈怎么样了，孩子有没有保住？”
<br/>
<br/>        医生摇头，“孩子没了，我们刚才给病人清了宫，病人本来就是大龄产妇，需要养胎的，你们自己不注意，这孩子肯定保不住。”
<br/>
<br/>        说完医生就离开了。
<br/>
<br/>        夏振国大受打击，他怎么也没有想到他随手一推，竟然将自己的儿子给推掉了。
