　　“做的不错！”
<br/>　　罗昊揉了揉罗战的小脑袋，毫不吝啬的对他进行夸奖。
<br/>　　“妈妈！”
<br/>　　云悠看到云熙，也是从蓝若依身后跑了出来，跑向云熙，扑到云熙怀里，委屈的哭了出来。
<br/>　　云悠再聪明，空间思维能力再怎么好，她也只是一个孩子。在外人面前，云悠尽可能的表现坚强，可是在看到自己妈妈的时候，脆弱的坚强瞬间就碎裂开来。
<br/>　　云熙把云悠抱起来，对着她安慰道：“没事了，没事了，妈妈在这儿。”
<br/>　　云悠两只手紧紧的抱着云熙的脖子不肯松手，生怕自己一松手，云熙就消失了。
<br/>　　“哟，我当是谁呢？”
<br/>　　罗昊看到诸钟伟，故作吃惊的样子，说道：“原来是猪头啊！”
<br/>　　“我现在是格斗教员！”
<br/>　　诸钟伟向罗昊说明自己的身份，似乎是想要找回一点面子。
<br/>　　“拉倒吧。”罗昊丝毫不给诸钟伟面子，说道：“叫你‘猪头’你还不服气？不服气没关系，我们两个再练练，这么长时间没打过架了，看你有多少长进。”
<br/>　　“老爹，你跟他打过架啊？”罗战对罗昊问道。
<br/>　　“你老爹的手下败将。”罗昊向罗战介绍起自己当年的光辉战绩，说道：“当年，他带着人欺负你姨夫，我冲上去就把他揍了一顿，从此以后，他看到我就跑！”
<br/>　　“姨夫，是真的吗？”罗战抬起头，看着齐凡问道。
<br/>　　“当然是真的。”齐凡回答道：“知道他为什么叫‘猪头’吗？就是因为被我们揍得像猪头。”
<br/>　　诸磊站在一旁，抬头看着自己的叔叔。在诸磊心中，自己的叔叔就是自己的偶像，可是发现自己叔叔还有这么一段黑历史后，诸磊心中的偶像形象开始崩塌了。
<br/>　　诸钟伟的脸色变得很难看，可是却也没办法，因为他根本打不过罗昊啊！
<br/>　　“儿子，说说，到底是怎么回事，我相信你不会无缘无故打架。”罗昊对着自己的儿子说道，这一句话直接就摆明了罗昊的立场，他是无条件相信罗战，虽然现在是诸磊挨揍了，那也肯定是诸磊先惹得事。
<br/>　　真是不讲道理啊！
<br/>　　老子不讲道理，儿子也同样不讲道理！
<br/>　　从罗昊、齐凡、云熙三个人走进办公室，沈琛就一言不发的坐在旁边，有些头疼的揉着自己的太阳穴。自己的确是红旗学校的校长，可是管不住罗昊和齐凡这俩货啊，更何况，这次显然是罗战和云悠姐弟俩占着理的，就罗昊那护犊子的脾性，肯定不会善罢甘休！
<br/>　　诸磊拉了拉诸钟伟的衣服，投去询问的眼光。诸钟伟现在恨不得一巴掌拍死诸磊，事情到了这个地步，已经是骑虎难下了。现在就算是诸钟伟想要把事情就此打住，也是不可能的。
<br/>　　“这小子实在是不要脸。”
<br/>　　有了自己老子在这里站场，罗战立刻底气十足，指着诸磊说道：“这小子知道打不过我，就找小悠儿麻烦。你要打那就好好打，可这混蛋被小悠儿抢攻了几招就用‘猴子偷桃’这种下三滥的手段，还有他那个叔叔也是不要脸，说什么在战场上敌人不会跟我们将规矩，只要能赢，不管用什么办法都是正确的。”
<br/>　　罗昊一听罗战的话，脸色顿时就变了，蓝若依、齐凡、云熙三个人的脸色也是变得难看起来，诸钟伟的话的确没错，可是这毕竟是学员和学员之间的比试，更何况，云悠还是个女学员。虽然大家都是孩子，可终究男女有别！
<br/>　　“如果只是这样也就算了。”罗战继续说道：“上次小悠儿还跟我说，她去厕所的时候，看到那混蛋带着几个平时跟他一起玩的，在后面偷偷跟着她。小悠儿什么时候受过这样的委屈，我当然不能放过他啊，所以就把他揍了。如果不是他叔叔拦着，我非把他打得连他妈都不认识。”
<br/>　　罗昊看向诸钟伟，说道：“你自己已经很不要脸了，你这个侄子比你更不要脸，还真是青出于蓝而胜于蓝啊。”说完，看了眼藏在诸钟伟身边的诸磊。
<br/>　　诸磊原本还想要争辩，可是对上罗昊的眼神，话到了嘴边，最后还是咽了下去，那个眼神实在太可怕了。
<br/>　　“罗昊，他还只是个孩子，你这样做是不是太过分了！？”诸钟伟对着罗昊质问道。
<br/>　　诸磊只是能感觉到罗昊的眼神可怕，可是诸钟伟却明白罗昊那道眼神真正厉害之处，这可是会对诸磊心理造成阴影的；以后诸磊再看到罗战或者云悠，就会想起罗昊这道令他灵魂都感觉到颤抖的眼神。
<br/>　　“那他对我女儿做的事情就不过分！？”罗昊盯着诸钟伟反问道。
<br/>　　“格斗课上的事情我承认。”诸钟伟说道：“但是罗战说诸磊尾随云悠去厕所的事情，也只是罗战自己在说，谁知道，这是不是罗战自己编出来的？”
<br/>　　“小悠儿，告诉妈妈，他是不是跟你去厕所？”云熙拍着云悠的后背，小声问道。
<br/>　　云悠两只小手依旧抱着云熙的脖子，然后轻轻点了点头。这一下，云熙的脸色就变得更加难看了。
<br/>　　“她、她胡说！”
<br/>　　被罗战打断了鼻子的诸磊，这个时候极力否认，自己这次说什么都不能承认以前做过的事情，不然事情就麻烦了。
<br/>　　沈琛回到办公桌背后，盯着诸磊说道：“你真的没有做过？”
<br/>　　其实，沈琛心中已经是相信了罗战和云悠的话，现在缺少的，就是证据。不管是物证，还是第三方人证。
<br/>　　“没有！”
<br/>　　诸磊还是继续否认，诸磊倒是不担心沈琛去找其他几个人过来对峙，他们其他几个人也很清楚事情的严重性，知道如果事情曝光，他们会怎么样。
<br/>　　罗战咬定了诸磊尾随云悠去厕所；而诸磊则是矢口否认，坚决不承认，一时间，情况陷入了僵局。
<br/>　　咄！咄！咄！咄！
<br/>　　拐杖落在地面上，才会有的声响从办公室外的走廊传来，这有节奏的拐杖声，吸引了办公室中所有人的注意。
<br/>　　使用拐杖的人很多，但是在红旗学校用拐杖的，只有一个人——“琴魔”方牧南！
