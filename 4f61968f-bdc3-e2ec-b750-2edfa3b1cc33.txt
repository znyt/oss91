　　紫色灵石上，光华流转，极其美丽，慕清霄瞥了一眼手中的感应器，是紫灵晶没错了，将紫灵晶收起，瞥了一眼地上的小紫晶狮王，便原路返回。
<br/>　　上空，战斗中的云韵，见慕清霄大摇大摆的走出来，顿时有些无语，右手在身前微微旋转，剧烈的青色龙卷再现，转眼间向紫晶翼狮王攻去。
<br/>　　在紫晶翼狮王抵挡的瞬间，云韵也不恋战，更没有问紫灵晶的事情，抱起慕清霄，能量羽翼一振，以最快的速度离开紫晶翼狮王的势力范围。
<br/>　　见云韵携带陌生的人类离开，紫晶翼狮王有种不好的预感，斗皇级别的它，智慧已经比低于人类，化作一道紫光向洞内窜去。
<br/>　　几秒后，洞内就传来一阵暴怒的咆哮声：“伴生紫晶源，紫灵晶都不见了，都不见了......该死的人类，我紫晶翼狮王，与你们誓不罢休！”
<br/>　　......
<br/>　　......
<br/>　　离开紫晶翼狮王的洞府，云韵直接带着慕清霄来到两人居住的山洞中，轻轻落在地上，收起能量羽翼，担忧的问道：“你没事吧？”
<br/>　　慕清霄似笑非笑的说道：“你希望我有事？”
<br/>　　“混蛋。”云韵瞪了他一眼，见他一副没事的样子，伸出纤纤玉手，道：“紫灵晶给我。”
<br/>　　也不再调戏她，将紫灵晶递给了云韵，至于伴生紫晶源，倒是只字未提，因为对于斗皇级别的她来说，并没有什么效果。
<br/>　　将紫灵晶收进储存戒指中，云韵美眸望着他，语气温柔，道：“接下来，你有什么打算？”
<br/>　　慕清霄知道她的意思，替她捋了捋遮住美眸的青丝，道：“我也有自己的事情要做，有缘我们自会相见，或者等有了实力，我会去找你。”
<br/>　　云韵并不知道慕清霄口中，所谓的实力是何等层次，自储存空间中取出那件流转着奇异光芒的淡蓝色金属内甲，脸颊羞红，叠好后递了过去。
<br/>　　“这是海之心甲，防御力极强，你独自一人闯荡，我很不放心，我知道你有自己的骨气，不肯跟我一起离开，这内甲应该对你有用。”
<br/>　　慕清霄也不客气，接过内甲放在鼻子上嗅了嗅，要知道，这内甲可是云韵贴身穿在身上的，上面果然有残留着一股淡淡的幽香，与她身上的香味一致。
<br/>　　见他流氓一般的举动，云韵无可奈何，表面依旧雍容华贵，但芳心却极其羞涩，想了想，又取出两本玄阶中级斗技交给了他。
<br/>　　“我们，有缘再见吧。”
<br/>　　话语刚落，云韵毅然转身，背部能量羽翼一振，离地而起，美眸温柔的看了一眼慕清霄，化作一抹青色流光消失在天际。
<br/>　　站在山洞前，慕清霄将内甲与斗技握在手中，目送云韵离去。
<br/>　　最终，无奈一笑，这些东西对他来说根本没用，若是他会受伤，那就表示他遇到了极其恐怖的敌人，这种层次的敌人，内甲如何抵挡。
<br/>　　至于斗技更是如此，最多也就算是大量的金币，因为他修炼的不是斗气，而是灵气。
<br/>　　摇了摇头，他也知道，云韵是在关心自己，将内甲与斗技装进储存空间最深处，至于卖了，或者送人，这绝对是不可能的。
<br/>　　正当慕清霄想转身离开的时候，却感应到一道恐怖的气息迅速接近，抬头望去，嘴角牵起一丝邪异的笑容，来者不是别人，正是去而复返的云韵。
<br/>　　云韵直接收起能量羽翼，他刚想开口说话，一道柔软的娇躯就携带者一阵香风扑进他的怀中，温香的粉唇立刻吻了上来，自己这算是被强吻了吗？
<br/>　　回过神来，慕清霄双手攀上她纤细的腰肢，逮住一条润滑的小舌，细细的品味起来。
<br/>　　十几分钟后，唇齿分离，带着几丝晶莹玉液，云韵气喘吁吁的抬起头来，脸颊上浮现绝美的嫣红，凤凰形发饰都有些凌乱，雍容高贵中参杂着一丝慵懒的气质。
<br/>　　“我真的要离开了，记得照顾好自己，遇到危险千万别莽撞，等我办完事情，会去找你。”
<br/>　　说罢，云韵好不拖泥带水，不舍的挣脱他的怀抱，双翼一振，化作一抹青色流光，消失于天际。
<br/>　　“莽撞？”
<br/>　　慕清霄微微摇头，若自己性格算是莽撞的话，她自己可真算是无脑了，明知紫晶翼狮王实力不低于她，却要正面激斗，而不是另想办法。
<br/>　　她担忧慕清霄，慕清霄何尝不是担忧她，毕竟如此极品的女子，在弱肉强食的世界中，可是最危险的，哪怕她身份是云岚宗宗主。
<br/>　　摸了摸嘴唇，上面还残留着佳人的幽香，那种销魂蚀骨的感觉，当真让人回味无穷：“放心吧，相信我们很快就能见面了。”
<br/>　　原本，慕清霄是想来找药材，在此遇到云韵，绝对是意外，佳人离去，他自然也没有留下来的必要，在魔兽山脉中待了三天，也不知道小医仙怎样了。
<br/>　　在青山镇的事情基本已经办完，净莲妖火残图也拿到手，规划了下路线，他所知的异火中，距离此处最近的就是塔戈尔大沙漠了。
<br/>　　而塔戈尔大沙漠中，就有他的目标‘青莲地心火’。
<br/>　　傍晚时分，慕清霄的身影再次出现在青山镇上，小镇依旧很热闹，只是刚进城门，却有不少异样的目光停留在他的身上。
<br/>　　“喂喂，你们快看，这不是狼头佣兵团正在通缉的人吗，听说只要有他的信息，上报就有一大笔钱啊！”
<br/>　　“你想找死就去吧，听说他可是杀了狼头佣兵团的少团长穆力，而且去魔兽山脉中的狼头佣兵团，有一半都死在他的手中。”
<br/>　　“对了，我还听说，这家伙跟小医仙有着不清不楚的关系，不过得罪了狼头佣兵团，他就死定了。”
<br/>　　“杀了穆力竟然还敢回来，这小子简直是不要命了。”
<br/>　　走在大街上，各种议论声传入他的耳中，甚至有许多新的版本，比如他杀了穆力，想要强占小医仙啥的。
<br/>　　慕清霄看了一眼城门上，挂着一张与自己三分相似的画像，面色平淡，径直向城中走去。
