　　武道碑林外，一众武者原地等待着，不时看一眼林劲等人，眼中尽是羡慕。
<br/>　　林劲、秦飞扬、花剑、王虎等十九人，从开始参悟玄界碑的那一刻，就代表着通过考核，能成为内门弟子。
<br/>　　内门弟子，待遇之高，足以让任何人狂热。
<br/>　　更重要的是，进入内门，几乎就等同于敞开了一道强者之门。
<br/>　　如今的裂天剑派十大长老，以及历代的长老，皆是破玄境，也皆是曾经的内门弟子，最终成为名震北山郡的一代强者。
<br/>　　可想而知，内门，底蕴何其之深，又是何等地让人梦寐以求。
<br/>　　这时，林劲没有理会其他人的羡慕目光，而是紧紧地盯着碑林内，如今的唯一一道身影，萧逸。
<br/>　　“萧逸还在参悟着。”林劲说道。
<br/>　　“呵呵。”铁牛憨笑道，“那种非人的折磨，俺是坚持不下去了，萧逸兄弟能坚持到现在，而且仍旧在参悟着，领悟力和心智怕是远超我们。”
<br/>　　柳嫣然笑了笑，脸上尽是喜意，似乎比其他人更为开心。
<br/>　　秦飞扬撇撇嘴，笑骂道，“如果说我们是天才的话，那家伙简直是个妖孽。”
<br/>　　四人打趣地聊着天，却没有注意到，不远处，顾长风也在盯着萧逸。
<br/>　　只不过，他的眼中尽是怨毒的目光。
<br/>　　“萧逸，是你逼我的，一而再再而三地破坏本公子的事，还让林劲那群混蛋来针对我，让本公子颜面尽失。”
<br/>　　“今日，我要你百倍奉还。”
<br/>　　顾长风阴蛰地想着。
<br/>　　另一边，十位执事也在紧紧地盯着萧逸。
<br/>　　“已经过去三个时辰了，也不知此子还能坚持多久。”
<br/>　　“以先天境武者的能耐，三个时辰，只够参悟一半多些的内容。若是能坚持六个时辰，便能将整块玄界碑彻底领悟。”
<br/>　　“若真是这样，那可是了不得的大事。”
<br/>　　“裂天剑派历史上，还从未有弟子在参加这个考核时能完全悟透一块玄界碑。”
<br/>　　剑堂执事得意地笑着，“如果他真能完全悟透一块玄界碑，毫无疑问，将成为剑派有史以来天赋最高的弟子。”
<br/>　　“你们剑堂这次捡到宝了。”其他九堂的执事笑道。
<br/>　　“若此子真能成功，怕是长老们都会惊动到。”
<br/>　　......
<br/>　　十大执事期待着，也紧张注视着。
<br/>　　可是，十分钟后，他们脸上的期待完全消散，面露遗憾之色。
<br/>　　因为，萧逸已经从玄界碑上站了起来，停止了参悟。
<br/>　　“结束了吗？”剑堂执事叹了口气，“才三个时辰多点，顶多参悟了七成左右。”
<br/>　　“可惜了。”其他堂的执事也摇了摇头，道，“剑派历史上，最出色的弟子，可是能参悟到八成。”
<br/>　　“也罢，七成也很厉害了，起码比其他十九人要厉害得多。”
<br/>　　.....
<br/>　　碑林内，萧逸笑了笑，目光随意地在这块玄界碑上扫视，却没有丝毫不适和痛苦。
<br/>　　因为，整块玄界碑，已经被领悟完毕。
<br/>　　碑上的内容，他全部都理解，自然不会有任何不适。
<br/>　　正常来说，完全参悟一块玄界碑，确实要六个时辰以上。
<br/>　　可他的参悟速度远超其他人，便只用了三个时辰就完成了。
<br/>　　参悟剑道知识，体内剑武魂品阶越高，越强，速度越快。
<br/>　　当然，自身的领悟力和心智也起着重要的作用。
<br/>　　“出来。”碑林外的执事，大手一挥，将他凌空提了出来。
<br/>　　“执事。”萧逸对着面前的执事们行了一礼。
<br/>　　“嗯。”十位执事点点头。
<br/>　　萧逸眉头一皱，他明显看到十位执事脸上的惋惜之色一闪而过。
<br/>　　“怎么回事？”萧逸暗暗疑惑。
<br/>　　十大执事，都在认真地看着萧逸，心中暗暗遗憾。
<br/>　　“可惜了，若是他能再坚持一个时辰，领悟到八成以上的内容，长老们都会被惊动出来。”
<br/>　　“若是能完全领悟，怕是长老们会争先收他为徒。”
<br/>　　执事们暗暗遗憾，殊不知，萧逸已然领悟了整块玄界碑，只不过领悟速度太快，不需耗费那么多时间罢了。
<br/>　　半晌，剑堂执事率先问道，“你叫萧逸是吧？”
<br/>　　“是。”萧逸点点头，道，“不知执事有何吩咐？”
<br/>　　“你可愿意入我剑堂？”剑堂执事问道。
<br/>　　“少年，先不急回答。”其他九堂的执事连忙道，“虽然你参悟的是剑道玄界碑，但你同样是可以参加其他内堂的。”
<br/>　　在其他执事看来，萧逸能参悟七成玄界碑内容，已经非常不错了，乃是今年二十位通过考核的弟子中最出色之人。
<br/>　　无论哪一堂，都对这样的天才尤为看重。
<br/>　　“你们瞎掺和什么。”剑堂执事瞪了其他人一眼，而后看向萧逸，等待着回复。
<br/>　　事实上，他几乎可以确定萧逸一定会选择剑堂。
<br/>　　裂天剑派，乃是以剑术闻名的剑派，剑堂自然是内门十堂中最厉害，也是最出色的。
<br/>　　加入剑堂，几乎是所有人优先考虑的选择。
<br/>　　果然，下一秒，萧逸点了点头，道，“我愿意进入剑堂。”
<br/>　　“好，有眼光。”剑堂执事笑笑，虽然心中感到遗憾，但总体来说，对萧逸还是相当满意的。
<br/>　　“从今日起，你就是剑堂弟子。”剑堂执事笑着说道。
<br/>　　萧逸面露喜意，他来裂天剑派，本来就是想修习剑术的。
<br/>　　“慢着。”
<br/>　　忽然，一道人影凭空而现，而后大喝了一声。
<br/>　　“嗯？”剑堂执事皱了皱眉头。
<br/>　　萧逸也是眉头一皱，心头涌现了一道不妙的预感。
<br/>　　不远处，一道人影缓缓走来，身姿优雅，风度翩翩。
<br/>　　来人是一年轻男子，样貌俊朗，气度不凡。
<br/>　　身上那股自然存在的高傲气息，仿佛瞬间就能让人折服。
<br/>　　眼中似蕴含着惊天剑意，锐利无双，只需淡淡一眼，就能将人绞成粉碎。
<br/>　　“好强。”在场众人，无不吃了一惊。
<br/>　　十位执事，则面露疑惑，道，“长空，你怎么来这了？”
<br/>　　年轻男子面对十位内门执事，却没有行礼，只是礼貌性地点点头，算是打个招呼。
<br/>　　而执事们，也没有任何不满。
<br/>　　年轻男子负手而立，神色淡然，但语气中却带着高傲与不可一世，乃至目空一切。
<br/>　　“我来此，一，是为舍弟讨个说法；二，便是不想让某些无能之辈入我剑堂，免得日后损了我剑堂的名声。”
<br/>　　说着，年轻男子走到了萧逸面前。
<br/>　　另一边，一众武者们也疑惑地看着年轻男子。
<br/>　　只有花剑以及王虎，忽然神色大变，“糟了，萧逸有麻烦了。”
<br/>　　“嗯？”林劲四人一惊，连忙问道，“怎么回事？”
<br/>　　花剑惊疑道，“你们四人不是百武城附近的人吧？”
<br/>　　“嗯。”林劲、铁牛、秦飞扬三人点点头。
<br/>　　柳嫣然道，“我曾经在百武城居住过一段时间，但没多久。”
<br/>　　“难怪你们不知此人是谁。”花剑脸色难看地说道。
<br/>　　王虎接话道，“他是顾长风的哥哥，也是剑堂首席，顾长空。”
<br/>　　花剑沉声道，“十年前，顾长空成为裂天剑派内门弟子，顾家可是大肆宣扬了一番。“
<br/>　　“十年时间，此人都在潜心修炼，极少离开剑派，你们不是百武城本土武者的话，确实不知道他。”
<br/>　　林劲四人顿时脸色一变，“顾长空，剑堂首席？”
<br/>　　另一边，萧逸明显可以感觉到，面前的年轻男子，正以一种无比轻蔑的姿态俯视着自己。
<br/>　　“长空，到底怎么回事？”剑堂执事注意到气氛有些不对劲，连忙道，“萧逸即将进入剑堂，你们也将成为师兄弟....”
<br/>　　顾长空打断道，“执事此言差矣，但凡我剑堂弟子，皆是北山郡赫赫有名的剑道天才，可不是什么阿猫阿狗都能随便加入的。”
<br/>　　“起码，一个控火兽武魂的废物，没有这个资格。”
<br/>　　顾长空冷声说道。
<br/>　　十位执事顿时脸色突变，“什么，控火兽武魂？”
