　　一转眼，
<br/>　　叶尘在黑岩城已经待了三天时间，
<br/>　　虽然时时刻刻都有黑岩城的人，在监视着他，不过他并没有放在心上，只是专心调养自己的伤势。
<br/>　　那位叫灵溪的小丫头，倒是对他很上心，一直在用她那微薄的医术，为叶尘疗伤。
<br/>　　她又哪里知道，叶尘身怀万古长青珠，又自己可以炼制高级灵丹，何须她一个小丫头医治？
<br/>　　不过，叶尘将这小丫头心地善良，又活泼可爱，所以倒也没有拒绝她的一番好意。
<br/>　　短短几天下来，叶尘的伤势已经好了三四成，小丫头还以为自己医术大进，高兴的不得了，于是便越发的上心了。
<br/>　　而两人的关系，也越来越熟络，叶尘从小丫头的口中，也逐渐得知了一些当地的情况。
<br/>　　原来，眼下他们所在的这片星域，名为巨门星域，仅仅只是一个四级星域而已，在中央大星域的最边缘位置。
<br/>　　而这灵溪小丫头所在的势力，名为黑岩城，在整个巨门星域，勉强可以跻身二流势力。
<br/>　　小丫头是黑岩城城主的二女儿，所以之前那些人，才会称呼她为二小姐。
<br/>　　“叶尘哥哥，等我把你身上的伤治好，我就可以向父亲大人证明，我虽然没有姐姐那样的修炼天赋，可是却可以当一名救命治人的药师，哼哼！”
<br/>　　庭院之中，小丫头灵溪，一边为叶尘敷着药，精致的小脸上，透着兴奋的笑容。
<br/>　　叶尘不由得暗暗好笑，
<br/>　　其实他身上的这些外伤，早就可以通过万古长青珠，在瞬间就可以治愈，不过为了满足这小丫头治病救人的愿望，所以才故意保留着。
<br/>　　根据这小丫头自己所说，她自小体弱多病，导致体内经脉阻塞，所以修炼起来，比普通人要困难的多。
<br/>　　而在她的上面，还有一位天资绝顶的姐姐，如今拜在一个大宗门门下，是整个黑岩城的希望！
<br/>　　因而相比之下，这小丫头一直都比较自卑，活在她那个天才姐姐的阴影之下。
<br/>　　由于修炼天赋欠佳，所以小丫头便一行钻研医术，希望将来可以成为一名药师，让自己的父亲和姐姐，对自己刮目相看。
<br/>　　原本叶尘打算，等伤势好一些，便立即动身离开的，可是望着眼前那张犹如瓷娃娃一般的俏脸脸庞，还有额头的汗水，心里顿时大为不忍，忍不住揉了揉小丫头的脑袋，道：
<br/>　　“灵溪妹子，其实你只是体内经脉受阻，只要服用一颗破灵丹，便能助你重塑经脉，这并不是什么难事啊！”
<br/>　　此时，灵溪已经为叶尘包扎好伤口，顿时眨了眨明亮的大眼睛，叹了一口气，道：
<br/>　　“叶尘哥哥，哪有你说的那么轻松啊！那破灵丹，可是四品丹药，我们整个巨门星域，也仅仅只有一位四品炼丹师而已，而且那位丹药大师，性格孤傲，父亲大人也曾经托人相求，却连见他一面都难！”
<br/>　　叶尘听到这话，不由得微微一愣，旋即眉头紧皱，道：
<br/>　　“区区一名四品炼丹师而已，也敢这么狂妄？”
<br/>　　灵溪一脸理所应当道：
<br/>　　“那是自然！叶尘哥哥，这你就不懂了，炼丹师可是一个很高贵职业呢！像我们黑岩城的桑大师，只是一个三品炼丹师，就连父亲大人，都要对他礼让三分！”
<br/>　　说到这里，灵溪的一双大眼睛中，透着一抹向往的神色，幽幽叹了一口气，道：
<br/>　　“我其实最大的愿望，就是成为一名炼丹师，可惜每一个品阶的炼丹师，需要相应的修为辅助，像我这样无法修炼之人，是注定一辈子成了炼丹师的！”
<br/>　　说到这里，小丫头那明亮的双眸，开始迅速暗道下去，小脸上尽显失落。
<br/>　　叶尘见状，也顿时更加怜惜，正想说，叶尘哥哥可以帮你实现这个愿望！
<br/>　　然而就在这时，
<br/>　　一名城主府的下人，从外面兴冲冲的走了进来，
<br/>　　“二小姐！好消息！大小姐从飞雪宗回来了，连同飞雪宗的首席大师兄一起，如今正在前厅，老爷让我喊您过去呢！”
<br/>　　灵溪听到这个消息，顿时高兴的一下子跳了起来，
<br/>　　“姐姐回来了？太好了！”
<br/>　　小丫头兴奋之下，便要赶去前厅，不过才刚刚走出几步，似乎又想起了叶尘，于是转身回到叶尘身旁，一把拉住他的胳膊，
<br/>　　“叶尘哥哥，你也随我一起去吧！我刚好可以介绍我姐姐，和你认识一下，你说好不好？”
<br/>　　叶尘本想拒绝，可是见这小丫头一脸期盼，拒绝的话最终还是没有说出口，只好点了点头，答应了下来。
<br/>　　……
<br/>　　不一会，
<br/>　　叶尘被灵溪拉着胳膊，来到了黑岩城城主府的前厅。
<br/>　　大厅里十分开阔，里面有不少人，已经各自入座。
<br/>　　坐在大厅主位上的，是一名气势很足的中年男子，一身的金色长袍，神色威严。
<br/>　　此人便是黑岩城城主，灵溪小丫头的父亲，叶尘之前曾远远见过一次，所以倒也认得。
<br/>　　而坐在客席首位上，则是一名一袭白衣的英俊青年，气质飘逸不凡。
<br/>　　英俊青年的旁边，还有一名姿色、气质都十分不俗的青年女子，五官容貌倒是和灵溪有几分相似。
<br/>　　美貌青年女子看到两人进入大厅之中，当即冲着灵溪招了招手，
<br/>　　“灵溪，快到姐姐这边来！”
<br/>　　“姐姐！”
<br/>　　灵溪也当即喊了一声，然后一脸欣喜的扑到那位青年美貌女子的怀里。
<br/>　　“原来她就是灵溪常常提及的那位天才姐姐灵韵？嗯，金丹境三重的修为，在一个四级星域，的确算得上是天才了！”
<br/>　　叶尘大眼一扫，便已经将大厅众人的修为，全都尽数了然于胸，同时也缓缓走进大厅，来到灵溪的身旁。
<br/>　　姐妹两人一番亲热之后，姐姐灵韵这才看到一旁的叶尘，顿时秀眉微微一蹙，
<br/>　　“灵溪，父亲大人说，你前几天救了一个来历不明的男人，就是他吗？”
<br/>　　叶尘见状，顿时眉头一皱，因为他隐隐感觉到，这女人对自己，似乎有着莫名的敌意。
<br/>　　“嗯！”
<br/>　　灵溪显然没有意识到这一点，当即兴奋的点了点，热情拉着叶尘的胳膊，道：
<br/>　　“对呀！他叫叶尘，来自外星域！叶尘哥哥，这位就是我姐姐，灵韵！”
<br/>　　“噗~”
<br/>　　灵溪才刚说完，还没等叶尘开口，大厅里，忽然响起了一道充满嘲讽的声音，
<br/>　　“难怪穿着如此古怪！原来是来自外星域的乡巴佬啊！”
<br/>(本章完)
