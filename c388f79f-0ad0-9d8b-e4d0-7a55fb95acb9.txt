　　第324章
<br/>　　嘟，恭喜完成十次山寨防守……
<br/>　　嘟，奖励战旗五幅
<br/>　　嘟，奖励四级山寨建设图纸
<br/>　　……
<br/>　　这么多日下来，程大雷也不记得挡住了多少波攻击，但很显然系统帮他记得。程大雷回到自己办公室，把门关上后清点这次的系统奖励。
<br/>　　四级山寨建设图纸先不去管它，目前还没有开四级本的条件，最关键是系统奖励的五面战旗。
<br/>　　程大雷眨了眨眼睛，就看到那五杆战旗摆在自己书案上。他抄手打开一副，毫无意外的看到上面绣着蛤蟆纹章。
<br/>　　程大雷无可奈何的撇撇嘴，这当然在他预料之中，所以也并不觉得惊讶。令程大雷眼前一亮的是，这五面战旗都附带特殊技能。
<br/>　　破军：提升己方冲锋速度
<br/>　　鼓舞：提升己方士气
<br/>　　战神：提升己方战意，同比例削弱敌方士气
<br/>　　铁血*2：提升己方体力
<br/>　　这类似于游戏中的群体BUFF，有增益BUFF，也有削弱地方势力的负面BUFF。眼前正好五面，正好交给秦蛮五人。
<br/>　　程大雷将银眸唤来，令她磨墨，要在旗上书下秦蛮五人的名字。银眸不会磨墨，好吧，程大雷又将柳芷唤来，摆开笔墨纸砚。
<br/>　　将破军旗交给赵子龙，上书斗大一个龙字；将鼓舞旗交给秦蛮，上书蛮字；将战神旗交给关鱼，上书一个鱼字；将铁血旗分别交给张肥和高飞豹。
<br/>　　到晚上开会时，程大雷把战旗交给五人，同时继续商讨明日的战斗。守住了连日来的攻击，守得很艰难，同时大家对敌人的战斗力已经有了一定了解。有这五面旗助阵，接下来蛤蟆寨的战略便该由守转攻。
<br/>　　……
<br/>　　战斗持续半月有余，蛤蟆寨的坚韧出乎莫鸣米的预料，时至今日，还未有一人活着登上蛤蟆岛。但莫鸣米为此感到沮丧嘛？
<br/>　　一点都不。
<br/>　　连续的战斗，伤亡很大，但并没有伤及莫鸣米的根本。因为每日攻打蛤蟆寨的人，都是自己手下十八家首领的人。
<br/>　　这些人军粮自备，武器自带，不花莫鸣米一分一毫。
<br/>　　莫鸣米毫无损失，却一直在消耗程大雷。久守必失，攻破蛤蟆寨对莫鸣米来说已不是太大问题。
<br/>　　夜来，大帐之内。
<br/>　　莫鸣米坐在帅案之后，左右两手坐着此地的十八家首领。
<br/>　　人人表情肃穆，即便内心愉悦的莫鸣米也是一样。
<br/>　　“半个月都打不下一座蛤蟆寨，各位当真是好大的本事。”莫鸣米不阴不阳道。
<br/>　　气氛凝重起来，人人垂头无语，大帐内鸦雀无声。
<br/>　　“云中龙，乔鹤，游九楼。”
<br/>　　莫鸣米点了三个人的名字，云中龙三个立刻站了起来。
<br/>　　“你们三个有什么话说？”
<br/>　　三人互相对视一眼，云中龙开口道：“将军，不是兄弟们无能，实在是程大雷太狡猾。”
<br/>　　“哈。”云中龙打了个哈哈：“太狡猾，怕是你们心里对姓程的佩服得紧吧，据说，你们这些日子跟着程大雷混得不错。”
<br/>　　云中龙三人踉跄跪倒，三人几乎同时道：“我们生死给将军效忠，与那程大雷势不两立。”
<br/>　　“好啦，好啦。”莫鸣米摆摆手：“起来说话吧，我也是随口说说，你们何必如此。我也知道，你们跟着程大雷，不过是形势所迫。现在能分清形势就是好的。既然如此，明日的战斗，就有你们三家主攻。”
<br/>　　“这……”
<br/>　　三人一楞，然后异口同声道：“誓死拿下蛤蟆寨。”
<br/>　　大帐的人散去，想到最近几日就可攻破蛤蟆寨，莫鸣米心情愉悦。但他不明白的是，世间的悲喜是相对的，有人开心，必然就有人不开心。
<br/>　　不开心的不是程大雷，而是十八家首领。
<br/>　　自己来替莫鸣米打仗，不管吃不管穿，手下人天天有人死，世间哪有这样的事儿。
<br/>　　诸人心里都积累着怨气，表面上表现着对莫鸣米多忠诚，内心的怨气就有多重。
<br/>　　而在所有人中，以云中龙三人的怨气最盛。
<br/>　　夜来，三人凑在一起，人人面沉似水。
<br/>　　“姓莫的太不是东西，谁不知道蛤蟆寨难打，竟然让我们送死！”乔鹤气急败坏。
<br/>　　“嘘，噤声。”云中龙道：“这可是别人的地盘，小心被别人听见了。”
<br/>　　“云老大，你说句话吧，明天咱们怎么办？”游九楼。
<br/>　　云中龙却也不知该如何是好，这半个月三人是出工不出力，吆喝声最大，但却绝对不上前打。之所以如此，是因为了解蛤蟆寨的实力。
<br/>　　秦蛮、赵子龙几个也不是没接触过，都是以一敌百的主儿，拿什么和人家打，总不能用脑袋往人家脑袋上碰吧。
<br/>　　“实在不行，咱们撤吧。”乔鹤：“我可不愿为了姓莫的送死。”
<br/>　　“撤？往哪里撤，琴川一地可都是莫老大的地盘。”游九楼。
<br/>　　“是呐，咱们总不能舍了这家业，去做丧家犬。”乔鹤道：“以莫老大的人性，要是抓住我们，我们连骨头渣子都留不下。”
<br/>　　“要我说，如果程当家能做了这片地盘的主人，我们的日子还会好过些。”云中龙发出一声感慨。
<br/>　　乔鹤和游九楼点点头，这些日子跟着程大雷混，程大雷为人还算是仗义的，只要你替他出力，他就不会令你白干活。相比之下，莫鸣米简直不是个东西。
<br/>　　“可程当家这次能赢么？”游九楼问。
<br/>　　云中龙沉默良久，最后只能无奈的吐出一句话。
<br/>　　“说不好。”
<br/>　　次日黎明，天蒙蒙刚亮，晨雾未散，战鼓擂响，莫鸣米再次发动进攻。
<br/>　　云中龙、乔鹤、游九楼三捎人马出阵，走在队伍最前方。
<br/>　　身后是莫鸣米的大军，前面是程大雷的重镇。程大雷不好惹，可往回走，莫鸣米的大刀也不是吃素的。
<br/>　　三人领着手下走在冰面上，人人哆哆嗦嗦，快已走到蛤蟆寨的射程，三人不约而同的都停住脚步。
<br/>　　程大雷在瞭望台上看到这一幕，撇撇嘴，令人冲下面喊道：
<br/>　　“呔，蛤蟆寨蛤蟆大王在此，你们放下武器，蛤蟆大王大人大量，饶你们不死。”
<br/>　　铛！
<br/>　　所有人的兵器同时落在冰面上，云中龙高举双手向蛤蟆寨奔来。
<br/>　　“投降了，我们愉快的投降啦。”
<br/>　　
<br/>(本章完)
