　　想好了详细的计划，苏莫便喊来了那名中年下人，命对方前去传巫天御。
<br/>　　夺舍逃离，最好的人选莫过于巫天御了。
<br/>　　如此的话，不仅能逃离，还能杀了对方报仇雪恨，真是一举两得。
<br/>　　他现在身为少司，地位崇高，传唤巫天御，对方不敢不来。
<br/>　　望着中年下人离去的背影，苏莫眸中寒光微闪，这名下人估计也是大祭司的眼线吧。
<br/>　　苏莫没有等多久，紧紧是盏茶的功夫，巫天御便跟随中年下人来到了他的府苑之中。
<br/>　　随即，苏莫在一间客厅之中接见了巫天御，并且挥退了那名中年下人。
<br/>　　“不知巫尘少司找在下前来又何要事？”巫天御面色不卑不亢，坐下之后沉声问道。
<br/>　　“巫天御，我记得上次你要和我约战！”苏莫端坐在主座上，面带笑意的问道。
<br/>　　“这个……一场误会而已！”
<br/>　　巫天御面上露出尴尬之色，随即摇了摇头，继续道：“巫尘少司天赋盖世，实力逆天，我哪里是对手！”
<br/>　　言语虽然如此，但巫天御的心中却是阴郁到了极点，他知道这次苏莫是要找他麻烦了。
<br/>　　所以，他只能尽量放低身段。
<br/>　　“哼！误会，我没成为少司之前，你怎么不说是误会？”苏莫冷笑着说道。
<br/>　　巫天御的脸色沉了下来，他没想到巫尘成为了少司，居然还如此额记仇，不想放过他。
<br/>　　“巫尘少司，之前是我不敬，还望原谅！”巫天御沉声说道。
<br/>　　“想要原谅，倒也可以，你需要拿出一点补偿！”苏莫笑道，言罢，他立刻知会阎魔鬼圣行动。
<br/>　　霎时之间，阎魔鬼圣的魔魂之中，一缕颇为强悍的意念汹涌而出，化为了一个无形的符文，包裹住了苏莫手指上的储物戒。
<br/>　　“什么补偿？”巫天御问道。
<br/>　　“你的命！”苏莫声音陡然一冷，随即一挥手漫天魔气汹涌澎湃，瞬间便将整个客厅包裹了起来。
<br/>　　“你……！巫天御见此大惊失色，不过他还没有来得及有什么动作，身形便瞬间被一缕淡紫色的雾气笼罩住了。
<br/>　　而后，巫天御石化了，被拉近了梦魔空间之中。
<br/>　　幽暗的囚室，同样的场景，巫天御被绑在刑架上，苏莫冷漠的站在对方的身前。
<br/>　　“这是哪里？巫尘，你要干什么？”反应过来的巫天御，发现自己的处境，顿时大吼了起来。
<br/>　　“杀你！吞噬！夺舍！”苏莫冷漠的说道，他也不急，虽然阎魔鬼圣只能屏蔽储物戒中少司令牌两三个呼吸的时间。
<br/>　　但梦魔空间之是梦境，在就算在这里待八个时辰的时间，外界也最多过去一个呼吸。
<br/>　　况且，哪里需要半个时辰，片刻便能搞定了。
<br/>　　“你……我不过是言语得罪了你，你居然要杀我，居然如此的狠毒！”巫天御眸中露出冲天的怒火，厉声大吼道，一边大吼他一边查探此地的情况。
<br/>　　不过，他感觉全是无力，怎么也挣脱不了束缚。
<br/>　　“言语得罪了我？”
<br/>　　苏莫闻言嗤笑一声，森然道：“巫天御，你死一万次都不为过，不杀你对不起青璇，不杀你对不起苍穹世界亿万生灵，不杀你我心难安！”
<br/>　　听闻苏莫的话语，巫天御愣住了，一脸呆滞的表情。
<br/>　　少倾之后，巫天御顿时脸色狂变，眼眸瞪得溜圆，不可思议的大喊了起来：“你是苏莫！你是苏莫！你居然混进了族中！”
<br/>　　“不错！我是苏莫，让你死个明白！”苏莫眸中寒光一闪，就与催动吞噬战魂，直接将对方的战魂吞噬。
<br/>　　不过，转念一想，他又不想让对方死的这么痛快。
<br/>　　唰！
<br/>　　苏莫手中光芒一闪，出现了一柄寒光森森的长剑，一剑刺在了巫天御的胸膛上。
<br/>　　啊！
<br/>　　巫天御顿时口中顿时发出一声凄厉的惨叫，疼痛的脸色都扭曲了起来。
<br/>　　噗嗤！噗嗤！噗嗤！
<br/>　　苏莫心狠手辣，手持长剑不断的在巫天御身上猛捅，将巫天御捅的不断惨叫。
<br/>　　“苏莫，你不能杀我，杀了我你永远也不会知道你朋友的下落！”巫天御满脸惊恐，突然大声的吼叫了起来。
<br/>　　“你知道青璇的下落？”苏莫闻言顿时停止了出手，面带惊讶之色。
<br/>　　“我知道，整个族中，只有我和大祭司知道！”巫天御虚弱的说道，他已经受了重伤。
<br/>　　当然，他只是在梦境中受了伤，外界的身体并没有什么伤害。
<br/>　　“青璇在哪里？”苏莫急忙问道。
<br/>　　“你放了我，我便告诉你！”巫天御咬牙说道。
<br/>　　苏莫闻言嗤笑一声，放了对方？那自己纯粹的找死了！
<br/>　　随即，他不由分说，一掌按在了对方的天灵盖上，立刻便运转了搜魂术。
<br/>　　虽然这里是梦境，但无论是他，还是巫天御都是精神意念体，所以完全可以搜魂。
<br/>　　随着苏莫开始搜魂，巫天御繁杂的记忆，如同潮水般涌进了他的脑海之中。
<br/>　　普通的记忆苏莫丝毫不去查看，只查看关于宏青璇的记忆。
<br/>　　不多时，苏莫终于找到了关于宏青璇的记忆。
<br/>　　记忆之中，宏青璇本来一直被存放在巫天御的空间宝器之中，巫天御虽然已经将宏青璇的情况告知了大祭司等族中高层，但大祭司并没有带走宏青璇。
<br/>　　后来，巫天御被大祭司传唤，来到了宗府之内。
<br/>　　来到宗府后，大祭司让巫天御放出了宏青璇，然后交给了一名女子。
<br/>　　这名女子是一名三十岁许的宫装少.妇，容貌姿色俱佳，身上布满了阴寒的气息。
<br/>　　大祭司将宏青璇给予这名少.妇，而这名少.妇同样给予了大祭司一样重要的宝物。
<br/>　　只不过，巫天御地位低下，并不清楚大祭司和宫装少.妇是何种交易。
<br/>　　再然后，宫装少妇居然抹除了宏青璇的记忆，将宏青璇带走了。
<br/>　　“太阴族！”
<br/>　　苏莫松开了搜魂的手掌，眼眸微眯，呢喃自语。
<br/>　　根据巫天御的记忆，那名宫装少.妇乃是‘太阴族’之人，换言之，宏青璇被太阴族的人带走了。
<br/>　　太阴族，乃是天荒星域内一支强大的种族，比巫族都要强大十倍不止，位列太古万族榜第八十七名。
<br/>　　“可恶！”苏莫双拳捏的咯咯作响，眸中杀机爆射，那名女子带走青璇倒也罢了，居然敢抹除青璇的记忆，这让他怒火中烧。
