　　第240章
<br/>　　皇城，朝华殿。
<br/>　　“不嫁，不嫁，不嫁！”
<br/>　　一个枕头丢在李乐天头上，他挠挠头，在父王那里自己挨了砚台，在这里自己又被丢了枕头。
<br/>　　李乐天揉揉额头，笑着道：“婉儿，莫要激动么，看把你高兴的，那草原王呼延跋可是戎族的英雄，你不是最崇拜英雄么？”
<br/>　　李婉儿没有说话，以一副看神经病的眼神看着李乐天。
<br/>　　“这么好，怎么你不嫁过去？”
<br/>　　“胡说了，我能嫁么。”李乐天：“他们只要你，其他什么婉华，玉乐，根本就看不上。嘿，谁不知道咱们明玉公主花容月貌，还知书达理，最是招人喜欢。”
<br/>　　那种看神经病的眼神又浮现在李婉儿脸上。
<br/>　　李乐天挠挠头，也觉得自己的话可信度不高。
<br/>　　“你是不知道啊，戎族愿意拿三座城池给帝国做礼物，都说倾城，可倾城算什么，咱明玉公主能值三座城。”
<br/>　　“不嫁不嫁不嫁，你若再逼我，我就悬梁自尽。”
<br/>　　“悬梁死相很丑的。”
<br/>　　“那我割腕？”
<br/>　　“割腕多疼啊，你受不了这个苦。”
<br/>　　“我投湖。”
<br/>　　“湖水都把你泡肿了。”
<br/>　　“我，我绝食。”
<br/>　　李乐天看着李婉儿，无奈的摇摇头：“婉儿，你大了。”
<br/>　　“大哥……”那眼里的柔软令李婉儿一动。
<br/>　　“长大就要做一些自己不喜欢的事情。”
<br/>　　李乐天起身，拖着长袍离去。
<br/>　　“我不嫁。”李婉儿冲他背影喊道。
<br/>　　李乐天没有回头，只是用背影甩出三个字。
<br/>　　“必须嫁。”
<br/>　　随之他的身影消失在门口，朝华殿的大门关上，封闭了外面的光线。
<br/>　　李婉儿忽然明白，这不是幼时兄妹间的玩闹。你吵着我不要学琴，我不要学琴……然后他就会说：好啦，好啦，今天不学就不学吧，可明天不能再偷懒喔。
<br/>　　最后，还是把琴学了，不仅学琴，还有书画，围棋，茶道，以及剑术，身为帝国公主，总会杂七杂八学一些东西，虽然并不喜欢。
<br/>　　包括这一次，依旧不喜欢，但却必须。
<br/>　　……
<br/>　　牢内点着烛火，而且是檀香味的鱼油烛，这在天牢内可是第一次发生，平日有盏菜油灯已经相当不错了。
<br/>　　老者将酒杯添满，程大雷却无动于衷，只是用双目注视着老者。
<br/>　　“为什么不喝，怕有毒？”
<br/>　　“不是。”程大雷当然不担心酒中有毒，因为完全没有这个必要好不好，不仅没毒，还要保证程大雷在行刑前吃好喝好，活蹦乱跳。
<br/>　　“司机一杯酒，亲人两行泪。”
<br/>　　“呃……”
<br/>　　老者等心中一万只草泥马狂奔而过，方才在程大雷面前落座。
<br/>　　“婉儿与我说了一些你的事情，长安城原来有四位大盗，各遮一方风雨，我还是第一次知道，原来我头上的天空，是你遮着的。”
<br/>　　“呃……有时候出来混，是需要一些人设的。”
<br/>　　老者没有理睬程大雷的胡言乱语，自顾自饮了一杯酒，见程大雷端坐在那里，没有饮酒。
<br/>　　他也没有说什么，又将自己的酒杯斟满。
<br/>　　“七月十四，就是你的死期，临死前，你可有什么想对我说的？”
<br/>　　“是不是长夜漫漫，你睡不着觉，所以特地来找我说话？”
<br/>　　“说话？”老者顿了顿：“大概就是想找个人说些话吧，坐上那个位置，就听不到别人说话了。我执掌天下，天下人的命都是我的，而你是将死之人，将死之人不在乎自己的命，也许你能和我说几句真话。”
<br/>　　老者喝了几杯酒，或许已有些微醺，便与程大雷说了很多话，酒也一杯杯喝尽。
<br/>　　“你一直没有喝酒？”明帝叹口气：“原来你并不愿意和我说话。”
<br/>　　“如果，你能把酒杯稍稍往前推一推，让我可以碰着，我可能会喝一杯。”
<br/>　　程大雷向前伸手，感觉像只身躯超大，两臂奇短的玩偶。
<br/>　　呱呱呱
<br/>　　老者头顶仿佛有乌鸦飞过，额头浮现三根黑线。
<br/>　　可是他突然俯身，身体向前倾，给程大雷带来巨大的压力。
<br/>　　“你看不起我。”
<br/>　　这不是疑问，而是肯定。因为肯定所以愤怒，我高高在上，威加海内，你凭什么看不起我。
<br/>　　您老别生气，我也没有其他本事，不就因为我是个穿越者么。
<br/>　　“你当朕是个昏君么！”
<br/>　　老者用上朕这个称呼，身子重重压在桌上，于是铺天盖地的威压如一座大山向程大雷倾倒，他不再是个垂垂老者，现在他是手握乾坤的大武天子。
<br/>　　“原来你真当朕是个傻瓜。”明帝冷笑一声：“林问天力战无援，冤死当街，杨龙停怯懦惧战，冒领战功，这些事情你以为朕不知道，朕一年前便已知道。”
<br/>　　“朕生在帝王之家，弱冠登基，从小与父母斗，与权臣斗，与诸侯斗，甚至与自己儿女斗，朕从出生开始学的就是与天下人斗。”
<br/>　　明帝伸展双臂，拥抱山河。
<br/>　　“可你们都觉得朕是个傻瓜，高高在上，耳聋目盲，听不得真话，辩不明是非，觉得朕是人人都能唬弄的昏君。”
<br/>　　“……”
<br/>　　程大雷错愕的睁大眼睛，情绪第一次出现波动。他心里冒出疑问：你什么都知道，为何什么都不做。
<br/>　　明帝的气力像是被耗尽，颓然坐在椅子上，醉意又浮上脸颊。
<br/>　　“可知道又能怎样，帝国败了，一败涂地，毫无颜面。帝国需要一个人来承受天下人的愤怒，所以林问天要满门抄斩，因为帝国需要一个人带来希望，所以杨龙停必须是战神。”
<br/>　　“我与林问天很小就认识，那时我们都还年轻，常纵马田野，醉……”明帝苦笑着摇摇头：“既然身在此位，便要替帝国做一些事，他被斩首之前，我们也有一场像今夜这样的交谈，那时他坐在你的位置，我们都喝的大醉，最后他嘱咐我，莫要难过。”
<br/>　　“任何一个位置都有他要做的事，即便身为天子，也难逃这个宿命。林问天用自己的死，履行了自己的责任，而杨龙停作为帝国战神，享受天下崇拜，就是他的责任。因为帝国败得太惨，必须有个人给天下希望。”
<br/>　　“而你……你是与帝国有功的，本无罪，不该死。可你不该杀杨龙停，战神死了，帝国的偶像就崩塌了，需要有个人承受失去偶像的愤怒，所以……”
<br/>　　明帝打了个酒嗝，用含糊不清的语气说：
<br/>　　“为国为民，请你去死。”
<br/>　　程大雷看着明帝，撞上明帝的目光，那眼睛里隐约有火光跃动。程大雷小看了明帝，和他相比，自己还是忒幼稚些。而程大雷也相信明帝是真诚的，或许他此生从未如此真诚。
<br/>　　程大雷叹口气，用比明帝更加真诚的眼神看着对方，嘴里郑重其事吐出一句话。
<br/>　　“我去你马勒戈壁。”
<br/>　　
<br/>(本章完)
