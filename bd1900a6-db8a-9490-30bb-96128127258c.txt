        &lt;h3&gt;第三卷第四百零七章  学长周泽瑞&lt;/h3&gt;
<br/>
<br/>        江俞轩应邀去接杨东一起吃饭，谁知道杨东看见他说：“走，去我家。”
<br/>
<br/>        “要去接嫂子吗？”
<br/>
<br/>        “接什么嫂子？晚上就在我们家里吃，你嫂子做饭。”
<br/>
<br/>        “呀，今天是什么贵客，要嫂子亲自下厨？”
<br/>
<br/>        “你嫂子姐姐家的儿子来了，你嫂子就想让他来家里吃顿饭。”
<br/>
<br/>        “我就说呢，不是贵客怎么可能在家里吃饭，啊，我今天又沾了贵客的光。”
<br/>
<br/>        “别贫嘴了，你在我们家也吃了好几顿饭了。”
<br/>
<br/>        “是，每次都是沾了别人的光。”江俞轩边开车边笑。
<br/>
<br/>        “好了，我给你嫂子说，改天专门给你一个人做饭吃。”杨东和江俞轩在一起是真的很开心，俩个人完全就是朋友一样随意。
<br/>
<br/>        “啊，我受宠若惊！”江俞轩夸张的叫着。
<br/>
<br/>        “你这臭小子，我服了你。”
<br/>
<br/>        “糟老头子，你最近有没有时间，我也想请你吃顿饭，我家老爷子要来西城。”
<br/>
<br/>        “我又不认识你家老爷子吃个什么饭？”
<br/>
<br/>        “咳，你这话说的，我想请你见见我家老爷子，给我家老爷子上上课，让我家老爷子也像你一样开放思想。”
<br/>
<br/>        “哈哈哈，你的意思你家老爷子是老顽固？”
<br/>
<br/>        “我可没有说，我家老爷子和你一样有趣，对，没错，就是有趣。”
<br/>
<br/>        “江俞轩，我发现你现在是越来越皮了啊。”
<br/>
<br/>        一老一少说说笑笑的到了杨东家。
<br/>
<br/>        候玉屏正在厨房忙着做饭，一个年轻人坐在客厅看书，见到杨东和江俞轩进门，连忙站起来：
<br/>
<br/>        “姨夫，您回来啦？”
<br/>
<br/>        “哈哈哈，泽瑞，来，给你介绍一个新朋友，这个是江俞轩，俞轩，这个是周泽瑞。”
<br/>
<br/>        俩个年轻人相视一笑，拥抱了一下。
<br/>
<br/>        “俞轩。”
<br/>
<br/>        “泽瑞学长。”
<br/>
<br/>        “哦豁，你们认识？”杨东睁着大眼睛。
<br/>
<br/>        “我们不但认识还是好朋友。”周泽瑞笑着说。
<br/>
<br/>        “这个世界还真的是很小，我随便拉一个人出来你都认识。”杨东笑着说。
<br/>
<br/>        周泽瑞替江俞轩和杨东各倒了一杯茶，杨东看着俩个年轻人认识，便对江俞轩说：“竟然你们俩个人认识，那你们就聊着，我去厨房陪你候姐姐了。”
<br/>
<br/>        “那我也去打声招呼先，学长你先坐。”江俞轩随着杨东到厨房和候玉屏打了声招呼，便又出来坐在客厅和周泽瑞聊起来了。
<br/>
<br/>        “学长这几年很低调啊，我都很少听到你。”
<br/>
<br/>        “哈哈哈，本本分分的干工作就好，要那么多人知道干什么？倒是你，这一两年成长的很快，房产界一支新秀啊。”
<br/>
<br/>        “咳，仗着老爷子的名号呗，要是我自己我也不知道会混成什么样子的。”
<br/>
<br/>        “你这可是谦虚了啊，你之前的那个公司就很不错啊。”周泽瑞看着江俞轩笑了笑。
<br/>
<br/>        “之前的那个公司是别人的，我只是打工而已。”
<br/>
<br/>        “哈哈哈，听说你们俩个人搭档无往不利。”
<br/>
<br/>        “我也算是从底层做起吧，隐藏了几年身份，说实话真的很能锻炼人。”江俞轩叹了口气。
<br/>
<br/>        人生有很多东西都不能从来，如果当初没有碰见张倩楠，如果没有成为张函的卧底，他现在会不会和陈子昂在一起？
<br/>
<br/>        “底层确实是很能锻炼人的，我也像你一样，从学校毕业就被派到边远的地方工作，最近这一两年才调到上海的。”
<br/>
<br/>        “你现在在上海上班啊，你看我这几年都没有你的信息。”江俞轩有点难为情的笑了笑。
<br/>
<br/>        周泽瑞是他的学长，俩个人在学校还经常见面打球，聊天，周泽瑞毕业后就没有了他的信息，基本就不来往了。
<br/>
<br/>        “哈哈哈，我们家故意将我外放，到一个小山村呆了两年，那个地方很清贫的，基本没有通讯，你想联系我也联系不上啊，回来之后就开始忙着各种工作，我这次是来西城出差，要不然我们还是联系不上。”
<br/>
<br/>        “伯父他们是让你体察民情呢。”
<br/>
<br/>        “说真的，俞轩，我在那个小山村工作的两年，才知道我们现在的生活是有多幸福，而我们现在幸福的生活都是老一辈打下来的，也是一些守在各行各业的人们为我们争取到的。”周泽瑞感慨着。
<br/>
<br/>        “是啊，只有经历过，才知道我们现在的生活是有多幸福，当初我离开家里在外打工的时候，也是经历了一番困苦，如果不是那一段经历，我可能不会觉得现在的生活是多么的幸福，人生是多么的幸运。”
<br/>
<br/>        “你还和原来的公司有来往吗？”
<br/>
<br/>        “嗯，我其实一直都没有离开过那个公司，目前陈子昂去外地拓展业务了，公司的事情都交给我了。”
<br/>
<br/>        “哈哈哈，你真是好精力，江家做的如此好，还将原来的公司也带得很好。”
<br/>
<br/>        “主要是都成气候了，公司的每个员工都是公司的一份子，员工只有将公司当做自己的事业来做的时候，就好管理了，他们已经形成了一套体系，我只管看着就行，况且现在的网络很发达，有些工作就在网络上处理了。”
<br/>
<br/>        “陈子昂是去哪里拓展业务了？”
<br/>
<br/>        “甸城，甸城要全部开展信息化项目，他们已经筹划了三年了。”
<br/>
<br/>        “甸城啊。”周泽瑞淡淡的应。
<br/>
<br/>        “泽瑞，你也认识陈子昂吗？”候玉屏端着一盘菜到餐厅，听见了周泽瑞的话。
<br/>
<br/>        “我不认识，听说过，商界的一皮黑马，堪称典范。”周泽瑞笑着对候玉屏说。
<br/>
<br/>        “我很喜欢子昂的，可惜子昂出国了，要不然我就介绍你们认识。”候玉屏笑着对周泽瑞说。
<br/>
<br/>        “你这是恨不得将世界上所有的好东西都分享给我呢。”周泽瑞见候玉屏端着菜便站起来朝餐厅走去：
<br/>
<br/>        “俞轩，请！”
<br/>
<br/>        “你请！”
<br/>
<br/>        “快点过来，搞得那么生疏，就我们家四个人，过来。”候玉屏笑着招手。
<br/>
<br/>        “泽瑞，我都不知道你们认识，看来今天将俞轩叫到一起叫对啦。”候玉屏在厨房听杨东说周泽瑞和江俞轩认识。
<br/>
<br/>        “咳，臭小子还傲娇的不来，还是我说晚上和陈子昂视频他才来的。”杨东挤兑江俞轩。
<br/>
<br/>        “昨天才视频，今天又视频，你不让陈子昂睡觉了啊？”候玉屏看了一眼江俞轩。
<br/>
<br/>        “来，让陈子昂看看我们家的饭菜，保证流口水。”杨东催促候玉屏。
<br/>
<br/>        “还真视频啊？”候玉屏拿出笔记本电脑。
<br/>
<br/>        打开电脑和陈子昂点击视频通话，过了一会接通了。
<br/>
<br/>        “候姐？有事吗？”陈子昂帅气的出现在镜头前，看得江俞轩一怔。
<br/>
<br/>        “没事，就是想你了。你看看我身边的人是谁？”
<br/>
<br/>        陈子昂看到的是周泽瑞，她不认识，便微微一笑。
<br/>
<br/>        周泽瑞也是看到了一个帅气的小伙子，但是声音却是女的。
<br/>
<br/>        “你好，我是周泽瑞！”
<br/>
<br/>        “子昂，子昂，我在这。”江俞轩挤到镜头前，冲着陈子昂笑。
<br/>
<br/>        “俞轩，你也在候姐家啊。”
<br/>
<br/>        “糟老头子说今晚和你视频我就来了。”江俞轩心里很欢喜，见到陈子昂便着急的挤到镜头前。
<br/>
<br/>        “得了，想蹭吃了就找借口，你们先吃饭，吃完饭再说。”陈子昂看到了桌上的饭菜，知道他们在吃饭。
<br/>
<br/>        她要和落雪商量事情，便微笑着和大家道了再见，主动挂了视频。
