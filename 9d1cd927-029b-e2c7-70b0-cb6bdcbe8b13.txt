　　优先，老头子的御书房当然要优先。
<br/>　　而且还有什么方式比刨了老头子的书房更好的减压方式么？好像没有把？
<br/>　　于是乎李承乾大手一挥，带着几乎已经转职成了工程兵的东宫禁军浩浩荡荡杀入大明宫，直奔老头子的书房而去。
<br/>　　到了地方啥也不用唠，直接就是一顿刨，看的老李揪心不已。
<br/>　　毕竟他的书房和李承乾的那个东宫不一样，这里的地面都是上好的花岗岩打磨而成，别的不说，光运输这一块就花了不下数千贯。
<br/>　　所以当看到李承乾这个败家子‘咣咣’几下就刨碎了好几块，瞅的老李这个心疼啊。
<br/>　　而且这还不算，那个爬到屋顶上的家伙在干啥？为啥要把那么长的铁杆子杵在上面？
<br/>　　啥？避雷针？那又是啥？
<br/>　　老头子看着一群人在自己的院子里肆虐，越看越生气，等看到自己的灯时，更是几乎气炸了肺。
<br/>　　“逆子，这就是你给朕装的灯？”指着头顶上五个圆圆的大灯泡，老头子咬着后槽牙说道。
<br/>　　“昂，是啊。”李承乾点点头。
<br/>　　“那为啥你母后那里的就是那种带玻璃装饰的？朕这里就是这几个破玻璃泡！”
<br/>　　“呃……”李承乾抬头看看头顶上那五个他自己也不知道是多少瓦的大灯泡，似乎没觉得有什么不好。
<br/>　　不过老头子既然问了，那自然是不满意，‘这样显的比较亮堂’的解释他犹豫了半天也没敢说出口，最后只能纠结着说道：“父皇，您这不是着急么，所以儿臣就直接把‘灯芯’给您装上了，不过您放心最，最多五天，儿臣一定让玻璃坊那边给您弄一套最顶级的灯具出来！”
<br/>　　老头子已经被李承乾折腾的有些无语了，心里想着反正都这样了，外面已经被刨的跟个乱葬岗子差不多了，里面五个破灯泡似乎也没啥不能接受的。
<br/>　　但是与以往不同的是，李承乾这次惹了麻烦之后竟然没走，反而一直在书房里走来走去，像是有什么事情没有办法决定一样。
<br/>　　老头子被李承乾转的有些头晕，敲了敲桌子说道：“你在那里转什么，有什么话就说，朕又不吃人！”
<br/>　　是啊，您是不吃人，可您比吃人还邪乎呢！李承乾暗中吐槽了一句，然后抽了抽鼻子，走到老头子身边压低了声音说道：“父皇，想发财么？”
<br/>　　“滚一边儿去，鬼鬼祟祟的像个什么样子！”老头子的回答总是那么出人意料，一声大吼瞬间把个李承乾吓的跳出老远，看了看书房中的确没有什么外人便说道：“父皇，儿臣这几天就在想，您说咱把那电风扇和这个电灯还有和电炕卖了怎么样？”
<br/>　　“卖了？怎么卖？”老头子对土地没了欲望之后变的财迷了许多，听说卖了立刻就想到了钱，反正李承乾这小子卖东西从来就没便宜过，这次估计又是一大笔进帐。
<br/>　　“您给打个广告，儿臣每卖出一台电风扇给您这个数。”李承乾伸出五根手指。
<br/>　　“五成？”老头子眼前一亮。
<br/>　　“五百贯！”
<br/>　　“不干，还是就这样吧！”老头子摇头鄙视的看了李承乾一眼：“你一个电风扇卖给朕一万贯，就给五百的广告费？”
<br/>　　“那，那您要多少？”李承乾换上一副死猪不怕开水烫的表情。
<br/>　　“五成！少于五成免谈！”这次老头子伸出了五根手指。
<br/>　　“没问题，成交！”但让老李同志纳闷的是，李承乾竟然毫不犹豫的答应了，这让李二有种上当的感觉，可一时间又想不起到底哪里上当了。
<br/>　　……
<br/>　　是夜，老头子回了长孙皇后的寝宫，喜滋滋的把白天的事情一说，结果发现老婆竟然用一种十分奇怪的眼神在看着自己。
<br/>　　“观音婢，怎么了？为何如此看朕？”上上下下打量了一会儿自己，发现身上似乎没有什么不妥之后，老头子疑惑的问道。
<br/>　　“陛下，您可知道五成的分红只有四百贯？”长孙皇后并没有让李二多等，很快揭晓了答案。
<br/>　　“什么？五成只有四百贯？可是明明那小子的电风……”话说了一半，老头子突然想是想到了什么，懊恼的拍了一下大腿：“坏了，朕竟然没有算成本。”
<br/>　　“是啊陛下，乾儿的电风扇可以卖出一万贯不假，但那是包含成本在里面的，实际去掉成本，能有八百到九百贯的利润就已经不错了。”长孙皇后为老头子解释道。

<br/>　　不想长孙皇后的回答却让老头子十分意外：“因为那个风扇厂本身就是后宫的产业，妾身自然知道这里的价格到底差多少。”

<br/>　　“那是朕的儿子，朕就是生气又能怎么样！不过……”老头子说到这里顿了顿，犹豫了一下说道：“那小子只把风扇厂交到你手里了？”
