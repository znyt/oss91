　　剑祖荒，鹿战神，荒圣君。
<br/>　　这三个人，都是数十年前，曾经在九域之中崛起的超级天才。
<br/>　　如今这三个人，都是进入到了这天羽神域之中。
<br/>　　如此一来，到底是有着多少曾经的天才，都进入到了天羽神域之中，都是一个未知数。
<br/>　　而且归墟天的资源，要远远超越九域，他们在其中修炼，如今实力达到哪一步，都无法想象。
<br/>　　“不好，九魔城的其他武者，过来了。”
<br/>　　鲜血魔尊突然开口道。
<br/>　　“方才我已经发出玉符，让其他人赶来支援，还请少主躲避一下，我来应付他们。”
<br/>　　“躲避？不用躲避，一会我就出手，杀了他们，你将他们引来便是。”
<br/>　　楚风眠平静的开口道。
<br/>　　随后他的身形，便是逐渐的隐藏了起来，跟周围的空间，融为一体，令人根本无法察觉到楚风眠的存在。
<br/>　　“是。”
<br/>　　鲜血魔尊点了点头，便是突然传音几声，不一会，两道身影便是突然过来。
<br/>　　这二人，也都是一袭血衣，跟着鲜血魔尊极为相似，每个人的身上都是蕴含的滔天魔意。
<br/>　　每一个人的实力，都不亚于剑苍穹，跟眼前这鲜血魔尊，都是在伯仲之间。
<br/>　　“疯魔，杀狂，你们两个来了。”
<br/>　　鲜血魔尊看到这两道身影来，便是打了个招呼，看起来极为平常的样子。
<br/>　　如今他被种下诸天生死印，便是全身心的臣服了楚风眠，自然不会有着任何反抗，按照楚风眠的吩咐，装出一副受伤的样子。
<br/>　　“鲜血。”
<br/>　　那疯魔魔尊，杀狂魔尊二人到来，打量了一眼鲜血魔尊，看着鲜血魔尊身上的伤势，突然吃了一惊道。
<br/>　　“你不是碰到了一个新人吗？怎么会被打伤？”
<br/>　　“这新人，极难对付，不过如今我虽然受伤，但是他也已经重伤，你们二人现在赶来，我们一起去追逐他！”
<br/>　　鲜血魔尊开口道。
<br/>　　“此人的身上，有着大奇遇，曾经得到过颜古龙帝洞府的传承。”
<br/>　　“什么？颜古龙帝的传承？”
<br/>　　“颜古龙帝，可是曾经纵横九域的无敌人物，相传龙族至宝，祖龙至尊剑都是属于他，此人居然是得到了颜古龙帝的传承？”
<br/>　　疯魔魔尊，杀狂魔尊都是一惊，随后大喜道。
<br/>　　“这可是一个好事，杀了他，他身上奇遇，都是我们的了！”
<br/>　　“别在耽误了，我们现在直接追上去再说！”
<br/>　　“野心不小，一群九魔城的小魔头，也想要杀楚某？”
<br/>　　突然之间，楚风眠的声音突然响起。
<br/>　　只见他的身影，便是缓缓的从一旁，走了出来，身上也再也没有任何的遮掩。
<br/>　　“你的谁？”
<br/>　　“怎么回事？”
<br/>　　这疯魔魔尊，杀狂魔尊二人，看到楚风眠的一刻，皆是脸色一愣，随后眼神中露出几分杀意。
<br/>　　如此陌生的一个人，显然并非是如今天羽神域中的武者。
<br/>　　“这是我的主人，楚风眠，你们二人，速速归顺了主人，这才是你们二人的大机缘！”
<br/>　　鲜血魔尊在一旁，看着他们二人，突然开口道。
<br/>　　“什么？主人？”
<br/>　　“鲜血，你居然敢背叛九魔城？”
<br/>　　疯魔魔尊，杀狂魔尊二人，都是勃然大怒道。
<br/>　　“该死！背叛九魔城的下场，只有一个，便是死，今日我们就杀了你们两个人！”
<br/>　　疯魔魔尊，杀狂魔尊二人，同时大怒，一股股魔意在他们二人的身边，爆发出来。
<br/>　　疯狂的向着楚风眠跟鲜血魔尊，轰杀过来。
<br/>　　“两个废物，就连臣服与我的资格都没有！都给我死！”
<br/>　　楚风眠面对着他们二人的出手，头也不抬，就是猛然大手一挥，战龙之爪便是凝聚出来，从天而降。
<br/>　　他们二人身上凝聚出来的魔意，在战龙之爪面前被全部粉碎。
<br/>　　“怎么可能？”
<br/>　　“这是什么力量？”
<br/>　　一招之下，直接破解了那二人的魔意，他们二人都是脸色大惊。
<br/>　　楚风眠的境界，不过是生死境三重，在他们看来，简直如同蝼蚁一样的人物。
<br/>　　谁能想到，居然足以爆发出如此可怕的力量。
<br/>　　“废话真多！”
<br/>　　楚风眠冷哼一声，懒得在浪费口舌，随着他心神一动，战龙之爪的力量便是被凝聚到了极限。
<br/>　　顿时突然落下，这疯魔魔尊，杀狂魔尊，便是都被直接轰杀，化为一团血雾，就连挣扎都无法做到。
<br/>　　他们二人的空戒，在血雾中，被楚风眠一把抓到了手心之中来。
<br/>　　“果然，在这归墟天中待了这么久的时间，一个个都是有着不少的宝藏。”
<br/>　　楚风眠打开空戒，一眼望去，其中什么皇品灵石，各种圣药，简直是数不胜数。
<br/>　　在归墟天中待上数年的时间，任何人都会有着如此富足。
<br/>　　“看来在这归墟天中，还是杀人最容易得到宝物。”
<br/>　　楚风眠嘴角露出几分冷笑。
<br/>　　看来也该是他大开杀戒的时候了，七大宗门的弟子，一个个都是有着无数财富，如同一座座宝藏。
<br/>　　“走，带我去你们九魔城的所在。”
<br/>　　楚风眠看着鲜血魔尊，直接开口道。
<br/>　　这九魔城的锁魂魔尊，拥有这回到风雷谷的玉符。
<br/>　　这玉符，楚风眠必须要得到，不然他就算是在这天羽神域中，得到再多的宝物，这里也如同是身处在一处监狱一样，令楚风眠无法离开。
<br/>　　“是。”
<br/>　　鲜血魔尊点了点头，便是带着楚风眠，向着一座山峰走去。
<br/>　　与此同时，在这周围平原上，一道道空间玄门，便是开启。
<br/>　　无数身影，皆是从这空间玄门之中，一跃而出。
<br/>　　正是小魔主，盘纯等人，还有其他一些武者，如今他们也都已经是度过了风雷谷，来到了这里。
<br/>　　“这里是？这山谷背后，居然还有这样一处地方？”
<br/>　　“那是？云羽天宫？”
<br/>　　一些武者，抬起头，正好是看见在天空之上，漂浮这的无数宫殿，极为雄伟的云羽天宫。
<br/>　　任何武者，在第一次看到这云羽天宫的时候，都会被深深的震惊。
<br/>　　就算是楚风眠当时，也不例外。
<br/>　　“云羽天宫，真的是云羽天宫！”
<br/>　　“我们来的没错！”
<br/>　　一个个武者，都是兴奋的非凡，如今这云羽天宫，就在他们的面前。
<br/>　　“走！我们去寻找这云羽天宫的宝藏！”
<br/>　　“谁可以得到这云羽天宫的宝藏，就彻底的发了！”
<br/>　　一尊尊武者，观察了一下，都是向着云羽天宫之下的群山进发。
