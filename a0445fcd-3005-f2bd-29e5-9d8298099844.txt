　　第87章
<br/>　　没费多少时间，程大雷就从自己屋子翻出刀谱，封皮上三个大字令程大雷怔住。
<br/>　　『无极刀』
<br/>　　本以为系统会奖励二爷的春秋八刀，有了本命武学升级速度还不是蹭蹭地。不得不说，理想和现实之间还是有差距的。
<br/>　　无极刀，武术大家李尧臣为西北军大刀队所创，仗此刀法，喜峰口一战，西北军天下扬名。
<br/>　　系统这一竿子支得可真够远的，民初的武学也跑出来了。
<br/>　　程大雷打开翻了翻，老实说，看不太懂，上面的招式和口诀令他眼花缭乱。
<br/>　　他将刀谱交给关鱼：“喏，看得懂么，看不懂就问？”
<br/>　　关鱼认真翻阅着，两只眼睛眯成一条线：“略懂，略懂。”
<br/>　　呵，你以为你复姓诸葛字孔明么……
<br/>　　过了半晌，关鱼将刀谱合上，沉吟道：“这是步战刀法，不适合马战，可惜了。”
<br/>　　啧，草草翻一遍就能看出这些，不错么。
<br/>　　“大当家，不知这刀谱你是从何处得来？”
<br/>　　程大雷坐直身子：“你可知我身上武学从何处来？”
<br/>　　“倒也听说一二，据说是神仙梦中传授……”
<br/>　　“着！正是神仙入梦，传我八八六十四路神斧，马上步下无人能敌。”程大雷含笑捻须，虽然他没有胡子：“天下武学皆在我胸中，何况区区一本刀谱。”
<br/>　　“那……大当家，可有厉害些的武学，这刀法传给他们还行，于我而言，委实太简单些。”
<br/>　　“……”
<br/>　　程大雷查看关鱼信息，忽然怔住。
<br/>　　姓名：关鱼（初出茅庐的普通山贼）
<br/>　　年龄：18
<br/>　　技能：练兵，无极刀
<br/>　　隐藏属性：无
<br/>　　难道随便看看，就能将刀谱精要掌握，就算知道你是天才，也不需要这么变态吧。
<br/>　　想想自己连看都看不懂，程大雷此刻的心情还真是……
<br/>　　好气哦。
<br/>　　“你先多熟悉熟悉，以后你就代替秦蛮训练山寨弟兄，把无极刀传给他们。”
<br/>　　“已经不用熟悉了，大当家真没有更利害的刀法么？难度高些的。”
<br/>　　程大雷愤然起身，忽然很不想和你说话也不知道怎么会事。
<br/>　　离开关鱼后，程大雷心里琢磨着，自己是否也想办法提升一下自己的战斗力。优秀等级虽然还算不错，但自己多少斤两，程大雷心里还是清楚的。如果也能达到秦蛮那样的程度，方圆百里几乎就可以平趟。
<br/>　　边想边走，前面碰上孙九。自从湖底捞迈入正轨后，孙九就回到山寨。这次见到他，看他站在那里畏畏缩缩的模样，似乎专门在等程大雷。
<br/>　　“大当家，有件事不知当讲不当讲？”
<br/>　　“不当讲。”
<br/>　　“呃……我觉得还是该告诉大当家。”
<br/>　　孙九的声音低下来，听罢之后程大雷也没有太大反应。他挥挥手，冲孙九道：“喔，知道了，你忙你的去吧。”
<br/>　　“大当家，这件事你可不能不小心呐……”孙九最后还这样补充一句。
<br/>　　……
<br/>　　孟子云最近一直不安分。
<br/>　　走投无路才会加入蛤蟆寨，从开始到现在，他没有一刻将自己当成山贼。占有绝对人数优势的柳家来到蛤蟆寨，蛤蟆寨姓程姓柳都不好说。
<br/>　　程大雷假若有什么不服，一刀砍了便是。
<br/>　　但事情的发展超出他的预料，他以为自己是林冲，程大雷可不是那白衣秀士王伦。
<br/>　　程大雷用各种手段给大家洗脑，现在柳家人完全没了武将之后的骄傲，俨然是以程大雷为救世主。
<br/>　　尤其当柳芷搬到程大雷房中以后……
<br/>　　一颗毒火的种子就在孟子云心里埋下了。
<br/>　　孟子云和柳芷从小一起长大，青梅竹马，两小无猜，更何况自己义父早就有将柳芷许配给自己的心思。如果柳家没发生这样的祸事，自己若能娶了柳芷，日后说不得还能继承义父的官位，成为柳家的一家之主。
<br/>　　可那一场灭门之祸，让一切成为泡影。柳芷还搬到程大雷房中，成了恶贼的房里人。
<br/>　　如此，不得不让柳芷在孟子云心里从青梅竹马变成小贱人。
<br/>　　他要毁了这座山寨！
<br/>　　这个念头一在孟子云心底埋下，便生根发芽，再也无法遏制。
<br/>　　孟子云并不是一个很有办法的人，他所谓的破坏，也就是在茶余饭后说一些程大雷的坏话，训练的时候偷些懒而已。
<br/>　　类似于『你们想想程大雷怎么逼死柳老的』『身为名将之后，怎可委身做贼』等等这些话。
<br/>　　然而，并没有什么卵用……
<br/>　　对于多数人而言，蛤蟆寨的生活还算不错，吃得饱，穿得暖，也不必担心人身安全问题。所以大家都很满意现在的生活，在填饱肚子的前提下，那些仇啊恨啊，其实并不重要。
<br/>　　不安分的孟子云很快引起了大家的主意，由刘悲亲自给他做思想工作，开口第一句话是：“小孟啊，你最近的想法有些小问题呐……说出来，说出来我开导开导你。”
<br/>　　那一刻，孟子云想死的心都有。
<br/>　　他复仇的火焰差点就在刘悲面前熄灭了，亏得他意志坚定，才没有遭了刘悲的毒手。
<br/>　　在孟子云一直努力的『破坏』下，蛤蟆寨不仅没受到影响，反而发展的劲头越来越好。
<br/>　　与之伴随的，是孟子云心底的火焰越烧越旺盛。
<br/>　　终于，他决定铤而走险，放火烧掉山寨的粮仓。凛冬已至，而幽州的冬天总是比其他地方更冷些，如果能一把火烧掉粮仓，让仓库内几万斤粮食化为灰烬，那么没有食物的众人，只能在这个寒冷的冬天饿死。
<br/>　　孟子云知道自己不是智将，为这一天他谋划了很久，每天轮岗的人数，以及自己逃跑的路线。在心中暗暗模拟了几次行动，终于在某一天，孟子云决定动手。
<br/>　　冬天的夜总是格外漫长些，在三更时分，孟子云悄悄从床上爬起来。屋子里的孙九酣睡不止，打着沉闷的呼噜。
<br/>　　本想一刀给他个了结，但怕杀死他造成的动静太大，孟子云只好忍下连日来从孙九身上遭遇的怨气。
<br/>　　他提着刀出门，门外风正紧，露正重，静悄悄的一点声音没有。
<br/>　　孟子云深吸一口气，走进寒风当中。
<br/>(本章完)
