　　木天瞭望着远方，那是安罗城外狼牙基地的方向，斜前方的大·片空地隔着狼牙墓地，高耸的巨大墓碑冲入云霄，整个狼牙地界的版图无论从哪个角度看，都透露着此处首领的行·事作风，张狂又霸道，只要是他的人哪怕是死人，也要逼着别人抬头仰望。
<br/>　　身处这样的环境里无论是异种王还是丧尸王，都相当的不舒服，同为两方势力的王者，长期杀戮而站立在顶端的自尊心，不允许他们去佩服或仰望任何人。
<br/>　　“既然不如你，应当是无法破开那层限制才对，不怕死是什么意思？”木叶随意说着，同时看向旁边的房屋，想要站得高一点，此处的平地让他很不适应。
<br/>　　木天看向木叶：“意思就是那人明知道不敌，还非要硬撞，因此受到的精神反噬不小，一般人一天就会变成痴·呆，但这个人似乎信念异常强大，已经七天了还在继续，并且不知道什么原因，那道屏障是双面都在变薄。”
<br/>　　前面的话还不能引起木叶的重视，但最后一句却让木叶瞬间回神：“双面是什么意思？你别告诉我楚涵他自己也在尝试破开？他不是出于昏迷中么？”
<br/>　　“这就是最古怪的地方。”木天目光染上了一层浓雾，星星点点的黑色渐渐铺开蔓延整个瞳孔：“所以接下来的行程我就不跟你同路了，我对这些人类食欲不大。”
<br/>　　话落，木天的身影就猛然一个残影消失，速度快到旁边路过的一人都没能察觉到异样。
<br/>　　木叶面上浮现了一丝怒色，对于这个他亲手转化成类尸，后又成为了丧尸王的亲弟弟，木叶此时已经丝毫感受不到亲情，与丧尸王一路同行的时间里，更多的是两个完全不同的生命在争锋相斗。
<br/>　　若非双方一战引发的后果不堪设想，木叶甚至想过木天最想杀的人，应该是他……
<br/>　　没有思考太久，木叶的身影也随着木天离开的方向而去，恐怖的速度在城池街道上带起一阵狂风，两兄弟之间哪怕反目成仇，也应当是在解决了后患之后。
<br/>　　楚涵这个人，怕是这世上所有势力组织头目都不可能容得下的人。
<br/>　　无论是神秘家族还是丧尸异种，都不约而同的将第一目标放在狼牙，将楚涵先解决掉！
<br/>　　道路中心的人皆面面相觑，东张西望了好久都没发现到底怎么回事，还以为是突然起风，只是……
<br/>　　“啊——”忽的一声尖叫，在安静的道路上凄惨无比。
<br/>　　众人惊吓之间循着声音望去，只见一名穿着狼牙防守队制服的人倒在墙边，胸口一个大窟窿正在喷血，地上一大·片都是血肉模糊的场景。
<br/>　　安罗城已经太久没有出现过如此血腥的场面，众人大惊失色的一阵慌乱，夹杂着各种乱喊哭嚎声。
<br/>　　附近的一批防守队迅速赶到，在看到这名同伴的死亡状态时都心头一个咯噔！
<br/>　　“心脏没了。”一人声音颤抖的开口。
<br/>　　“异种。”
<br/>　　“高阶异种……”
<br/>　　领队的人表情阴沉无比，同时默默的举起枪，黑黝黝的枪口对准地上死去的人。
<br/>　　其余人心中沉痛，却默契的背过身，将此处围起来隔断了不远处居民的视线。
<br/>　　不多时，只听得‘嘭’的一声枪响，紧接着便是防守队快速的清理现场。
<br/>　　能在无形之中取人心脏，并且还是一名新人类的防守队成员心脏，下手的显然就是高阶异种无疑，还是没几个人能对付的了那种可怕大异种。
<br/>　　而五阶以上的异种皆带有极强的丧尸病毒，指甲和牙齿破开人类的血肉，便会留下毒素在体内蔓延，无论此人是死了还是重伤状态，只要大脑组织没被破坏不久后都会尸变。
<br/>　　这是每一名狼牙战团成员都铭记于心的常识，防守队守护的安罗城的确常年无忧无患，但防守队从未因此放松过警惕，就是为了以防万一！
<br/>　　这个被异种掏出心脏的人如果不及时处理，后果便是丧尸病毒的传播，倘若防守队的成员来的哪怕再慢一点点，都有可能导致不可挽回的后果。
<br/>　　这座城市，太多的人类，太过于新鲜的午餐！
<br/>　　在现场清理完毕后，安罗城便全面进入了高度警备状态，有异种混入的事不容小觑，每隔一条街道都有人在不断巡逻，同时城池也进入了紧闭模式，任何人不得进出。
<br/>　　杨天带了一批防守队成员随着狼牙战团远走征战，留下来的人身负的便是整个狼牙地界的安危，所有非狼牙地界的居民都被一一调查，很快安罗城少了两名幸存者的事便被彻查而出。
<br/>　　但在防守队成员察觉到那古怪的两兄弟异样时，木天和木叶却早已远离安罗城很远，步步靠近了狼牙基地。
<br/>　　“城主怎么还没回来？”一名狼牙少校神色焦虑：“我怎么总觉得事情不对。”
<br/>　　“城主留下话说，管好城市，护好幸存者。”旁边的一人开口，同样忧愁。
<br/>　　——————
<br/>　　阴阳谷内，白忧等人依然被困在火族开启的那座出口迷宫·内，他们一步步死循环的走着，已经有人在长时间的高度紧张里昏迷过去，神秘家族长期高枕无忧的生活方式在这一刻将自身的短板暴露无遗。
<br/>　　白忧还是一点点的在寻找出路，虽然这期间已经与罗杰爆发过不止一次的争吵，而两家族之间的成员更是好几次动手，甚至有了伤亡。
<br/>　　时间一点点过去，焦躁不安和彼此间的怀疑排斥，让这两个家族的同行人数减少到了三十不到，中途有被遗弃的人，也有直接被打死的人。
<br/>　　无论是白忧还是罗杰都没有多管，此刻他们自保都不及还要彼此防备着对方，根本顾不上家族中人的死活，那些人愿意跟着就跟着，不愿意那就有多远滚多远。
<br/>　　白忧没有心思解决罗杰这个隐患，他全部的精力都放在了破解迷宫上，导致一度的精神不济。
<br/>　　终于在这群人被困了不知道多久，同行的总人数也只剩下了二十出头之后，忽的眼前一阵视野宽阔，原本死死困住他们的植被墙壁就此移向两旁，露出了前方通往出口的宽阔道路。
