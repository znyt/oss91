　　夜幕降临，皎洁的月光倾洒在豪华的别墅顶上。
<br/>　　别墅院落中，王语嫣等人迎着月光，盘坐于游泳池边，看似柔弱的娇躯上，却散发出一股令人余悸的气息。
<br/>　　慕清霄则坐在一旁护法，结界笼罩着别墅，自然不会闹出一丝的动静。
<br/>　　就在刚才，他已经将所剩余的灵液分配均匀，分别给王语嫣，小依她们服下，并且助她们炼化，一举突破金丹境界。
<br/>　　他现在出去历练一次，最少需要半年的时间才能回来，往后尚未可知，王语嫣她们的修为，自然是越高越好，这样他才能够放心的离开。
<br/>　　随着时间的推移，先是小依，然后是语嫣和雅儿，最后阿碧和阿朱也相继突破，全部迈入金丹境界。
<br/>　　并且，因为灵液中蕴含的能量庞大，她们的修为也停在了金丹四重与金丹五重之间。
<br/>　　等她们气息稳定，慕清霄才撤去结界，满意的点了点头，除了陆雪琪，她们的资质都由龙元，凤血改造过，提升至此实属正常。
<br/>　　修炼过后，免不了一番翻云覆雨，虽然荒唐，但欣喜过望的妻子们都极其努力的配合他。
<br/>　　......
<br/>　　......
<br/>　　竖日清晨，慕清霄习惯性的睁开双眼，望着怀中两具完美无瑕的娇躯，眼中尽是温柔。
<br/>　　穿戴好衣物，简单洗漱一番后，慕清霄就来到楼下，阿碧已经准时的做好了早餐，蔬菜沙拉搭配煎蛋和牛奶。
<br/>　　吃过早餐，打过招呼后，慕清霄就离开金帝别苑，化作一道剑光向静海市东侧飞射而出。
<br/>　　出了静海市，大概几公里的样子，此处有几座小山，山高几百米的样子，周围树木倒是茂密，最主要的是这里荒无人烟。
<br/>　　慕清霄来此处的目的，自然是修炼，毕竟修为到了元婴境界，稍微不小心，别墅可能就保不住了。
<br/>　　小麻烦倒是其次，整个别墅区都是他的，毁个别墅算什么，可最主要的是，那栋别墅已经是他的家，完全住习惯了。
<br/>　　突然之间搬走，他自然不可能接受。
<br/>　　落在最中央的山林间，慕清霄神识扩散而出，将两公里范围全部覆盖，随手一挥，阴阳灵气就扩散而出，化作无形的结界。
<br/>　　如此一来，便能够掩人耳目，哪怕他修炼起来动静再大，外界也不可能有人察觉。
<br/>　　随后，慕清霄便在小森林中，随意找了块较为干净的地面，就地盘坐而下，然后取出储存空间中的容器。
<br/>　　容器打开，浓郁的清香散发而出，这是源于神药的药香，闻上一口就让人心旷神怡，精神为之一振，可见药效有多么的强大。
<br/>　　眼睛直勾勾的盯着神药，慕清霄喉咙微微滚动一下，原本平静的心态，再次跌宕起伏，在这等神药面前，他定力在高，也免不了心动。
<br/>　　捏着手中晶莹剔透的神药，慕清霄深吸了口气，他来此的目的，正是想炼化神药。
<br/>　　接下来的世界，谁也不知道，但肯定不会低于青云志位面，他现在的修为，在低等仙侠位面能够称王称霸，但到了高等位面，根本就是个默默无闻的小人物。
<br/>　　他喜欢将命运捏在自己手中，所以必须尽快提高实力，而神药就是一条捷径。
<br/>　　当然，神药中蕴含的能量极其庞大，根本不是他现在能够吞服的，若是直接服下，恐怕他下一秒就会爆体而亡。
<br/>　　不过，只要不吞服不就行了？
<br/>　　想到此处，慕清霄微微一笑，手中握着神药，荧光缭绕，晶莹剔透的丹药中，划过一抹流光，仿佛带有无穷的魔力。
<br/>　　慕清霄深吸了口气，控制阴阳灵气，将整个神药包裹，随后小心翼翼的尝试，将神药中的力量，一丝丝的牵引出来。
<br/>　　阴阳灵气微微一波动，一股庞大的热流，随着灵气自神药中泄出，这是神药中的能量。
<br/>　　慕清霄直接张嘴，卷动灵气，任由能量流入喉咙中。
<br/>　　下一秒，澎湃的能量犹如洪水一般，铺天盖地的涌入四肢百骸中，慕清霄紧咬着牙关，紧守心神。
<br/>　　能量在经脉中奔腾，犹如刀割，慕清霄额头上已经出现一层细汗。
<br/>　　“刚刚只是牵引了一丝能量，没想到会如此庞大，神药不愧是神药，但这一丝药力，却不足以让自己提升太多，看来剑走偏锋，总是伴随着风险。”
<br/>　　想到此处，慕清霄狠下心来，控制灵气，继续牵引能量，一丝两丝，只见透明色的能量源源不断的灌入他的喉咙中。
<br/>　　几分钟后，慕清霄的脸色已经涨红，连他自己都不知道，体内充斥着多么庞大的能量，但他清楚，这已经是自己的极限了。
<br/>　　于是，连忙停止牵引神药中的能量，双手快速结印，运转起《御女心经》，开始炼化吸收神药的能量。
<br/>　　心法开始运转，体内浑厚的能量也荡起波澜，逐渐浮动，随后顺着心法路线流动起来，只是流动的速度很慢。
<br/>　　慕清霄的身体，现在就是一个容器，而体内的能量却已经溢满，在也没有多余的空间，四肢百骸，无数经脉中都被浑厚的能量注满。
<br/>　　慕清霄隐约能够体会到，能量在微微的流动，如此一来，他才松了一口气，能够炼化便好，哪怕速度极慢。
<br/>　　同时，也有些庆幸，好在他及时收手，否则身体非得撑爆不可，随着时间的推移，能量逐渐与血液，肉身相融。
<br/>　　慕清霄身体猛的一颤，他能够感觉到，体内精纯的能量，融入血液肉身的同时，也开始冲刷自己的身体。
<br/>　　由于能量实在过于庞大，慕清霄甚至感觉皮肤都有些胀痛起来。
<br/>　　然而，庞大而精纯的能量并未因他的痛觉有所停滞，随着能量炼化，经脉中空间越来越大，能量冲刷的更加凶猛起来。
<br/>　　直到最后，能量顺着四肢百骸，不断的钻入细小狭窄的经脉中。
<br/>　　慕清霄盘坐在地面，强忍着痛苦，眉头也不皱一下，身体上的皮肤也由于能量的关系，鼓起一道道四处乱窜的痕迹。
<br/>　　他现在的模样，就犹如皮肤下面隐藏着一条条小蛇，看上去极为狰狞恐怖。
