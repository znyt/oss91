　　长安距离咸阳并不算远，抓住几个倭国奸细的消息很快就传了过去。
<br/>　　黑子现在已经完全接手了情报科，这种事情自然是第一个知道，而黑子既然知道了，那么李承乾自然也就知道了。
<br/>　　如果不考虑倭人的动机和李承乾对倭人的成见，单就事情事身来说，小李觉得这帮倭人特傻，特没有脑子。
<br/>　　你说你偷东西也得看看自己什么条件是吧？没有一点工业基础，甚至社会也还停留在半奴隶半封建的社会，连手工作坊都没有几个的小国家，竟然想造枪？还有比这更扯的么？
<br/>　　哪怕是他想要火药技术也算是他们有点脑子，要造枪造炮算怎么一回事儿。
<br/>　　不过想归想，该给的教训还是要给的，首先就是全部驱逐出境，今后十年之内倭人不得踏足大唐土地半步。
<br/>　　第二，当初既然已经答应卖给他们武器了，这一点大唐不会反悔，不过价格加五成，也就是原本的一点五倍。
<br/>　　至于第三，好吧，没有第三，李承乾实在不知道那个破国家还有什么可让自己惦记的了。
<br/>　　就资源来说，现在的挖出来的那点东西与风险根本不成正比，就土地来说，海外的一块飞地让李承乾十分不放心。
<br/>　　所以，就这样吧，让倭人自己打去吧，回头再给那个什么山背大兄的运点武器装备，或者让北海舰队去倭国走一圈，给那个秦吉胜长点教训，让倭岛的战事再均衡一些也就是了。
<br/>　　……
<br/>　　秦吉胜万万没想到出了这么大的事情，竟然李承乾竟然处理的如此简单，没死一个人，只是被驱逐了，这简直是不幸中的万幸。
<br/>　　所以他也顾不得被打的鼻青脸肿的儿子，带上人用最快的速度在情报科的监视下离开了长安，坐上火车直奔釜山。
<br/>　　列车之上，秦吉胜问起事情的经过，倭人青年顶着两个黑眼圈说道：“父亲，孩儿完全是按照您的意思，到了咸阳之后，便控制了一个从大唐科学院里出来的家伙，那家伙是一个负责卫生清洁的，算是一个人物，应该没有什么见识。”
<br/>　　“可是，孩儿万万没想到，那小子竟然扮猪吃老虎，硬生生的瞒了我们三天，等到我们再去他家里的时候，便被那些唐人给抓到了。”
<br/>　　秦吉胜木然的听完儿子的讲述，淡淡看了他一眼说道：“那么你是如何控制那个人，让他老老实实听话的？”
<br/>　　“我觉得他应该不会乱说，所以……”青年吱吱唔唔的说道。
<br/>　　“你觉得他不会说？你是从哪得得出这样一个结论的？”秦吉胜对自己这个儿子的智商表示了隐隐的担忧，如果他这么容易就会想信别人，估计以后死都不知道怎么死的。
<br/>　　“他，他说，说的。”青年犹豫了一下，最后颓然说道。
<br/>　　“果然是这样。”秦吉胜无奈的叹了口气，闭上眼睛想了一会儿才说道：“这次回去你好好反醒一下吧，如果轻易的就会相信别人又没有主见，天照大神是不会经常眷顾你的，早晚有一天你会因为你的愚蠢而死无葬身之地！”
<br/>　　“是的父亲，我，我知道了。”青年也知道自己这次失分很多，再也不敢如以前那么狂傲，低眉顺眼的退了出去。
<br/>　　……
<br/>　　青年退走之后，一个老者来到了秦吉胜的身边：“王上，您觉得此事还有挽回的余地么？”
<br/>　　“没有，唐皇给做到这个样子已经给了我们很大的面子，没有直接将我们全部抓起来就是我们最大的幸运。”秦吉胜苦笑着说道。
<br/>　　他现在的脑子里很乱，根本拿不出一个好办法来，一切只能等到了釜山再说。
<br/>　　但让他十分意外的是，老者并没有离开，依旧坐在他的面前，似乎有什么话要说。
<br/>　　疑惑中的秦吉胜皱了皱眉头：“怎么？还有事情？”
<br/>　　“王上，我在大唐打听到一件事情。”老者说到一半顿了顿，隔了好一会儿才说道：“我听说大唐的太上皇如今已经驾船出海，正在南海那边漂泊。”
<br/>　　“你想干什么？”秦吉胜眼中精光一闪，正色问道。
<br/>　　“我觉得一不作二不休，不如我们先下手为强，直接……”
<br/>　　“不行，大唐的战船可不是我们能比的，真的打起来估计我们根本就不是对手，而且我们的船也跑不了那么远，万一被人发现可就全完了。”秦吉胜很清楚老者是什么意思，但考虑再三他还是否定的他的意见。
<br/>　　但那老者显然还不死心，兀自继续说道：“王上，事情到了这一步您还在犹豫什么？如今我们手里没有一点筹码，只能任由大唐宰割，如果我们可以把老皇帝控制住，局势翻转，一切都是以我们为主，难道这样不好么？”
<br/>　　“那么你怎么对付大唐的船队？你又怎么对付唐皇的那些护卫？”秦吉胜被老者说的有些不耐烦了，声音不由大了一些。不过好在他们现在至少算是一个使团，火车上有他们单独的包厢，秦吉胜的话并没有被外人听到。
<br/>　　而那个老者显然已经考虑了很多问题，面对秦吉胜的提问慢条斯理的说道：“我们其实不必在大唐上与唐军作战，我们只要到了岛上装成土著就好，到时候假意归顺，只要一有机会立刻一起上去，将大唐的老皇帝控制住，这样一来唐军投鼠忌器就不敢真的围攻我们。”
<br/>　　不得不说，秦吉胜对老者的话还真是有点动心，如果真的能控制住李世民，到时候还不是条件随便自己开，李承乾那小子如果不想背上一个不孝的名声，那就只能乖乖听话。
<br/>　　想到这里，秦吉胜不由定了定神，压下心中的激动，认真的问道：“你有多大的把握能够找到大唐老皇帝的位置，又有多大把握能够把他抓住？”
<br/>　　“把握不大，可以说完全是凭运气，可是试一下总比不试要好吧，至少试了有可能成功，而不试一下的话，我们就只能面对大唐无休止的欺压。”老者握紧了拳头，正色说道。
