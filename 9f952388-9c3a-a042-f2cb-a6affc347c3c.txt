　　“嗯？那家伙怎么没有气息？”
<br/>　　那火牙宗弟子，疑惑地看着萧逸的背影。
<br/>　　但下一秒，那火牙宗弟子又摇了摇头，“呵，错觉罢了，区区一个圣王境八重武者，也没什么好担心的。”
<br/>　　前方，萧逸行走着，宽大黑袍之内，双眼一眯。
<br/>　　幸好他刚才反应快。
<br/>　　他倒是忘了，戴着幽魂面具的情况下，他身上气息会全消。
<br/>　　平日里行走，自是无所谓。
<br/>　　再者，也不会有人无缘无故去感知别人的气息。
<br/>　　但现在他要到火牙宗内查探个清楚，这才特地换上宽大黑袍。
<br/>　　还好他反应快，连忙在身外凝聚出些许气息。
<br/>　　这股气息，凝聚得并不强，大概只到圣王境八重的层次。
<br/>　　这也是那火牙宗弟子感知出萧逸的修为在圣王境八重的原因。
<br/>　　正常来说，武者这种办法是无用的。
<br/>　　因为武者自身就有气息存在，再凝聚出一股更弱的气息，根本掩盖不了自身气息。
<br/>　　但他现在戴着幽魂面具，本身身上是无气息的。
<br/>　　故他现今凝聚出一股气息，便成了他身上唯一的气息。
<br/>　　自然的，外人感知他，也会是感知到这个层次。
<br/>　　而且，萧逸谨慎下，并没有用自身元力凝聚。
<br/>　　而是手中捏着几株皇品天材地宝和皇品丹药，利用这些精纯之物，凝聚出气息。
<br/>　　这股气息，沾在了宽大黑袍之外。
<br/>　　如无意外的话，外人如果对他感兴趣，想要感知他，感知到的，会是精纯的草药气息。
<br/>　　也就是说，他会被当成是一个圣王境八重修为的炼药师。
<br/>　　宽大黑袍之下，萧逸捏过一道得意笑容。
<br/>　　他自信，自己这身乔装足够了。
<br/>　　……
<br/>　　一路在火牙宗的山门外往上行走。
<br/>　　陆续能看到大量武者也在前往火牙宗。
<br/>　　这时，萧逸倒是注意到，大部分正常武者中，不时也会夹杂着几个邪修。
<br/>　　当然，这些邪修穿着正常服饰，面容也如普通武者一样。
<br/>　　但萧逸这些天来，与那么多邪修打过交道，自然一眼就能从各种细节，认出这些邪修。
<br/>　　偌大个地域下，他不知道那些邪修的踪迹，也就罢了，只能以天地之力的形式去感知。
<br/>　　可面对面近距离看到这些邪修，萧逸却一眼能认出。
<br/>　　或许，八殿武者中，现今也无人能比他更了解这些邪修了，包括殿内比他更强的武者。
<br/>　　接触得多了，早有几分直觉。
<br/>　　萧逸边走，边思索着。
<br/>　　忽然，一道身影从一侧迎面撞来。
<br/>　　不过，撞来的力度并不大，而且身影显得有些娇小。
<br/>　　故萧逸也并无影响，倒是这道身影，被反震了两步。
<br/>　　“啊，抱歉。”身影连忙说了一声。
<br/>　　身影，是个女子。
<br/>　　一身粗布麻衣，倒也没有什么特别。
<br/>　　看起来，跟周遭在火牙宗山门行走的普通武者没什么区别。
<br/>　　这时，女子说了声抱歉，迎面而走。
<br/>　　啪…
<br/>　　孰料，萧逸的手掌，疾速从宽大黑袍中伸出，捉住了女子的手腕。
<br/>　　“你做什么？”女子脸色一怒。
<br/>　　萧逸隐藏在宽大黑袍内的眼眸，眯了眯，“姑娘走前，不如先将在下的乾坤戒还回。”
<br/>　　女子脸上显然闪过一丝惊色，随后皱眉道，“我不知道你在说什么。”
<br/>　　萧逸的眼眸，蓦地一冷，“小女娃，在火牙宗外偷东西，你好大的胆子。”
<br/>　　“你…”女子想反驳些什么，但看着萧逸那浑身黑袍的阴森模样，还有黑袍之内完全被遮盖的面容，只余一双冰冷的眼眸，不禁身躯一颤。
<br/>　　而且，这把声音如此沧桑嘶哑，袍子外又带着丹药与天材地宝的气息。
<br/>　　想来，这是个性格古怪的炼药师。
<br/>　　不少成名的炼药师，都是耄耋老者。
<br/>　　而其中，又有一些性格怪异者，不喜言谈，生人勿近，一生至于药草、毒草、妖兽尸体等死物打交道。
<br/>　　总之，直觉告诉她，面前这个黑袍人，很危险。
<br/>　　“咕。”女子不禁咽了口口水，颤巍巍地摊开手掌。
<br/>　　手掌之上，正有一枚乾坤戒。
<br/>　　“那个…前辈，乾坤戒还给你。”女子略带求饶。
<br/>　　萧逸取过，女子转身便要走。
<br/>　　萧逸仍旧捉着她的手腕，冷声而嘶哑道，“不是这一枚。”
<br/>　　这般说着，萧逸心头暗暗苦笑，这女子，有够机灵的，给他换了个空戒指。
<br/>　　若非他谨慎下一直没放手，恐怕自己刚取过乾坤戒，这女子就跑得没影了。
<br/>　　“哦，抱歉前辈，我拿错了。”女子脸上闪过一丝狡黠，竟是轻飘飘就将事情带过去了。
<br/>　　女子手中重新取出一枚乾坤戒。
<br/>　　萧逸接过，也收回了捉住女子的手。
<br/>　　女子松了口气，转身便走。
<br/>　　“慢。”萧逸嘶哑地说了一声。
<br/>　　女子疑惑停下。
<br/>　　萧逸沉声道，“随我走一遭，参加完这火牙盛事，我放你离去。”
<br/>　　“前辈你…”女子陡然一怒，“前辈，你的东西我已还你，你还想做什么？”
<br/>　　泥人尚有三分火，何况武者。
<br/>　　“桀桀。”萧逸阴冷一笑，“空素风狐，可是极其稀有的武魂。”
<br/>　　“对炼药师而言，拥有这等武魂的武者，其鲜血可是宝贝…”
<br/>　　萧逸话还未说完。
<br/>　　女子脸色大变，连忙做了个‘噤声’的手势，随后惊疑地看着四周行人，似是怕被周遭武者听到。
<br/>　　“前辈，说实话吧，你到底想做什么？”女子苦笑道，“我一无重宝，二又只是冒犯前辈半分，前辈何必为难我。”
<br/>　　“让你跟着便跟着。”萧逸嘶哑的话语中，带着不耐烦的语气。
<br/>　　“若无我允许，胆敢离开半步，我保证你离开火牙宗山门前，便会成为一具鲜血流尽的干尸。”
<br/>　　女子身躯一颤，连忙跟上，不敢多言。
<br/>　　萧逸瞥了眼女子，冷哼一声。
<br/>　　女子面露惧色，紧跟萧逸身旁，“前辈有何吩咐？”
<br/>　　“你是火牙地域的本土武者？”萧逸沉声问道。
<br/>　　“是的。”女子点点头，眼中闪过泪花，“小女子幼时便父母双亡，自幼孤身一人。”
<br/>　　“好不容易有了点微末修为，可也终日受人白眼欺凌，也只能靠偷窃度日…”
<br/>　　“闭嘴。”萧逸冷哼一声。
<br/>　　“我问你，这火牙盛事是怎么回事？”
<br/>　　“啊？”女子愣了愣，眼中泪花，说没就没。
<br/>　　“前辈是外来武者？”
