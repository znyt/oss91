　　“几位有事吗？”
<br/>　　萧逸看着围来的众人，微微皱眉。
<br/>　　“我只想问问，吞灵冰鲸皇，是否已被你擒获。”沐冰沉声问道。
<br/>　　萧逸还未说话。
<br/>　　韩苍率先道，“小子，你休想狡辩，我和沐冰都亲眼看到了。”
<br/>　　“吞灵冰鲸皇，已被你擒获。”
<br/>　　“然后呢？”萧逸沉声问道。
<br/>　　“没什么。”沐冰微微摇头，脸色冷漠道，“只是得个确认。”
<br/>　　说罢，沐冰转身离去。
<br/>　　韩苍怒道，“沐冰，什么叫没什么？”
<br/>　　沐冰没有回答，已然远去。
<br/>　　韩苍将目光看向萧逸，道，“小子，吞灵冰鲸皇，乃是我等三人一齐追击。”
<br/>　　“怎可你一人独吞。”
<br/>　　“识相地，便…”
<br/>　　韩苍话还未说完。
<br/>　　一把锋芒而寒气逼人的神剑，凭空而现。
<br/>　　“便怎样？”萧逸手持寒霜剑，冷声问道。
<br/>　　韩苍的意思，其实已经很清楚。
<br/>　　萧逸自不会对他客气。
<br/>　　“怎么？你想跟本公子动武不成？”韩苍脸色一冷。
<br/>　　“真当你裂天剑宗在极寒之地还能算得了什么？”
<br/>　　一旁九位冰宫长老连忙阻拦，横亘于萧逸于韩苍之中。
<br/>　　“两位稍安勿躁。”冰宫大长老沉声说道。
<br/>　　“冰缘大会第一轮，冰海之缘，已经结束。”
<br/>　　“此事，或许可以回不夜城再做商榷。”
<br/>　　“没必要再在这极寒冰海上争斗，免得再生无谓的波澜。”
<br/>　　韩苍闻言，皱眉思索了一下，冷声道，“小子，本公子稍后再与你算账。”
<br/>　　说罢，一个闪身，瞬间离去。
<br/>　　“哼。”萧逸冷哼一声。
<br/>　　他明显看到韩苍脸上一阵潮红，并压下了自身伤势。
<br/>　　刚才追击吞灵冰鲸皇时，韩苍本就受了不轻的伤势。
<br/>　　另外，那强大的秘法，也给予了韩苍不少的反噬。
<br/>　　韩苍作为极寒之地最负盛名的天才之一，会施展如此秘法，并不奇怪。
<br/>　　但，他不过天元一重修为，却爆发出媲美木青云那等巅峰强者的战力。
<br/>　　不可谓不惊人。
<br/>　　而这得益于极寒冰海深处极为骇人的极寒气息，让得他的秘法威力大增。
<br/>　　那种极寒气息的程度，连萧逸这等战力都不敢说在里头无限下潜，或者待上许久。
<br/>　　可想而知，其恐怖程度，恐怕更在高空中的极寒罡风之上。
<br/>　　韩苍那秘法，类似于‘吸元掌’，吸了那些寒冰气息，实力大增。
<br/>　　却也受了不轻的反噬。
<br/>　　这才是他现在罢手并离去的原因。
<br/>　　言归正传。
<br/>　　萧逸对着九位冰宫长老点了点头，算是打个招呼。
<br/>　　随后身影一闪，回到了冰海边缘的岸上。
<br/>　　“萧逸，你没事吧。”白冰雪四人，一直在冰海边缘等待。
<br/>　　“没事。”萧逸摇了摇头，淡淡一笑。
<br/>　　“没事就好。”叶铭松了口气，显然对于之前萧逸潜入深海追击吞灵冰鲸皇的事感到担忧。
<br/>　　一旁钟无忧撇撇嘴，冷酷着脸，不语。
<br/>　　凌羽脸下，也因此着些许担忧，虽掩饰得很好，却还是被萧逸发现了。
<br/>　　萧逸笑笑，道，“先回不夜城吧。”
<br/>　　……
<br/>　　不夜城内，萧逸一行人回到客栈。
<br/>　　冰缘大会第一轮已结束。
<br/>　　不夜冰宫的执事们，会将记录的分数，进行整理。
<br/>　　两天后，公布成绩与排名。
<br/>　　随后，参赛者休息三天，备战第二轮。
<br/>　　也就是说，冰缘大会第二轮，将在五天后举行。
<br/>　　客栈中，萧逸的房间内。
<br/>　　一行五人，在清点着这次的收获。
<br/>　　所谓的收获，自然是吞灵冰鱼以及吞灵冰鲸。
<br/>　　一开始，众人都是边捕捉，边吸收里头蕴含的精纯力量。
<br/>　　可到了后面两天。
<br/>　　吞灵冰鱼出现的数量，太多，太过密集。
<br/>　　几乎是刚捉完一堆，又引来一堆。
<br/>　　捉之不尽。
<br/>　　故，众人自然无暇浪费时间去吸收，而是一直在捉。
<br/>　　而捉到的吞灵冰鱼和吞灵冰鲸，便暂时保存于乾坤戒和乾坤袋中。
<br/>　　乾坤戒，对于大势力的天骄来说，几乎是标配。
<br/>　　五人中，除了叶铭没有乾坤戒外，其余人，人手一个。
<br/>　　另外，乾坤戒和乾坤袋，是无法保存活物的。
<br/>　　但，吞灵冰鱼，只是一种由精纯力量形成的神奇之物。
<br/>　　其存在，大概和上次药楼中暴走的星辰之火差不多。
<br/>　　拥有一定灵智，但本身只是一股力量。
<br/>　　言归正传。
<br/>　　冰海之缘开始的前几天，众人在吸收了大量的吞灵冰鱼，以及一头吞灵冰鲸后。
<br/>　　修为就已经大增。
<br/>　　白冰雪，突破地元八重。
<br/>　　凌羽和钟无忧在地元八重巅峰。
<br/>　　叶铭，突破地元一重。
<br/>　　而之后的两天，众人又吸收了大量吞灵冰鱼，以及数头吞灵冰鲸。
<br/>　　实力再次大增。
<br/>　　如今。
<br/>　　凌羽、钟无忧与白冰雪三人，已是地元九重修为。
<br/>　　叶铭，也达到了地元四重修为。
<br/>　　修为一直没有变化的，只有萧逸一人。
<br/>　　“哈哈哈哈。”钟无忧狂傲一笑。
<br/>　　“萧逸，我听闻你的气泉极大，足有七八百丈。”
<br/>　　“这般大的气泉，赋予了你远超其他同境界武者的战力。”
<br/>　　“却也让得你迟迟无法突破修为。”
<br/>　　“武者，修为才是重中之重。”
<br/>　　“看来，要不了多久，我就能将你踩在脚下。”
<br/>　　边说着，钟无忧摇了摇头，“即将少一个对手，想想，还真是无趣。”
<br/>　　萧逸淡淡一笑，道，“成，我等着你。”
<br/>　　此时，各人的乾坤戒和乾坤袋内，拥有大量吞灵冰鱼，自不必多说。
<br/>　　而吞灵冰鲸，则各自还有15头左右。
<br/>　　吞灵冰鲸王，则各有一头。
<br/>　　一番清点，众人纷纷离去。
<br/>　　房间内，只剩下萧逸。
<br/>　　萧逸叹了口气，其实，他的收获和众人一样。
<br/>　　乾坤戒内的吞灵冰鱼和吞灵冰鲸数量，也是一样。
<br/>　　但，这些收获对白冰雪等人来说，足以让他们修为大增。
<br/>　　可对他萧逸，却还不足以。
<br/>　　内视了一番，那三千三百三十三丈的气泉，着实惊人。
<br/>　　而在气泉一旁，还有一头吞灵冰鲸皇。
<br/>　　不过，这头吞灵冰鲸皇，已经缩水了七八成以上。
<br/>　　冰鸾剑，虽然将正头吞灵冰鲸皇都吸收了，并用于修补自身裂缝。
<br/>　　但这个过程中，始终会有一丝力量，在同一时间化为元力，增幅自身修为。
<br/>　　恰恰在吞灵冰鲸皇被冰鸾剑吸收了七八成力量之时。
<br/>　　气泉内的气态元力，悉数化为了液态元力。
<br/>　　这个时候，萧逸已处于突破天元境的临界点。
<br/>　　而这时，萧逸选择暂时压下了这剩下的数成精纯力量，暂时不吸收，保存于小世界。
<br/>　　他并不打算现在突破天元境。
<br/>　　他在等。
<br/>　　……
<br/>　　两天时间，转瞬便过。
<br/>　　今天，是宣布第一轮排名的日子。
<br/>　　白冰雪等人，纷纷从房间中走出。
<br/>　　所有吞灵冰鱼与吞灵冰鲸，已被他们悉数吸收，修为大增。
<br/>　　而恰在此时，一位冰宫长老，来到了客栈，找上了萧逸等人。
<br/>　　“剑宗诸位天骄，我不夜冰宫宫主有请。”冰宫长老说道。
