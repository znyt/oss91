<br/>　　而后，巫人王身上的气息立刻大减，不断的爆降，很快便降到了武帝境三重的层次。

<br/>　　巫人王闻言眸光一闪，心中闪过一丝阴霾，他感觉大祭司明显是偏袒巫尘啊！

<br/>　　这不是偏袒巫尘，还是什么？
<br/>　　因为以巫尘的身份和实力，根本不可能有太强大的宝器，而他身为武帝境九重武者，身上强大的尊器足有十几种。
<br/>　　“嗯！”即便心中不忿，巫人王还是点了点头。
<br/>　　而后。
<br/>　　嗖！
<br/>　　破空声响起，巫人王身形一掠，闪电般的掠到了万里之外。

<br/>　　巨大的龙爪直接炸裂开来，化为漫天气流，涌荡八方。
