　　“你是何人，怎有资格与我同坐，还不退下！”
<br/>　　准提那愤然的声音在大殿之中响起。
<br/>　　“混账！你……你说什么？！”
<br/>　　鲲鹏道人大怒，吼叫道。
<br/>　　“是啊，鲲鹏，你有什么资格坐在上方，还不下来。”
<br/>　　就连太一与帝俊都鄙视的看了一眼鲲鹏，叫嚣道。我们乃妖族的皇者，都没有蒲团，你一个北冥之修，竟然抢在我们的前头，这怎么可以？不坐就大家一起不坐，若是没有机缘，大家一起没有。
<br/>　　人性就是如此自私。
<br/>　　当下，众修士纷纷起哄，嚷着要让鲲鹏让座，鲲鹏目露一股强烈的愤怒之色，眼中更带有不甘之意，可碍于众怒，他不得不起身让位，狠狠地瞪了众人一眼，面色阴翳地走开。
<br/>　　“多谢这位道兄的相让，贫道感激不尽！”
<br/>　　面色疾苦的道人急忙坐下，对着鲲鹏感谢道。
<br/>　　大殿之中发生的一切，元始都看在眼里，不管是红云道人的不正常，还是准提的不要脸，亦或者是鲲鹏的不甘，元始都没有去管，众人的恩怨，与他何干？他只是来听道，顺便捡些宝贝的，还有顺便看一场好戏，其他的****什么事？
<br/>　　这时，先前看门的两个童子童女走了进来，扫视了众人一眼，然后对着众修士打了一个稽首，道：
<br/>　　“众位道友，老爷马上就要讲道了，还请众位道友做好准备，切勿大声喧哗。”
<br/>　　话毕，忽然在众人没有发现丝毫的波动的情况下，大殿云台之上悄无声息地出现了鸿钧的身影，他在众人身上扫了一圈，看到元始之时，瞳孔微微一缩，之后便不再多管。
<br/>　　最后他的视线落在接引与准提两人身上，微微皱了皱眉，却还想起了西方大战之时所欠下的因果，看来是要落在此两人身上了。可惜，他们注定不能成为我玄门中人。
<br/>　　元始看了看鸿钧，眼神微微闪烁，他知道，鸿钧此刻的修为，至少是圣人三重天，鸿钧之前的积累太深厚了，一旦突破，不会只是突破一重，如今果然如此。而且，之后他的修炼只会越来越快，特别是合了天道之后，修为更是进步神速。
<br/>　　不过，合道之后，他亦不能随意出现在洪荒，这等于有了限制。这就算是一得一失，有所得，就必须有所失吧。
<br/>　　“尔等既是进入紫霄宫，便是与贫道有缘，彼时天地初开，吾有幸得到造化神器造化玉牒，如今成圣，天道之下，当有教化众生之责，是以贫道当讲道三次，此为第一次，能明悟多少，就看你们自己的了。”
<br/>　　鸿钧淡然开口，眼神淡漠，扫视了众人一眼之后，目光微微在元始身上停顿一下，便立刻转为严肃，看着蒲团上的元始等人，道：
<br/>　　“日后就按这般座位，不可更改，昊天、瑶池，关闭宫门，贫道开讲。”
<br/>　　“是，老爷。”一旁的两位童子对着鸿钧恭敬地说道。
<br/>　　哐啷一声，紫霄宫大门关闭了，众修士屏息凝神，就连昊天与瑶池两个童子都随意找了一个地方，盘腿坐下，准备听道。
<br/>　　“叮咚——”
<br/>　　一声钟响，却是那昊天童子敲响左边高台上的紫金小钟，宣布讲道开始。
<br/>　　鸿钧亦不废话，在众修士期待目光之下，便开始了他的第一次讲道：
<br/>　　“天地初开之前，有一物，为混沌也，混沌之前，有一物，为鸿蒙也。道者，鸿蒙之中成也。”
<br/>　　“有物混成，先天地生。寂兮寥兮，独立而不改，周行而不殆，可以为天地母。吾不知其名，强字之曰道。强为之名曰大。大曰逝，逝曰远，远曰反。故，道大，天大，地大，人亦大。域中有大，而人居其一焉。是故，人法地，地法天，天法道，道法自然。”
<br/>　　“混沌生无极，无极化太极，太极化阴阳，两仪四象循……”
<br/>　　“混沌生一，一生二，二生三，三生万物，万物负阴抱阳……”
<br/>　　“大道以天地为烘炉兮，万物滋养成以为道。故道无常相，无所为知道。”
<br/>　　“道者，无名也，万物母也，天地之始也……”
<br/>　　鸿钧讲道，一时间紫霄宫中，天花乱坠，地涌金莲，有无数紫气升腾而环绕大殿，又有空中显现天女散花异象。无数灵气幻化为一个个珍禽异兽，有真龙腾空，有凤凰舞天，有麒麟咆哮，有仙鹤长鸣，有白虎纵跃，有朱雀啸天……
<br/>　　种种异象纷呈不断。
<br/>　　真是圣人讲道，天地同庆。
<br/>　　鸿钧讲道，将元神三花，头顶庆云，尽皆显现出来，各种大道玄妙，放出万丈神光，天道之音。他将自己对各种各样的大道的理解与感悟转化为各种法则之奥展现出来，直接呈现在众人脑海，不会出现错误理解。
<br/>　　一众先天大神只感觉自己来到了那混沌未开的鸿蒙之中，整个混沌都按照那道的规律，道的指引在运转。他们就感觉那虚无缥缈的大道便在自己的眼前，触手可及一般，却又仿佛远在天边，抓之不住，看之不明，见之不清。真是说不清，道不明，只可意会，不可言传。
<br/>　　众多修士，表现各不相同，福缘深厚者，一个个听闻大道，茅塞顿开，如同恍然大悟，脸露微笑。福缘不够者，时而大哭，时而大悲，时而大喜，时而大笑，各种姿态，尽显其中。
<br/>　　紫霄宫中三千客，此刻无不沉浸在那玄妙的意境之中，炼精化气，炼气化神……虚仙，真仙，地仙……大罗金仙，众多修道境界的玄妙被一点点剖析出来。
<br/>　　可惜这次鸿钧所讲的是修道之始到大罗金仙境界有关的，中间穿插着三千大道法则与八百旁门左道，与如何突破大罗金仙境界却没有半点关系。
<br/>　　不过，也正是因为如此，众人才大部分能听懂，要知道此时大多数修士都是靠着自己摸索修炼的，图中不知道犯了多少错误，通过此次听到，将这些错误一一发掘出来，重新改善，同时对各个境界有了全新的了解，厚实了根基。
<br/>　　鸿钧所讲大道种类甚多，众人只能从中挑选适合自己的来听。
<br/>　　下方听道的众人异象纷呈，元始因为境界与积累都已经足够多了，故而，这些人中，他听到的最多，也能够理解。而且他还用鸿蒙元始印，将鸿钧所讲，一一记录在册，补全鸿蒙元始印中缺失的法则，这一趟，对于他来说，可谓是收获颇丰啊。
<br/>　　元始头顶太上无极庆云显现，开天神斧，劈开混沌，天地初分，日月星辰显现，大地山川，河流湖泊，之后又是灭世异象，灭世大磨转动，将天地毁灭，重返混沌。
<br/>　　五行无方，甲木神树，丙丁神火，戊己化莲，庚辛金莲，壬葵神水，接连出现。之后演化四象。
<br/>　　四象地水火风肆虐，朱雀仰天长啸，神火漫天，玄武四脚镇世，大浪滔天，青龙腾云驾雾，行云布雨，白虎操割杀伐，破碎虚空。
<br/>　　四象破碎，三才并出，天地人三才并列，日月星三才横贯，演变星辰恒宇。三才之后，便是阴阳，少阴少阳，太阴太阳，光明黑暗，生命死亡，造化毁灭，各种异象，全部化为阴阳太极。
<br/>　　阴阳衍太极，太极合道一，道一之后，又是无极，不周神山，诛神神枪，各种异象，尽在其中。鸿钧所讲，迅速就被他吸收，在太上无极庆云之上演化出来，异象甚是惊人。
<br/>　　就连高台之上的鸿钧看了都极为惊讶，他想不到元始对道的领悟如此之深，只是，看他的异象，鸿钧并不能看出元始所走的道是何种大道。
