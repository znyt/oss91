　　流星郡郡都，流星商行后堂。
<br/>　　这里，便是炎武卫据点之一。
<br/>　　后堂内一个个房间，便是炎武卫的办公地方。
<br/>　　最中央那间，属于副统领。
<br/>　　此时，房内，李元等三位正队长，以及萧逸。
<br/>　　正站于副统领面前。
<br/>　　“北山剑主，你果然没有让我失望。”副统领林重满意地笑道。
<br/>　　“失踪的三位正队长，数百名炎武卫。”
<br/>　　“一个不少，全部被救了回来。”
<br/>　　“此次，你当记大功。”
<br/>　　“谢副统领。”萧逸行了一礼。
<br/>　　林重一番赞扬后，李元等三位正队长，将霍迪之事，详细汇报了一番。
<br/>　　“嗯。”林重点了点头，脸色有些沉重。
<br/>　　林重思索了半晌，也沉默了半晌。
<br/>　　数分钟后，才说道，“霍迪之事，已经超出了你们能处理的范畴。”
<br/>　　“稍后，我会将这个任务，从任务单中删除。”
<br/>　　说罢，看向金甲卫那位统领。
<br/>　　“此事，你回去禀报流星郡王。”
<br/>　　“我则上报王都炎武卫总部。”
<br/>　　“事情当如何处理，现在暂时无法下定论。”
<br/>　　“是。”金甲卫统领点了点头，而后转身离去。
<br/>　　“副统领。”这时，萧逸拱手说道，“如果可以的话。”
<br/>　　“我想继续追查霍迪之事。”
<br/>　　有些疑惑，萧逸想亲自去查明。
<br/>　　“不。”林重摇了摇头，“霍迪之事，太过危险。”
<br/>　　“待我禀报王都总部后，自会有其他强者去处理。”
<br/>　　“你不必涉险。”
<br/>　　萧逸闻言，皱了皱眉头。
<br/>　　林重见状，笑道，“不用皱眉。”
<br/>　　“霍迪之事，我不会再让你查下去。”
<br/>　　“若这一代最强剑主在我麾下出了什么差池。”
<br/>　　“我可没办法向裂天剑宗交代。”
<br/>　　“不过，此次你的功绩，已经足够，我可以将你晋升为正队长。”
<br/>　　“哦？”萧逸眼睛一亮。
<br/>　　成为正队长，他便能单独行动。
<br/>　　做什么事都方便了许多。
<br/>　　“恭喜萧逸分队长…哦，不，现在该改口叫正队长了。”
<br/>　　李元三人，对着萧逸拱了拱手，以示祝贺。
<br/>　　萧逸笑笑，回敬一礼。
<br/>　　林重笑道，“晋升队长，有一系列复杂手续。”
<br/>　　“当然，这些手续我会替你办好。”
<br/>　　“三天后，拿令牌过来任职即可。”
<br/>　　“谢副统领。”萧逸拱了拱手。
<br/>　　“好了，你们可以下去了。”林重笑道。
<br/>　　四人行了一礼，转身离去。
<br/>　　……
<br/>　　据点外，周平等人兴奋等待着。
<br/>　　“队长，副统领肯定好好赞赏你了吧。”
<br/>　　“额，还好。”萧逸淡淡道，“三天后，升正队长。”
<br/>　　“啊，要升正队长了吗？”众人先是一喜。
<br/>　　而后满脸低落。
<br/>　　他们和萧逸一起行动有大半个月了。
<br/>　　也知道萧逸的心思，一直想单独行动。
<br/>　　萧逸知道他们的想法，笑道，“我一日还在炎武卫。”
<br/>　　“就一日当你们是手下队员。”
<br/>　　“日后有事，可随时来找我。”
<br/>　　“休息一天，明天，继续执行任务吧。”
<br/>　　这大半个月，萧逸带着他们全郡四处执行任务。
<br/>　　众人也该休息一下了。
<br/>　　“不。”孰料，众人纷纷摇头。
<br/>　　“只剩下三天了。”周平恋恋不舍道。
<br/>　　“这三天，我们想和队长你继续并肩作战。”
<br/>　　众人，忽然格外珍惜这剩下的三天。
<br/>　　“这…”萧逸迟疑了一下，道，“好吧。”
<br/>　　随后，众人接了些任务。
<br/>　　再次踏上了执行任务的路途。
<br/>　　三天时间，众人格外卖力。
<br/>　　连胡三这样老是抱怨的人，也没有叫苦过半句。
<br/>　　……
<br/>　　三天后，众人完成了最后一个任务，往郡都而回。
<br/>　　而此时，流星郡郡都炎武卫据点。
<br/>　　却迎来了一位不速之客。
<br/>　　“周副统领。”林重对着面前中年人，拱了拱手。
<br/>　　“林副统领。”周副统领，也回敬了一礼。
<br/>　　周副统领，从王都而来。
<br/>　　三天前，林重将霍迪之事上报王都总部。
<br/>　　王都总部，便派遣了一位副统领过来。
<br/>　　“不知总部那边，对霍迪之事如何处置？”林重问道。
<br/>　　“是王都总部亲自派强者追杀。”
<br/>　　“还是其它据点前来相助我们呢？”
<br/>　　“单凭我们流星郡据点，可无法继续追杀他们。”
<br/>　　“此事不急。”周副统领摆摆手。
<br/>　　“总部那边，自有决断。”
<br/>　　“我此次来，是奉万统领之命，前来办些事儿。”
<br/>　　“哦？万统领之命？”林重惊疑一声。
<br/>　　炎武卫，五位统领，万统领就是其中之一。
<br/>　　再下面，才到25位副统领。
<br/>　　至于大统领，传言，已经多年没有出手了。
<br/>　　常年坐镇王都，替国主分忧。
<br/>　　当然了，大统领的威名和威信，是无人能比的。
<br/>　　那是真正处于炎武王国最上层的巅峰强者之一。
<br/>　　所以说，炎武卫总部的事。
<br/>　　其实很多时候都由五位统领决定。
<br/>　　“不知万统领谴周副统领前来，所为何事？”林重问道。
<br/>　　“自然是为了北山剑主。”周副统领说道。
<br/>　　“据我所知，林副统领打算今日将他升为队长吧。”
<br/>　　“不错。”林重点了点头。
<br/>　　周副统领说道，“万统领的意思是，暂时不让他当队长。”
<br/>　　“为何？”林重疑惑道。
<br/>　　“不为何。”周副统领声音变得坚冷。
<br/>　　“万统领不想他当队长，他便不能当。”
<br/>　　“什么？”林重皱起了眉头，“万统领可是剑宗的长老。”
<br/>　　“为何要打压北山剑主？”
<br/>　　林重是人精了，自然知晓周副统领的意思是要打压北山剑主。
<br/>　　周副统领沉默着，没有解释。
<br/>　　半晌，林重皱眉道，“难道，传言是真的？”
<br/>　　“剑宗宗主以及诸位长老，都对北山剑主不待见？”
<br/>　　前些天，王都就有流言。
<br/>　　剑宗宗主以及诸位长老，对这一代最强剑主颇有微言，甚至很不待见。
<br/>　　在有心人的宣扬下。
<br/>　　似乎是这一代最强剑主，持才傲物，目中无人。
<br/>　　但后来，被宗门发现，他不过是个控火兽武魂拥有者。
<br/>　　之后，他怒火中烧，当众顶撞宗主以及长老，不敬长辈。
<br/>　　宗主一气之下，打算将其逐出宗门。
<br/>　　后实在是这位最强剑主武道天赋极高。
<br/>　　又身怀一代传奇裂天剑魔的武道传承。
<br/>　　宗主心中敬仰前辈，看在这一份上。
<br/>　　才给出考核，让这一代最强剑主，暂时去当炎武卫。
<br/>　　待得成为副统领才回来。
<br/>　　这便是如今在王都流传的流言。
<br/>　　至于谁信谁不信，就不得而知了。
<br/>　　这时，周副统领说道，“林副统领只需照办即可。”
<br/>　　“你该清楚，万统领不让他当队长，他便当不了。”
<br/>　　“哪怕你现在提拔了。”
<br/>　　“万统领只需一道命令，即可撤去他的队长职务。”
<br/>　　万统领，便是剑宗长老之一，全名万剑一。
<br/>　　“我知道了。”林重点了点头，“北山剑主晋升队长之事。”
<br/>　　“即刻取消便是。”
<br/>　　“嗯。”周副统领满意点了点头。
<br/>　　“如此，事情便交由林副统领去办了。”
<br/>　　“我还有要事，先回王都了。”
<br/>　　“周副统领慢走。”林重点了点头。
